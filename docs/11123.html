先决条件

在开始之前，我们将使用create-react-app命令创建一个React Typescript应用程序项目:

npx create-react-app react-styled-components-examples --template typescript

安装

首先，我们需要添加带有类型包的 样式化组件 :

yarn add styled-components && yarn add @types/styled-components -D

⚠️ 注意:如果可以的话，建议(但不是必须)也使用样式组件巴别塔插件。它提供了许多好处，比如更易读的类名、服务器端呈现兼容性、更小的包等等。

styled-components利用带标签的模板文字来设计组件的样式。

全局样式

要添加全局样式，您可以使用createGlobalStyle辅助函数→生成一个特殊的 StyledComponent 来处理全局样式。通常情况下，样式化组件的作用域自动限定在一个本地CSS类中，因此与其他组件隔离开来。在createGlobalStyle的情况下，这个限制被移除，可以应用CSS重置或基本样式表之类的东西。
