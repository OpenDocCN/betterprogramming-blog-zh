## 寻找创造性的方法脱颖而出，创造你的机会

在你放弃实习或工作机会之前，读一下这个…

你觉得你没有任何东西可以提供给科技实习或工作，因为你没有任何经验？

别担心，我理解你的想法。也许你是一名即将完成学位的计算机科学学生，或者你正在自学编程，并且刚刚开始。

因此，你的简历上没有太多的经验。没关系。当我开始我的职业生涯时，我也在那个位置上。

所以，今天我就分享一些我第一次实习申请和落地时做的事情。你现在就可以开始做的事情，并添加到你的简历中，这可能会帮助你得到你想要的实习机会。

此外，我将分享我多年来的一些见解，你可以对你的简历做些改变。

我今天要分享的策略背后的想法是找到从其他候选人中脱颖而出的方法。

根据我的经验，在申请与技术相关的实习和工作时，例如在移动和 web 开发领域，同一职位可能会有数百甚至数千人申请(取决于你申请的公司或组织)。

因此，找到脱颖而出的方法至关重要；这样，你就能吸引别人对你的申请的注意，并能开始面试。

另一方面，根据我的经验，我了解到，打算雇佣你的公司或组织最关心的是你能给他们的团队带来成果和价值。

因此，我在这里分享的策略的另一部分是想办法向潜在雇主或实习提供者展示你可以用你的才能带来成果和价值。此外，表明如果你被录用，你很有可能会成功。

这个策略的另一个关键因素是利用你没有太多经验的事实，并将其转化为优势。换句话说，找到你创造经验和展示技能的方法。

事不宜迟，我们开始吧。

# 添加辅助项目

兼职项目是在简历中添加新技能的好方法，也是展示你的才华和技能的好方法。

我喜欢副业是因为它们是你工作的有形产品。也就是说，兼职项目是你在职业生涯中获得的技能的结果。

你可以做的一件事是在网上的某个地方主持你的副业。或者，你可以将其添加到 Google Play 或 iOS 应用商店(如果是移动应用)。

然后，您可以选择一个 URL，并将其添加到您的简历中项目名称的旁边。通过这种方式，实习生或招聘人员可以看到他们想要的项目。

此外，如果话题出现，你可以在面试中谈论这个项目，并把这个网址作为参考，这样你就可以解释为什么你选择做这个兼职项目。

您可以解释为什么您决定使用您选择的工具来开发应用程序。你也可以谈论你想要解决的问题。

兼职项目的另一个好处是，你可以开始自己的项目，或者继续像黑客马拉松这样的活动。有了黑客马拉松，你可以和其他开发人员(或者你自己，取决于黑客马拉松)一起做一个项目，并把它添加到你的简历中。

我喜欢在小组中完成的兼职项目，因为你可以解释你在团队中的角色，并谈论与他人合作的经历。我相信有这样的经历来谈论是很棒的，因为你可以作为团队的一员工作。

因此，有这样的团队经历会让你受益，并展示你的团队合作技能。

另外，关于副业的另一个建议是让它们尽可能的独特。这样，你就能引起招聘人员的注意，脱颖而出。如果你想出一些附带项目的想法，集中精力让它们独一无二。

尽量展示个性；我相信这有帮助。此外，如果你可以添加使用公司正在寻找的编程语言或工具制作的副业项目，我认为这对你有利，即使你没有经验。

# 你拥有的技能和公司需要的技能

我在和我一起工作的学生身上看到的一个普遍现象是，他们加入了他们过去学习过的所有技能，包括非技术性技能。

我所做的和为我工作的是只包括潜在雇主或实习提供者所需要的技能。这些是我想补充的第一个技能。

我相信，只包括与职位**相关的技能，会让招聘人员在查看数百甚至数千份简历时，容易看到他们需要的技能。此外，这可能有助于你从不具备这些技能的候选人中脱颖而出。**

例如，如果你要申请一份前端开发人员的工作或与前端相关的实习，我会添加像 HTML、CSS 和 JavaScript 这样的技能。

我还会在职位描述中添加雇主要求的任何技能(到目前为止我所具备的)。

然后，我会删除所有其他与工作描述无关或对你没有任何帮助的技能。此外，我会把这些技能放在简历的顶部，这样可以很容易看到。

# 创造你的机会

如果你想开始获得经验，这里有些现成的东西。

如果你是大学生，你可以问行政部门，如你的部门主任，如果他们需要帮助他们的网站或任何其他项目。

看看是否有有偿的开放，如果没有，提供你可以免费帮助。你唯一需要的是一封推荐信或推荐信，你可以把它作为你申请实习或你选择的工作的一部分提交。

我知道免费工作听起来可能不太吸引人。然而，这是让你开始，并增加一些经验到你的简历。我相信这是双赢的局面。你帮助了部门，也获得了经验。

当你申请工作和实习的时候，这份推荐信是很珍贵的，因为它们会给你的工作留下很好的印象。同样，如果你工作出色，推荐人可以写下你的总体表现，并可能在你提交申请时对你有利。

请记住，这个想法是寻找创造性的方法脱颖而出。

创造自己的机会可以帮助你获得经验，并通过跳出框框思考来实现。

不要担心，如果你不是大学生，你可以联系当地的非营利组织，看看他们是否需要帮助。当然，你可以用你的技能帮助他们。不管怎样，策略是一样的。

# 结论

仅仅因为你没有任何工作相关的经验，不应该成为你获得想要的实习或工作的障碍。

寻找创造性的方法脱颖而出，创造你的机会。此外，想办法证明你能给你申请的公司或组织增加价值。

这就是我过去写简历时所做的事情，以及这些年来我在旅途中学到的东西，我想与你分享。

你可以开始着手申请你想要的实习，即使你觉得自己没有太多经验。

走吧。你能做到的！

希望这有帮助！
