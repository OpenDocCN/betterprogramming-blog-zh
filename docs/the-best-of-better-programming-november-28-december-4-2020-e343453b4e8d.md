## 我们本周阅读量最大的文章！

![](img/c767108133195f9a579904561beea756.png)

由 [Rudney Uezu](https://unsplash.com/@rudney?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

大家好，

我希望你的一周充满了深入的工作，没有太多的会议，也希望感恩节后没有为你的家人做太多的技术支持😃

我们本周阅读量最大的文章是一个名副其实的学习、课程和资源宝库，所以让我们开始吧:

2.  [**4 个你可能不知道的关于 Python 函数的高级技巧**](https://medium.com/p/d1214d751741) 作者[Erik van Baaren](https://medium.com/@eriky)——Erik 在这里将向你传授更多关于强制关键字参数的技巧，在你的函数、装饰器和匿名函数中使用`*`和`**`。本周有很多好的 Python！
3.  [**每个程序员必备的数据结构**](https://medium.com/p/7d1511eba838) 作者[Mahdhi rez VI](https://medium.com/@mahdhirezvi)——数据结构听起来可能令人生畏，但事实并非如此。Mahdhi 没有深入研究所有的数据结构([参见 Wikipedia](https://en.wikipedia.org/wiki/List_of_data_structures) )，而是专注于一个有用的子集，除了数组和字典之外，您实际上还会用到它。
4.  [](https://medium.com/p/f3be5f6c9701)**作者 [Erik Engheim](https://medium.com/@erik-engheim) —这是一篇来自时事通讯新人 Erik 的有趣文章，他研究了不同的语言以及每种语言给我们如何编写和构造代码带来的新的或新鲜的想法。**
5.  **[**如何使用 Ref 解决 React 性能问题**](https://medium.com/p/4d92a8120c09) 作者 [Sidney Alcantara](https://medium.com/@notsidney) —本周的另一位新人，Sidney 在这里向我们讲述`useRef`如何通过从 React 上下文中获取控制权来解决他们的性能问题。如果你正在升级一个 React 应用，它值得一读。**
6.  **[**Android 开发中的 8 个常见错误**](https://medium.com/p/2edcf5179ec0) 作者[Siva Ganesh kanta mani](https://medium.com/@sgkantamani)—Siva 本周回来了，带来了一份作为 Android 开发人员应该避免的有用错误列表。我们所有的代码库都可以再好一点。我希望这能激发一些小的清理或重构的灵感。**
7.  **[**即使不能解决问题也要在编码面试中表现出色**](https://medium.com/p/91a950947226) 作者[布雷特·法齐奥](https://medium.com/@faziobrett)——我们都经历过:你被问到一个你知道自己无法“正确”回答的问题。好消息是，你还可以做其他的事情来告诉你的面试官，即使你不能得到正确的答案，你也知道如何打破僵局，尝试解决问题。你可以相信还有其他方法可以得到这份工作。记住:完成比完美更好。**
8.  **[**作为一名开发人员，你每天可以用来成长和学习的 5 种方法**](https://medium.com/p/6da1aa56bb59) 作者[扎卡里·米诺](https://medium.com/@zack_minott)——这里我最喜欢的是#1:尽可能以最小的方式出现。一位伟大的虚构总统曾经说过:“决策是由出现的人做出的”。有时候呆在房间里就足够继续前进了。**
9.  **[**10 必须为前端开发人员提供 Chrome 扩展**](https://medium.com/p/8f841034506e) 由[Harsha vard Han](https://medium.com/@harshaktg)——从 JSONView(我大部分时间使用的东西)，到 Meta SEO Inspector 等等，这里有一个有用的 Chrome 扩展列表，可以增强你的浏览器功能。**
10.  **[**我从使用 M1 MacBook Air**](https://medium.com/p/a77f93c50381) 中学到的 5 件事作者[杨梦](https://medium.com/@ericleiyang)——埃里克有一台苹果的新款 M1 MacBook Air(我很嫉妒)，但他的开发环境中并不是所有东西都能工作。让我们来看看在苹果闪亮的新硬件上什么可行，什么不可行。**
11.  **[**2021 年开发者应该知道的 5 个有用的 Golang 模块**](https://medium.com/p/340ed21177e5) 作者[艾美奖·张伯伦](https://medium.com/@emmyc)——艾美奖本周带着有用的 Go 模块回来了，你可以将它们合并到你的代码库中，让你的生活变得更轻松、更有效率。**
12.  **[**投资者的 6 个教训让程序员赚得更多**](https://medium.com/p/dcbd63db8187) 作者[Anna Anisienia](https://medium.com/@anna.anisienia)——沟通投资和编程的世界，新人 Anna 在这里告诉我们 6 个同样适用于工程师的强有力的想法。**
