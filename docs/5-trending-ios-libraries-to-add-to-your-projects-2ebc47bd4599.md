# 2.纵火案

![](img/2d3870863c64e8fec73c250ed85b4d65.png)

类似于著名的 Instagram Stories 功能，你可以使用这个库来显示照片和视频。

下面是一些要点来重述它的众所周知的用法:

*   点击*介质*跳过。
*   向前/向后滑动以查看下一个/上一个媒体。
*   视图顶部的进度条表示经过的时间，图像或视频长度为 0 到 5 秒。

# 5.TinyCreditCard

![](img/e98d5c4ec28cd001dd010f7bb74bda54.png)

这个小而强大的库可以让你轻松地添加信用卡到你的应用程序中。通过清晰而生动的工作流程，它指导用户输入所有必要的信息。

它还可以识别信用卡类型，并在右上角显示相应的标志。
