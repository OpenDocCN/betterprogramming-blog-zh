# 2020 年在科技行业成功求职的 5 个策略

> 原文：<https://betterprogramming.pub/5-strategies-to-run-a-successful-job-search-in-tech-in-2020-c665bcb62249>

## 这份指南可以帮助你在职业生涯中迈出下一步

![](img/75439c3cf290331486febbb2702ec3f5.png)

[安德鲁·尼尔](https://unsplash.com/@andrewtneel?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍照。

新冠肺炎疫情改变了每个人的求职方式，但仍然有可能找到一份有回报的技术工作。为了让你的求职成功，同时运用多种策略。让我们来看看最有效的。

# 1.花时间更新和改进你的简历

大多数求职者不会更新和改进他们的简历来走在潮流的前面。平均来说，你有[7 秒](https://www.theladders.com/static/images/basicSite/pdfs/TheLadders-EyeTracking-StudyC2.pdf)的时间来吸引招聘人员或招聘经理的目光。如果你想让你的技术简历脱颖而出，遵循这些建议。还有各种各样的[简历增强服务](https://cvcompiler.com)可以极大地帮助你实现这一目标。

# 2.列出你想去的公司，然后联系他们的人力资源人员

首先，列出潜在雇主的名单。然后，跳上 LinkedIn，搜索你的目标公司。从招聘团队中找一个人把你的简历发给他。理想情况下，这个人应该是为你想要的职位招聘的人。否则，联系其中一名团队成员，说明哪个职位引起了你的注意，并请他们为你联系合适的人。

一个小提示:通常，你可以在招聘人员的 LinkedIn 个人资料中找到他们的联系方式。如果没有，可以在谷歌上快速搜索(例如 name-of-the-recruiter@company.com)来找到所需的电子邮件地址。

更进一步，为你的目标公司写一个代码样本，连同你的简历、你活跃的 GitHub 个人资料/个人网站/技术博客的链接以及一封给相关人员的求职信一起发送。是的，许多科技公司仍然需要求职信，至少根据这项调查是这样的。

或者，使用 LinkedIn 找到你的老朋友和关系，他们目前在你想找工作的公司工作，问问是否有可能被介绍到他们的公司。

# 3.完整填写您的 LinkedIn 个人资料

科技行业内外 90%的招聘人员都使用 LinkedIn 来寻找和审查候选人。如果你在 LinkedIn 上的个人资料 100%完整，你更有可能被招聘人员发现。

> “拥有完整个人资料的用户通过 LinkedIn 获得机会的可能性是普通用户的 40 倍。”— LinkedIn

请特别注意你的个人资料标题，因为它是 LinkedIn 和 Google 搜索结果中显示的文本。

努力写一份强有力的、吸引人的摘要，因为这是招聘人员/招聘经理访问你的个人资料时首先看到的东西之一。[这些小贴士](https://outwittrade.com/how-to-write-a-linkedin-summary/)会帮你打造一个有效的方法。

确保你的个人资料——尤其是你的标题和“关于”(摘要)部分——包含正确的关键词。招聘人员在为他们的职位空缺寻找潜在候选人时，强烈依赖布尔搜索和关键字。下面是一个用于寻找 Python 开发者的布尔搜索字符串示例:

```
(programmer OR developer) AND python AND (mysql OR postgresql) AND (django OR flask) AND (chef OR puppet OR sentry)
```

# 4.网络！不要完全依赖网上求职

与你所在领域的有趣的人联系的一个好方法是参加[在线和离线聚会](https://www.meetup.com/)，研讨会，会议等等。即使这不会马上带来一份工作，但在将来可能会有用。添加您在 LinkedIn 上认识的人。

> “网络很重要，原因有很多。…首先，当然，这是找到工作机会的重要途径。许多工作从未公开发布。如果你完全依赖公开发布的广告，你会错过很多机会。”— [LinkedIn 的招聘简讯](https://www.linkedin.com/pulse/networking-still-important-heres-how-do-while-youre-stuck-seaman/)

找到相关的 LinkedIn、脸书、 [Slack](https://standuply.com/slack-chat-groups) 、 [Gitter](https://gitter.im/scala/job-board) 或其他在线社区加入。在线社区也是你可以建立有用联系的地方。此外，几乎所有的网站都有一个专门介绍就业机会的板块/频道。以下是一些拥有活跃#jobs 频道的 Slack 聊天群:

*   [开发聊天](https://devolio-devchat.slack.com/)
*   [TechMasters](http://techmasters.slack.com/)
*   联邦政府官员(前端开发者社区)
*   [PHP 聊天](https://phpchat.slack.com/)
*   [iOS 开发者](https://ios-developers.slack.com/)(#求职版频道)
*   [安卓聊天](https://androidchat.slack.com/)
*   [Ruby 开发者](http://rubydevelopers.slack.com/)
*   [Ruby on Rails 链接](https://rubyonrails-link.slack.com/)(#工作机会频道)
*   [地鼠](https://gophers.slack.com/)(Golang 开发者社区)
*   [Python 开发者](https://pythondev.slack.com/)
*   [Cpplang](https://cpplang.slack.com/)
*   [DevOps 聊天](https://devopschat.slack.com/) (#jobs，#jobs_remote)

建立人脉，利用这些人脉建立关系，然后“推销”你的简历。这些建议会让你开始。

# 5.在各种网站上搜索工作列表，但直接通过公司网站申请

使用传统的求职网站、小众求职平台(例如[寻找国外的技术工作](https://relocate.me/blog/job-relocation/a-practical-guide-to-finding-a-tech-job-with-relocation-to-europe/#more-1211))、社交网站等搜索职位空缺。然而，在申请工作时，尽量在公司网站上进行。当你在公司网站上申请一个职位时，你的简历会直接发送给该公司的招聘人员/招聘经理。而且，它可以被不止一个团队成员注意到。

# 结论

当然，求职的成功取决于多种因素。这些方面包括你申请工作的时间。然而，我确实相信这些建议，加上一份出色的简历，将帮助你更快地找到你在科技领域的下一个富有挑战性的角色。
