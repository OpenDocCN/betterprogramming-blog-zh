一个使用PHP、MySQL和MAMP的例子

有时我们正在进行的iOS项目会看到web平台的移动版本的构建，在那里我们会找到注册、登录或任何其他模块。

这些平台的任务是将相关数据保存在数据库中，并使用web语言构建，例如，逻辑部分使用 PHP、 Python、C #、JS，存档部分使用 MySQL 、SQL、MongoDB、PostgreSQL。

在这种情况下，我们将介绍两个广泛使用的应用， PHP 和 MySQL 。
