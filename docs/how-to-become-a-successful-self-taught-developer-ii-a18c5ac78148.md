# 采取系统的方法

如果计划不是建立在一个系统之上的话，它们是不会起作用的。

当你昨天已经说过的时候，不要再告诉自己你明天会做这件事。当你还不能从一数到二的时候，你就不能活到 100 岁。如果您对前端或后端的理解还不太适应，就不要学习全栈。当你还不熟悉 JavaScript 的时候，不要跳入 JavaScript 框架。不要仅仅因为它很难或者它不是最令人兴奋的事情就跳过最重要的部分。

有时候，我们需要做我们不想做的事情，这样我们才能做我们喜欢的事情。最坚强的人在内心进行了一场不可能的战斗。他们有纪律，接受事情需要时间，事情是有条理的，我们需要尊重这一点。

停止思考，开始行动。只要简单地写下来，组织起来，制定计划，围绕它建立一个系统，并设定一个合理的期限，你所能想象的一切都将开始成为现实。

如果你想成为一名优秀的前端开发人员，从基础开始:HTML、CSS、JS、Bootstrap、更多的 Javascript，然后是更多的 JavaScript、API、数据结构、数据处理、HTTP 请求、部署、版本控制、测试、JS 框架或库，然后是后端。事物的结构是有原因的。

选择五年后、十年后你想要的是什么，并围绕它建立一个系统。你打算怎样从 A 点到达 B 点？你必须把事情分解。当你把目标分成几个步骤时，你的目标更容易实现。

当我还在努力寻找第一份工作时，一位招聘经理问我，十年后我能在哪里看到自己。我告诉他我会成为一名全栈开发者。尽管我没有得到那份工作，但渐渐地，不知不觉地，我已经围绕它积累了财富。

有趣的是事情是如何发生的。我目前从事的工作最近给了我们一个即将到来的项目的文档，该项目将需要 MERN 堆栈。全栈梦想正在成为现实。这不是一夜之间发生的，而是通过慢慢建立系统的小细节。有了合适的结构，它就更加现实和可伸缩。

还有一点要记住，你就是你的生态系统，你花大部分时间在上面的东西，就是你未来的样子。拥抱纪律，培养正确的习惯，因为你的生活是一系列的选择。你的时间太宝贵了，不能花在电子游戏和电视节目上，因为你今天可以学到一些东西。

# 多出现一会儿

它总是归结于你有多想要它。

我可能很幸运，在六个月后找到了我的第一份工作，但旅途很艰难。在我职业生涯的头两年，困难的事情发生了。我差一点就要放弃了，因为我觉得无法继续下去了。刚开始做 web 开发的时候，怎么做手机 app？

如果当时我放弃了，我就不可能实现那些我从来不认为可能的事情。

当事情变得艰难时，记住你最初开始这段旅程的原因。记住所有的不眠之夜，压力，以及你为此付出的所有努力。可能只是糟糕的一天或者糟糕的一个月。休息一下，收工，重新开始。不管发生什么，每天继续出现就好。你已经尽力了，只有你坚持下去，生活才会回报你。

如果第二天我没有出现，我永远也不会知道我真正的能力。

坚持相信你的梦想。生命可以在一瞬间从零到 100 岁。有时候，你会花上 5 年甚至 10 年的时间来获得你生命中最美好的一年。你只是还看不到，但你必须去感受它，相信你的直觉。他们有时最清楚。俗话说得好:

> “累了就不要停。完了就停。”

# 这是一场漫长的比赛

你能做的最好的事就是你自己。

> "如果上帝让你等待，那就准备好得到比你要求的更多的东西."

这不是短跑。这是一场马拉松。谁先到达那里并不重要。最重要的是完成——把事情做完。相信我，旅程本身就是奖励。

这不仅仅是获得第一份工作，不仅仅是获得开发人员的头衔，也不仅仅是成功地学习这些语言。它是关于意识到如果你投入工作，你可以做任何你想做的事情。

这是关于你相信自己，因为你可以，这也是关于学习成功的公式。无论你进行什么样的冒险，只要你想得长远，只要你接受失败和失望是不可避免的，是计划的一部分，这些事情会让你变得更强大，你总能成功。我向你保证，你可以做任何事情。

成为开发者是一个漫长的过程。它是如此复杂，有如此多的事情要做和学习。我花了差不多三年的时间才有自信称自己为开发者。

完成一件事的最好方法是把任务分成几个部分，让它看起来更容易完成，更真实。你必须用那种方式去接近它。赢得战争就是赢得小战役，一次一个。

最后，无论如何都要相信自己，因为如果你不相信，就没有人会相信。
