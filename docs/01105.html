使用集合处理数据集中的唯一元素

集合是ES6中包含的一种新的对象类型，允许创建唯一值的集合。集合中的值可以是简单的原语，如字符串和整数，也可以是更复杂的对象类型，如对象文字或数组。把集合想象成一个没有重复元素的数组。

在许多情况下，我们有一个包含许多元素的数组，我们希望将这些元素转换成一个唯一的数组。虽然它可以通过数组上的暴力、检查元素、使用循环、不同的回调方法等来完成。，最好的方法是使用set()。

语法:

new Set([iterable]);

设置方法

现在让我们看看一些基本的set方法，你可能不需要它们，但是最好熟悉它们以便优化你的代码:

  • add
  • size
  • has
  • forEach
  • delete
  • clear

非常简单，add方法添加元素，delete方法删除元素，has方法与数组中的includes具有相同的功能，clear方法清空集合。

正如我们在语法中看到的，set方法将iterable作为参数，因此字符串也可以用作参数。
