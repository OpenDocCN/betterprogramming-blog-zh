了解如何创建ML以及如何立即开始构建模型

Create ML是苹果公司最新推出的一套强大的第一方开发工具。

从我有机会获得奖学金参加的WWDC17开始，苹果推出了Core ML——他们的第一方ML框架，使ML模型能够无缝嵌入到iOS和macOS应用程序中。Create ML，既是框架又是应用，在一年后的WWDC18上亮相。

根据 Core ML文档，Core ML的目的是“将机器学习模型集成到你的应用程序中。” 太好了！但是，如果您是一名经过培训的iOS开发人员，并且没有任何Python/TensorFlow技能来自己创建模型，该怎么办呢？那么，创建ML作为一个补充。

Create ML的文档称Create ML的目的是“创建在你的应用中使用的机器学习模型。”因此，现在iOS和macOS开发人员有了一个端到端的途径来创建用于他们应用程序的模型。

在Create ML中可以创建哪些类型的模型？

Create ML支持五种不同类别的模型:

  • 图像
  • 声音
  • 移动
  • 文本
  • 桌子

在图像中，用户可以建立分类和检测模型。声音和动作只支持分类。文本支持分类和标记。表支持创建分类器、回归器和推荐器模型。

结论

归根结底，Create ML是一种简单有效的方法，可以创建简单的ML模型用于iOS和macOS产品。创建模型本身不需要编码(尽管将它集成到应用程序中需要一些编码)。您所需要的是，可能的话，调整模型并将数据组织到文件夹中。

如果你正在为苹果产品开发软件，这是一个快速启动并运行模型的好方法——所有这一切都有无缝的第一方体验。
