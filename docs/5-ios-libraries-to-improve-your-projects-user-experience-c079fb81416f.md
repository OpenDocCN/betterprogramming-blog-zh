# 4.sp 权限

![](img/876eab20d559a062de69d5f8add05771.png)

[SPPermissions](https://github.com/ivanvorobei/SPPermissions) 是一个 API，通过三个不同的选项询问用户权限:列表、对话框和本地。

以苹果风格创建的界面支持 iPhone、iPad、黑暗模式和 tvOS。支持的权限请求有:

*   照相机
*   照片库
*   通知
*   位置
*   麦克风
*   日历
*   联系人
*   提醒
*   移动
*   媒体
*   演讲
*   蓝牙
