过去50年中最有影响力的10种编程语言及其创造者

我列出了过去50年中最有影响力的10种编程语言及其创造者。

软件世界中有许多编程语言，而且每年都有越来越多的语言被开发出来。最近我们已经发布了 Scala 、 Kotlin 、 Go 、 Kotlin 和Closure，但是历史表明只有极少数能够存活下来。

这些语言对软件开发世界做出了巨大的贡献，这也是它们入选过去50年中最有影响力的10种编程语言的原因。

由于编程语言是软件开发世界中唯一最重要的东西，并且多年来经常被讨论、批评和改进，当有人使用一种编程语言却不知道它背后的历史时，这让我很困扰。

虽然这种情况在流行的编程语言中不太常见，比如人人都知道詹姆斯·高斯林是Java之父，但并不是每个开发人员都知道是谁创造了Perl、Pascal、Lisp或Erlang。

这促使我收集并分享十种最流行、最成功的编程语言背后的程序员和设计师的名字。

十大编程语言及其创造者

以下是我列出的十大编程语言及其创造者。语言排名不分先后，但由于我是一名 Java开发人员并且从 Java 中受益匪浅，我毫不犹豫地将它放在首位。

我知道许多C程序员不会同意我的观点，因为 C 是仍然经常使用的最古老的语言，但这个列表不是为了排名，而是为了了解和记住在编程语言和软件开发领域做出贡献的大师级程序员。

1.爪哇—詹姆斯·高斯林

Java是最流行和最成功的编程语言之一。James Arthur Gosling博士发明了Java，并被誉为Java编程语言之父。

Java最初是由Sun Microsystems开发和支持的，在2010年1月收购Sun Microsystems后，现在由Oracle开发和支持。

Java是带着WORA的使命“一次编写，随处运行”而Java的平台独立性是其在企业世界中成功的支柱之一。

迄今为止，它是最流行的应用程序编程语言之一。如果你对学习Java感兴趣，那么 完整的Java Masterclass 课程是一个很好的起点。

2.丹尼斯·里奇

美国计算机科学家丹尼斯·麦卡利斯泰尔·里奇于1967年至1973年间在美国电话电报公司贝尔实验室创造了C程序语言。

C 仍然非常流行，在系统编程中被广泛使用。它比Java古老，但仍然保持着它的优势。

对了，丹尼斯·里奇还和他的老同事肯·汤普森一起创造了世界闻名的 UNIX 操作系统。

如果你把他的受欢迎程度与比尔·盖茨或史蒂夫·乔布斯相比，他是望尘莫及的，但他对软件世界的贡献却无人能及。

每个程序员都应该知道丹尼斯·里奇和他对编程世界的贡献。如果你想学C，你应该加入Coursera上C专业 的 编程入门。像其他Coursera认证一样，所有课程都是免费的，但你需要为认证付费。

还有如果你想学习数据结构，那你就享受C 这些 免费的数据结构和算法课程吧。

3.C++ —比雅尼·斯特劳斯特鲁普

比雅尼·斯特劳斯特鲁普，1950年12月30日出生于丹麦的奥尔胡斯，是一名丹麦计算机科学家，以创造和开发广泛使用的C++编程语言而闻名。

C++ 顾名思义，是在C流行的时候作为下一代语言开发的。它带有面向对象的编程特性，与更结构化的C编程相比，这被认为是非凡的。

C++ 仍然是最流行的语言之一，因为它与本地系统非常接近，并且具有流行的面向对象特性，所以在高频交易领域被广泛使用。

如果你想从零开始学C++的话，我推荐Udemy上的 开始C++编程——从入门到超越 。它将理论和实例恰当地结合在一起。

如果你不介意从免费资源中学习，那么你可以查看这些免费的C++课程来开始学习。

4.Python吉多·范·罗苏姆

Python是由CWI的吉多·范·罗苏姆设计的，是一种通用的高级编程语言，其设计理念强调代码可读性。据说它的语法清晰而富有表现力。

在美国， Python 在学术层面已经大部分取代了Java。与上一代不同，学生们已经开始学习使用Python而不是C或Java编程。

如果你还不确定开始编程时是用 Python 还是 Java ，这张信息图或许能帮到你。

Python广泛用于web应用程序开发、软件开发和信息安全。

Python也被谷歌、雅虎和Spotify等科技巨头广泛使用。如果你想学习Python，那么何塞·马西亚尔·波尔蒂利亚的《完整的Python 3训练营》 是最好的开始课程之一。

5.JavaScript —布伦丹·艾希

如果你问我在过去的五到十年中哪种语言是赢家，我会说是JavaScript。它通过像jQuery这样的库明显地统治了客户端脚本空间，现在正通过像NodeJS这样的库转向服务器端开发。

JavaScript 是一种基于原型的脚本语言，是动态的、弱类型的、具有一流功能的，由Brendan Eich设计，Netscape communication s Corporation开发。

JavaScript广泛用于客户端脚本、验证、动画、事件捕获、表单提交和其他常见任务。它运行在浏览器内部，几乎被所有主要网站使用。

我强烈建议所有程序员学习JavaScript。JavaScript简介& jQuery 是最好的起点。

6.PHP —拉斯马斯·勒德尔夫

无论你有多讨厌PHP，它代表个人主页，你都不能忽视这样一个事实:一半的互联网都在运行这种奇妙的网络语言。PHP最初是由拉斯马斯·勒德尔夫在1995年创建的。

PHP 的主要实现现在由PHP小组产生，并作为PHP语言的正式参考。

PHP是微软的动态服务器页面(ASP)服务器端脚本引擎和类似语言的竞争对手，例如 Java Server Pages (JSP)，但逐渐被接受，现在安装在2000多万个网站和100万个Web服务器上。

它也是开源的，被脸书、维基百科、 WordPress 和Joomla等互联网巨头使用。PHP被广泛用于构建动态网页和服务器端开发。如果你想学PHP，网上有很多课程，我推荐jorge edwin diaz prado的 PHP for初学者 。

如果你想使用免费资源学习PHP，那么这里有一些 免费PHP和MySQL课程 的列表。

7.Perl —拉里·沃尔

Perl是一种高级、通用、解释型、动态编程语言。Perl由拉里·沃尔在20世纪80年代中期设计和开发，由于其出色的文本处理能力而声名鹊起。

它仍然是在UNIX系统上开发报告和脚本的主要语言。Perl 以解析和处理大型文本文件而闻名，用于CGI、数据库应用程序、网络编程和图形编程。

Perl也被IMDB、Amazon和Priceline等公司广泛使用。如果你想学习Perl，那么我建议你看看Udemy上的 Perl初学者编程 课程。

对于Java开发人员来说，将Perl或Python添加到他们的投资组合中是一个很好的补充，因为出于维护和支持的目的，你经常需要一种脚本语言来完成特定的任务。

8.红宝石——松本幸弘

Ruby最初是由日本的松本幸宏(Yukihiro“Matz ”)在20世纪90年代中期设计和开发的。使用Ruby很有趣，如果你尝试过Ruby with Rails你就会明白我的意思。

Ruby受Perl、Ada、Lisp和Smalltalk的影响，是为高效和愉快的编程而设计的。

Ruby主要用于web应用程序开发，并被Twitter、Hulu和Groupon等主要网站使用。

如果你想学习Ruby，那么Alex Korban在Pluralsight上的 Ruby基础 课程是最好的起点。

9.Lisp —约翰·麦卡锡

Lisp，代表列表处理器，由约翰·麦卡锡开发，是第二古老的高级编程语言。

我从未尝试过Lisp，但据说它是函数式编程语言之父，如 Haskell 、 Erlang 或 Scala 。它主要用于人工智能开发和防空系统。学习Lisp的课程不多，特别好的，但是Nikoloz的 学习Lisp编程基础知识 是一个很好的开始。
