我的5步计划是在2021年作为一个完全的初学者学习如何编码

没有训练营或在线课程

在当今的数字经济中，编程是你工具箱中最重要的技能之一。

毫无疑问，学习编程可以在技术领域打开就业机会，同时它也可以帮助你在未来的职业生涯中获得一个令人兴奋的创业角色，比如内容创作或数字营销。

如果你梦想周游世界，科技行业正在引领远程工作革命。学习编程还可以让你更容易从事自由职业，成为自己的老板。

和任何技能一样，你能获得的机会取决于你如何创造它们。

我内心深处有一种感觉，我应该学习编码一段时间。在我的研究中，我简要地学习了Python的一些方面，但是因为太难而放弃了。我似乎不像我的同学那样容易“理解”编码。

我认为它不适合我，并在毕业前继续学习其他课程。

三年后，我想用一本傻瓜程序设计教科书来打击19岁的我。如果我那样想，我将一事无成。

我最近回到了编程，因为学习这项技能是我2021年目标清单的一部分。我最初需要它在我的工作中做一些数据分析，并开始再次查看基础知识。

我惊讶于第二次我是如此的喜欢它。最后，我觉得我在工作中做了一些有趣的事情。我意识到，一直以来，我最怀念的一件事就是积极学习新技能。

现在，我发现自己在早上和晚上的空闲时间编程，并尽可能多地学习。我肯定想充分利用我的时间和网上的所有资源，所以今年，我计划继续尝试，看看我能在编程方面走多远。

以下是我今年作为初学者学习编码的五步计划，没有训练营或在线课程。

1.定义你为什么想学习编码

如果这看起来像是基本的第一步，那是因为它确实是。然而，像所有的第一步一样，花些时间和精力考虑你的目标是值得的。

定义你为什么想学习编码是在新爱好的最初热情消退后保持动力和一致性与迅速把它扔到一边做别的事情之间的区别。

对我来说，我目前正在规划我职业生涯的下一步。我喜欢自由职业、远程工作，或者甚至回到学术界攻读博士学位的想法。学习编程为我打开了所有这些可能性，同时帮助我将一项新技能带到我当前的工作中。

决定你为什么要学习编码。你想要升职还是换个工作岗位？你愿意辞职自己创业吗？你想从事学术研究吗？

写下你的目标和你想达到的目标，看看学习编码在哪里与它们一致。如果你这样做了，你会有更清晰的愿景和前进的动力。

2.决定一门与你的目标一致的语言

将学习编程的原因藏在心底，现在你必须决定开始使用哪种语言。

对我来说，我选择了Python。这是因为我在本科时已经接触过它，它是我的物理和工程学术领域中使用最广泛的语言之一。

它也是对初学者最友好的语言之一，因为它的简单语法类似于普通英语。

如果你是一个完全的初学者， Python和JavaScript 是很好的起点，因为这两者的需求量都很大。

JavaScript 在网站和应用程序开发中最受欢迎，所以如果你的目标是获得一份软件开发员的工作，熟悉它是很重要的。

如果你对应用程序开发感兴趣，无论是作为职业还是追求自己的伟大想法，Swift 也值得学习。

如果你对数据分析或数据科学更感兴趣，无论是学术研究还是专业工作， R或SQL 可能是你的最佳选择。

不管你的长期目标是什么，每个人都有自己的语言。

要做的主要事情是决定一种编程语言并坚持下去。一旦你有了一个，接下来就容易多了。

3.通过可用资源让自己熟悉基本概念

当你第一次开始学习编程时，花很多钱去参加训练营甚至是在线课程是很有诱惑力的。

然而，在进行重大的金融投资之前，有必要看看你是否喜欢编程，是否愿意花很长时间去学习。

有无穷无尽的免费资源可以帮助你学习编程背后的基本概念。我从 freeCodeCamp 和 W3Schools 开始。

这两个网站都概述了基本概念，并有多项选择题，允许您参与内容并测试您对知识的掌握程度。

也不要注销书籍。作为起点， Head First 系列尤其受欢迎。

还有更多的互动在线资源可供年费使用，如 Codecademy 和 Dataquest 。如果你想更多地致力于学习编程，支付年度会员费是一种相对低成本的利用自己的时间进行交互式学习的方式。

你也可以通过Coursera或LinkedIn Learning了解课程。

找到一些免费或低成本的资源是深入了解并获得一些实际入门经验的好方法。这将帮助你认识到学习编码是否是你想要做的长期承诺，以及它是否适合你未来的目标。

4.独立练习并应用所学知识

学习编码不仅仅是通过阅读书籍或观看视频就可以完成的事情。

编程首先是一种动手能力。最好的学习方法是边做边学。这意味着开始建立自己的项目，独立解决在线可用的问题，即使你觉得没有准备好。

一开始可能会有一个陡峭的学习曲线，但是和任何事情一样，在你和实现你想要的东西之间唯一的障碍就是时间和练习。

培养自信并把你的技能运用到工作中的最好方法是在没有书本或老师的情况下独立练习。

如果你还没开始，网上有一些好主意。你还可以在 GitHub 和堆栈溢出上获得灵感、帮助或想法。

通过设定一个目标并想出用你选择的语言解决问题的方法来建立一些实际的东西。这是学生和程序员的最终区别。

5.重复！

最后一步是重复步骤1-4，并定期出现，投入自信地学习编码所需的时间。

有了编码，没有一个时刻你会神奇地知道一切并掌握你选择的语言。

科技领域在不断发展变化，其中最美妙的事情之一就是你可以不断地学习新技能，拓宽知识面。

如果你致力于学习编程，你需要对你学习的每一门新语言重复上述步骤。你还需要利用网上资源，致力于独立解决问题和构建项目。

如果你有动力，并把“为什么”藏在心里，你会成功的。

继续走。

从这里去哪里

恭喜你！如果你已经按照上面的步骤做了几个月，很有可能你正在成为一名自学成才的程序员。

如果你已经决定编码是你未来的一部分，你可能想知道从这里去哪里。这可以归结为你的编程目标，但也可能是:

  • 训练营或证书。
  • 申请免费培训中心，如 Recurse中心或创始人和程序员。
  • 跳进来寻找工作机会——无论是在研究领域还是在工业领域。

在你和你想得到的编码之间唯一的障碍是你投入时间和激励自己的能力。没有学位或训练营也不是不可能得到一个软件开发的角色。

如果你不去要求机会或者自己创造机会，它很少会凭空出现。现在就开始规划你的未来，学习编码。
