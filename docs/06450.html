设置

本质上，Electron是一个 Node.js 应用程序。像任何节点应用程序一样，它将有一个包含指令和依赖项列表的package.json文件。

让我们创建一个新的节点项目:

yarn init// ornpm init// I prefer yarn

它将用新应用程序的初始信息创建一个package.json。我们需要在我们的package.json中添加一个脚本列表，这样我们就可以运行电子图书馆。这是最初的package.json的样子:

接下来我们需要安装电子图书馆:

yarn add electron
