JavaScript 是你可以学习的最流行的语言之一。正如许多人所说:“如果你打算只学一门编程语言，那就去学 JavaScript 吧。”

在最近的一次采访中，FreeCodeCamp 的创始人昆西·拉森被问及开发人员应该首先学习哪种语言。他回答:“JavaScript。”：

> “软件在吃世界，JavaScript 在吃软件。JavaScript 正逐年变得越来越占主导地位，没有人知道最终什么会取代它。
> 
> 如果你没有非常好的理由去学习一门新语言(比如你的工作要求你维护一个非 JavaScript 代码库)，我的拙见是专注于更好地学习 JavaScript。"

如果这对你来说听起来很有说服力，这里列出了从这个项目中摘录的 127 个有益的片段，你可以立即学习和使用。

# 1.全部

如果谓词函数为集合中的所有元素返回`true`，则该代码片段返回`true`，否则返回`false`。如果想使用`Boolean` 作为默认值，可以省略第二个参数`fn` 。

# 2.allEqual

这段代码检查数组的所有元素是否相等。

# 3.大约相等

这个代码片段检查两个数字是否彼此近似相等，只有很小的差别。

# 4.arrayToCSV

该代码片段将没有逗号或双引号的元素转换为具有逗号分隔值的字符串。

# 5.arrayToHtmlList

这个代码片段将数组的元素转换成`<li>`标签，并将它们附加到给定 ID 的列表中。

# 6.试图

这个代码片段执行一个函数，返回结果或捕获的错误对象。

# 7.平均的

该代码片段返回两个或更多数值的平均值。

# 8.平均地

在最初使用给定函数将每个元素映射到一个值之后，这个代码片段返回一个数组的平均值。

# 9.分叉的

这个代码片段将值分成两组，然后将一个 truthy 元素`filter` 放在第一组，否则放在第二组。

您可以使用`Array.prototype.reduce()` 和`Array.prototype.push()` 将元素添加到基于`filter`的组中。

您可以使用`Array.prototype.reduce()` 和`Array.prototype.push()` 将元素添加到组中，基于`fn` 为每个元素返回的值。

此代码片段检查页面底部是否可见。

# 12.字节大小

# 15.castArray
