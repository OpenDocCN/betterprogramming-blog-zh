## 如何用 clear，pop，remove，del 从列表中删除

在 Python 中，有几种方法可以从列表中移除元素。

让我们来看看四个主要的方法:`clear`、`pop`和`remove`方法，以及`del`操作符。

在下面的例子中，我将使用一个公共列表来演示每种方法。为了简单起见，让我们假设使用的列表是`l = [2, 3, 4, 5]`。

# 清楚的

方法从列表中移除所有元素。当您不再需要列表的任何元素，而只想对新数据重复使用同一个列表对象时，这很有用。

## 例子

在我们调用`clear`之后，列表是空的——没有返回任何内容:

```
l
# l = [2, 3, 4, 5]l.clear()
# l = []
```

# 流行音乐

`pop`方法移除特定索引处的列表元素，并在移除后返回元素的值。如果您想从列表中删除一个元素，但仍想将该元素的值用于其他目的，这将非常有用。

## 例子

当我们调用`.pop(0)`时，索引`0`处的元素从列表中移除并返回:

```
l
# l = [2, 3, 4, 5]res = l.pop(0)
# l = [3, 4, 5]
# res = 2
```

# 去除

方法通过值而不是索引来删除列表元素。如果有多个元素与提供的值匹配，则第一个匹配的元素(索引最低的元素)将被删除。当您预先知道需要删除哪个元素，并且不想在删除它之前浪费资源查找它的索引时，这很有用。

## 例子

当我们调用`.remove(3)`时，列表中的第二个元素被删除(`3`)，并且不返回任何内容:

```
l
# l = [2, 3, 4, 5]l.remove(3)
# l = [2, 4, 5]
```

# 倒三角形

最后，我们有`del`操作符。这是 Python 内置的关键字，而不是 list 方法。`del`操作符允许你通过索引从列表中删除元素。这种方法很有用，因为它既可以通过提供单个索引来删除单个元素，也可以通过提供一系列索引来删除多个元素。

## 例子

首先，我们将从列表中删除一个元素。当我们`del l[0]`时，我们从列表中移除索引`0`处的元素。

```
l
# l = [2, 3, 4, 5]del l[0]
# l = [3, 4, 5]
```

现在，要删除多个元素，我们可以使用列表的一部分。当我们`del l[0:2]`时，我们正在移除索引`0`和`1`处的元素，因为我们指定了索引的范围— `0, 2`(下限包含，上限不包含):

```
l
# l = [2, 3, 4, 5]del l[0:2]
# l = [4, 5]
```
