## 如果你正在创业，回答关于栈溢出和其他平台的问题会有什么帮助

![](img/81d50f4e75e56e93af15ba7557143685.png)

在[dribble](https://dribbble.com)上由[居荣](https://dribbble.com/jujung)拍摄的照片。

今天，我们想稍微偏离一下与编码相关的文章，谈谈如何帮助其他开发人员并回答不同平台上的问题，如 Stack Overflow、Quora、GitHub、Reddit 等。我想很多人已经知道程序员互相提供很多支持，在大量的资源中你可以找到很多问题的答案。与这种响应相关的主要结果之一也是开源项目。但是我们将在以后的文章中讨论这个问题。

现在，让我们开始讨论回答 dev 问题和与他人分享您的知识的重要原因。

# 1.通过帮助他人来提高你的知识

这是最重要也是最有价值的原因。通过帮助他人解决他们的疑问和问题，你将提高你的知识，并以更好的方式理解这个问题的解决方案。通常，这种方法已经在学校使用。在全班同学面前在黑板上解决问题向其他人解释了任务的解决方案，因此他们可以开始更好地理解你所做的每一个细节。

> 知识就像金钱:要有价值，它必须流通，在流通中，它的数量会增加，而且有希望增值。— [路易斯·拉摩](https://www.brainyquote.com/quotes/louis_lamour_138868)

这同样适用于编程。通过帮助别人，你开始看到微妙的细节，并在更深的层次上理解问题。这允许你考虑更新更好的解决方案。

# 2.帮助让你感觉和其他开发者有联系

第二个原因与社会因素有关。通过回答不同平台上的问题，您将加入与您的编程兴趣相关的小组和主题。当你加入他们的时候，你会发现一群和你有相似兴趣的人。和与你做同样事情的志趣相投的人交流可以帮助你发展得更快。他们还可以帮助你创业和出主意。

![](img/b6ace12459e71075498e8b1d4d1ce1e5.png)

它还可以帮助你重新考虑你的决定，并在咨询了同样了解这些问题的人后，你可以找到更合适和完美的解决方案。因此，我们认为这一点非常重要，如果你想变得更好，并与那些对你所解决的问题感兴趣的人朝同一方向前进。

# 3.良好的声誉可以帮助你营销

当然，当你是一个创业公司的时候，你想赚钱，想推广你的产品。在你非常精通的领域，也许是世界上最好的领域之一，帮助别人会明显地推动你前进。成为不同平台的专家，用你的答案帮助别人，展示你知识的深度，显然可以提高你获得新客户的机会。

![](img/8c2dd68ff7b13e010f02edf020d5ea87.png)

在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上由 [Merakist](https://unsplash.com/@merakist?utm_source=medium&utm_medium=referral) 拍摄的照片。

大多数科技创业公司的首席执行官也是精通技术的人，经常访问这些平台——或者至少他们的首席开发人员可能会从这些平台上与他们分享一些解决方案。如果他们注意到你比他们更了解这些问题，他们可以向你的创业公司寻求帮助，成为你的客户。因此，这条道路也是非常重要和有用的。

# 4.帮助回旋镖

每次你能帮助别人的时候，记住你也会发现自己处在他们的处境中，最终会向更善于理解这个问题的人寻求帮助。因此，一定要提供帮助——至少提供一个链接或其他可以为他人提供一些指导的东西。将来，有人会帮助你，这个世界会变得更加愉快和快乐。

# 结论

虽然本文不包含任何技术细节或代码，但我认为告诉您我对此事的看法是很重要的。这不仅适用于程序员，也适用于其他行业的专家。

我希望这篇文章会对你有所帮助，你可以通过撰写不同主题的文章和教程，以及回答不同平台上的问题来帮助其他开发者。
