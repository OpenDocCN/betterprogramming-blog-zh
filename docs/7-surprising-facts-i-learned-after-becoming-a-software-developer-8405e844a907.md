## 现实和我想象的大相径庭

![](img/80b5e4bddfed5a14966964b3b5111ef5.png)

Avi Richards 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

我学的是计算机科学，从 2015 年开始做 web(前端)开发。在我的职业生涯中，我慢慢开始意识到，作为一名软件开发人员的现实与我的预期相差甚远。

因此，如果你有兴趣成为一名软件开发人员，在这篇文章中，我将尝试给出一些关于这个职业的细节(基于我的经验)，希望能帮助你了解你能从软件开发中得到什么。

# 1.我们没写那么多代码

![](img/fd19c7331992e34ec549ac2fea2a072c.png)

照片由[潘卡杰·帕特尔](https://unsplash.com/@pankajpatel?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

当我还是学生的时候，我认为软件开发人员每天要写 7-8 个小时的代码。

事实上，我们不会写那么多代码，除非我们在做一个全新的项目。在大多数情况下，找到工作后，我们的雇主会将我们分配到现有的项目中，该项目有以下要求:

*   会议
*   解决技术问题
*   实施新功能

**会议**

我们每天都有团队会议，此外，一周中的某些日子，我们还会与公司的其他部门、产品经理和其他非技术人员召开大型会议。

会议是我们日常工作的一部分。

**技术问题**

在日常生活中，这是我们大部分时间去的地方。我们需要解决 bug 和其他技术问题。

首先，我们需要定位问题，为什么会发生问题，搜索 Google，阅读，询问他人，学习，实施解决方案，测试，最后部署。

这个过程可能会让人不知所措，我认为大多数开发人员并不喜欢这个过程。如果严重的问题发生得太频繁，那么程序员将会改变他们的项目或者他们将会找到另一份工作。

**实施新功能**

最后，当有新的特性要实现时，我们编写代码。甚至这部分不仅仅是写代码，而且:

*   测试
*   代码审查
*   重构(如有必要)
*   与产品负责人/经理讨论这是否真的是客户想要的
*   部署

# 2.学校和工作是两个不同的世界

![](img/4205119eef0504c7da8f4524363606e5.png)

查尔斯·德洛耶在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

我于 2013 年从计算机科学专业毕业(学士学位)，但一旦我开始工作，我很快意识到学校和工作是两个不同的世界。

在学校，我们学到了一些东西，比如:

*   高等数学(微积分、线性代数等。)
*   c 程序设计和数据结构
*   数据库系统介绍
*   面向对象编程
*   操作系统介绍

我今天在做什么？前端 web 开发。所以我不得不自学以下几个方面:

*   网站如何工作
*   关于前端、后台和 API 的一般信息(以前从未听说过这些术语)
*   HTML、CSS 和 JavaScript
*   有角度，反应 JS
*   饭桶
*   排除故障
*   敏捷方法(Scrum 和看板)
*   协力
*   通讯技能
*   更多

因此，在我看来，无论你的背景如何——大学毕业、自学或参加训练营，都不会对你的职业生涯产生巨大影响。

在学校，是的，我学会了如何写代码，但这对工作来说是远远不够的。

# 3.仅仅学会编码是不够的

![](img/1496f835bc1320db1c0db3a77d2f789d.png)

照片由[对比](https://unsplash.com/@comparefibre?utm_source=medium&utm_medium=referral)[上的纤维](https://unsplash.com?utm_source=medium&utm_medium=referral)不飞溅

现在你可能在问“如果学习如何编码还不够，那我该怎么办？”

> 简而言之:获得经验。

经验是最好的老师。你可以从学校或者自学学习编码，但是没有经验，抱歉这么说但是你会吃亏。这就是为什么尽快找到一份工作或一个项目是很重要的。

获得一些经验后，你将开始学习:

*   如何学习(字面意思)
*   如何谷歌
*   如何解决问题(问题、错误)
*   如何团队合作
*   要多少钱
*   您应该选择哪个项目/技术组合
*   不要害怕失败
*   不要害怕实验
*   不管你有没有激情

# 4.激情是关键

![](img/6b8ef6b59fbf8fe1847c7e3f1de06a90.png)

照片由 [Randalyn Hill](https://unsplash.com/@randalynhill?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

我学到的另一件重要的事情是:

> 如果你想成为一名软件开发人员，你不仅要对写代码有热情，还要不断学习和解决问题。

如果你不觉得自己对这三样东西没有激情，就不应该成为软件开发人员。做点别的。

因为没有激情，你就没有动力，没有动力，你就没有效率，不快乐。激情是关键。

# 5.无尽的学习

技术进步很快，编程也是如此。自从我在这个领域工作以来，没多久我就意识到我永远不能停止学习新东西。

即使编程语言持续的时间更长(谢天谢地)，版本更新也会有很大变化(取决于语言)。

在你的职业生涯中，会发明新的库、框架和工具，你可能也需要学习它们，但是你几年前已经学过的东西可能会过时。

所以我们不能停止学习。

对我们中的一些人来说，这可能会让人不知所措。这就是为什么我在这里的建议是在同一个项目上工作至少 1-2 年，这样你就有时间思考下一步该做什么/学什么。

# 6.编程是反社会的，但是…

![](img/87ccefa9cb0c7d86706691fe5e2001b2.png)

普里西拉·杜·普里兹在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

我们每天花很长时间解决 bug，开发新功能，编写代码。此外，现在有更多的人在家工作，所以这比以前更加不合群。

我上学的时候，认识新朋友很容易。开始工作后，才知道变得更难了。但是在疫情之后，事情变得更加困难了。

人类是群居动物，或多或少我们需要周围有其他人。我实际上在社交酒吧里更内向一点。

但即使对我来说，这种反社会的环境变得如此之多，迫使我提高我的社交技能。

以下是我应对孤独的方法:

*   我打电话给我的父母、亲戚和老朋友聊天(每天)
*   我与同事建立了更密切的关系
*   我参加我感兴趣的活动(在线或离线)并结识新朋友
*   我更多的出去认识新的人

变得更社会化，建立一个人际网络会在很多方面帮助你:

*   这会增强你的信心
*   你可以找到新朋友
*   你可以为你的职业生涯找到一份工作和其他的好机会
*   如果你没有，你可以见见你的女朋友/男朋友，甚至你的妻子/丈夫

我们生活在一个全球化的世界里。外面有无尽的机会。编程和在家工作可能是反社会的，但如果你投入一些时间和努力，你会克服它。

# 7.编程极大地改变了我的生活

![](img/7b9fd6b9e5bf1baa446ec4f0b52c7c5d.png)

由[伊莉莎·肯纳莫](https://unsplash.com/@elisalou?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

当我成为一名软件开发人员后，我的生活发生了巨大的变化，并且仍在向更好的方向发展，这也是我没有预料到的。

编程可以是快乐的，但也是具有挑战性的。这个职业的挑战迫使我决定去过更好的生活。

以下是编程给我的生活带来的两大改变:

*   我搬到了另一个国家，找到了一份全新的工作，开始了新的生活
*   我开了一个博客和 YouTube 频道，这是我从未想过会做的事

我做这些选择是因为我的编程生涯。

这些不必对每个人都完全一样，但我确信编程也会改变你的生活，希望是以一种更好的方式。即使这个职业很辛苦，我也很感谢我成为了一名软件开发人员。
