## 打个盹，提前计划

您可以与团队成员进行有意义的一对一会议，即使您身在异地。

远程会议带来了大量的挑战，包括技术、时间、数字干扰等等。[掌握与直接下属的一对一](https://medium.com/better-marketing/how-i-learned-to-run-better-1-on-1-meetings-c1bb9f80177e)也是成为优秀经理和领导高效团队的关键。

没有什么比与团队成员面对面更好的了，但即使在偏远的地方，你也可以采取许多步骤来更好地进行 1 对 1。以下是实现这一目标的五种方法。

# 1.每周至少见一次面

一对一的会议*必须*至少每周一次，这取决于你的直接下属负责多少，甚至可能更频繁。

一对一会议最简单的形式是状态更新。有人在做什么？他们需要什么帮助？有什么即将到来的担忧吗？这些类型的更新可以通过电子邮件发送。

如果你不提前投入工作，每周例会很快就会变得无聊和重复。你需要问一些问题，这些问题能够快速超越他们的工作量，更深入地了解流程、职业发展和正在发生的问题。频繁的会议将这一点作为成功的衡量标准。如果你们都害怕它们，而它们又令人厌烦，那么它们就没有得到有效的管理。

# 2.提前计划

寒冷的到来会导致很多“嗯”和不确定性。如果一个员工带了他们自己的议程，你仍然可以从一个你没有准备的会议中得到很多。如果他们不这样做，对每个人来说都只是浪费时间。

在一对一谈话之前，至少花 15 分钟回顾一下那个人自上次见面以来所做的工作，上次你们讨论的话题，思考一下你想要解决的任何具体可行的问题。

想出五个伟大的问题，关于他们的职业目标，他们希望事情如何改变/改善，或者其他有意义的事情。避免是/否问题或任何可以通过文本轻松回答的问题。这些是谈话的开始。如果它们建立在你上次讨论的主题上，会加分。

# 3.打盹懈怠

打盹松弛(或任何你使用的工具)是必不可少的。在远程工作之前，我在所有的会议室都挂了钟。我也会带着一个笔记本和一支笔去一对一。我的电话或电脑只是分散我与会面者的注意力。甚至用数字方式记笔记也意味着通知控制了局面。

处于远程位置使您无法切换到全模拟模式，但将状态设置为“离开”或在会议期间从繁忙的应用程序中发出休眠通知，将有助于您专注于正在与您交谈的团队成员并进行更深入的对话。

# 4.不要让 WFH 成为话题

就像含糊的抱怨不会帮助一个人成长一样，对被困在家里工作表示同情(或者说这有多棒)不会让你们两个有任何进展。始终解决战术问题或可以改变或改进的事情。

尽管彼此相距遥远，你仍然可以帮助他们发展和壮大他们的事业。对工作的反馈更加重要。如果“在家工作”出现了，解决可行的问题(更好的软件，工作/生活平衡，可用性)或转移到下一个话题。

# 5.布置作业

每一次一对一都应该以某种行动项目结束。当整个团队都在远程时，确保包括和其他人交谈。给你正在辅导的人找借口，与他们的同龄人和同事建立关系，甚至寻找更多的导师。

# 结论

经理和员工之间的一对一会谈仍然非常有意义，即使是在遥远的地方。与面对面辅导一样，他们需要提前做好准备，并专注于成长和改进，这样才会有效。
