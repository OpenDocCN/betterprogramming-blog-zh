> “许多优秀的人都是自学成才的”——赫伯·瑞茨

术语*自学成才的开发者*对很多人来说被认为是一个矛盾修饰法。事实上，每个开发人员在一定程度上都是自学成才的。然而，我对自学成才的开发人员的看法是，在计算机科学、工程或 IT(很像我自己)方面具有最低限度/没有大学背景的同时，能够编写生产级代码。

在本文中，我们将会看到一些我发现对构建/提高软件开发技能有用的存储库。

# 4.编码-面试-大学

**⭐️⭐️⭐️—183k
⑂⑂⑂—49.6k**

这个知识库是由 John Washam 创建的，他是亚马逊的一名软件开发工程师。

据他说，这个回购的主要目的是创建一个学习计划，从一个软件/网站开发人员过渡到一个 FAANG 公司的软件工程师。

回购涵盖的主题包括:

*   日计划
*   大 O 符号
*   数据结构和算法
*   哎呀，设计模式、线程/流程、网络、测试
*   系统设计
*   简历，回答一般性问题的技巧，向面试官提问的技巧
