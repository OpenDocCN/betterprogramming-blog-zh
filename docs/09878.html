7个未被发现的YouTube频道来学习React

React初学者的有用资源

在这篇关于在YouTube 上从哪里开始学习编码的后续文章中，我将告诉你一些具有良好知识库的频道，它们的关注者数量低于它们应有的数量。您将要阅读的列表由主要关注React前端开发的优秀教师组成。

如果您刚刚开始使用React作为框架，建议您先学习基础知识，然后再随意浏览其他资源。

  • 你可以从Mozilla的React入门指南开始，它向你展示了构建一个待办应用的基础。
  • 之后，在''入门'指南中阅读React官方文档，通过文档了解更多理论方面的内容。
  • 务必注意“挂钩部分。

我将提供的一些资源提供了结构化的课程，指导您从头开始构建应用程序的整个过程，而其他资源提供了更广泛的教程供您挑选。

事不宜迟，让我们进入列表。

Leerob

Lee Robinson 是一名前端开发人员，擅长清晰完整地传达编程概念。他几年前开始创作YouTube视频，主要关注React和Next.js，目前是Next.js背后的公司Vercel的开发者关系主管

也可以看看他的网站，里面有很多很酷的功能，还有关于如何在你自己的网站上实现这些功能的博客文章。

我最喜欢的东西

  • 他的初学者系列上 Next.js
  • 他与Next.js合作建造SaaS的新系列， React 2025

错误的阿克兰

我真的很喜欢Akram Khalid的频道，因为他给了YouTube一些它所缺乏的东西，一些我很喜欢的东西，即向我们这些外行人展示你如何能够创建与Awwwards.com的T2不相上下的互动网站，使用动画库，如T4的GSAP和T6的Framer Motion，并用React开发它们。

Akram有关于使用Figma准备您的设计、使用React和Framer Motion的基础知识的教程，甚至还有获奖网站的完整再现。

他发帖子的频率比以前少了一点，但你仍然可以在他以前的视频中找到很多精华。

我最喜欢的东西

  • Awwwards重建第1集|使用React重建Awwwards网站& GSAP |第1部分
  • React和Three.js (React三纤)入门

阿里·斯皮特尔

Ali是AWS的高级开发专家，也是web开发社区的长期发言人。她有自己的 YouTube频道，在那里她教初学者React、Next.js、GraphQL，还有AWS Amplify。

她在 alispit.tel 也有自己的网站，并在welearncode.com教授代码，在那里她分享关于上述主题的完整初学者指南。

我最喜欢的

  • 她在Next.js上的初学者视频，比如这个短片
  • 在AWS | Amplify Storage、Auth、Hosting和Data上创建Fullstack Next.js应用

她在她的频道上也有一个“我在别处”的播放列表，在那里她和其他讲座一起在我们的下一个主题频道上出现。

杰森·伦斯托夫

Jason Lengstorf目前在Netlify担任首席开发人员体验工程师。Jason的YouTube频道主要是他未经编辑的常规直播流的集合，其中他与一位知识渊博的嘉宾讨论了各种编程主题，主要是与 JAMStack 相关的现代前端技术，简而言之就是JavaScript、API和标记，这是一种前端与后端分离的开发理念，允许灵活的开发人员体验。

虽然在他的信息流中有相对大量有趣的闲聊，但也有硬技术质量编码。对于每一个可以想象的与现代前端开发相关的话题，Jason都有涉及。如果你想了解更多的现代网络技术，Jason有一个专门的视频。

还有，看看他俏皮的个人网站；他也在FrontendMasters.com大学教书(没有从属关系)。

我最喜欢的

有太多的直播流充满了可行的建议，但我只选择几个:

  • Contentful，GraphQL，&付费内容(与Stefan Judis一起)——与Jason一起学习
  • react-three-fiber中的动画和3D(与Paul Henschel一起)——与Jason一起学习

威廉·坎迪隆

William为我们提供了来自“美丽的瑞士苏黎世”的源源不断的React Native教程，重点不仅仅是代码的功能方面，还有开发有吸引力的应用程序的视觉方面。

他的系列能在React Native中完成吗？很好地介绍了React Native在设计和功能上模仿原生应用的可能性。非常透彻，非常容易理解。

我最喜欢的东西

  • 带React Native的SVG动画
  • 比特币基地专业烛台擦洗器——“它能在React Native中完成吗？”

利·哈利迪

Leigh的频道充满了有价值的中等长度的视频，直接进入许多有趣的React技术和库的肉和肉汤，您可以立即将其合并到您的项目中。

利是一个伟大的老师，他以平易近人的方式讨论这个主题。

我最喜欢的

  • Next.js with GraphCMS:静态站点生成&使用无头CMS的API路由
  • React中的谷歌地图——构建互动地图

肯特·多兹

如果你在React世界有任何经验，你一定会在某个地方碰到Kent。他是React世界的知名人士，提供课程，讲授React，并编写了 react-testing-library 。

他的YouTube上充斥着直播和其他与节目相关的内容。他已经记录了许多与自己项目相关的材料，每次会议都要处理几个与项目相关的任务，例如将Tailwind添加到他的书架应用程序。

他的办公时间被现场直播，他回答观众的问题，深入具体React技术的本质，因为它们发生在现实生活的项目中。

我最喜欢的

  • 他的会议讲话，只是简单地作出反应
  • 最近一次办公时间直播

这就结束了

使用YouTube的搜索功能，查看您最迫切需要的视频，并开始应用他们的经验教训。祝你好运！
