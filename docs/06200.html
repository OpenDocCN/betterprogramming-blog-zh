不能在VS代码中缩进你的html.erb文件是一件非常痛苦的事情。ERB格式化程序依赖于htmlbeautifier gem，所以我们也需要安装这个依赖项。

gem install htmlbeautifier

1.原子一号黑暗主题

你编辑的主题是个人选择。选择你觉得好看的，并提供有用的语法高亮。或者，检查材料主题。

2. VS代码—图标

漂亮的图标使得区分文件夹和文件类型变得容易。

3. DotENV

您的.env文件的语法高亮显示。

4.输出上色器

用彩色日志文件使阅读日志变得容易。

5.自动重命名标签

为自己节省50%选择结束标签的时间。

6.高亮匹配标签
