# 一般建议

我对这些垃圾的建议是不要在什么都不懂的情况下学习和重复。这在很多方面都是行不通的:首先，它没有用，而且你会浪费时间，因为很少需要字段内存。第二，AWS 是一家聪明的公司。亚马逊是一家 FAANG 公司，所以不要指望他们在学习时不会改变他们写问题的方式。

但我可以向你保证，除非 AWS 改变考试的方式，否则这些转储将非常有用，因为答案和评论有助于支持你背后的推理过程。所以，开动你的大脑，让我们开始吧。

# 解决方案架构师—助理

我在这个领域有六个月的经验，所以拿到了这个证书。

解决方案架构师助理级别是目前最好的认证。不算太深；不算太浅。它为您提供了了解 AWS 服务及其应用所需的适量知识。它提供了有效而简单的用例，其中充满了在云行业中使用的好例子。

奇怪的是，与专业认证相比，我对这个认证感到更加疲劳。也许是因为当时我的阅历太浅。

我在准备认证时使用的课程是由 Stephan Mareek 教授的。

斯蒂芬是一个真正的好老师；他非常冷静地慢慢解释这些概念。有时候在某些部分可能会有点慢，也许是你更有信心的部分。在这种情况下，我的建议是将视频加速到 1.5x，这样你仍然可以听到他要说的话。

课程结束后，你可以查看我的转储来重复问题，并熟悉评论。我的垃圾场并不完美；可能会有错误，可能会有一些错误的答案。但是，我可以向你保证，我已经在考试题目上浪费了很多时间来深入研究社区部分，以获得每个问题的正确答案。

# 解决方案架构师—专业人员

理论部分可以与[斯蒂芬·马瑞德的](https://www.udemy.com/course/aws-solutions-architect-professional/)或[尼尔·戴维斯的](https://www.udemy.com/course/aws-certified-solutions-architect-professional-training/)课程整合，只需几美元。如果你真的想投资一些钱在一门课程上，不仅为了考试，而且为了对所有科目的总体理解，寻找阿德里安·坎特里尔的课程。

首先，我们肯定会推荐解决方案架构师助理，如果可能的话，还会推荐认证开发人员。这是因为 SAP-C01 考虑到您已经了解 AWS 及其服务。这是一个更多地与用例以及在已经可行的答案中找到最佳答案(根据问题给出的上下文来定义最佳，即具有成本效益)相关的考试。你很少会找到错误的答案；更常见的是你找到一个更差的答案。

有不同种类的问题:单项选择、双项选择和三项选择。

单项选择题是最常见的，也是最容易快速摆脱的。也就是说，你至少要仔细阅读两遍问题的要求，这一点很重要。双重和三重选择题不太常见，也更难理解。我的建议是把重点放在消除选项和耦合一些选择上，以解开谜题并找到正确的组合。

考试时间 190 分钟。它需要快速阅读和理解。将 190 分钟除以 75 个问题，每个问题大约有 2.5 分钟的时间来按时完成考试。让你的身体和精神做好准备(吃饭、上厕所、参加早上的考试)，以便反应更快、注意力更集中。

为了避免在每个问题上花费太多时间，通过检测问题的主要关键词来阅读是很重要的，我认为这很有帮助。找到一个问题的关键词，就是直接把目光指向加快阅读过程的某些方面。例如，阅读问题的必要目标，如成本效益、最佳性能等等。

我第一次尝试使用这个垃圾场就通过了认证。但是问题的范围很广，所以问题中会有一些变化。

嘿，卢卡，这是不是意味着这个垃圾场没用了？

如果是，我就不会发表了。转储是有用的，因为它是一个很好的起点，让你了解问题的领域，加快你的过程，并习惯考试的模式。

至于备考时间，我大概花了两个月的时间。

祝您好运获得 SAP-C01。这可能是最难的 AWS 考试，但值得花时间参加。

# DevOps 工程师-专业

这个认证有种 [odi-et-amo](https://en.wikipedia.org/wiki/Catullus_85) 的感觉。它非常非常特定于 AWS 的 DevOps 工具，这是有意义的，因为它是 AWS 认证。但大多数时候，DevOps 或 SREs 不仅使用 AWS 的工具，还采用其他技术(Terraform、Jenkins、Ansible、Puppet、Chef 等。).

因此，这些很容易获得，但它们用于特定的技术。你可能想先买 SAP-C01 或 SAA-C02。

# 其他考虑

请注意，认证考试可能会随着时间的推移而变化。AWS 每隔几年就会改变其内容，使其更符合当前市场，提供全新的考试。这也是它们能持续三年的原因。虽然重考很无聊，但掌握最新的知识也很重要。
