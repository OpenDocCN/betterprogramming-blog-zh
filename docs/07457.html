我们本周出版的最好的东西

嘿大家好，

这将是我几周内的最后一篇时事通讯，因为我将在周一休陪产假，花些时间和我们即将到来的新家庭成员在一起！

不要害怕，亲爱的读者，你在伟大的手中。本周，我给我们的临时编辑、资深编程作家阿努帕姆·丘格写了一篇介绍文章。请热烈欢迎他！

事不宜迟，以下是我们本周发表的最佳文章:

JavaScript:

  1. Node . js 15中的新功能作者 Jennifer Fu —本周由Jennifer Fu带领我们，谈论Node 15中的新功能、新特性、错误修复等！
  2. React的未来:服务器组件 作者 Donovan So — Donovan看看服务器端React会是什么样子。我自己对此也很兴奋。
  3. 一种读取Node.js中文件的内存友好方式 作者Kasper Moskwiak——Kasper本周加入了时事通讯，教我们如何在没有内存泄漏的情况下读取Node.js中的千兆字节数据。好像有用！
  4. Michael Chi2021年React学习路线图——最后，如果你刚刚开始学习React，Michael Chi将在今年教你如何开始以及如何成为React大师。

软件工程:

  1. 4种不同寻常的提高编程技能的方法 作者 Szymon Adamiak — Szymon有一个有趣的提高编程技能的方法列表，这些方法可能与你想象的不同。我最喜欢的是第三条——限制自己。我有一些美好的编程记忆，在我下飞机的时候，我没有可靠的互联网，在事先没有任何想法的情况下，我做了一些很棒的东西。它通常只需要使用我的机器上的文件和大量的试验和错误，但我肯定在那次飞行中做得更好。
  2. 5个让你与众不同的非编码技能 作者:Fernando Doglio——很容易认为成为一名优秀的开发人员就是写干净、简洁的代码，但其实远不止这些。费尔南多本周回来探索这一点。
  3. 2021年PHP还相关吗？ 由布拉克·卡拉坎——“X死了。X万岁！”2021年该不该学PHP？当然可以！成千上万的公司使用它，社区很大，很活跃。Wordpress仍然很庞大，为互联网的大部分提供动力，并且是建立在PHP之上的。不要沉迷于TypeScript、Swift、Dart或你的同事本周向你推销的任何东西。
  4. 介绍和应对缓存挑战的策略 作者 Aastikta Sharma —缓存是开发人员众所周知的棘手部分。我们中的许多人都遇到过这种情况，想知道为什么有些东西不能清除缓存，或者可能是在处理由于意外缓存数据引起的bug。Aastikta在这里教我们更多关于缓存的知识。
  5. 理解工厂方法设计模式 作者:卡洛斯·卡瓦列罗 —卡洛斯本周回来教我们一个有用的设计模式——工厂。在您的项目或代码库中使用有意义吗？你是法官！

面试:

  1. 如何在不到10分钟的时间内构建一个LRU缓存，100行代码 作者孙李贝蒂 —孙李本周在这里讲一个著名的面试问题，构建缓存。如果你正在为一个技术面试而学习，这可能值得一读！

Python和数据科学:

  1. Python异步编程入门 作者:【崔永】 — Yong本周回来谈论Python中的异步编程，你应该如何考虑它以及用Python进行异步编程的细节。
  2. Django、RabbitMQ和Celery的分布式任务队列 作者: Louis de Bruijn — Louis是本周新闻简报的新成员，谈论分布式系统以及如何在Python、Django、RabbitMQ和Celery中正确使用队列。

杂项:

  1. K8s:Vanilla All Way作者 Luc Juggery — Luc本周回来，谈论Kubernetes中的Vanilla堆栈，以及为什么它可能对您的K8s设置有意义。
  2. 现在是2021年——这是你在艾伦·赫尔顿的《技术 》中应该关注的地方——最后，艾伦本周回来谈论你的API标准化、一切自动化，以及总体上变得更加以API为中心。如果你正在偿还API技术债务，绝对值得一读。
