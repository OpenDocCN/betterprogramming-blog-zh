# 问题陈述

对于数组中的每 k 个连续数字，打印最大的数字。这个挑战很容易蛮干，但是要努力找到最优解。

*   简单的解决方案:O(nk)
*   更好的解决方案:O(n log k)
*   最优解:O(n)
