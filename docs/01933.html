每个开发人员都应该知道的React库

无论您是一名专业的React开发人员还是刚刚开始使用React，您都希望尽可能地简化开发人员的工作。跟上React生态系统的所有最新趋势很难。项目正在快速发展。所以很容易错过一些很棒的React库。

这就是为什么我们将仔细阅读每个开发人员都应该知道的前七个React库。

1.洛达什

Lodash 通过消除处理数组、数字、对象和字符串的麻烦，使JavaScript变得更简单。在GitHub上有超过40k个星星，Lodash是一个受欢迎的库。根据其网站，Lodash是一个现代JavaScript实用程序库，提供模块化、高性能和额外功能。

尽管Lodash不是一个特定的React库，但它是我个人最喜欢使用的库之一，因为它非常优雅。我强烈建议你去Lodash看看。

2.反应引导

React引导库顾名思义:用React构建的引导组件。Bootstrap是一个用于HTML、CSS和JS开发的开源工具包。Bootstrap是世界上最流行的前端组件库。

React Bootstrap的伟大之处在于它取代了旧的Bootstrap JavaScript。该库中的每个组件都是作为真正的React组件从头开始构建的，没有像jQuery这样不必要的依赖。

作为最古老的React库之一，React Bootstrap已经随着React一起发展和成长，使它成为您的UI基础的一个很好的选择。

3.react-i18接下来

如果您需要一个多语言应用程序，那么您肯定需要 React-18next 库。React-i18next是React基于i18next的强大国际化框架。它为您提供了一个完整的解决方案，将您的产品从web本地化到移动和桌面。

i18next库具有翻译功能，如复数、格式化和插值。最重要的是，这个库适合服务器端渲染。

4.反应DnD

反应DnD (拖放)帮助你构建复杂的拖放界面，同时保持组件的解耦。与大多数其他伟大的拖放库不同，React DnD是建立在现代HTML5拖放API之上的。

这个库专注于正确的拖放交互，把它的视觉方面留给你。

过去，这个库有一个主要缺点——不支持触摸屏。如果你想要触摸支持，你可以使用React DnD与触摸后端，而不是HTML5后端。

5.材料用户界面

谁没听说过谷歌的材料设计？材质UI 是一个包含一组实现著名材质设计的组件的库。

GitHub上超过50k颗星，素材UI库绝对要在这个列表上。它是目前最流行的React UI包。它已经存在好几年了。这个库易于实现，并且组件引人注目且用户友好。

这个库有太多的组件，无法一一列出，所以我建议你查看Material UI本身。

6.反应旋转器

在你的应用程序中已经没有空间容纳无聊的旋转器了。 React spinners 是一个提供各种各样花式微调器的库。可以调整这些微调器的属性，以便您可以按照自己的方式完全自定义微调器。

这是一个小库，在GitHub上有大约1k颗星星，但它可以在UX方面给你的应用程序带来真正的提升。

7.酶

Enzyme 是一个JavaScript测试工具，用于测试React组件的输出。它由Airbnb工程团队创建，并于2015年作为开源软件发布。酶的API意味着直观和灵活。这个库应该使断言、操作和遍历React组件的输出变得更加容易。

Enzyme可以与各种各样的测试跑步者一起跑步，如Mocha、Jest、Karma。

如果你是测试React组件的新手，我强烈建议你去检查酶。
