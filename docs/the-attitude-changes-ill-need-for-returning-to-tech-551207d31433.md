当我辞去上一份软件工程师的工作时，我非常清楚一件事。

如果我回到科技行业，我需要改变态度。我不知道它到底是由什么组成的，但是如果我以现在同样的态度回来，我会继续痛苦下去。我将无法生存。

除了大量的阅读、大量的日志和大量的白日梦，我没有特别的计划去发现这种变化的本质。我不知道还有谁对自己的职业生涯得出了类似的结论——包括科技行业的其他女性。所以我没有人可以求助。

失业期间，我有过几次“恍然大悟”的时刻。其中至少有两条指出了必要的态度转变。

# 你可以从书本中学习，而不仅仅是在工作中

出于我无法向任何人解释的原因，我坚信唯一有价值的学习发生在工作中。我觉得其他大部分信息来源都不可靠或没用。

由于这些信念:

*   许多学习机会被锁在人们可能不会给我的高不可攀的工作和项目后面。我受其他人的支配。
*   当工作中出现问题时，我经常听取别人的意见，因为我想从中吸取教训。我没有从外部来源提供可能有助于解决我们面临的问题的信息。

在为求职面试而学习的时候，我的眼界终于开阔了。我回答了一个假设性的问题“所以你想学习如何扩展后端服务...你是如何朝着这个目标努力的？”我震惊地发现，我来到了相当空。我意识到我没有全力以赴去实现我的目标。我只是在等待合适的机会出现。那个回答似乎很无力。所以我开始改变它。

我在网上发现的东西让我大吃一惊。有很多相关的书籍、博客、YouTube 视频、面试学习指南。那里有大量(几乎)免费的信息，只是我没有意识到。软件公司正在谈论他们所面临的工程挑战，以及他们是如何解决这些挑战的。一切都在那里，在我的指尖。

于是，学习开始了。有时候，我发现自己在学习全新的东西。但我也在阅读一些书籍，将我偶然的工作经历放入一个结构化的背景中。我在为自己的职业生涯打下更坚实的基础。

# 你必须大声说出来

我的意思不仅仅是“我需要大声说出来让别人听到。”我的意思是“我必须大声说出来，否则我会很痛苦。”

当我在写自由分享想法的重要性时，我意识到了这一点。

我的线索是:当我打字时，手上的湿疹突然爆发。显然，我触到了痛处。

我的湿疹第一次出现是在几年前，当时我开始对工作感到不开心。正是在那些日子里，我第一次说出了“我觉得我的工作没人听我说”这句话。直到最近，我才意识到这两种现象紧密相连。我把我的湿疹归因于一种我无法控制的普遍的“性别歧视”和“有毒的工作场所”。

多年来越来越严重，我的湿疹在我辞职后有效地痊愈了。在过去的几个月里，每当我快要找到一份软件工程的工作时，它就突然成为一个警告信号。所以我很确定湿疹和我的精神健康紧密相连。

现在我知道，为了自己的幸福和心理健康，我需要大声说出来。我需要分享我的想法。我需要一个意见。我需要诚实。我不应该让自己被吓得保持沉默。

# 我又乐观了

我拥有这些主要的脑波需要很多因素的共同作用。我很高兴我终于找到了在科技世界生存所需的态度转变。

下周我有现场面试。我希望我做得好，被录用。这听起来像是一个惊人的工作机会——我一直在争取的工作。如果我被录用，我可以把上面的“啊哈”时刻付诸实践。

几个月来，我害怕重返工作岗位。现在我可以昂首挺胸地走进那份工作，乐观地认为我会快乐。有了正确的态度，我可以让它工作。

有可能我还缺少一块拼图。我可能会遇到新的问题。但是至少上面的想法会让我有一个好的开始。每当我有这样的脑波时，我就会对自己在科技世界生存的能力更加自信。
