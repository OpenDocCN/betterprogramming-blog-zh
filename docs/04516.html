如果编程语言有诚实的口号

现在给我们所有人来点喜剧

软件工程是一个经验可以变化的领域。一些人喜欢编程，而另一些人会觉得编程令人沮丧。毫无疑问，构建软件会让任何人紧张。

为了避免这种情况，这里列出了一些描述一些流行编程语言的诚实的口号——无意冒犯任何人。幽默有助于建立联系，我希望其中一些能让你发笑。

Java 语言(一种计算机语言，尤用于创建网站)

我们老了，总是啰嗦。但是付钱给我们，我们会给你长期的支持。

C

我可能是最大的，但你还是得给我分配空间。指指点点是我最大的爱好。

C++

你可以讨厌我们，鄙视我们，但我们会找到进入你的框架的方法。我们相信工作中的friends。

C#

我既不像我父亲，也不像我音符。如果有跨平台的GUI支持，我认为比Java更清晰。

左上臂

初学者喜欢我们，因为数组索引从1开始。但是数组不是数组，对象也不是对象，因为我们把一切都看成表。

Java Script语言

我们保证，这不是Java的脚本或从它派生的脚本。对了，苹果的Safari引擎还是会给我们回电的。

计算机编程语言

即使你不理解伪代码，我们也可以用真代码来写。不是分号或括号，而是额外的空格会引起混乱。

去

谷歌不仅对我的名字偷懒，他们还抢走了我们的仿制药和lambdas。但是我们有最好的国歌。

服务器端编程语言（Professional Hypertext Preprocessor的缩写）

要利用我，你不需要学太多，但只要少一个括号，你就会恨我。默认情况下，我们不提供命名约定或样式，只有您让我们看起来有多糟糕，我们就有多糟糕。

迅速发生的

我们很好看，简洁，比很多人都快。不像Objective-C那么混乱，直到你耗尽内存。

科特林

我们是每一个Java开发者应得的语言。将您从可怕的NPEs和冗长中拯救出来，同时运行在优秀的旧JVM上。但是不能有fun。

锈

不像我的名字那样过时，我提供内存安全，避免可能的竞争条件，并反对垃圾收集器。我爱borrowing，我比“C”更强大，只要你有时间了解我。

商用计算机编程语言

我们每隔几年就复活一次，即使在今天，整个失业体系都依赖于我们。另外，我们在招人。

编程笑话从一开始就存在，没有一种语言反对它。不同的语言处理不同的问题，这就是为什么有这么多的语言。针对流行的语言是人的天性。作为结束语，我想说——尽管上述口号确实描绘了事实，但不要把这些笑话当真。每种语言都有它自己的优点和缺点，我希望这篇文章能给你更多一点动力去挖掘一门语言的基础。
