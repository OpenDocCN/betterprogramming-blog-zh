## [为什么我不再阅读“我如何成为一名软件开发人员”的文章](/why-i-stopped-reading-how-to-become-a-software-developer-articles-dba17222f071)

“如果我能做到，你也能”这句话你可能在互联网上的许多编程文章中读到过。老实说，它已经有点饱和了，对于有抱负的软件开发人员来说，它几乎不能提供可操作的建议。正如 Sun-Li Beatteay 通过这个故事(我最喜欢的故事之一)所传达的那样，“我如何在 X 个月内成为一名软件开发人员”只是描绘了一幅关于软件开发的美好画面。它美化了少数人的成就，但可能不会惠及所有人。每个人都有自己的旅程，你的里程可能会因许多变量而异(这些变量通常会被忽略)。
