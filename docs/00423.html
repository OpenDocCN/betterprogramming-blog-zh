初级读本

你的电脑有一个特殊的地方存放程序使用的东西。它被称为内存，更确切地说是随机存取存储器，或简称RAM。随机存取存储器是分块布局的，想想乐高积木；每个块可以容纳这么多数据。数据以位或字节存储。

8位= 1字节

1024字节= 1千字节(KB)

1024 KB = 1兆字节(MB)

1024 MB = 1gb，以此类推。

我知道如果你以前从未见过这个，你可能会流泪，但如果你想学习如何编写程序，这是你需要了解的。

我们将在稍后讨论类型(以及其他内容)时讨论这个问题，但我想在继续之前先讨论一下这个问题。

先说够了。开始吧！

在我的Swift系列中，我立即开始谈论变量。这很容易，因为它是一种编译语言。Python是不同的，因为它是一种解释型语言。在编译语言中，你的代码被编译成你所使用的处理器的机器码。

解释语言被编译成字节码，当程序运行时，字节码随后被转换成机器代码。这可能会导致代码的优化，而这在其他语言中通常是必须考虑的。这些优化使得Python相对容易掌握。

变量

Python中的变量通常在声明时被实例化。声明一个变量意味着你创建一个桶来存储一个值，实例化意味着给变量一个值。变量存储可以改变的值。写变量的时候应该用snake_case。这意味着变量中的单词由下划线(_)分隔。

在后台，当你创建一个新变量时，你的计算机会做一些事情。首先，它查看a并说“嘿，我需要向RAM请求内存，让我们看看需要多少。”然后它看着= 1说“好的，我需要向RAM请求足够的内存来保存数字1。”它向RAM请求空间；如果RAM认为给它空间没有问题，RAM会说“嘿，我可以接受。”并且该值被存储在存储器中。

同样的事情也发生在其他变量上。

如果你想改变一个变量的值，这很简单。

a = 3
name = "John"

Python的一个简洁的特性是编译语言做不到的，那就是可以使用不同类型的变量。(稍后将详细介绍类型)

a = 1
a = "Bob"

这仍然是有效的代码。如果您尝试用C或Swift来做这件事，您会收到一个错误，但是Python知道如何处理这个问题。这并不意味着一次又一次地使用一个变量，它只是为您提供了这种灵活性。但是，有了这些知识，要明白使用相同的变量名会导致问题，我们将在本系列的后面部分讨论这个问题。

常数

常量是不能改变的值。在讨厌的人来纠正我之前，Python中没有常量。如果你来这里是希望有不同的阅读方式，我很抱歉，但没有。然而，有一些方法可以直观地显示出你的常量应该是什么，这向其他人暗示这些值永远不应该改变。您可以通过在所有大写字母中使用变量名来定义它们。

MEANING_OF_LIFE = 42
FIRST_NAME = "Bob"

常量用于名字、地点、事物、出生日期、结婚日期、死亡日期(直到僵尸启示录来临)等。如果我要创建一个与特定数据库对话的服务，我可能会将服务器主机名、数据库名和表名作为常量。至少我会使用服务器主机名，因为我唯一需要改变它的时候是数据库放在一个新的服务器上。

类型

类型是每种语言的本质，它们是帮助你做程序员做的事情的基本构件；移动比特。我将在这里介绍基础知识，并根据需要在本系列的后面介绍更高级的类型。

  • 字符串 —字符串类型可以包含任意数量的字符。是abc还是supercalifragilisticexpalidocious都无所谓。它仍然只是一个字符串。从Python 3开始，我们可以包含类型注释来将变量定义为特定的类型。使用my_awesome_variable: str来确保你使用正确的类型。字符串通常使用2字节的内存。
  • 整数 —整数类型保存数字。Python的特别之处在于，在32位机器中，您不会像在编译程序中那样受限于20亿的数值限制。Python允许您使用所有可用的RAM来生成一个数字，这使它成为数据科学的一种优秀语言。这也是一个缺点，因为你没有上限来崩溃你的程序。相反，你要么在达到硬件的内存极限时让程序崩溃，要么因为内存不足而让计算机崩溃。不要试图用逗号来实例化大数值的变量，而是使用2000或更容易阅读的2_000。32位程序以32 / 8 = 4字节存储整数，64位程序以64 / 8 = 8字节存储整数。
  • 浮点数 —浮点数本质上是十进制数。与其他包含双精度浮点数的语言不同，Python中的浮点数就像双精度浮点数一样，使用大量内存来获得精度。它确实截止了，但是在到达那个点之前，你会失去了解浮点的全部值的兴趣。据我所知，浮点数总是被表示为64位或8字节。
  • 布尔 —布尔基本上是真或假，1或0。它们是以提出它们的乔治·布尔命名的。布尔值只占用 1字节的内存。
  • 注释——注释并不是一种真正的类型，但是它们可以帮助你理解你的代码中发生了什么。你不想写代码，一年后再来看，却不知道它在做什么。评论帮你解决问题。我将在这里介绍两种类型的注释，单行注释和多行注释。
