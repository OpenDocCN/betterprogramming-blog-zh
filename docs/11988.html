提升您的Python技能

Python程序是为了解决问题而写的。当我们使用像Python这样的计算机编程语言来解决问题时，我们通常可以将问题分成三个部分——输入、处理和输出。当这些部分放在一起时，它们组成了输入-处理-输出程序模板。

在本文和以后的文章中，我将使用一组程序模板介绍Python编程语言。如果你有兴趣学习更多关于程序模板的知识，这里的是我写的一篇文章，描述了它们是什么以及为什么它们对学习计算机编程很重要。不过，现在我想通过介绍第一个程序模板——输入-处理-输出，直接开始学习Python。

输入-处理-输出模板

当我们编写一个计算机程序来解决一个问题时，我们的程序通常遵循三个不同的步骤。第一步是把我们需要的所有数据输入程序。第二步是处理数据以解决问题。第三步，输出处理的结果，一般是问题的解决方案。

让我们从研究如何将数据输入Python程序开始。

Python数据

有几种方法可以将数据输入到Python程序中，但是我们在这里将只关注一种方法:直接赋值。

在我谈论存储数据之前，我需要讨论数据在Python中是如何组织的。Python可以识别几种不同类型的数据。第一种数据是数字。Python中的数字要么是整数(整数)，要么是浮点数，比如3.14159。整数数据类型称为int。浮点数据类型称为float。

下一种类型的数据是文本数据，称为字符串。字符串是用单引号或双引号括起来的零个或多个字符的集合。类型名是str。例如，“Joan”是一个字符串，“3000 West Scenic”和“123–45–6789”也是如此。

任何时候数字被用作字符串，你都不能用它们做算术。例如，“12”+2在Python中导致错误，而“12”+“2”变成了“122”，因为应用于两个字符串的加法符号导致这两个字符串连接在一起。稍后我会有更多关于字符串的内容要说。

您需要理解的第三种数据是布尔数据。这种类型的数据表示特殊值True和False。数据类型名为bool。当您编写做出决策的代码时，通常会间接使用布尔数据，例如决定一个数字是偶数还是奇数，或者根据某人工作的时间支付多少工资。

还有其他数据类型，但这些是我们开始学习Python所需的主要类型。

通过直接分配输入

Python程序将数据存储在变量中。变量是指一个值的名称。例如，我们可以用可变工资来表示某人在一份工作中挣的钱。我们可以使用变量fullName来指代某人的全名，而不仅仅是他们的名或姓。

要在变量中存储数据，可以使用赋值语句将数据赋给变量。这个过程叫做直接分配。通过在变量和数据之间放置赋值运算符等号(=)来进行赋值。赋值语句的一般形式如下:

variable = data

这被称为语法模板，我将用它们来介绍新的Python编程结构和程序模板。

以下是Python赋值语句的一些示例:

name = "Paul McCartney"
salary = 40000
pi = 3.14159

当谈到赋值语句时，你需要区分语句的“左边”和“右边”只有变量可以用在赋值语句的左侧。右侧可以是计算单个值的任何表达式。

我说的评估单个值是什么意思？当您编写赋值语句时，例如:

salary = 40000

40000是一个单一值。更具体地说，另一个术语是数字文字。在下面的赋值语句中:

name = "George Harrison"

“乔治·哈里逊”是一个字符串文字。

在文字之后，右侧数据还可以包含表达式，如算术表达式和字符串表达式。例如，以下代码行使用算术表达式将数据赋给变量:

gpa = (4 + 3.5 + 3 + 3.75) / 4

下面是一个使用字符串表达式的示例:

name = "John" + " " + "Lennon"

当Python遇到这些类型的赋值语句时，它会简化右侧，并将结果值赋给变量。

打印输出

要从程序中获取数据，可以使用打印功能。该函数将提供给它的数据作为参数，并在屏幕上显示数据。以下是一些例子:

print("Hello, world!")
print(3.14159)
print("Ringo" + " " + "Starr")

在您的Python系统中尝试这些，看看会发生什么。

使用print函数显示数据还有更复杂的方法，但这些例子足以让您入门。

输入-处理-输出模板

现在我们已经知道如何使用输入-处理-输出 (IPO)模板来编写一些简单的程序。

不过，首先让我们来看看IPO的总体模板。以下是模板的概要:

输入数据

处理数据

输出加工结果

我们来看两个例子。第一个例子通过输入一个人工作的小时数和他们的工资率来计算这个人的工资总额。计算总薪资，程序显示输入的数据和获得的总薪资。以下是该计划，以及标识IPO模板各部分的注释:

# InputhoursWorked = 40
payRate = 15.25# Process
grossPay = hoursWorked * payRate# Output
print("Hours worked:", hoursWorked)
print("Rate of pay:", payRate)
print("Gross pay:", grossPay)

下面是程序的输出:

Hours worked: 40
Rate of pay: 15.25
Gross pay: 610.0

下一个示例从名称的每个部分构建一个完整的名称。程序是这样的:

# InputfirstName = "James"
middleName = "Paul"
lastName = "McCartney"# Process
fullName = firstName + " " + middleName + " " + lastName#output
print("The bass player for the Beatles was", fullName)

以下是该程序的输出:

The bass player for the Beatles was Paul James McCartney

最后的想法

你写的每一个程序都会在某种程度上遵循这个模板，尽管经常会有变化。例如，一些程序会接收一些输入，进行一些处理，返回一些输出，然后要求更多的输入。

有时输入和输出会来自不同的地方。例如，一些程序将接收来自用户的输入(交互式输入),然后将输出写到不同于屏幕的设备上，例如文本文件或打印机。有些程序会从文件中接收输入，然后将输出写入文件，这样运行程序的人就永远看不到输入或输出。

尽管如此，所有的程序都需要输入、处理和输出，所以在学习如何编程的早期理解这些编程需求是有好处的。
