## 3.为自己建立一个团队

![](img/d1ee71f6d6be837e32ddebebd4a0bbc1.png)

照片由 [Fotis Fotopoulos](https://unsplash.com/@ffstop?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

2012 年，我开始了我的机器人工程师之旅，当时我甚至还不知道。我高中毕业后，一直计划去大学学习化学或机械工程。

但是，在麦克马斯特大学，第一年的工程是一般的，你必须从所有的工程流的课程。之后，你在第二年再次申请进入一个重点班。就是在这段时间，我爱上了编程。

事实上，你可以写下指令，并把它们转化为行动，这是令人惊讶的。在我的第二年，我没有申请化学或机械，而是申请了机械电子工程(通常被认为是机器人工程)，这是机械、软件和电气工程的结合。

这是艰苦战斗开始的旅程的一部分。当我刚刚发现编程的时候，我班上的许多学生，如果不是大多数的话，从他们会打字的时候就已经开始编程了，我的课程期望他们有那样的能力水平。

我知道我必须适应，而且要快！在大学的四年里，我逐渐养成了更好的习惯和技巧，用更少的努力获得了更好的成绩。哪个有抱负的工程师不喜欢好的效率故事？

我的目标是不断改进我的流程，使其更加有效。我不仅毕业了，还上了院长的光荣榜。更糟糕的是，在我毕业之前，我已经被加拿大最好的工程实验室之一录取了。以下是我希望从一开始就知道的习惯。

# 尽可能做兼职项目

![](img/392fce741270cf27ea7e74088fcd83a7.png)

照片由[蒂姆·班尼特](https://unsplash.com/@timbennettcreative?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 拍摄

兼职项目可以是测试你的软件技能的任何东西。它可能是与一些朋友一起建立一个应用程序或网站，或者制作一些很酷的 Arduino/Raspberry Pi 机器人。

你可能需要几个小时或几个月来建造。我先说，如果你找不到一个让你感兴趣的副业项目(即使是一个简单的项目)，那么这可能是这个领域不适合你的早期迹象。

这个想法是为你做一些有趣的事情，如果你找不到，就跳槽，减少你的损失…不管怎样，继续前进。在这些项目中，你将能够包含你在学校学到的知识。这会让你对所学的工具有很好的理解。

此外，你将学到更多在学校里学不到的实用技能。这可能会导致一个你甚至不知道的软件新领域的发现。

对我来说，是机器学习/深度学习。我们在学校学过一些理论，但只有当我做自己的机器学习项目时，我才意识到我有多喜欢这门学科。我后来在本科毕业后获得了机器学习和深度学习的认证，现在我从事许多深度学习项目，作为我职业的一部分。

这就引出了我的最后一点:在简历中使用副业。在我简历的最后，我有大约一页半的篇幅来记录我参与和完成的所有兼职项目。这包括我共同拥有的一个网站、我的机器学习项目、我的亚马逊 Alexa 开发、我的 Arduino 项目和我的 Xamarin 应用程序。

在我本科毕业后的每一次面试中，我的副业项目的话题总是会成为话题。从统计数据来看，如果雇主尊重我在兼职项目上的工作量，我的成功率是 100%。一个好的雇主会尊重你在空闲时间做兼职的主动性。他们会把这看作是你渴望并愿意在你的手艺上做得更好的标志。

# 使用修订控制

![](img/a46bbf1ef36e9fc76739d864f6ba0b5d.png)

照片由[扬西·敏](https://unsplash.com/@yancymin?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

如果你不知道，修订控制，也称为版本控制，是一个跟踪代码变更的系统。这意味着您可以持续地保存代码在那个时刻的状态，并在将来返回到那个状态。

这也是一个备份你所有代码的系统。因此，如果出现问题(您的计算机被盗或其他原因)，您可以关闭您的存储库并继续工作。版本控制是我希望他们在大学里多推广的概念之一。

在最初的几年里，我的编码任务过程是类似于“好的，我刚刚完成了任务的第一部分。让我将这段源代码复制到一个“保存”文件夹中，并将其标记为 assignment-1.0.py。我会这样做 10 或 12 次，然后忘记哪个版本是哪个。

修订控制允许您将消息附加到代码的每一个快照上，使得浏览历史记录和找到您想要返回的状态变得容易。版本控制也使得团队项目的协作变得非常容易。

如果您都在项目的不同部分工作，您可以创建一个共享存储库。每个人都可以将代码推送到 repo，而不是通过 Slack 或其他方式发送代码，实际上，您可以通过命令行或 GUI 合并代码。这也让团队中的每个人都有责任。

代码的作者与每一次推送都有关联，这使得确定团队项目中谁在偷懒变得非常容易。此外，每个有自尊的软件企业都会使用某种版本控制，所以提前练习是有好处的。最常见的是 Git 和 Team Foundation Server (TFS)。

# 为自己建立一个团队

![](img/0fb5b9f120bfd4c1bbf6548e0b1e72c4.png)

普里西拉·杜·普里兹在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

你很快就会发现，努力完成大学学业是一项集体努力。尝试和与你有相似目标的朋友在一起是很重要的。

我的意思是，如果你的目标是在大学期间尝试创办一家公司，并愿意牺牲一点成绩，那么找一群有同样雄心壮志的人。

或者，如果你想获得博士学位并进入学术界，找到那些人。你会在大学里找到各种各样的人，利用这一点。此外，检查一下你的同学是否有脸书小组，如果没有，就创建一个。

你可以通过你的大学开设脸书团体。我们有一个机电一体化工程的脸书小组，人们会问作业问题，张贴网络事件，考试日期，学习会议，或者只是工程迷因！这是一个无价的资源，直到我第二年才发现。不要一个人读完大学。

# 对作业使用自动化测试

![](img/2dae43857d2da79fd8effe7b1518e9fb.png)

照片由[费伦茨·阿尔马西](https://unsplash.com/@flowforfrank?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 拍摄

大多数编码任务都有 4-5 个步骤，有时，这些步骤会相互叠加。您希望确保代码的所有部分都完全按照预期工作，但是您对代码的构建越多，就越难跟踪什么在工作。

同样，编程赋值也有“极限情况”，如果给定某些输入，代码可能会灾难性地崩溃。这可能很简单，比如试图获取一个空列表中的元素，或者试图求一个负数的平方根。

这些因素将导致频繁地手工测试代码。为什么不试着开发自动化测试来确保你需要的所有功能都正常工作呢？

这个想法是将代码分解成更小的片段，并按照您期望的输出分别测试它们。这样，当出现问题或测试失败时，您就知道是哪个代码段出了问题。这个概念被称为[单元测试](http://softwaretestingfundamentals.com/unit-testing/#:~:text=UNIT%20TESTING%20is%20a%20level,and%20usually%20a%20single%20output.)，每种编程语言都有一个库来完成这个功能。

你可以将这个想法更进一步，进行[测试驱动开发](http://agiledata.org/essays/tdd.html#:~:text=Test%2Ddriven%20development%20(TDD),fulfill%20that%20test%20and%20refactoring.)，其中你实际上首先编写测试，然后创建通过测试的代码。这种技术有时很难实现，有一个相当大的学习曲线，并且在开始时会很耗时，但它会有很大的回报。

# 参与黑客马拉松和其他编码活动

![](img/418ced16455d7580b6ed51a9413bc9dc.png)

[MD 杜兰](https://unsplash.com/@mdesign85?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

黑客马拉松是软件开发人员聚在一起，试图在短时间内完成一个软件项目的活动，通常是在一个周末。然后，该团队将不得不向评委小组推销他们的项目。有奖品、游戏、免费赠品和新朋友。

参加黑客马拉松比赛与之前描述的兼职项目有相似的好处。黑客马拉松的好处是他们通常有一个主题或特定的目标，所以如果你在构思自己的项目想法时遇到困难，黑客马拉松将会激发一个想法。

黑客马拉松的短暂时间是福也是祸。这是时间管理方面的一个很好的实践，也不是一个承诺，但是，你不会在比赛中使用最佳软件实践。在我的硕士论文中，我发现了 [Hackster.io 竞赛](https://www.hackster.io/contests)。这些竞赛就像黑客马拉松，唯一的区别是它们通常持续几个月。这让你有机会设计和创建一个更先进的成品项目。我强烈推荐。你可以点击查看我的参赛作品[。](https://www.hackster.io/daquilnp)

# 相信自己，保持动力

![](img/f9b6cff2e4e602859f3c67e9409b014c.png)

照片由[冉贝尔科维奇](https://unsplash.com/@berko?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

我知道这听起来很俗气，但它非常重要。获得本科学位将是对耐力的考验。就像跑马拉松一样，你会有疑虑，可能会有放弃的想法。尤其是前 2 年，这是你的大学试图淘汰学生的时间段。

到第一学期结束时，我的第一年班级从 1200 人增加到 600 人，第二年增加到 300 人左右。此外，还有一种被称为“大一 15”的普遍现象，即一名学生在大学第一年的成绩将比高中时下降 15%，体重增加 15 磅。这是非常令人沮丧的消息。你是如何停止怀疑而不退出的？

你需要有动力和自我肯定。自我肯定的形式是定期提醒自己有多聪明和勤奋。尽可能的庆祝小胜，也许你的成绩没有你想要的高，但是你做的比平均水平好吗？你的成绩是不是整体在慢慢提高？为你的进步感到骄傲。

动力应该来自于用这个学位设想自己的未来。任何时候你需要动力的时候，你都应该问自己“这个软件学位对我有什么用？”。当你做作业时，你会停下来想“哇，我真想以此为生”吗？你对开始新的兼职项目或参加黑客马拉松感到兴奋吗？提醒自己为什么你属于你的项目，另一边是什么。

# 尽可能保持健康和活跃

![](img/3501674a6826a2406e94e43ea295213c.png)

照片由[维克多·弗雷塔斯](https://unsplash.com/@victorfreitas?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

我并不是说每周花 7 天时间在健身房里，然后变得筋疲力尽(当然，除非你愿意)。这里的想法是每周进行 3 到 4 次身体活动，每次至少 45 分钟。"但是尼克，我没有时间健身，考试就要到了，我下周有三个作业要交，而且……"，是的，我以前都听说过。

你猜怎么着，现实世界也有压力时期，如果有什么不同的话，那就是持续的压力状态。我们都知道无论如何你都会在考试周狂饮网飞！如果你现在忽视你的身体健康，你将会养成一辈子忽视它的习惯。

身体健康和心理健康是相辅相成的，你的整体健康将有助于你在学校和生活中取得成功。大学是养成好习惯的时候，这些好习惯会伴随你一生。确保你的整体健康至关重要。

我希望这些技巧和技术能让你在软件领域有一个漫长而成功的职业生涯。感谢阅读。如果你喜欢这篇文章，请查看我的页面，了解更多故事:)
