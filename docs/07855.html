11个JavaScript库来提高生产力

Lodash、ExpressJS等等

我们在日常生活中都会写很多代码。有时我们创建巨大的应用程序，有时我们花时间为简单的功能编写代码，我们可能根本不需要编程。

有成千上万个用于各种用例的库，使用这些包可以节省大量时间，您可以用这些时间来提高工作效率。

1.洛达什

Lodash 是一个用于快速完成任务的现代JavaScript实用程序库。这个库的主要用途是它广泛的数组和对象函数，它可以用于几乎所有的项目。

Lodash没有编写自己的数组函数，它一定会有您可能需要的东西。

2.UUID

UUID是一个非常小的包，用于快速生成uuid(或通用唯一标识符)。

在某些应用程序中，出于安全等原因，应该避免使用自动递增的id。UUID是一种众所周知的创建ID的方法，一些语言提供了现成的方法。

3.快递. js

Express是Node.js的一个最小的web框架，它是众所周知的，被认为是大多数项目的最佳选择之一。

由于Express的极简特性，许多框架都是基于Express的。启动服务器只需要不超过11行代码。文档不是最棒的，但是社区很大。

4.引导程序

制作网站时，程序员可能不想重温创建布局的基础知识。 Bootstrap 解决了这个问题。使用Bootstrap，您可以从官方文档和庞大的社区中获得广泛的可用组件。

它处理样式和响应没有太多问题。它广受好评，有一个提供主题和组件的大型社区。

5.Next.js

Next.js 是一个使用React快速构建服务器渲染的静态网页的框架。它通过读取pages目录中的文件来处理路由。任何子目录都是URL的另一个步骤。

这是一个JAMstack框架，可以通过Vercel从GitHub轻松部署。它的工作方式类似于Gatsby，也是一个JAMstack框架。它们都有很多现成的功能。

6.顺风CSS

这是我最喜欢的样式库。 Tailwind CSS 基于实用程序类，提供了比Bootstrap更大的灵活性，无需编写一行CSS代码。

响应来自于CSS类的前缀。Tailwind提供了一种简单灵活的方式来创建漂亮的网页。您还可以创建自己的类，为可重用组件“应用”Tailwind类，以增加可读性和可维护性。

7.Axios

Axios 是一个允许用户轻松执行HTTP请求的包。

Axios的强大之处在于它的简单性，而且它仍然很灵活，因为您可以添加您需要的所有选项。

8.护照

Passport 是一个广泛应用于Express和其他Node.js应用的认证中间件。

它支持所有主要类型的身份验证—从脸书到OAuth，再到您自己的身份验证。可以很容易地将其放入Express应用程序中。这是一个广受好评、规模相当大的社区。

9.白垩

这个终端样式库从列表中的其他选项中脱颖而出。粉笔可用于设计终端样式，增加可读性。

它易于阅读和设置。向错误消息添加红色文本或向成功消息添加绿色文本可以减少滚动整个终端所需的时间。

10.国际光子

Luxon 是一个简单而现代的时间和日期处理包。它有很多与Moment.js相同的功能，但它使用了更现代的Intl对象，而不是通常的Date对象。

如果你需要日期和时间操作，Luxon是一个不错的选择。现代、经得起未来考验、易于书写。

11.Nodemon

Nodemon 是一个帮助你开发基于Node.js的应用程序的工具，当检测到文件夹中的变化时，自动重启应用程序。

有些库(比如Next.js)自带快速刷新功能，但是如果没有，Nodemon是一个不错的选择。

结论

在开发大型应用程序时，库会有很大帮助。虽然我们大多数人喜欢为小任务编写自己的代码，但Lodash等库可以帮助我们利用这段时间做更好的事情。你在你所有的项目中使用任何其他的库吗？让我知道。
