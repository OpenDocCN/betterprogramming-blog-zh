你很有才华，努力工作，想成为大人物。

不要在创业的时候做。

我为许多初创公司工作过，经历了从三个人在一个共同工作的空间到获得 1 亿美元资金的扩张。

对于有能力的工程师来说，投资的时间回报是没有意义的。有更快的方法去学习，赚钱，过得更好。

五年前，我会欣然接受为另一家创业公司工作的机会，但现在不会了。

# **你要放弃师徒关系**

你不会从高级工程师那里得到你需要的指导。

在足够长的时间内，师徒关系是双赢的。但当流动率很高，工作太多，公司每天都在烧钱的时候，情况就不一样了。

我对这种关系的双方都有不好的指导经验——作为一名大三学生，我必须独自解决问题；作为一名大四学生，我没有时间指导。

相比之下，为一家老牌公司或 MRR 不断成长的初创公司工作会给你更多向高级人才学习的机会。

导师是成功职业生涯中最被低估的投入。甚至沃伦·巴菲特[和本杰明·格拉哈姆](https://en.wikipedia.org/wiki/Warren_Buffett)[手下的学徒](https://en.wikipedia.org/wiki/Benjamin_Graham)。不要在这里妄自菲薄。

# **宽而不深**

要做的事情太多了，还没有人知道什么有效。

创业公司的价值在于它解决的问题。因此，在产品符合市场需求之前，资源以“准备、发射、瞄准”的方式部署，从一个项目跳到下一个项目。

每周学习新东西很有趣。一段时间。但这不会让你成为专家，也不会给你推销自己的技能。

如果你还不知道专攻什么，这会很有帮助。它让我从软件工程师转变为数据科学家。但这也让我意识到，在开始未来的工作时，我对这两者知之甚少。

用四种不同语言编写 CRUD 应用程序的能力很酷，但不会让你赚很多钱。

直到我开始在一家成熟的公司工作，我才有机会优化代码，了解它真正是如何工作的，并为规模而构建。我无法推荐这种体验。

# 你会努力工作而不是聪明

我们在我之前的创业公司开玩笑说，晚上 9 点离开办公室是“半天”但这并不意味着我们一天有 12 个多小时是有效的。

我要在这里做一个巨大的假设。

技术人才认为有一个最佳的工作时间，而商业人才认为工作时间越长越好。

埃隆·马斯克(Elon Musk)因说“没有人会每周工作 40 小时来改变世界”而闻名[，但我更喜欢站在](https://twitter.com/elonmusk/status/1067173497909141504)[卡尔·纽波特](https://www.calnewport.com/)、 [*深度作品*](https://www.calnewport.com/books/deep-work/) 的作者一边。

> “对于新手来说，每天大约一个小时的高度集中似乎是一个极限，而对于专家来说，这个数字可以增加到四个小时，但很少会更多。”—加州新港

推动球前进的不是一天中的小时数，而是专注的小时数。

年长的创始人更容易理解这一点。

如果有选择的话，为 40 岁而不是 25 岁的人工作。前者有工作之外的责任(家庭、朋友、健康等。).因此，他们会比 20 岁没有责任的人更有效率。

# **招得快，炒得更快**

我不知道是谁创造了这个短语，但它总是感觉不负责任。

如果你之前已经说服某人离开现在的工作，加入你的“火箭计划”，你欠他们好处和足够的时间来加速。

也许我太像一个工程师，而不够像一个经理，但这种文化总是让我难过。

老牌公司明白领域知识是王道，需要几个月的时间才能变得真正高效。

*注:我一直讨厌解雇人，所以对这件事要有所保留。*

# **你的股权一文不值**

你现在放弃了你的工资，希望以后能兑现股权。

我从未从股票中赚过一美元。大多数早期创业的工程师都有同样的经历。这就是十个创业公司中有九个失败的现实。

从期望值的角度来看，你在上市公司工作会更有前途。亚马逊的朋友们一直在做股权银行，尽管它已经多年没有创业了。

在公司上市或被收购之前，你的股份一文不值。这可能需要比你想象的更长的时间。

因此，考虑到股权与薪酬的浮动比例(这在初创公司很常见)，尽可能多拿些薪酬。

# **也有一些例外**

有时候加入创业公司是有意义的。

1.  这家初创公司由一位富有、人脉广泛的长期投资者出资。如果他们愿意支持公司，帮助销售产品，那就完全不同了。文化压力会更小，工作会更专注。
2.  你需要你的第一份编码工作，并且没有获得计算机科学学位。创业公司更可能雇佣没有经验的人，因为他们只需要廉价劳动力。这可能是通向更大成就的垫脚石。
3.  这是你的创业公司，你在建立它之前已经预售了这个想法。

在这种情况下，忽略我说的话。

# **最终……**

无论创始人对自己的想法多么有信心，初创企业都远非一条通往财富的必经之路。

创业经历对培养全面的技能有价值吗？绝对的。但是要知道[的机会成本](https://www.investopedia.com/terms/o/opportunitycost.asp)。这对有能力的工程师来说更是如此，因为他们有很多机会。

我的建议。如果你必须加入一家创业公司，对这个想法和团队持怀疑态度，在工作与生活的平衡上加强界限，尽可能多拿薪水。

你的创业经历如何？
