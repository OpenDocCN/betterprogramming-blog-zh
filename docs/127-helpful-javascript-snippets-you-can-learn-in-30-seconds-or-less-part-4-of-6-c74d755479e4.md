# 64.isBrowser

这个代码片段可用于确定当前运行时环境是否是浏览器。这有助于避免在服务器(节点)上运行前端模块时出现错误。

# 65.isBrowserTabFocused

该代码片段可用于确定浏览器选项卡是否被聚焦。

# 66.isLowerCase

这个代码片段可以用来确定一个字符串是否是小写的。

# 67.伊斯尼尔

这个代码片段可以用来检查一个值是`null`还是`undefined`。

# 68.isNull

这个代码片段可以用来检查一个值是否是`null`。

# 69.isNumber

这个代码片段可以用来检查提供的值是否是一个数字。

# 70.isObject

这个代码片段可以用来检查提供的值是否是一个对象。它使用对象构造函数为给定值创建一个对象包装。

如果它已经是一个对象，那么将返回对应于给定值的对象类型。否则，将返回一个新的对象。

# 71.等物体状的

这个代码片段可以用来检查一个值是否不是`null` 以及它的`typeof`是否是“对象”。

# 72.isPlainObject

这个代码片段检查一个值是否是由对象构造函数创建的对象。

# 73.isPromiseLike

这个代码片段检查一个对象是否看起来像一个`Promise`。

# 74.伊萨梅达特

这个代码片段可用于检查两个日期是否相等。

# 75.isString

这个代码片段可以用来检查一个参数是否是一个字符串。

# 76.isSymbol

这个代码片段可以用来检查一个参数是否是一个符号。

# 77.I 未定义

这个代码片段可以用来检查一个值是否是`undefined`。

# 78.isUpperCase

这个代码片段可以用来检查一个字符串是否大写。

# 79.isValidJSON

这个代码片段可以用来检查一个字符串是否是有效的 JSON。

# 80.最后的

这段代码返回数组的最后一个元素。

# 81.比赛

此代码片段比较两个对象，以确定第一个对象是否包含与第二个对象相同的属性值。

# 82.最大日期

这个片段可以用来获取最新的日期。

# 83.maxN

这个代码片段返回列表中最大的`n`个元素。如果`n` 大于或等于列表的长度，那么它将返回原来的列表(按降序排列)。

# 84.minDate

这个片段可以用来获取最早的日期。
