## 不全是自动气象站和数字海洋

克里斯蒂娜@ wocintechchat.com 在 [Unsplash](https://unsplash.com/s/photos/server?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上的照片[。](https://unsplash.com/@wocintechchat?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText)

亚马逊网络服务，谷歌云平台，微软 Azure。你很有可能听说过这些名人。他们是云计算领域的主要参与者。他们拥有巨大的市场份额、高效的成本结构和真正巨大的规模。

许多大型企业公司使用知名提供商，因为他们提供了所需的所有功能。PCI 合规性或复杂的计费策略是大型企业的标准，但如果您不需要所有这些呢？外面的那些云呢？那些更小、更小众的云呢？

他们没有在世界的每个角落都有数据中心，也没有提供最新的一键部署机器学习产品，但他们确实有勇气。

我想看看云计算领域中一些鲜为人知的名字。独特的云。精品云。在一个由亚马逊和谷歌这样的庞然大物主宰的空间里，供应商们做着有趣的、不同的事情。

# 大型电网

![](img/77869c29aa6a786474cac7c914fb6c97.png)

图片来自 [MassiveGRID](https://massivegrid.com/) 。

这是一家专注于可用性的云提供商。他们目录中的几乎每一项产品都包含某种形式的内置高可用性:

*   高可用性虚拟主机
*   高可用性共享服务器
*   高可用性专用服务器
*   高可用性存储托管
*   …这样的例子不胜枚举。

MassiveGRID 与 Equinix 等公司合作，这些公司提供一些世界上最好的数据中心。虽然 Equinix 提供了出色的 DC，但它们不提供点击部署公共云。在 MassiveGRID 上构建您的云意味着您可以获得 Equinix 质量，而不会受到与它们共处一地的一堆服务器的冲击。

MassiveGRID 总部位于英国，根据初步检查，他们在全球范围内提供了四个部署基础设施的位置。虽然他们没有 AWS 拥有的令人眼花缭乱的不同实例类型，但他们确实以有竞争力的价格提供了高度可用的基础设施。

# 绿色的云

![](img/a4ae2e9d0417fe5a9d58379500487b48.png)

图片来自[绿云](https://gogreencloud.com/)。

这是一家位于南卡罗来纳州格林维尔的小型云提供商，在美国有六个区域数据中心。绿色云面向中小型企业，为这些企业迁移到云提供了较低的准入门槛。

绿云给人一种温暖的小城镇的感觉。他们有几十份证明支持他们的优质服务和支持。他们的网站甚至有一个介绍视频，展示其规模较小、灵活的员工队伍。

绿云将自己标榜为“云支持合作伙伴”，这意味着他们提供的不仅仅是一套基本的云基础设施。他们提供从 Office 365 备份/迁移到网络工程的“端到端”服务。

尽管他们的名字，他们似乎没有提供任何特别的“绿色”服务器。更有可能是因为他们的总部设在格林维尔。

# 服务器中心

![](img/ababd88274b677f73946a9efd47256c2.png)

照片来自[服务器中心](https://www.servercentral.com/)。

ServerCentral 是另一家数据中心和云提供商，像大多数其他公司一样提供各种托管产品，但有一个独特的区别。ServerCentral 最近收购了图灵集团。图灵集团是一家 MSP ( [托管服务提供商](https://en.wikipedia.org/wiki/Managed_services))，提供基于 AWS 等其他云提供商的定制解决方案。

随着最近对图灵集团的收购，这使 ServerCentral 处于一个独特的位置，既提供底层基础设施，又提供实现它的专业知识。

较小的企业客户可能希望捆绑一些服务，这使得 ServerCentral 成为所有云计算服务的一站式商店。

# CloudatCost

![](img/fbb97087f4eb2f618c4efd5249cde257.png)

图片来自 [CloudatCost](https://cloudatcost.com/) 。

这是一家 100%加拿大所有的公司，提供一些最令人震惊的低价。这家云提供商采取了完全不同的方法来处理基础设施和定价。

交易是这样的:你为大多数服务支付“一次性”费用，然后它们就变成你的了，你再也不用付钱了。是的，你没看错——永远。有什么条件？服务和支持。

不要指望花 20 美元买一台服务器就能获得 100%的正常运行时间。甚至不要指望 80%的正常运行时间。

CloudatCost 是一项尽力而为的服务。是的，你得到一个云服务器。是的，该产品看起来与其他一些提供商相同，但实际上并非如此。对于正常运行时间、资源、可靠性，甚至是最新的硬件，都没有多少保证。

就像自助餐一样，CloudatCost 允许用户挑选低价服务器，然后添加“升级”,如 IPv6 或脚本功能。

即使有这些警告，如果你只是想运行一个*《我的世界》*服务器，并不关心服务，这可能是你的供应商。
