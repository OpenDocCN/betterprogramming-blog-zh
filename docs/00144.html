材料主题

Visual Studio代码现在最宏大的主题是:

自动导入

自动查找、分析并提供所有可用导入的代码操作和代码完成。与打字稿和TSX一起工作。

进口成本

这个扩展将在编辑器中以内联方式显示导入包的大小。扩展利用 Webpack 和 babili-webpack-plugin 来检测导入的尺寸。

缩进-彩虹

一个简单的扩展，使缩进更具可读性。

HTML中CSS类名的智能感知

一个VS代码扩展，它为HTML class属性提供CSS类名完成，基于在您的工作区或外部文件中找到的定义，通过link元素引用。

SVG查看器

预览SVG的简单方法。

更漂亮—代码格式化程序

离不开这个！使用更漂亮的来格式化你的JavaScript / TypeScript / CSS的VS代码包。

反应原生工具

用于的代码提示、调试和集成命令对原生做出反应。

升华文本键映射和设置导入程序

该扩展将Visual Studio代码绑定转换为与ST3相同的。旋转一下，试试这个:

cmd ⌘ + P — Mac

ctrl + P —视窗

你可以进入文件，如果你在搜索前添加了>符号，你就可以搜索操作——比如切换内置终端、安装扩展等等。

对于我们这些习惯了ST3捷径的人来说，这是一个巨大的胜利。

npm智能感知

在导入语句中自动完成npm模块的VSCode插件。

Lit-html

在JavaScript和TypeScript标记的模板字符串中添加HTML的语法突出显示和语言支持，正如在 lit-html 和其他框架中使用的那样。

GitLens — Git增压

Git项目经理

Git项目管理器(GPM)是一个Microsoft VS代码扩展，它允许您打开一个新窗口，直接从VS代码窗口定位Git存储库。

Git历史

查看Git日志或文件历史，比较分支或提交。

文件实用程序

创建、复制、移动、重命名和删除文件和目录的便捷方式。

括号对着色机

用于匹配括号着色的可定制扩展。在处理大量回调时非常有用。

彩色高光

这个扩展设计了文档中的CSS/web颜色。

CSS Peek

允许窥视CSS ID和类字符串作为定义，从HTML文件到各自的CSS。允许查看并转到定义。

Chrome调试器

在Chrome浏览器或任何其他支持Chrome调试器协议的目标中调试您的JavaScript代码。

Quokka.js

用于JavaScript的Live Scratchpad(用于快速演示)。

结尾空白

突出尾部空格，瞬间删除！

打字英雄

瓦卡提时间

指标、见解和时间跟踪—从您的编程活动中自动生成。

韦图尔

VSCode的Vue工具

代码运行程序

运行多语言的代码段或代码文件。

PHP智能感知

PHP的高级自动完成和重构支持。

虚拟代码-图标

VS代码的图标。

玩笑

愉快地使用脸书的笑话。

React.js代码片段

使用ES6语法进行 React.js 开发的代码片段。
