# 扩展集合

![](img/8dc6e01b8eb0f44fc9608ba0f21ba7b0.png)

`expanding-collection`是`UICollectionViewCell`的定制细胞亚类。通过显示所有的细节，它提供了一个很好的动画，将一个单元格动画化为一个视图控制器。这有点类似于 App Store 对“今日”标签所做的事情

这个库兼容 iOS 9+和 Xcode 9+。
