我是如何在没有计算机科学学位的情况下通过顶级科技公司的面试的

有时间表和资源，所以你也可以做到这一点！

介绍

准备技术面试可能感觉是一项非常模糊的任务——至少对我来说是这样。在这篇文章中，我想阐明我个人的学习时间表和我为自己制定的策略。应该注意的是，为了成功，并没有一个你必须遵循的唯一策略。

我坚信让你周围都是志趣相投、积极上进的人。虽然我在自学期间有一些出路，但我想分配一些自己的时间，创建自己的社区，以帮助与我处境相似的人。这就是为什么我创建了 cscareers.dev社区。

在申请和面试这些公司之前，我强烈建议你看看我正在建设的社区。我们旨在为我们的社区成员提供免费的公司特定面试问题以及模拟面试和顶级技术公司工程师的简历评论，以帮助您获得并赢得面试。

挑选语言—2019年5月

进入面试准备阶段，我实际上没有一种语言可以用来面试。我过去是，现在仍然是一名JavaScript开发人员。在我看来，JavaScript是完成这些算法面试的次优语言。我可以解释为什么，但这值得单独写一篇文章。

不管怎样，这让我处于一个特殊的位置，我需要适应面试语言和解决问题。我发现Java因其冗长而吸引人，但事后看来，我宁愿投资学习Python，因为它简单。不管怎样，不要在选择一门语言上花太长时间。大部分人都是用C++/Python/Java面试。下定决心，开始关注更重要的事情。

code signal—2019年5月

由于LeetCode对我来说太难了，甚至连最简单的问题都无法完成，所以我开始使用的第一个资源是CodeSignal。该平台提供不同难度的几个不同类别。对我来说，我需要学习如何编写Java代码，并开始锻炼我解决问题的能力。我花了大约一周的时间完成了他们游戏区介绍部分的各种编码挑战。利用他们网站的这一部分，我可以一举两得。

leet code—2019年5月至6月

此时，我决定重温LeetCode。考虑到我不熟悉如何将数据结构应用到问题中，对我来说完成问题仍然很困难，但是真的没有办法绕过这个学习曲线。由于我没有上算法课程的奢侈，我不得不通过YouTube上的在线讲座来补充我的知识缺口。

我最大的建议是在做单个问题时尊重你的时间。花超过30分钟去思考如何解决问题会变得非常低效。如果你没有在正确的轨道上(随着时间的推移，你会获得这种直觉)，前往问题的讨论标签，阅读解决方案。LeetCode的讨论标签是了解其他人如何处理/解决问题的极好资源。在一个问题上花费几个小时和30分钟之间的差别是次优的。当学习不同的概念时，你需要非常尊重你的时间。

为了真正理解一个问题的解决方案，我会把这个人的解决方案写到我的代码编辑器中，并尽我最大的努力去理解每一行的作用。对于我看到解决方案的问题，我会把它们添加到以后要做的问题中，以加强这种方法。

模拟面试-2019年6月至8月

模拟面试是为面试做准备的好方法。此时，我正在参加CodePath的技术面试课程，这让我有机会每周进行一次模拟面试。这是一个可以利用的巨大资源。如果我现在就准备，我肯定会利用Pramp，试着为自己制定一个时间表，每周进行1-2次模拟面试。虽然失败很伤脑筋，但它是免费的，你希望在一个没有任何影响的环境中接受它。模拟面试是最好的工具。

在完成我自己的模拟面试时，能够看到其他成功人士是如何面试的确实改变了游戏规则。有几个YouTube频道对有经验的工程师进行模拟面试。

我亲自花时间模拟面试那些通过了这些高难度技术面试的人，并在网上播放这些面试，希望他们能像我发现的那样足智多谋:

leet code—2019年7月至10月

到了七月，我开始对LeetCode越来越适应了。五月份困扰我的问题变得微不足道，我终于能够开始解决盲顶75个问题列表中的问题。我建议解决这个列表上的问题，因为它们触及了这些面试问题所涵盖的许多概念。我甚至在实际面试中遇到过被问到同样的问题。

在 CSCareers Discord服务器中，有一个机器人，你可以与之互动，从流行的盲单中选择一个随机问题，以及任何公司特定的问题。

每当我要面试的时候，我都会花时间做公司经常问的问题。例如，脸书喜欢根据他们经常问的问题来问树的问题，所以我会确保我回答树的问题时非常舒服。这不是一个万无一失的方法，但它是为公司的特定面试做准备的一个很好的方法。

除了研究公司的问题，我会在白板上回答所有的问题。你不希望真正的面试是你第一次在白板上写字。你可以学习一些技巧来帮助你更快地在白板上写代码，但是如果你不花时间做准备，你就不会学到这些技巧。

编程面试的要素—2019年9月

我要特别感谢《编程访谈》书中的 元素。我经常看到有人推荐CTCI的书，但是当把这两本书放在一起比较时，我真的相信这本书本身就足以作为在面试中取得成功的学习材料。我不能再对CTCI说同样的话了。EPI根据你需要准备的时间提供有针对性的学习计划。

结论

就像任何其他技能一样，技术面试是一项你必须磨练的技能。我真诚地相信，如果你在学习过程中与合适的人在一起，并有意识地努力提高，你将能够通过这些大型科技公司的面试。
