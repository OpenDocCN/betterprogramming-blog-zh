寻找 2021 年要学习的新编程语言？有许多很好的选择，但 Rust 可能是最好的选择之一。学习起来很有趣，可以提高你作为软件开发人员的技能，并且有一个非常受欢迎的社区。

在本文中，我将研究为什么您应该学习 Rust，列出如果我是您，我将采取的学习途径，然后包括资源(全部免费！)可以在旅途中使用。

# 为什么要学 Rust？

我不会说世界上每个软件开发公司都在排队等着抢新的 Rust 开发者。再过几年左右，这可能是真的。

目前，Rust 是软件开发人员成长的最佳语言。

首先，使用像内存管理、借书检查器、生存期和表达式类型这样的东西无疑会让你成为一名更好的软件开发人员。与大多数其他编程语言相比，Rust 是一成不变的良好开发实践。

但这还不是全部。虽然 Rust 既不是 Haskell 也不是 Lisp，但它对各种有趣的活动提供了强大的支持，比如函数式编程和宏，您可以在学习 Rust 时尝试一下。

此外，Rust 拥有一个由乐于助人的人组成的庞大社区和大量的学习材料。这也很重要。

# 如何学习铁锈

虽然没有人会因为我建议你从正式的 Rust book 开始(这是一个合理的选择)而解雇我，但我认为有一个更好、更实用的方法。

仅仅用一本书来学习语言是一个旅程，经常会在第 70 页左右停下来。因此，我相信在开始长途跋涉之前，先完成一些小事，获得一些实用技能，这就是 [*生锈的编程语言*](https://doc.rust-lang.org/book/) 。

首先，我建议查看一个 Rust 入门教程:要么[半小时学习 Rust](https://fasterthanli.me/articles/a-half-hour-to-learn-rust) (文本)，要么[Rust 之旅](https://tourofrust.com/)(互动)，要么 [Rust 速成班](https://www.youtube.com/watch?v=zF34dRivLOw)(视频)。这应该能让你了解 Rust 的语法和这门语言的主要哲学。

之后，用[exercisem](https://exercism.io/)练习编码来提高你的锈印章，阅读更广泛的资源，如[锈书](https://doc.rust-lang.org/book/)(标准选择)或 [*易锈书*](https://dhghomon.github.io/easy_rust/) (可访问选择)。

如果您觉得已经准备好构建自己的玩具应用程序了，那就去做吧。有一些不错的在线社区，你可以在那里寻求反馈和支持，所以不要担心陷入困境！

下面，我将介绍这些以及您在学习 Rust 时可能用到的所有其他资源——书籍、教程、视频等等。每一个都有一个小的描述，所以你可以决定哪些对你有用，哪些没用。

# 生锈的书

## Rust 编程语言

Rust 编程语言 是包含该语言所有基础知识的官方巨著。如果你想学习 Rust，你可以在这里找到所有必要的信息，这是一个很好的资源。

## 生锈的例子

[*Rust By Example*](https://doc.rust-lang.org/rust-by-example/)是一组可运行的示例，涵盖了与官方 Rust book 或多或少相同的功能。这可能是一个很好的地方来寻找如何使用一个特殊功能的例子。

## 铁锈食谱

[*Rust Cookbook*](https://rust-lang-nursery.github.io/rust-cookbook/)包含了一些实用的代码示例，可以完成一些你可能在 Rust 中完成的基本编程任务。不过，这些收藏确实让人感觉有些杂乱无章。

## 易生锈

[*Easy Rust*](https://dhghomon.github.io/easy_rust/) 这本书原本是给非英语母语的人看的，但老实说，我认为很多人会发现从这本书里学习 Rust 更容易。官方书籍有点密集，虽然这本书可以使用一点编辑来获得一个恒星推荐，我认为它更容易理解。

# Rust 初学者教程

## 半小时学会生锈

本教程中没有任何内容[。在相当接近标题中提到的半小时的时间范围内，您将习惯于通过浏览大量不同的代码片段来使代码生锈。](https://fasterthanli.me/articles/a-half-hour-to-learn-rust)

## 对铁锈的温和介绍

[Rust 简介](https://stevedonovan.github.io/rust-gentle-intro/)是一个广泛的在线教程/书籍，以一种非常有趣的方式编写，涵盖了大多数 Rust 基础知识。

## 铁锈之旅

[Rust 之旅](https://tourofrust.com/)是一个有趣的关于 Rust 基础知识的互动教程。它也是多语言的(有 15 种语言！)，这对非英语母语的人来说至关重要。

## 铁锈入门

如果你还不太了解 Rust，我写了一篇[小介绍](https://serokell.io/blog/rust-guide)，讲了 Rust 为什么很酷，它是用来做什么的。这对于在使用其他学习资源之前获取更多的背景信息是很有用的。

# Rust 视频

## 生锈速成班

一个关于 Rust 基础知识的两小时简明视频，有助于学习其语法和熟悉 Rust 代码。

## 易生锈

这是一个视频系列，涵盖了前面提到的书，*易锈*。如果你喜欢从视频中学习，这可能是使用书本的一个很好的选择。在撰写本文时，该项目尚未完成，仅涵盖了全书的五分之一。

## 锈皮

rust 的外壳是一个大约 90 分钟的视频集合，涉及更中级的 Rust 主题，如生存期、迭代器等。这些视频展示得很好，信息量也很大，所以我强烈建议在某个时候去看看。

# 其他资源

## 练习

如果你想获得更多的语言经验和直觉，练习是一个不错的选择。我建议尽可能早地做练习，因为这是巩固概念和完成事情的一个极好的方式，而不用为要建造什么东西而苦恼。

## 沙沙声

[锈蚀](https://github.com/rust-lang/rustlings)基本上是锈蚀特有的。在大部分练习中，代码中存在某种让编译器生气的错误。你的工作是安抚编译器。对 Rust 编译器如何工作以及最常见的代码问题建立直觉是很好的。

## 铁锈操场

懒得下载 Rust 编译器设置？可以在 [Rust 游乐场](https://play.rust-lang.org/)试玩 Rust。

## Rust 语言备忘单

[锈语小抄](https://cheats.rs/)顾名思义。一大页纸上有很多关于铁锈的东西。您可能会发现它是编写代码时有用的参考资料。

## 新鲁斯塔西亚人

虽然仅仅通过听播客来学习编程语言并不是最棒的主意，但在散步或(如今不太常见的)通勤时， [New Rustacean](https://newrustacean.com/) 是学习或强化基本 Rust 概念的绝佳资源。

## 在线社区

Rust 有一个非常友好的社区。可以和人见面，就 [r/Rust](https://www.reddit.com/r/rust/) 、 [Slack](https://rust-slack.herokuapp.com/) ，或者 [Discord](https://discord.com/invite/aVESxV8) 求教。都很酷很活泼。

我希望这些资源可以帮助你的旅程。祝你好运！
