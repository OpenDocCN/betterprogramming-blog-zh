# 飓风

在他拳击生涯的巅峰时期，鲁宾“飓风”卡特的生活在被错误地指控犯下一项他没有犯下的滔天罪行:三重杀人罪后，直接跌入了绝望的深渊。

大多数人都会在这种不幸局面的压力下崩溃，但卡特不会。一到监狱，他就向典狱长明确表示，他不是囚犯，因为他不会也永远不会无能为力。

卡特拒绝放弃的东西——他的态度、他的信仰、他的选择——让他变得比以前更强大、更坚持、更优秀。他能够将通常用于自怜的所有精力转向通过书籍来磨砺自己的思想和精神，最终帮助建立他的法律案件。19 年后，他的刑期被撤销，重新开始生活。

我们可能不会面临像卡特那样严峻的形势，但有一点是肯定的，我们对生活的反应总是有决定的。我们的决定塑造了我们的命运，增强或削弱了我们的精神。无论你发现自己处于什么样的境地，你都不会无能为力。

# 环境的监狱

在我们作为开发人员的职业生涯中，我们经常渴望在声誉良好的公司或充满才华和卓越领导力的有前途的初创公司工作。当我们工作的现实不符合我们在工作环境中的期望或梦想的标准时，我们往往会灰心丧气，随波逐流。我们允许生活中的所有外部因素使我们陷入困境。

我知道有太多有前途的开发人员因为缺乏“激情”或“动力”而在“错误”的地方结束，导致他们落后他们成为环境的囚徒，无法前进和创造新的机会。他们走上了与卡特相反的道路，因此被关进了他们自己制造的监狱。

有太多的开发人员依赖优秀的领导、地位或良好的教育来帮助他们加速和发展职业生涯。事实是，你的个人成长与你在哪里无关。你的个人成长更多地取决于你想要达到的目标，以及为了达到目标你将决定做些什么。接下来的问题在于，你是否专注于你应得的，而不是你的能力和你能为自己赢得的职业。

我在两个失败的创业公司工作过。我去了一所大学，它的计算机科学课程并不是为了输出科技天才、工程大师或创新思维而设计的。然而，在我现在的公司，我成功地成为了一名优秀的开发人员，创造了记录，并在两年内获得了高级架构师和首席开发人员的职位。

> 我没有成为今天的开发人员，不是因为我在哪里工作，也不是因为我如何学习。尽管我有过去的工作经历，我还是成为了今天的开发人员。

# 挣脱束缚

我以前说过，现在我要再说一遍，你不需要出色的教育，也不需要出色的工作，就能成长为一名出色的开发人员，既有非凡的能力，又像冠军一样肩负着巨大的责任。

与其靠经验教育你，不如靠自己成长。你需要领导你自己。你需要强迫自己去转变和进化。你需要创造你自己的机会，而不是等着机会从你身边溜走。

对我来说一切都不容易。在大学期间，我平衡了作为一名学生运动员和我个人的学习。一天中只要有空闲时间，我就会跳上电脑，专注于学习游戏开发或其他技术、干净的代码和设计原则/模式。在练习和阅读之前，我会起得特别早。我在自己身上投资，它给了我超越他人的优势，让我获得了巨大的回报。我从多方面理解了努力工作和磨砺心智的重要性。

当涉及到现实世界时，我在两个独立的创业公司的垃圾箱里开始了我的开发生涯。第一个，我的任务是在没有上级指导的情况下构建一个完整的增强现实应用程序。在第二次面试中，我在一家缺乏组织、方向或智慧领导的公司受到了非常差的待遇。然而，我很感激。这给了我更多的时间来思考。

他们让我看到了是什么让一个公司失败。他们让我意识到，我有能力抓住机遇，通过我的行动改变现状。他们让我根深蒂固地意识到，除了你自己，你不能完全依赖任何人来建立你的职业生涯，扩展你的开发/编码/领导技能。这种经历每天都存在，是一个拓展你技能的机会。

作为一名开发人员，成长只需要掌握自己的教育、自己的行为和自己的命运。

这仅仅意味着，当你对解决方案有意见或见解时，大声说出来。这意味着通过你的编码实践大胆地表达你自己。这意味着要足够谦虚，从自己的错误中学习，将自己编写的代码与比自己更熟练的人编写的代码进行比较，以及从任何建设性的批评中学习。

这意味着在严峻的工作环境中打开你的思维去寻找机会。这意味着在一天中找出时间，让自己变得比昨天更优秀 1%——无论是通过在线课程、阅读、观看教育视频/讲座，还是与一位更懂技术的上司进行坦诚的交谈。

不管你的环境如何，成长的机会总是存在的。接下来的问题是:你会抓住它吗？你会像鲁宾“飓风”卡特一样，掌控自己命运的轨迹吗？
