# 如果编程语言有工作

> 原文：<https://betterprogramming.pub/if-programming-languages-had-jobs-ad14239a40cc>

## 你可能会雇佣其中的一些人

![](img/d038ccb6484dafa772f552f52de6d502.png)

图片来自 [Pixabay](https://pixabay.com/vectors/recruitment-interview-selection-job-6838250/)

我们最终都要加入劳动大军，让自己成为对社会有用的人。现在许多需求量最大的职业都需要一种或另一种编程语言，但是如果不是工作需要编程语言，而是编程语言需要工作呢？

以下是我收集的一些最流行的编程语言，以及他们在必要时会从事的工作。

# Java 语言(一种计算机语言，尤用于创建网站)

## 潜在工作 1:当地咖啡店的咖啡师

## 潜在工作 2:大学教授

![](img/bfc9ae2bc8f99b950ddd04512b27f234.png)

图片来自 Canva

不过说实话:作为最受尊敬的编程语言之一(大量大学将 Java 嵌入其计算机科学课程就证明了这一点)，如果 Java 决定宁愿在大学教书，我们也不会感到惊讶。毕竟，没有什么能阻止这个雄心勃勃的女人去攻读咖啡豆的博士学位！

# 计算机编程语言

## 潜在工作 1:社交媒体影响者

![](img/001d8fd18222efe08fb0e2dc0c9a2804.png)

图片来自 Canva

Python 是现在的趋势，而且已经有一段时间了，它的[年增长率为 27](https://stackoverflow.blog/2017/09/06/incredible-growth-python/#:~:text=With%20a%2027%25%20year%2Dover,shows%20similar%20growth%20is%20R.)。Python 让它的壮举看起来很容易，因为它们令人印象深刻，所以这不是什么秘密，这种语言在它选择的任何社交媒体平台上都拥有成为下一个头号热门的一切。小心世界！

## 潜在工作 2:向导

![](img/257c0d4a204af38130ba2d0c24765752.png)

图片来自 Canva

我知道你在想什么:“魔法不是真正的工作”。在你不赞成地点击退出这篇文章之前，请听我说完！巫师是存在的。我发誓，他们会的，他们努力让我们的世界变得更好(不信就去问 J.K .罗琳)。奇才是存在的，Python 以其不可思议的能力在如此短的时间内吸引了如此多开发人员的注意力，看起来它拥有成为奇才所需的所有技能。

# C

## 潜在工作 1:政治家

![](img/90d40a306a1d715ebab1fabe37f6ead3.png)

图片来自 Canva

政治家们强硬且要求高，但设法对现代社会产生了巨大的影响，尽管他们通常都很老了。你知道还有谁符合这个描述吗？你猜对了:C。通过变异成 C#和 C++，C 是一种不知何故仍然和以前一样相关的语言。

## 潜在工作 2:法官

![](img/e7ab49f8c137ff7271ab907d94c799a7.png)

图片来自 Canva

由于 C 非常热衷于让开发人员遵守其严格的法律(我可以补充一下，这些法律显示了对分号非常奇怪的偏见)，我们都不得不承认没有比 C 更好的法官了。老实说，如果 C 以某种方式成功进入最高法院，我们也不应该感到惊讶

# HTML/CSS

## 潜在工作 1:技术支持借口

![](img/e7acc3ebc2253dc75e74aadd053d2d80.png)

图片来自 Canva

虽然被认为不是真正的编程语言，但我相信这两种语言会在技术支持领域蓬勃发展，在那里它们可以继续愚弄人们，让他们认为自己是他们不是的东西！

## 潜在工作 2:艺术家

![](img/0278a3f2a36fa48be9c478306bf51cac.png)

图片来自 Canva

尽管 HTML/CSS 可能不是编程语言，但它们仍然很有价值。没有前端开发，用户体验会很烂。不管我们承认与否，我们需要他们，他们的工作很困难。我们不得不承认，他们是我们这一代的顶尖艺术家，他们值得我们尊敬。

# Java Script 语言

## 潜在工作 1:教师

![](img/212c243d3ef2210571478068f3f0ed2e.png)

图片来自 Canva

老师被低估了，他们经常不得不照顾那些有些人认为不值得照顾的叛逆学生。他们相信学生的潜力，就像 JS 相信 HTML/CSS 的潜力一样，即使说实话，他们也继续为社会服务，就像 JS 一样，他们没有得到应有的认可。

## 潜在工作 2:土木工程师

![](img/a9efbfe326ed04299c179a530317e164.png)

图片来自 Canva

不管出于什么原因，如果 JavaScript 不适合她，她可以成为一名土木工程师。没有她，大多数网络架构都会崩溃。谢谢你 JS。

# 装配工

## 潜在工作 1:建筑工人

![](img/2854080823f9575fbeb8160b1c70e768.png)

图片来自 Canva

我们需要一个建筑商来建造哪怕是最基本的设施:我们头上的屋顶。编程语言需要汇编程序才能运行。那么，装配工成为本世纪的建造者就不足为奇了。

## 潜在工作 2:医生

![](img/a3f8e750a2315a5f32a0d40e2b5b79d6.png)

图片来自 Canva

像建筑一样，医学是一项像装配工一样永恒的工作。就像医生知道我们生存所需的所有器官一样，汇编程序知道所有高级编程语言都离不开的机器代码。
