## 自定进度的课程只是安眠药；行业专家是正确的选择

我在 2016 年开始了我的数据科学之旅。那个时候，机器学习训练营要流行得多。这门课程的唯一要求是基本的 Python 知识。我感到非常高兴，因为我那时经常玩 Python 代码，所以我选择了这门课程。

当我们学习新东西时，人类通常的做法是尝试从不同的地方收集尽可能多的材料。我们可能认为这些资源会帮助我们了解更多。而且，你拥有的材料种类越多，你就有越多的选择来选择合适的资源。但是，这是错误的。

我早在 2016 年收集的大部分资源还在我的 D: drive 里，还没看。但是，是的，有一门课我学了 70%，那就是机器学习训练营。我完成那门课的原因是——这门课很容易学，一切都简单明了。

当我们在机器学习领域的时候，在不知道算法背后的思想的情况下，我们可能会有建立模型的幻想。训练营的课程主要集中在实施部分，而不是概念部分，这就是我犯的大错误。我抱着学习机器学习的虚幻希望。

> 机器学习不是导入库、输入数据和获得结果。更多的是算法背后的知识，帮助我们为数据选择正确的算法，以获得更好的结果。

# 自定进度的课程只是安眠药

自定进度的课程给我们提供的是被动学习，不适合机器学习。在这里，你可以找到一个神经网络，它有不同的层、激活函数、反向支撑、正向传递以及许多单人无法在自定进度的课程中涵盖的内容。没有人会在自定进度的课程中分享这些技巧——通常，这些技巧是免费的或者便宜得多。我和其他人一样，相信机器学习的自定进度课程，在这些课程中，所有事情都由另一个人来完成。我坐在这里，只是看视频教程来建立一个完整的机器学习模型，而不是自己尝试哪怕一行代码。然后，仅仅通过覆盖那些视频来获得学习机器学习的虚假希望。

我不反对自定进度的课程。这取决于学习的领域。一个学习吉他的自定进度的课程可能是一个很好的练习，因为我们只需要一把吉他在手来尝试。

# 行业专家是正确的选择

在尝试了三年来自不同资源的自定进度课程后，我发现我甚至不能决定最适合特定问题的算法。我不知道简单的建模过程。我所知道的只是导入某些库并传递数据以获得回归和分类问题的输出。但是，如何解释这些输出，我没有线索。

终于有一天，我的一个同事建议加入一个由行业专家来教我们的机构。那天我想，这只是浪费钱，因为他们收取的费用是那些自定进度课程的十倍。

总之，我加入了线上直播班，观察变化。通过实时课堂的主动学习给了我动力。构建神经网络的一步一步的过程给了我另一个动力。最大的好处是每天分配给我们的任务。

我感觉到的主要区别如下:我可以和老师取得联系，澄清任何疑问；在实时行业场景中工作，享受专业质量的内容，而我是必需的，这是我们无法从自定进度的课程中获得的。

网上大部分免费的课程结构都不太好，这就是机器学习的秘密。构建任何模型都有一个循序渐进的过程。首先，我们必须了解算法，获取数据，清理数据，找到最佳特征，分割数据，训练模型，验证模型，然后进行推理。如果我们想要更好的结果，我们不能错过任何一步。
