# 保持匿名和清白

到目前为止，这是在房间里创造心理安全的最关键的方面。如果被特别指出，尤其是受到攻击，人们下次被要求反馈时会退缩。

回顾是集体讨论情况并看看我们如何一起改进的时候。通过保持复古无可指责，我们可以攻击问题，而不是造成它的人。

# 停止防御

促进追溯的经理的一个常见错误是在追溯中围绕一个敏感话题进行辩护。我曾经遇到过一个产品经理，他在一次复古中猛烈抨击并解雇了一名工程师，因为两人对项目结果的看法不同。当有人试图变得有成效和有帮助时，没有比变得有防御性更好的停止谈话的方法了。

当一个话题变得困难时，通过注意紧张来保护房间和复古。尽可能提出中立的问题，听取双方的意见。模仿你希望看到的行为，警惕防御姿态。

# 从破冰船开始

我参加过的一些最好的团队在回顾开始时都有非常深刻和个人化的破冰船。我发现这让我们所有人都进入了这样一种心态，即我们在复古中让彼此变得更好，我们可以安全地表达我们的观点。

一个新成立的团队或一个仍在“风暴”中的团队不会很舒服，所以从一些软球开始，并加入更个人化的破冰船，随着时间的推移进行测试。一些入门问题:

*   泰德·拉索的“第一场音乐会，最佳音乐会”
*   适合工作的问题
*   你目前在 repeat 上有什么专辑？
*   你吃过的最好的一顿饭是什么？
*   你现在最想读的三本书是什么？
*   进阶:你最大的伤心事是什么？

# 记得检查“哪些进展顺利”

太多时候(我对此也感到内疚)，我们忽略了“进展顺利的事情”。

“事情进行得很顺利，所以没什么可借鉴的。让我们拍拍自己的背，继续前进。”

**错了。**改问这些后续。

*   为什么会顺利？
*   我们能继续做些什么来让这成为一件好事？
*   是某个特定的队友的努力吗？
*   什么变了？
*   我们如何复制这一成功？

如果你正在寻找更多关注亮点的理由，请阅读 [*Switch:当改变很难时如何改变事情*](https://www.amazon.com/Switch-Change-Things-When-Hard/dp/0385528752) 。它深入探讨了如何复制成功，以及为什么尝试这样做会产生更好的结果。

# 改变格式

单调是沉默的杀手。我们习惯了事物一直以来的样子，并养成了舒适的习惯。它可能以追溯的形式变得不那么有意义，所以我喜欢不时地改变事物和格式。复古有很多种方式。我的五大最爱:

*   开始、停止、继续
*   高兴，悲伤，疯狂
*   海盗船(帆船)
*   喜欢，爱，学习，渴望
*   保留、添加、减少、增加

只要你为人们一起解决问题和讨论成功与失败留有空间，进行复古就没有错误的模板。

# 旋转主持人

实际上，你应该在所有的会议上考虑这一点，尤其是在回顾会议上，一旦你为一个好的回顾会议打下了基础，就应该轮流主持会议。当有人新的促进，他们带来了一个新的视角，并可能引入一些新的东西，用于未来的追溯。

轮流主持人也给那些对管理或领导力感兴趣的人一个机会，让他们坐在驾驶座上，处理棘手的讨论。不要选择这个时候做一个缺席的经理，做一个安全网，以防谈话失败。

不然就在团队复古期间看看团队动态，看能不能学到新东西。

# 小心复古猪

复古猪很容易被发现。他们对每张…单…卡片都有话要说。在追溯期间密切关注，并在追溯完成后跟进。在许多情况下，个人可能不知道他们说了这么多。在其他情况下，可能是因为没有其他人，所以他们想打破沉默。

继续与复古猪一起工作，帮助确定他们的贡献是否有益，或者你是否可能错过了其他人的声音。

如果我注意到某个人在整个复古过程中一直保持沉默，我可能会直接问:“查尔斯，你对此有什么想法吗？”在偏远的环境中，他们可能觉得声音传得不够快，所以他们会保持安静。帮助把人们拉进对话，同时把复古猪拒之门外。

# 投资一种工具

如果工具是不灵活的，那么你的改进也会是不灵活的。关键组件应包括:

*   卡片的匿名输入
*   轻松分组/分类
*   匿名投票
*   轻松显示当前主题
*   导出行动项目的能力

我最喜欢的追溯工具是[teamrutro](https://www.teamretro.com/)。我和他们完全没有关系，但它的成本相当低，而且当我的前任老板不想投资时，我最终为我的团队自掏腰包购买了这个软件，因为我太喜欢它了。

当然，你可以在黑板、壁画或其他能让你靠近的工具上手动完成，但主要目标应该是减少参与的摩擦。特别是在远程环境中，一个易于导航的复古工具在移动设备上运行良好，并且保持对反馈的关注是必不可少的。

# 跟进行动项目

关闭未来追溯的最简单方法是对创建的措施项不做任何操作。如果一个团队不相信任何事情会改变，他们将停止提供他们的输入。让你自己和你的团队负起责任，努力在两周内完成行动项目。

# 最后的想法

回顾不必是枯燥和麻木的。找点乐子，提醒团队复古的意图，整理他们的想法和反馈。
