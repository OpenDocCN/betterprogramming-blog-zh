> "出生在笼子里的鸟认为飞行是一种疾病."—亚历桑德罗·佐杜洛夫斯基

众所周知，编程是最难的工作之一，成为一名程序员，尤其是当你自学成才时，可能是你一生中最具挑战性的事情之一。

从经验来看，我可以说这也是最有收获的。

所以，如果你觉得自己在挣扎，要知道每个人都在挣扎。这通常发生在中间。当我们要开始新的、不同的事情时，我们都会受到鼓舞，但当这种兴奋开始消退时，也是真正的事情发生的时候。

我们开始质疑一切。看不到任何结果很容易让你崩溃，所以你必须摆正心态。这段旅程需要你付出很多。这趟旅程将带走你所拥有的一切。

当事情感觉要分崩离析时，后退一步有时是获得洞察力的唯一方法。与其苛责自己，不如休息一天，如果有必要的话，休息一下，但是重新站起来，振作起来，只是出现。

不管有多吓人，不管有多恐怖，你都会挺过去的。这只是你怀疑你自己，你的能力，你的可能性。更大的梦想有时会让我们感到无力，但这是一种伪装的兴奋，因为我们有可能变得非凡。

当你开始看到你努力工作的结果时，你就会看到了。在那之前，继续学习，因为我们必须为这个梦想做我们必须做的事情。

# 我们总是玩小游戏

这是一个老生常谈的话题，我们总是小心翼翼地玩游戏，并想知道为什么我们的生活中没有什么伟大的事情发生。

这是难以下咽的药丸之一。

然而，这不是我们的错。这就是我们被父母、我们的社会、我们的教育系统设定的方式。他们想让我们认为错误是错误的，不舒服会导致灾难，与众不同是错误的。

与众不同是一种赞美。当别人都在嘲讽你与众不同的时候，你可以尽情地笑，因为他们都是一样的。

大多数人渴望被接受，这不是什么大新闻。我们需要同伴，因为我们是社会人:我们必须像其他人一样生存，感到安全，被接受。我们必须像其他人一样，所以当我们的创造性思维开始飞翔和漫游时，我们会掐自己并试图再次脚踏实地，因为我们选择相信我们不允许要求更多，不允许想得更多。这就是可悲的生活现实。

我们是自己的极限，但前提是我们让自己成为自己。

我们都生活在一个笼子里，一个盒子里。有人会说我们应该开始跳出框框思考，但是为什么呢？谁说有盒子了？

你应该像没有盒子一样思考，有了这种心态，你就会让自己自由。

所以不要玩小游戏了。你值得拥有生命中最好的东西。你值得拥有比你想象的更多，你也有能力拥有比你想象的更多。你比那更聪明，你比那更优秀，所以开始想要更多，开始有更大的梦想。把它做得比你的闹钟更早起床。人生苦短，不能儿戏。

是时候站出来和巨人队一起比赛了。如果我做到了，你也可以。

# 当事情变得艰难时，我们为什么要退缩？

事情总是可能出错，但如果我们退缩或为之奋斗，找到方法，并让它发生，这将永远取决于我们。

如果你知道这是你应得的，那你为什么要放弃呢？你为什么要放手？

事情是这样的:当它已经感觉像地狱时，为什么还要呆在那里？前进，穿过它，穿过它。你牺牲了很多东西，你为之如此努力，然后你就要停下来？

当你可以顺其自然，感受一切时，你为什么要让它白白流逝？害怕吗？顺其自然吧。累吗？休息一天。没有灵感？回到你一开始为什么要这样做。这不是灵感有没有的问题。最终会消失的。你只需要做该做的事。

重要的是你要坚持做重要的事情，即使你并不想做。因为不管你愿不愿意，世界都会继续运转。

作为一名开发人员不仅仅需要头脑。就像我说的，这需要你付出很多，需要你付出一切。大多数时候，成功的不是天才或有才华的人。通常是那些适应力强的人，那些欲望最强烈的人赢了。只有那些努力奋斗、不接受否定答案的人才能到达终点。

如果你不能做到所有这些，如果你没有所需要的东西，那么你可能不值得自学。但是如果你这样做了，那么就打个盹，呼吸一下，出去走走——这只是暂时的失败。休息一下吧。

如果你想赢，那就赢下每一场小战斗，每一天。这就是你赢得战争的方法。

# 无论如何都要做

你感到害怕吗？无论如何都要做。

你觉得你会把事情搞砸吗，就像以前一样？无论如何都要做。

你是否觉得自己不具备成为一名开发者的条件？无论如何都要做。

你觉得你来的太晚了吗？对这个来说太老了？无论如何都要做。

你有没有因为责任太多而觉得没有时间学习？无论如何都要做。

你认为它会失败还有其他原因吗？无论如何都要做。

都是脑子里想的。不是你对抗世界，是你对抗自己。

这段旅程不仅仅是为了找份工作。这不仅仅是成为一名成功的开发人员，也不仅仅是金钱或生活方式的问题。这都是关于我们自己的。

意识到我们可以做得更好，我们可以付出更多，我们应该拥有更好的生活，这只是未来更好的开始。

成功只是最终目标。作为一名软件开发人员工作只是许多其他山峰的顶峰。

这是我成功征服的山峰之一，就像我之前设定的其他最终目标一样。你知道，我开始梦想得到我的第一份网络开发工作，我在六个月的时间里实现了。几年后，我成为了一名移动开发人员，就在去年，我终于找到了一份软件开发人员的工作。

一座又一座山，一个又一个目标——这是一个永无止境的旅程。

我也有了新的目标，我一直在追求。这个开发者之旅不会在你得到第一份工作或第二份工作后就结束。你最好从大处着眼，从长远考虑。培养纪律、毅力和思考自己应得的能力。因为成为一名开发者意味着成为最好的自己，过上你能想象的最好的生活。
