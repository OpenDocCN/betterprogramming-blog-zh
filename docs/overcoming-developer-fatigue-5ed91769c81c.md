# 克服开发人员疲劳

> 原文：<https://betterprogramming.pub/overcoming-developer-fatigue-5ed91769c81c>

## 如何重新发现你刚开始时感受到的魔力

![](img/de84209f4245ca0894b16d0d76d8d846.png)

图片来自[皮克斯拜](https://pixabay.com/?utm_source=link-attribution&amp;utm_medium=referral&amp;utm_campaign=image&amp;utm_content=2912424)

对于我们大多数人来说，设计师和程序员之间的界限是非常模糊的。我们要么是一个人的商店，要么在团队中工作，我们必须身兼数职。当我听到“开发人员”一词被用来形容一个人时，我认为他们正在做一点点事情来使项目启动，我赞扬他们。

乍一看，戴很多帽子似乎有助于减少无聊的可能性，但我发现事实恰恰相反。与直觉相反，信息和选择的过载会导致严重的*开发者疲劳*。

开发人员疲劳可以通过多种方式表现出来。有时这是当你坐在键盘前，盯着你最新的项目，感觉没有灵感或没有想法时的感觉。其他时候，是无法开始一个新项目，因为感觉你只是在一遍又一遍地做同样的事情。更令人疲惫的是，开发人员疲劳会让你觉得你没有任何好的想法，或者你永远不会像其他开发人员一样优秀。

所有这些想法在那一刻会让人感到难以置信的真实和强大，并且会严重影响你的效率和产出。然而，克服这些困难并不像“埋头苦干”那样容易。我和你认识的所有其他开发者都经历过。让我们来看看一些屡试不爽的克服开发人员疲劳的方法。

# 重温你的旧项目

对我来说这总是第一步。当你感到特别疲惫或缺乏灵感时，花点时间提醒自己已经完成的不可思议的事情是很重要的。

重温旧项目会让你对过去解决难题的聪明方法有新的认识。它可以向你重新介绍你已经建立的真正优秀的设计和 UI/UX 解决方案，你应该感到自豪！

如果你做这项工作的时间足够长，以至于患上了开发疲劳症，那么你已经擅长这项工作足够长的时间，以至于拥有一个你很可能已经忘记的自我激励的目录，只等着你去重新发现。

# 探索新的语言

还记得你第一次让电脑说“你好，世界”吗？这种感觉太棒了。这是一个关键时刻，你第一次开始掌握代码的力量。很鼓舞人心！

我今天的代码日大部分时间都是用 JavaScript 写的，有时候我真想把我的笔记本电脑扔向墙壁。当这种情况发生时，我会去玩 Python。我根本不是 Python 开发人员，但是看到其他语言如何完成类似的任务，或者看到不同的语言可以完成哪些新任务，对程序员来说可能是一剂肾上腺素。

大多数时候，它只不过是灵感的一瞬间。然而，有时它可以为全新的开发范式或方法打开大门。理解其他语言的工作方式对任何开发人员来说都是一个积极的结果，即使你除了涉猎之外从未使用过这些语言。

# 阅读开发人员/程序员/设计博客

有时候，你已经知道做某事的最佳方法。更多的时候，有新的更好的完成任务的方法在等待被发现。

我相信开发人员依靠发现而繁荣。我们的工作是为人们的问题构建解决方案，在这样做的过程中，我们必须在最终选择一个解决方案并构建它之前，发现问题的全部本质以及解决问题的许多方法。

每天都有成千上万的其他开发人员、程序员和设计师写下他们是如何解决问题的。花些时间阅读他们的流程，看看哪些部分可以融入到你自己的流程中。

这不仅会让你成为一名更好的开发人员，还会帮助你重新点燃发现和潜力之火，正是这种发现和潜力让我们大多数人一开始就进入了这个行业。

# 尝试对等编程

最近，我发现自己越来越多地向开发人员推荐对等编程。在一个项目中与另一个开发人员合作可以打开全新思维方式的大门。

让自己接触别人的专业知识可以让你很快学到很多东西，以至于很难夸大它的价值。很有可能他们会用全新的方法来解决你多年来用同样的方法解决的问题。你可能会发现他们的方法更好，在这种情况下，你可以从中学习，或者更差，在这种情况下，这是对你工作质量的认可。

对等编程还为你提供了一个机会，让你可以给不如你熟练的人当老师，这将是一次非常有益的经历。看着他们对你展示给他们的东西感到兴奋，不可避免地会让你对自己的才能再次感到兴奋。兴奋是会传染的！

# 结论

克服开发人员疲劳没有完美的答案。重要的是要记住这是一种暂时的状态。这可能会让人感到疲惫不堪，但这是道路上的颠簸，而不是道路本身。

花些时间记住你成为开发人员的所有原因，分享你的经验，探索新的方法都是摆脱疲劳的好方法。

开发者是建设者。那是一件不可思议的事情。有太多令人难以置信的理由让我们对自己的工作感到兴奋，但人类状况的真相是，你不可避免地会觉得自己对此感到厌倦。这并不意味着你是一个糟糕的开发人员，或者你会一直有这种感觉；这只意味着是时候重新发现你刚开始时的那种魔力了。

你能克服它，你会克服它，你会继续创造更多不可思议的东西！
