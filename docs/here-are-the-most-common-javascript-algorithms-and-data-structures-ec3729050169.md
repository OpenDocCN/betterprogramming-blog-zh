# 数据结构

数据结构是在计算机中组织和存储数据的一种特殊方式，以便可以有效地访问和修改数据。

更准确地说，数据结构是数据值、数据值之间的关系以及可应用于数据的功能或操作的集合。

这里有一个算法和数据结构的列表，你应该学习来提高你的编码技能。

每种算法和数据结构都有自己单独的`README`以及相关的解释和链接，以供进一步阅读(包括 YouTube 视频)。
