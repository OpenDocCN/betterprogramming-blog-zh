## 不用每个月写一万字，甚至不用每天写

程序员同事经常问我为什么开始在线写作，他们是否也应该这样做。让我从一开始就告诉你一件事:是的，你应该！

作为一名程序员写作有很多好处，比如额外的收入、个人发展、提高声誉等等。不需要每个月写一万字。你甚至不用每天都写。而且你绝对不需要写作背景或者一些隐藏的天赋！你只需要一个写作的理由，这就是这篇文章的内容。

所以在下面的文章中，我写下了我个人认为程序员应该开始在线写作的五大理由。

# 1.名声

许多程序员开始写作的一个很好的理由是建立某种声誉。假设你是一名程序员，能够编写 JavaScript 代码并将其转化为功能应用程序，但没有人知道你能做到这一点。

通过开始在网上写你所做的事情，你给了其他人了解你和你所拥有的技能的机会。这是一种极好的自我营销方式，并不断带来新的机会——尤其是在工作方面。

为此，你甚至不需要自己的博客(我建议创建一个，但那是另一篇文章的内容)。你可以使用 Twitter 或 Reddit 等社交媒体或 Medium、Dev.to 或 Hashnode 等优秀平台来分享你的作品。

# 2.教育

作为程序员开始写作的另一个重要原因是希望帮助他人。世界各地对熟练开发人员的需求仍然很高，但获取学习材料和购买学习材料的资金绝对不够。

通过以博客帖子、教程或电子书的形式创建内容，您可以帮助其他人发现和学习新事物，并提高他们的技能。

你是否愿意收费完全取决于你，但是特别是在开始的时候，我会建议你免费创造一些东西。这不仅会帮助你成为一名更好的作家，而且是建立声誉的好方法，很有趣，帮助别人永远是一种很好的动力。

# 3.你自己

这其实也是我开始网文的原因。我想为自己做这件事——分享我的想法并获得反馈。但通常，写作对我有某种治疗作用。我的脑海里总是充满了想法和想法，把它们写下来在很多情况下真的对我有帮助。

不要害怕分享失败和你后悔但从中吸取教训的事情。生活并不总是充满阳光，总会有人试图拖累别人。在网上更是如此，但不要让他们打击你，因为当你写作时，你是为自己而写！

# 4.宣传

在线写作的另一个正当理由是获得公众的关注——无论是对你自己、你的产品还是你的公司。许多人写作当然是希望出名。以及 Twitter、Instagram 等平台。用他们的 clap 和 like 系统支持这一点。这些小奖励真的会让人上瘾，也很危险。

我觉得为了赶时间而写作是可以的，但是写作肯定有更好的理由。我相信，如果你开始写作时不去看那些喜欢的东西，不仅写作的压力会更小，而且从长远来看，回报会更多。

你也不应该试图过多地模仿别人。有灵感没问题，但最终还是要找到自己的写作方式。

# 5.金钱

你的写作可以产生大量的(额外)收入。我在 Medium 上写关于编程和技术的文章，每个月能挣几千美元:

![](img/a93ffd498dcd1fd770383e36eeebafc9.png)

但是在我看来，为钱写作本身就是一门艺术。它需要特别关注。对赚钱的关注。如果你想最大化你的投资回报率，你需要让你写的所有东西都朝着这个目标前进。声誉或教育之类的东西只是副作用——而不是主要目的。

当为了钱而写作时，你需要确保你的文章不断地被很多人阅读。我是说很多很多。至少在 Medium 上，你需要高曝光率的文章，但也需要有趣的内容，因为每个人阅读你文章的时间是决定你支出的算法的一个重要因素。中型出版物是扩大你文章覆盖面的好方法，所以你应该努力成为至少一个主要出版物的作者！

# 最后的想法

开始网上写作有很多好的理由。有些需要你特别关注才能成功，而且几乎所有这些都是相互受益的。

但我能给你的最好建议是开始写作。做就是了，不管什么原因。不是明天，而是今天！
