在技术领域，数据科学是最热门的工作之一，数据科学家几乎在每个领域都受到追捧。数据科学家是商业目标和产品策略之间的中间人。据《哈佛商业评论》报道，数据科学是 21 世纪最性感的工作，但它也带来了很多挑战。

你必须拥有正确的人际交往技能才能成功，因为你需要的不仅仅是技术技能来克服这些挑战。这些技能的结合使数据科学家能够获得洞察力，并通过有效地交流结果来帮助决策。

> “数据科学不仅仅是数字运算:它是应用各种技能来解决行业中的特定问题。”
> 
> Infosys 数据科学团队负责人 N. R. Srinivasa Raghavan 博士。

作为一名数据科学家，你需要有创造力和想象力。这些技能可以在不同的职业间转移。发展这些技能会让你从众多数据科学家中脱颖而出。

以下是数据科学家的五大软技能:

# 沟通

有效的沟通几乎在任何地方都是一种追求的技能。交流数据需要视觉、书面和口头交流技能。作为一名成功的数据科学家，传达信息至关重要，也是最关键的软技能。

在商业中，数据科学家需要精通数据分析，他们还必须能够流利地向技术和非技术受众解释他们的发现。组织依靠数据科学家来传达他们的结果并为他们的业务问题提供解决方案，这将使他们能够做出更好的业务决策。数据科学家可以用商业术语解释数据驱动的见解，这将有助于下一步的行动。

无论你在技术技能方面有多高超，当你能有效地向他人传达准确、正确的信息时，你的真正价值才会得到检验。你将不得不与很多人交流，从利益相关者到技术团队，再到非技术团队。你也几乎肯定需要写下你的分析和建议。

# **批判性思维**

除了从数据中获得洞察力，数据科学家还应该能够以不同的方式提问，并了解答案与业务的关系，以帮助做出最佳决策。

数据科学家应该有好奇心，有能力看到一个问题的不同角度，考虑数据来源等等。通过批判性思维，数据科学家可以分析问题并获得结果。

# 解决问题

数据科学基本上是解决问题。你应该具备作为数据科学家解决问题的技能。肯定会有问题，从杂乱的数据到 bug 等等，能够解决这些问题是一项核心技能。问题解决者很容易识别不同类型的问题，而且他们很快就能提供解决这些问题的方法，从而获得最佳结果。

大多数时候，公司并不关心你修了多少门课程，并且成绩优异。他们想知道你在现实生活项目中遇到的挑战，以及你是如何处理和解决的。你应该能够辨别需要解决的问题和潜在挑战，以维持和发展业务。

# 商业头脑

作为一名数据科学家，你需要了解你所在行业的具体情况。除了在你的领域了解很多，公司还确保数据科学家拥有领域知识，能够为他们将数据转化为结果。企业的主要目标是盈利。作为商业中的数据科学家，你被期望用数据为公司带来价值。

你需要敏锐的商业嗅觉来决定在哪些项目中应该和不应该应用数据科学，你需要跳出数据科学家的角色，去体验更接近财务业务的东西。您将与业务团队合作，深入了解业务运作和收入产生的方式。

# **求知欲**

数据科学家很好奇，他们试图根据给定的数据发现潜在的真相，他们有一种推动力量来回答被问及未被问及的问题。他们不断寻找答案，深入挖掘以了解更多。他们试图发现潜在的真相。

作为一名数据科学家，您的专业技能非常重要。然而，你需要发展以上这些软技能以及更多的技能，以便在你所在的领域茁壮成长，成为最优秀的人。
