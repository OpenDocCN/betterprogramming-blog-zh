## *关于更好的编程，我们详细讨论了以下主题*

![](img/e98247c42e0b5c8151caa807aefffc9d.png)

由[凯利·西克玛](https://unsplash.com/@kellysikkema?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

我们喜欢你的写作和提交给 [Better Programming](http://medium.com/better-programming) 。然而，我们已经广泛发表了某些主题，我们认为进一步提交是多余的。

关于以下主题的提交将被拒绝；我们恳请您不要提交它们(除非有一些新的和有趣的东西出现。如果是这样的话，请随时[提交一篇文章](https://medium.com/better-programming/write-for-us-5c4bcba59397)介绍新的进展。

*   `**this**` **和 JavaScript 中的 let/var/const**
*   **什么是/a/an:编程、数据库、API、MySQL、NoSQL、云计算、区块链、JavaScript、React、Angular、Node.js、Vue.js、计算机科学、数据科学、互联网、数据、Git、GitHub**

上面的术语、语言、库和框架的基本定义已经被更好的编程和互联网上的各种其他资源很好地覆盖了。

*   **与新冠肺炎相关的文章**——对许多受新冠肺炎影响的人来说，这是一个棘手而又令人痛苦的话题。我们将友好地传递关于仪表板，追踪器，或其他与新冠肺炎有关的项目的文章。谢谢！
*   **React 钩子简介**
*   **任何历史悠久的语言(Python、JavaScript、Objective-C、Swift、Ruby 等)中的 if/else 或控制流。)**
*   **Git 基础知识**
*   **如何用 X 创建博客**
*   **股票或加密货币价格预测文章**——虽然金融数据集为我们提供了广泛的数据，供我们在机器学习模型中使用，但我们对发布金融建议或感觉接近它的文章不感兴趣。

谢谢你的理解。我们期待您的提交并与您合作！
