## 如何发现和避开它们——或者在它们里面生存

![](img/568e87dcbbe8730747907834277d253c.png)

凯特·萨德在 [Unsplash](https://unsplash.com/s/photos/company?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上的照片

有不同类型的公司。有些公司依赖软件，你的代码就是他们的命根子。他们重视软件质量，并且他们提供良好的基础设施来支持软件开发。

另一方面，也有一些公司不适合软件开发人员。

作为一名软件开发人员，你有没有觉得自己在错误的公司工作？管理层不懂软件开发，你的工作没有得到重视，你觉得自己被困在了工作中？作为一名经验丰富的开发人员，我在以前的几个工作场所都有这种感觉。最近，我和一个有类似问题的朋友聊天，这促使我在这里分享我的想法。

# 你在一个你应该回避的公司吗？

这是一份清单。如果你的公司属于以上几类，那么你可能需要考虑是否应该留在那里。

1.  业务是否依赖于软件产品质量？好软件的价值能在商业结果中体现出来吗？如果答案是肯定的，那么你就是在为底线做贡献，公司很可能会把你当成一笔有价值的资产。否则，公司可能会将开发人员视为费用或只是一个“成本中心”软件开发商很贵。对某些人来说，它可以被视为发展业务的一项投资，或者对其他人来说，它只是一项没有可衡量利益的支出。
2.  你的团队领导或经理理解并重视好的技术工作吗？一个没有技术背景的管理者可能不懂软件开发。编程需要专业的技能、高度的专注和一定程度的创造力。但是如果管理层不理解这一点，他们可能会将软件开发视为生产线。他们倾向于要求一个不切实际的时间表，并将开发人员视为可有可无的资源。
3.  是否有既定的合理的软件开发流程和标准？如果公司有一个低质量的代码库，或者没有合适的过程或标准，你将很可能在一个不可维护的和错误的代码库上工作。这将是一项非常痛苦的工作。
4.  一个开发者有明确的职业道路吗？一个管理良好的公司应该为技术人员和非技术人员提供发展的途径。不提供清晰的职业道路是公司管理不善的标志。
5.  该公司是否使用过时的技术堆栈，并且没有升级它们的计划？如果一个公司不想投资它，他们对一个开发者来说是没有前途的。
6.  公司对软件的要求低吗？如果公司不要求定制解决方案具有深度和专门的领域知识，没有高容量的用户流量，那么他们对开发者的标准也会很低。在这样的环境下工作对你没有好处。

# 如何更早发现那些公司？

为了避开这些公司，我们需要首先识别它们。以下是一些早期发现它们的迹象:

*   如果你的面试太短太简单——只有很少的技术问题——那是一个警告信号。这是公司文化的体现。
*   询问你的工作环境将会有什么样的硬件和软件规格。如果你得到一台不适合运行开发工具的计算机，或者没有足够的软件许可证给每个开发人员，这就是一个危险信号。
*   研究 Glassdoor 上现有或以前的员工反馈。
*   通过 LinkedIn 或 Glassdoor 等社交媒体网站了解该公司是否有高离职率。
*   如果你是唯一的开发者，要小心。这并不适合所有人；没有知识分享和与他人的交流，这可能是一个孤独的旅程。

# 如何在这些公司生存？

如果你被困在这些公司中的一家，由于某种原因不能离开，这不是世界末日。

在恶劣的环境中，你的日常工作可能只是复制粘贴，你可能使用的是过时的技术堆栈，但你仍然可以通过以下方式尽最大努力实现价值最大化:

*   在你的工作场所发起一项倡议。它可以简单到编写一个脚本来自动执行您自己使用的手动任务。或者这也可以是一个大胆的建议来更新一个框架。改变是艰难的，但是如果你做好了充分的准备并下定决心，总会有成功的机会。
*   **不断学习**，更新你的技能。通过开始一个个人项目和应用新技术来实践你刚刚学到的东西。或者找一个有意思的开源项目，定期投稿。
*   **持续分享贡献**。当你学到新东西后——分享它并使用它！写技术文章是分享知识的好方法。写作不仅能巩固你所学的知识，还能向潜在雇主展示你的技能。此外，你可以参加技术用户群，甚至可以提供演示。你也可以在工作场所开始定期的知识分享。
*   **成为商业领域专家**。由于软件开发人员可能不会在一家公司呆太久，所以成为领域专家并不容易。但是积累深厚的商业知识是无价的。既有编程技能，又有领域知识，就能脱颖而出，更容易在同行业找到下一份工作。

# 最后的想法

作为一名程序员，你沉浸在编程的世界中。编程世界很简单。软件是可预测的:给定一个输入，一个写得好的程序将总是有相同的输出。

但是生活是不可预测的。你无法预测下一份工作会是什么样。不管你有多努力和贡献，你都可能得不到应有的认可。

在对你不好的公司工作很难。你可能会感到压力、无助和不开心。但这就是生活。生活充满了惊喜、竞争和冲突。

> “多一些微笑，少一些担忧。多一些同情，少一些评判。多一些祝福，少一些压力。多一些爱，少一些恨。”——罗伊·T·贝内特

我们能做的是保持积极的态度，为新的机会做好准备，继续前进。
