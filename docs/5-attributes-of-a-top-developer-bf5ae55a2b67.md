# 顶级开发人员的 5 个特质

> 原文：<https://betterprogramming.pub/5-attributes-of-a-top-developer-bf5ae55a2b67>

## 他们有共同的特点，使他们脱颖而出

![](img/5ed3d521468854743ed841088dcc9618.png)

迈克·丁尼翁在 [Unsplash](https://unsplash.com/s/photos/great-developer?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上的照片

顶级开发人员是那些引领技术方向并完成困难工作的人。一个顶尖的开发人员会给团队带来强大的正面影响。

他们很少见，但是我很幸运以前和一些人一起工作过。他们有一些共同的特点，使他们与众不同。

> “最好的程序员并不比优秀的程序员强多少。无论用什么标准来衡量，他们都要高出一个数量级:概念创意、速度、设计独创性或解决问题的能力。”——[*兰德尔·e·斯特罗斯*](https://www.quotemaster.org/q2a9caad0a35ea08e59f2c5cc9b8b4b3f)

在这篇文章中，我将列出我认为的顶级开发者的标志。了解它们让我们有机会反思自己，找到差距，成为更好的开发人员。

# 对编码的热爱

对于我们中的许多人来说，编程只是一份朝九晚五的工作，可以让我们获得报酬。如果你不喜欢这份工作，它可能是一份压力很大的工作。有些人在最初几年对编码有热情或热爱，但这种热情和热爱会慢慢消退，编码会变成一件累人的苦差事。

> "做伟大工作的唯一方法是热爱你所做的事情。"— [史蒂夫·乔布斯](https://news.stanford.edu/2005/06/14/jobs-061505/)

对于顶级开发者来说，编码是一种爱好而不是工作。他们真正喜欢编码，因为它给他们能量和满足感。

顶级开发人员有太多的理由热爱编码。他们喜欢创造一门新手艺的兴奋感。他们喜欢解决一个困难的 bug 后的成就感。他们喜欢用他们惊人的脑力创建一个运行程序的感觉。他们知道成千上万甚至上百万的用户会使用它。

# 问题求解程序

软件开发人员的工作就是解决问题。顶级开发人员是在压力下能够快速解决难题的人。换句话说，当关键问题发生时，他们是关键人物。当面对一个难题时，顶尖的开发者是不会放弃的。他们固执而执着。

批判性思维可以说是最难学习和发展的技能。所有顶尖的开发者都有惊人的批判性思维能力。

> “如果这个星球上的每个人都需要一种生活技能，那就是以批判性的客观态度思考的能力。”— [乔希·兰宁](https://libquotes.com/josh-lanyon/quote/lbw4b1d)

当面临一项关键任务时，他们使用系统的方法来过滤噪音，识别问题的本质，并评估各种解决方案。能力来自于多年的经验和无数小时的调试和学习。这成为顶级开发人员的第二天性，这就是为什么他们总是能有效地、创造性地解决问题。

# 谦逊和同情

一个顶级的开发者是一个谦逊的人。他们尊重每一个人。他们的经历告诉他们，每个人都可以学到一些东西，所有人都应该得到平等对待。

> “我足够谦虚，知道我不比任何人更好，也足够聪明，知道我与众不同”——未知

顶级开发人员在其职业生涯中获得广泛而深入的技术知识。他们知道的越多，就越意识到自己所不知道的。他们变得更加谦逊。

一个顶尖的开发者也是一个有同理心的人。

他们可以站在别人的立场上，因为他们在生活中经历了太多。同理心让他们变得耐心、善解人意、敏感。只要有可能，他们总是愿意提供帮助，并与团队的其他成员建立联系。这就是为什么他们对团队有很强的积极影响。

# 焦点

专注的能力在编程中极其重要。专注可以帮助开发人员进入高效的[流动状态](https://en.wikipedia.org/wiki/Flow_(psychology))。

我听到很多开发人员抱怨工作场所的注意力分散，但从来没有顶级开发人员抱怨过。他们可以完全沉浸在一段具有挑战性的代码中，就像我 3 岁的孩子在搭建一个新的乐高积木。

我不确定一些顶级开发人员是否生来就有这种能力，因为对编码的深度关注似乎是他们的第二天性。

对于一个开发人员来说，有很多关于如何保持专注的提示，尽管它们都忽略了最基本的一点:从深度流程工作中获得乐趣。当顶级开发人员将自己沉浸在工作中时，他们的世界被缩小到跳跃想法、钉牢代码块和让部件一起工作的思考过程中。时光飞逝，外面的世界不存在了。

> “水流动是因为它愿意。”— [马蒂·鲁宾](https://www.knowyourquotes.com/Water-Flows-Because-Its-Willing-Marty-Rubin.html)

我相信所有的开发人员有时都处于深度聚焦流中，我们都知道高效工作时的享受感。我们只需要增加我们到达那里的机会，就像顶级开发者一样。

# 不断升级

IT 行业一直在变化和发展，顶级开发人员也是如此。他们是总是对新知识充满好奇，也是伟大的自学者。

作为开发人员，我们都知道我们需要跟上新技术，这是一项艰巨的任务。

顶尖的开发人员有能力高效地学习新技术。他们不必强迫自己去学习；反而对新鲜事物充满了好奇。他们将通过询问主题专家和寻找所有可用的资源来弄清主题。他们不怕问问题，即使这意味着承认他们不知道。

> "好奇心是幸福的最大秘密之一。"— [布莱恩特·h·麦吉尔](https://www.brainyquote.com/quotes/bryant_h_mcgill_168220)

他们也热衷于将所学应用到工作中。新知识激发新的想法和解决方案，从而产生更好的产品，有时甚至是改变游戏规则的产品。

# 摘要

顶尖的开发者也是顶尖的执行者*。他们的成功不仅仅是因为他们惊人的技能，还因为他们谦逊、善解人意、充满好奇心。*

作为一名经验丰富的开发人员，我从不后悔我的职业选择。我很高兴早上起床(嗯，大部分时间)，因为我知道有新的事情等着我去解决。

我知道我不是顶尖的开发人员，但我正努力成为一名。使用这五个属性作为基准可以帮助我在旅程中成为更好的开发人员。

旅途无限。编程快乐！
