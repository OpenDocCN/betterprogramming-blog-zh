# iTerm2

[](https://iterm2.com/) [## iTerm2 - macOS 终端更换

### iTerm2 是 Terminal 的替代品，也是 iTerm 的继任者

iterm2.com](https://iterm2.com/)

# Zsh 和 OhMyZsh

[](https://ohmyz.sh/) [## 哦，我的 Zsh——一个令人愉快的开源 Zsh 框架

### Oh My Zsh 是一个开源的、社区驱动的框架，用于管理您的 Zsh 配置。听起来很无聊。让我们试试…

ohmyz.sh](https://ohmyz.sh/) 

多年来，我一直是 Zsh 的用户，部分原因是因为 OhMyZsh。

开箱即用的 Zsh 是 bash 的一个很好的替代方案，但是如果您希望进一步增强 Zsh，那么 OhMyZsh 将提供这一功能。

OhMyZsh 可以看做是`zsh`的框架。它有一个非常广泛的社区，提供了许多有趣和有用的软件包，您可以安装这些软件包，使您的终端体验更加个性化。

# 镜头

[](https://k8slens.dev/) [## 镜头 Kubernetes IDE

### Chris Kalmer@chriskalmar 这是一个不错的免费桌面应用程序，它将帮助你可视化和控制你的 Kubernetes…

k8slens.dev](https://k8slens.dev/) 

如果你使用过 Kubernetes，你就会知道重复输入`kubectl`命令是多么的令人厌烦。lens 是一个 IDE，允许您连接到多个 Kubernetes 集群，并查看当前在所述集群上的所有资源(包括 CRD)。您还可以检查给定资源的 yaml 定义，并使用内置的编辑器进行更改。

Lens 的另一个很好的特性是，如果你正在管理一个 Kubernetes 集群，你可以通过邀请用户到 Lens 内的协作空间来邀请、共享和配置用户访问。

# 邮递员

[](https://www.postman.com/) [## 邮差 API 平台|免费注册

### 超过 1700 万开发者使用 Postman。通过注册或下载桌面应用程序开始。什么是邮递员…

www.postman.com](https://www.postman.com/) 

如果你正在使用 API——公共的、私有的或者以微服务的形式，那么 Postman 应该在你的工具箱里。

Postman 消除了手动输入 curl 命令的麻烦。它提供了一个易于掌握的用户界面，在如何定制您的请求方面提供了很多控制。

此外，它还使 API 请求的存储、测试和协作变得容易，因为您可以将它们存储为提供另一个特性子集的项目。

# 红色尾灯

[](https://stoplight.io/studio/) [## Stoplight Studio | OpenAPI 设计、规划和建模工具

### 从头开始设计一个快速高效的 API。Stoplight Studio 缩短了学习曲线，因此您不需要…

交通信号灯. io](https://stoplight.io/studio/) 

Stoplight 允许您创建和定义带有 UI 的 API。这让您可以轻松地定义 URI 路径、请求主体、请求响应、特定路由上的身份验证以及它们之间的一切。

虽然这本身非常有用，但它实际上做的是获取你的 API 设计，并在`yaml`中将其加工成有效的 OpenAPISpec (OAS)设计。

这个 OAS 非常重要，因为它可以用于创建文档、将规范导入 postman、生成代码片段，或者用于验证您的 API。

# Todoist

[](https://todoist.com/) [## 待办事项:组织工作和生活的待办事项列表

### 通过把所有这些任务从你的脑海中抹去，放到你的任务清单上，重新获得清晰和平静(无论你在哪里…

todoist.com](https://todoist.com/) 

虽然不是一个软件相关的工具，但我会说这个工具对我来说可能是最重要的。

Todoist 让我在任何给定的时间都能保持有组织性并专注于多个项目。
