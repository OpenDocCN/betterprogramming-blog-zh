随着 2020 年接近尾声，我很高兴与你分享我在过去 12 个月中读过的最好的软件工程书籍。

如果你是一名软件工程师、数据科学家，或者在技术或软件行业工作的人，你会同意我的观点，如果你想在游戏中保持相关性，你必须不断地学习。

当你决定成为一名软件工程师的时候，你实际上就开始了一段终生学习的旅程。

学习或获取知识的方式有很多种，但书籍仍然是这一领域的主导力量。

# 1.雄辩的 JavaScript

JavaScript 在网站中被广泛使用，其受欢迎程度持续上升。现在是学习 JavaScript 的最佳时机！

[这本书](https://www.amazon.com/Eloquent-JavaScript-Modern-Introduction-Programming/dp/1593272820)分为三个主要部分。

第一部分是 JavaScript 本身的介绍。它涵盖了如何声明变量、数组、循环等内容。但是它也深入探讨了一些紧迫的问题，比如 JavaScript 中的空值和未定义值之间的区别。

本书的第二部分深入探讨了 JavaScript 在 web 浏览器中的工作原理。在这里，您将开始学习和理解文档对象模型(DOM)。

第二部分还涵盖了在画布上绘图等主题，这是通过图表和图形实现数据可视化的基础。

最后一节介绍 Node.js 以及如何在 web 浏览器之外运行 JavaScript。我还喜欢有整整一章专门讨论 JavaScript 的性能、垃圾收集的工作原理以及性能优化。

> “当人们问我如何学习正确的 JavaScript 时，我会分发这本书。”— [美国 JSConf 组织者克里斯·威廉姆斯](https://www.oreilly.com/library/view/eloquent-javascript-2nd/9781457189821/pr02.html)

# 2.Linux 圣经

在 Windows PCs 上引入 Linux 的 Windows 子系统(WSL)意味着 Linux 和/或 Unix 现在真的无处不在。

作为一名软件工程师，能够无缝地理解和操作基于 Linux 的系统是至关重要的。毕竟，很多基于云的服务器都是基于 Linux 的。

这本书将带你踏上从初学者到大师的旅程。您将学习像在机器上安装 Linux 发行版这样的基础知识，然后继续学习像管理和保护 Linux 机器这样的高级主题。

您还将学习并掌握命令行、命令行工具以及如何编写 shell 脚本。

如果你想对 Linux 操作系统有一个全面而深刻的理解，并且还想找一本参考书，那么这就是你要找的书。

# 3.JavaScript & jQuery:缺失的手册

JavaScript 的广泛流行导致了许多 JavaScript 框架和/或库的出现，如 jQuery、Angular、React、Svelte、Vue.js 等。

虽然 jQuery 可能不再像过去那样是占主导地位的 JavaScript 库，但[这本书](https://www.amazon.com/JavaScript-jQuery-David-Sawyer-McFarland/dp/1449399029)将帮助你了解 JavaScript 库和/或框架已经走了多远。

这本书也很好地以温和的方式介绍了 JavaScript。概念解释得非常好，非常清楚。如果你对 jQuery 部分感兴趣，这本书也不会让你失望。它很简洁。

通过学习 jQuery 库，您将体会到为更大的前端项目使用更结构化的库或框架而不是使用普通 JavaScript 的威力。

# 4.ASP.NET 核心在行动

[这本书](https://www.amazon.com/ASP-NET-Core-Action-Second-Andrew/dp/1617298301)涵盖了与你创建和部署跨平台的基于 ASP.NET 核心的 web 应用相关的所有主题。

这本书将教你相关的主题，如:

*   MVC —创建使用模型视图控制器和 razor 视图的 web 应用程序。
*   中间件管道，它是如何处理 HTTP 请求的核心。
*   为 web 客户端和移动应用程序创建健壮的 web APIs。
*   实体框架核心—用于访问和修改数据库的健壮的对象关系映射器(ORM)库。

还详细介绍了应用程序安全性和身份服务器的身份验证。

我也喜欢这本书涵盖了使用 xUnit 框架的单元测试，因为这个主题在软件工程书籍中经常被忽略。

# 5.对 Mongo、Express、Angular 和 Node 变得刻薄

很长一段时间，JavaScript 被认为是一种只适用于 web 的语言，但随着 Node.js 的引入，一切都变了。

如今，JavaScript 被用来创建全栈应用——这本书将告诉你如何做。

这本书主要关注如何使用以下技术构建 JavaScript web 应用程序:Node.js、Express、Angular 和 MongoDB。

我真正喜欢这本书的地方是，在继续构建之前，它会带您浏览一个示例 web 应用程序的架构和设计。这个过程帮助你理解技术的选择，以及为什么你会从一个特定的角度来处理一个项目。

此外，您还将学会使用 Bootstrap 和 pug 模板来创建响应迅速且美观的网页。

最后，您还将学习使用现代文档结构化数据库系统(MongoDB 及其关系映射器 Mongoose)。

# 6.在一个月的午餐中学习 Docker

码头工人席卷了我们的行业。它革新并简化了我们运送、部署和设置软件开发环境的方式。

这本书的格式是这样的，你在一个月内每天学习一个概念。每章都有练习题和动手实验来培养你的技能。

这本书将教你非常基本的 Docker 和更多的基础知识。您将了解什么是 Docker 图像以及如何创建图像。您还将学习在本地机器上将 Docker 图像作为容器运行。从那里，您将学习如何通过 Docker Hub 在云中托管和分发 Docker 映像。

这本书还涵盖了高级主题，如使用 Docker Swarm 和 Kubernetes 的容器编排。

最后但同样重要的是，您将学习优化和保护您的 Docker 映像以用于生产目的。

# 7.粉碎 WordPress:超越博客

WordPress 是一个开源的内容管理系统，主要用于创建网站(例如登录页面、网络商店和博客网站)。

尽管你可以不用写一行代码就能在 WordPress 中创建一个完整的网站，但是如果你想把你的开发技能提升到一个新的水平，理解基础知识是很重要的。

[这本书](https://www.amazon.com/Smashing-WordPress-Beyond-Blog-Magazine/dp/1119995965)将帮助你理解当你使用像 Elementor 这样的工具建立一个 WordPress 站点时，后台到底发生了什么。您还将了解哪些数据库表是由您的帖子更新的，以及如何根据您的喜好编辑配置。

此外，这本书着眼于高级主题，如创建用于扩展 WordPress 核心功能的自定义插件。

最重要的是，这本书非常小巧轻便，便于携带。

# 8.C#深度

现在是第四版，[这本书](https://www.amazon.com/C-Depth-Jon-Skeet/dp/1617294535)准确地传达了书名所说的内容。它深入涵盖了每个主题，可能不适合初学者。

另一方面，如果你打算把你的 C#技能提高到一个新的水平，那么这本书将帮助你做到这一点。你将彻底理解这种语言是如何运作的。

这本书向你展示了 C#是如何经过多年的发展成为今天这样成熟稳定的语言的。

大多数情况下，这本书提供了核心概念的深入分析，如 LINQ，异步编程元组等。这是彻底的，写得非常好。
