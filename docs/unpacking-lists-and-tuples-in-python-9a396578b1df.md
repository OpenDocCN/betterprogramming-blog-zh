# 扩展的可迭代解包

在解包的时候，我们不得不提到变量的个数等于序列的长度(list/tuple)。

为了克服这个问题，我们可以使用扩展的 iterable 解包。

扩展解包使用`*`操作符。以`*`操作符开头的变量用于扩展解包。这将是一个列表，包含 iterable 中所有未分配给变量名的元素。

*   `first,*rest` —将第一个元素解包到`first`，所有剩余元素解包到`rest`。`rest`将是除第一个元素之外的所有项目的列表。
*   `first,*middle, last` —将第一个元素解包到`first`，最后一个元素解包到`last`。除了第一个和最后一个元素之外，其余的元素将是所有项目的列表。
*   `*rest, last` —将`last`中的最后一个元素解包，剩余的元素全部解包到`rest`。`rest`将是除最后一个元素之外的所有项目的列表。
