# 作为开发人员分享知识的 5 大好处

> 原文：<https://betterprogramming.pub/5-amazing-benefits-of-sharing-knowledge-as-a-developer-7d6f3ce6b7ee>

我想鼓励你采取行动并分享你的行动

![](img/8859b4d076602c9181166316b64306c5.png)

克里斯蒂安·威迪格在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

在网上分享我的知识给我带来了很多好处。因此，我想鼓励你采取行动并分享你的行动。当然，如果你愿意的话。这不是强制性的。

那么，在线分享你的知识有什么好处呢？作为开发人员，分享您的专业知识有以下五个好处:

*   你加深了对一个主题的理解
*   你可能会得到工作机会和/或有偿演出
*   你帮助其他人
*   你成为了社区的一部分
*   当你将来遇到同样的问题时，你可以省去自己的麻烦。

事不宜迟，让我们深入了解每项优势。

# 巩固你的知识

最重要的好处之一是你巩固了你所教授的学科的知识。当你教授一个课题时，你必须研究它，以确保你没有教授错误的东西。这迫使你更深入，更好地学习这门学科。

此外，你可能会学到一些新东西。众所周知，在技术领域，事情发展得很快。也许你所教的科目在你第一次学的时候已经改变了。因此，这是一个保持与时俱进的好方法。

就我而言，每次我写或录制一个关于特定主题的视频，我都会学得更好，它会更长久地伴随着我。有一句名言完美地总结了这一节。大意是这样说的:“教就是学两次。”

# 获得机会

在网上分享你的知识是一个很好的机会。不管是什么——工作，写演出，等等。拥有一个博客和/或一个视频频道就像一个私人招聘人员不停地为你寻找机会。

看你内容的人可以大致了解你的知识和技能。例如，因为我写了关于 AWS 和 JavaScript 的文章，很多公司联系我为他们写文章。不是找工作，而是工作找到你。听起来不错吧？

当然，获得机会应该是你行动的结果；不一定是目标。你可能会也可能不会得到机会。不是给定的，但是可以发生！因此，在网上分享你的知识，看看这会把你带到哪里！

# 帮助他人

当你打开你的博客或视频频道时，感觉你在为任何人创造内容。起初没有人知道这件事。但是最终，人们会偶然发现你的内容，希望他们能从中学习到一些东西。知道你的内容也帮助了其他人是一种奇妙的感觉！

所以这一节就不多说了。你可以通过分享你的知识来帮助很多人。

# 成为社区的一部分

另一个好处是你可以加入并成为各种社区的一员。你会遇到志同道合的人，和他们一起出去玩。除了交朋友和娱乐，它还能带来合作和机会。

此外，和有知识的人在一起，你可以更好地学习和提高。他们可以指引你正确的方向，你也可以这样对待别人。因此，作为社区的一部分:

*   很有趣
*   帮助你交朋友
*   推动你更好更快地进步
*   确保你周围都是志同道合的人

# 这给你省了后患

你更有可能不止一次地遇到一个问题。因此，多次解决同一个问题可能是浪费时间。为什么不创作一篇文章或一个视频来解释你的解决方案呢？

每次你遇到同样的问题，你可以回到你的材料，并迅速解决它。除此之外，其他人可能也有同样的问题。因此，你也帮助了别人。

分享你的知识是记录问题解决方案的好方法。它拯救了你和其他人。

# 结论

文章中列出的好处并不详尽。这些是我亲身体验到的好处，但可能还有其他好处。

我鼓励你开始分享你的知识，因为你:

*   向机遇敞开心扉
*   更好地学习一门学科
*   成为社区的一部分
*   帮助你未来的自己和他人
