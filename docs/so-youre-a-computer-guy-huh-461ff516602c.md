一个老朋友最近介绍我是她的“电脑男”她已经认识我 20 多年了，但我肯定她无法准确地描述我实际上是如何使用计算机的。

当她第一次见到我时，我正爬到她的桌子底下，插上一根电缆，以确保她的网络打印机正常工作。当时我正在她工作的地方安装定制软件，连接嘈杂的点阵打印机。

从那以后，我在信息技术领域身兼数职，但是即使我的妈妈知道我获得了计算机科学学位，她仍然把我描述成一个只在计算机上工作的人。有一段时间，我现在的前妻以为我整天修电脑和打印机。

它被称为*信息技术*是有原因的——它完全是关于信息和数据的。

这里有一个对描述计算机男女的各种角色的过度简化。为了有一个大多数人可能会涉及的背景，让我们使用亚马逊网站，而不是河流作为我们的焦点，以及它如何与数据(信息)相关联。

# 节目编排者

也称为编码员、软件工程师或软件开发员，这个人编写代码接收你的搜索文本，然后向你显示一页又一页的匹配条目。

程序员负责确保网站上的*“添加到购物车”*按钮，正确地将正确的项目放入您的购物车。

你有没有注意到，有时候你想要的商品在某个特定的颜色上显示缺货？或者另一个卖家以更低的价格出售类似的商品？

你猜对了，网站提供的所有“智能”都是由程序员创造的。

当你点击“结帐”时，网站会显示出计算出税金和运费后的正确总数，这是程序员的杰作。

程序员提供收集您的数据的按钮和文本框，并确保您的订单正确使用您的数据(地址、支付信息等)。

# UI/UX 设计器

还记得“添加到购物车”按钮吗？为什么是橙色的？

当你在网站上时，你是很容易就找到了那个按钮，还是不得不在网站上到处寻找？

也许你还必须更改送货地址或添加新的支付方式。你能在不寻求帮助的情况下凭直觉做出这些改变吗？希望你也阅读了一些评论和其他购物者提供的一些问题和答案。

感谢用户界面/用户体验设计师让你能够轻松地在网站上四处走动。有时候 UI 设计师和 UX 设计师是不同的人；有时候两者都是同一个人做的。

这个人确保你能够**轻松找到并理解产品及其相关数据**。

# 数据库管理员(DBA)

所有这些产品数据来自哪里？当您点击“提交订单”时，它会转到哪里？

亚马逊如何不把你的订单和其他几百万的订单混淆？当你登录你的账户时，网站如何知道向*显示你的*运输和付款信息？亚马逊*记住*你的信用卡信息和你许许多多的送货地址，这不是很方便吗？

你有没有在亚马逊上找不到任何东西？好像什么都有！每件产品的每一个描述、每一个价格、每一条评论都相当于一吨的数据。

数据库管理员是所有这些数据的保管者。他们的工作是确保数据对你、顾客以及亚马逊都是可用的。DBA 确保这些数据被安全地存储起来，并能以极快的速度获得。否则，你可能会感到沮丧，转而去 Walmart.com 购物。

# 网络工程师

如果你能想象亚马逊网站上的所有数据，它包括你和其他人的所有购物历史。显然，还有数以亿计的产品可供搜索。

所有这些数据都驻留在世界各地的服务器上。这些信息是如何进入你的电脑或手机的？

网络工程师的工作是提供将亚马逊的服务器直接连接到你的设备的路径。在任何时候，您都可能连接到欧洲、北美或东南亚的服务器，这取决于各种条件。

网络工程师**确保路径清晰，以便信息快速顺畅地流动。**

程序员和 UI/UX 设计者创建的代码在这些路径上流动。DBA 管理的数据也通过这些“管道”来回流动

# 网络安全工程师

网络工程师管理的路径被设计成只包含亚马逊、他们的合作伙伴和你这个购物者之间的数据。

网络安全人员在这些路径上巡逻，以确保没有不良分子在查看流动数据，或试图窃取购物者的私人信息。他们还沿着这些路径安装了额外的护栏，以确保安全。

通常，他们还会对程序员的工作进行测试，以确保接收信用卡信息的文本框不会被全球黑客劫持。

网络安全部门还会检查数据库管理员，以确保他们的数据没有被窃取、破坏或盗用。

还记得《夺宝奇兵》中他试图用一袋沙子代替金偶像的场景吗？网络安全人员检查以确保数据库管理员确实在保护神像，而不是一袋沙子。在这个类比中，它们也是威胁摧毁任何入侵者的巨大岩石。

# 质量保证工程师

如果任何一个购物者用错了信用卡，亚马逊就会陷入一堆麻烦。或者是否收取了错误的金额。或者如果网站显示错误的库存信息，销售更多实际上已经售完的商品。

为了确保这些情况尽可能少地发生(错误确实会发生)，程序员的工作不是简单地一上交代码就对外发布。

对网站的任何更改必须通过质量保证工程师和他们的团队。他们是测试者和验证者。在公众看到最终的修改之前，他们就盖上了最后的批准章。当亚马逊第一次创建 OneClick 功能时，QA 人员对它进行了大量测试。

QA 团队确保网站的更改不会对购物数据的收集和管理产生负面影响。

# 数据分析师

这个人检索从购物者、供应商、第三方卖家、评论者和其他地方收集的所有信息。Amazon 创建或收集的任何信息都会被数据分析师合理使用。

所以数据被收集和查看。然后呢？从收集的数据中创建了更多的数据。

分析师为管理层、产品团队、营销团队和高管准备报告，以帮助他们做出商业决策。亚马逊需要雇佣更多的人吗？亚马逊需要扩张到另一个地区来缩短运输时间吗？消费者是否更有可能购买某种产品而不是另一种，因此需要大量储存？

数据分析师和他们的报告能够回答杰夫·贝索斯想知道的关于公司运营的几乎任何问题。

所有的数据从哪里来？来自数据库管理员管理的数据库。

# 系统管理员

“系统管理员”让一切为购物者和亚马逊员工服务。

他们为会计部门修理可怕的打印机。他们帮助首席管理人员解决经常忘记的密码。当服务器决定需要重新启动时，它们会重新启动服务器。

他们让房子保持运转。他们保持庞大的计算机网络相互交流。他们帮助找出为什么电子邮件不能到达正确的地方。

如果有人应该被贴上“电脑人”的标签，认为他可以修理任何有电源按钮的东西，那么这个人就是系统管理员。

# 我们都是电脑人

除了上面列出的角色和头衔，技术世界还包括几十个其他的角色和头衔。一些角色是行业专业化或其他角色的组合。这个列表仅仅是常见内容的一小部分。

仅仅因为我们都用电脑工作并不意味着我们都有同样的能力。某些角色之间有一些重叠，但是我们的日常职责可能会大不相同。

根据公司的规模和我们工作的行业，我们中的一些人会身兼数职，拥有上述各种头衔。我们中的一些人，在我们的职业生涯中，成长、进化并获得了许多上述技能。

我们可能有一个共同点，那就是我们有能力用我们高超的谷歌技能解决任何与技术相关的问题。我们的这种能力让我们永远被贴上“电脑人”的标签

然而，仅仅因为我们可以很快找到一个关于如何修理你的烤面包机的 YouTube 视频，并不意味着我们真的想这么做。
