使用JSX

由于默认情况下Vue CLI 3支持使用JSX，所以如果您愿意，现在可以在JSX编写代码(也可以从React编写)，这对于编写功能组件来说非常方便。如果你还没有使用Vue CLI 3，你可以使用babel-plugin-transform-vue-jsx来获得jsx支持。
