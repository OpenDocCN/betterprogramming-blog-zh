作为一个完全的初学者，我用来学习Vue.js的资源列表

在这篇文章中，我想向你展示我作为一个完全初学者用来学习Vue.js的五个资源。有些资源是免费的，而有些是付费的。

1. Vue精通

Vue Mastery是一个在线学习平台，每周制作关于Vue的课程，它有免费和优质内容。课程或教程有视频和文章两种形式。因此，你可以选择你最喜欢的格式。

课程由在行业中有多年经验的专业工程师讲授。而且，你可以找到Vue核心团队成员的教程。

话虽如此，以下是你可以从Vue Mastery中学到的一些东西:

  • 单元测试
  • Vuex
  • 带Vue的打字稿
  • 证明
  • Vue中的表单
  • 组件的设计模式
  • 反应性和成分API
  • Nuxt

当然，这些只是一些例子来突出科目的多样性，但还有更多课程。最后，值得注意的是，每月支付的25%用于Vue项目。我相信这是一个惊人的姿态！

话虽如此，如果你想学Vue，我推荐Vue Mastery。

2.官方文件

当涉及到技术、编程语言等等时，官方文档经常被忽略。然而，在某些情况下，文档对于了解该工具已经足够了。

Vue.js的官方文档也不例外；它很棒，是一门独立的课程。因为它是官方文档，所以涵盖了从基础到高级概念的所有内容。因此，它教你你需要知道的关于Vue.js的一切。

另外，需要注意的是，官方文档并不是一次性的。你会发现自己会时不时地想起它。

话虽如此，我还是强烈推荐。即使你不把它作为你的主要资源，也要用它来补充你从其他资源中的学习。

3.完整的Vue开发者

“零到精通”品牌发布的课程质量非常高，Vue.js课程也不例外。这是最全面的课程之一，你可以通过查看涵盖的科目来说服自己。

该课程时长超过29小时，涵盖以下主题:

  • Vue基础知识
  • 开发者环境
  • 高级Vue组件
  • Vuex
  • 形式
  • 证明
  • 按指定路线发送
  • 指令
  • 测试
  • 组合API
  • 组件设计模式

此外，您将构建一些项目来实践这些概念。我认为这是一个很大的好处，因为编程是一项实用的技能。你通过做来学习，而不是通过看或读。因此，你将学习概念，然后通过将它们付诸行动来巩固你的知识。

最后，我非常信任和欣赏他们的课程质量，所以我同意成为他们的会员。这个课程使用了联盟链接，这是我唯一接受的联盟链接。

4. Vue学校

Vue学校是学习Vue的另一个很好的资源。它有28门课程，449节课，34个小时的内容。课程由专业开发人员和行业专家讲授。

Vue School是一个免费增值平台，这意味着他们既有免费课程，也有高级课程。它的好处是提供了购买力平价折扣。因此，如果你来自一个购买力低的国家，你会得到很大的折扣。

我喜欢Vue学校的一点是，课程是按难度分类的:

  • 初级水平
  • 中间能级
  • 优等

这种分类方式使得从基础开始学习变得非常容易。也就是说，Vue学校是一个很好的资源！

5.布拉德·特拉弗斯的Vue球场

我第一次尝到Vue的滋味是从Brad的课程中得来的，这是一门非常优秀的免费课程。这个VueJS速成课程涵盖了开始使用Vue所需的所有基本概念。

本课程中遵循的一些概念如下:

  • 构建和使用组件
  • 组件属性
  • 处理事件
  • 使用Vue CLI
  • 实施和使用Vue路由器
  • 形式

概念清单并不详尽；该教程涵盖了更多，但我想强调最重要的。

如果您想知道首先选择哪种资源，我推荐您学习本课程。最后，您将能够轻松构建简单的Vue.js应用程序。

6.网络忍者的Vue课程

最后，我们有Shaun Pelling的免费课程，被称为网络忍者。尽管它不太全面，但它是一个很好的起点。共有12课，您将了解:

  • Vue.js基础
  • Vue CLI
  • 表单和输入
  • Vue路由器
  • 获取数据
  • 组合API

值得注意的是，肖恩有更深入、更全面的课程，但这是付费的。此外，我想提一下，我没有参加付费课程，但我听到了关于它的好消息。你可以在这里找到。

话虽如此，免费课程是一个很好的起点，我强烈推荐！

结论

这些都是我用来学习Vue.js甚至Nuxt的资源。每种资源本身都很棒，当你一起使用它们时，它们会更加强大。

在你为一些资源付费之前，我强烈建议你尝试免费的。有时候，你可以用免费资源学到足够的东西。如果你认为你需要更多，你也可以支付高级服务。

也就是说，如果你知道其他的资源，可以在评论中自由地提出来。
