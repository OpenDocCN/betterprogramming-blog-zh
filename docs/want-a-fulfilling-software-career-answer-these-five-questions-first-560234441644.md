# 想要充实的软件职业吗？先回答这五个问题

## 准确地根据你想要的来定义你的职业道路

![](img/267fa09f59da189a60e3a31ba8ace285.png)

我经常看到人们充满热情和兴奋地进入软件行业以做大。但很快，他们就成为了办公室文化日常琐事的牺牲品，失去了最初梦想的踪迹。最终，他们会走上一条便捷的道路，效仿其他人在工作场所的做法。

而当你随大流的时候，你也会成为人群的一部分。你会失去你的个性。因此，你不得不告别在行业中开拓自己的梦想。

然而，永远记住，你的软件生涯是一个独特的旅程，它将塑造你今天的样子和明天的样子。你需要思考你的人生历程，在你每天所做的事情中找到满足感和成功。

利用我在软件行业的经验，我编制了这个问题列表供您考虑，这将有助于引导您的软件职业朝着正确的方向发展。这些决定并不简单，但却很重要。

如果你想在工作中获得可持续发展，在生活中获得快乐，你需要花时间找到这些问题的诚实而准确的答案。

# 1.学习哪种编程语言、工具或技术

要回答这个问题，你能做的最重要的事情之一就是从你的优势开始。你最精通的语言、工具或技术是什么？或者说，如果你是初学者，那么哪一个最让你着迷？

如果你不确定学习哪种语言、工具或技术，请随时向你的前辈和导师寻求建议。有许多面向软件工程师的在线论坛和社区，在那里你可以讨论你所关心的问题。

我并不是建议你在职业生涯中只需要学习一种编程语言。但是当你开始的时候，你必须坚持学习一种特定的语言来学习编程的基础。一旦你对基础知识有了信心，你就可以通过学习更多的语言来扩展你的知识面。

我在软件工程师中看到的一个常见的愚蠢行为是，他们试图跟随潮流，根据他们在市场上的宣传从一种语言或工具跳到另一种语言或工具。他们在失去兴趣之前学习一项技术的点点滴滴，然后在市场中挑选下一个炒作对象。

但是这并不是在你的软件职业生涯中取得进步的最好方式。你的学习应该基于你的项目要求什么或者什么语言最让你着迷；而不是被周围人认为的时髦。

# 2.选择什么职业道路

![](img/4b7a51d08955f61e5f191569dc0f87dc.png)

照片由 Pixabay 从 [Pexels](https://www.pexels.com/photo/close-up-view-of-system-hacking-in-a-monitor-5380664/?utm_content=attributionCopyText&utm_medium=referral&utm_source=pexels)

如果你想有一个长久而充实的职业生涯，你需要扮演最能激励你并让你满意的角色。因此，在最初的几年后，你需要计划你想在你的长期职业生涯中扮演什么样的角色。

在决定你未来的角色时，不要按照建立在你的假设上的看法行事。尽职调查。与已经在扮演该角色的人交谈。收集所需的事实，并对其进行分析，以决定新职位是否能帮助你实现目标。

几年前，我的一个队友想成为产品负责人。当我询问原因时，她说这似乎是一个容易得多的角色。据她说，首席助理只是接听电话、与人交谈和撰写文件。他们不需要承担她作为程序员所面临的任何头痛问题。

我知道她对目前的角色不满意，想要一个出路。但是，在这个过程中，她认为 PO 角色可能是基于她在日常互动中所看到的解决方案。为了帮助她，我给她介绍了几个 POs 来讨论和纠正她的事实。

重点是，请不要基于自己的假设，通过转换角色，做一些完全陌生和意想不到的事情来惩罚自己。这会让你的生活更加痛苦。

向他人寻求建议时，广撒网——与开发人员、领导、测试人员、项目经理等交谈。一旦你知道哪种职业道路适合你的需要和能力，就去寻找那些道路上的人。

和他们谈谈他们的责任。他们是喜欢做这份工作，还是会去做别的事情？询问他们是如何决定选择这条路而不是别的。询问人们的职业道路，看他们是否会向像你这样的人推荐。

# 3.为你的事业付出了多少努力

你不必为了成功而在一天中工作最长时间。我也不相信任何人可以永远每天工作两小时而茁壮成长。你当然可以选择每天工作几个小时，但是你必须相应地调整你的期望。

在工作中付出 200%的努力是完全没问题的，同时，如果你决定享受生活，这也是可以接受的。你不必把醒着的每一个小时都花在电脑前。

这取决于你的决定，你想从你的职业生涯中得到什么，重要的是采取相应的行动来实现你的目标。

如果你渴望尽可能快地成长，并且对你所做的事情充满激情，那么每周超过 60 小时的工作可能会帮助你更快地在竞争中领先。然而，如果你的生活中有其他重要的事件或人，你可能会优先考虑花更多的时间和他们在一起，而不是经常在办公室过夜。

最终，你目前的人生阶段是这个问题的重要决定因素。几年后，随着责任的演变，你的决定可能会改变。并且，在你人生的那个阶段重新评估你的选择是正确的。

确保你的努力符合你的期望，这能保证你的职业生涯有所成就。

# 4.你希望自己的职业发展有多快

在你看来，事业成功是什么样子的？不仅如此，人们会如何看待或认可你是一个成功的人呢？

如果你有现实的想法，知道什么会更激励你——在职业方面——并激励你更加努力地实现你想要的，那会有所帮助。

我见过一些人在职业生涯中达到一定的水平，并希望继续担任相同的角色。他们不指望他们的称号会有更多的提高。有些人甚至拒绝进一步的晋升。他们对自己已经取得的成就和当时正在做的事情感到满意。

遇到这样的人感觉太不现实了——满足于生活中的一切。

这并不意味着你必须永远扮演同样的角色。你可以随时决定以后改变你的成长速度。关键是，你控制你的成长，而不是你的公司或社会设定的标准。

没有人能告诉你你应该做什么。如果你想从事新的工作，走出你的舒适区，每年获得更多的知识——然后加快你的速度，不断成长。

如果你想要生活中的平均增长，或者如果别人继续比你增长得更快没关系，那就以适合你职业的速度稳定下来——这没什么错。

但是一定要想想什么对你有意义！请创建您的职业轨迹，并决定您希望以多快的速度穿越它。当你确定了自己的道路和步伐，你无疑会在前进的过程中找到满足感。

# 5.你在什么样的公司文化中茁壮成长

![](img/d1d99583ad41af13e387f3ecd15fba64.png)

来自[派克斯](https://www.pexels.com/photo/photo-of-people-doing-handshakes-3183197/?utm_content=attributionCopyText&utm_medium=referral&utm_source=pexels)的[派克斯](https://www.pexels.com/@fauxels?utm_content=attributionCopyText&utm_medium=referral&utm_source=pexels)拍摄的照片

谈到公司文化，每个人都有不同的偏好。如果你和大多数人一样，很有可能你喜欢处在一个充满友情、合作和支持的环境中。你希望能够感觉到你是比你自己更大的东西的一部分。

或者，你可能更喜欢鼓励员工之间高度健康竞争的公司。

也许你想在一家与技术发展保持同步并在日常工作中使用前沿技术的公司工作。大多数软件工程师选择为对新的、新鲜的想法开放的组织工作，人们被鼓励自由地分享他们的观点。

如果你知道你在什么样的公司文化中茁壮成长，这将大大有助于确保你的职业道路包括在非常适合你的组织中的机会。

同时，永远记住，你是你工作环境的一部分。你对建立健康文化的贡献，就像找到一家拥有符合你气质的优秀工作文化的公司一样意义重大。因此，努力创造一个适合你的氛围，为你的其他队友提供最好的服务。

# 最后的想法

这是你的事业和人生。你有权决定你想从中得到什么。

当你违背自己的意愿听从别人的建议，或者盲目听从别人的建议以获得非凡的成功时，问题就开始了。

同时，我不会评判任何建议是好是坏。每个人都根据自己的经验和情况提出自己的观点。你的经历与众不同。

因此，根据你对职业的期望，你是最能判断什么适合你的人。听听大家的建议，结合自己的情况进行分析，自己决定要遵循哪些。最终，你的职业是你的责任。

感谢阅读这篇文章。我希望你对这些问题的回答能在你做出职业选择时带来清晰。
