我只是一个希望过上更好生活的人。然后我就成了一个梦想成为开发者的人。一晃四年过去了，我现在和其他高级开发人员一起工作。谁能想到呢？

然而，障碍总是存在的。下面的技巧将帮助你作为一名自学成才的开发人员驾驭它们并生存下来。

# 了解基本知识

> "有了好的基础，你会有无穷无尽的选择."

专注于建立一个坚实的基础——即使这意味着一遍又一遍地重复基础知识，直到那个学科开始。

把它想象成一项投资，复利的力量就在其中。起初你看不到它，但是通过那些熬夜和重复的项目，你很快就能开发出一种编程肌肉记忆。

直到它成为你的常态的一部分，没有比坚持编程更好的掌握编程的方法了。没有捷径，没有甜言蜜语。纯粹的努力工作和决心会给你每天的开发者生活提供燃料。

当你开始的时候，你能做的最聪明的事情是选择一种可靠的编程语言。它可以是 Java、C#、Python、JavaScript、PHP 或 Ruby on Rails。只要选择一个并专注于它，直到你看到那个模式。每个人都称之为“像程序员一样思考”的思维模式。

一旦你能够适应这种思维方式，你就可以采取下一步，开始寻找下一种编程语言。事实是，你不需要花那么多时间来选择你的第一语言。你等待的时间越长，迈出第一步就越难。

所以选择一个，投入其中，全力以赴。如果你重复这样做，有一天你会惊讶地发现自己成长了这么多。不要给自己太大压力。一步一步来，一天一天来。

专注于自己。花多长时间不重要。这是一场马拉松，而不是短跑。

# 多听少说

这个世界已经很嘈杂了。人们一直在谈论——大多是关于他们一厢情愿的想法或他们没有做的事情，这就是为什么他们都一样。

尝试相反的方法。

多听少说，多活在梦想中，冒险而不是稳扎稳打。生活应该按照你的方式生活。

所以，尽你所能地高飞吧——不是为了给任何人留下深刻印象，而是为了满足你充实生活的愿望。

当你开始真正倾听时，你会学到很多东西。你会理解很多，因为你会开始多用脑，而不是用嘴。

我的第一份工作已经做了三年了。我倾听，我观察，我观察我的同事，同时尽可能多的理解他们。一开始，这不是我的本意。他们都很吓人。他们随着节奏跳舞，我被震撼了。这让我更加热爱这份工作。我决定拥抱它。

我学到了很多，因为我成为了一个好的倾听者。我也开始少说话了。我变得不仅擅长观察人们如何移动，而且更擅长观察他们如何思考。

我成为了一名成功的自学成才的开发人员，过着我曾经认为不可能的生活。

# 保持理智

要想成功，你需要努力。但是就像在所有的运动中一样，你不能只打前锋。你还必须建立一个坚实的防御，以确保你不断前进。

道路是艰难的，所以你必须学会如何平衡事情。无论你多么热爱自己的工作，都要确保自己保持理智。

我是吃了苦头才知道的。相信我，学习和工作已经很难了。你不想让你的精神状态恶化。因此，即使只是锻炼 30 分钟或在公园散步，甚至只是把自己从工作站拉出来，也会有很大的不同。

可能发生的最糟糕的事情是牺牲几个月或一年的努力，只是意识到这太多了，你不能再做了。

一年每天学习 12 个小时并不能保证你能得到这份工作。

这既不健康也不可持续。如果你想从事这个职业，那就从长计议。这是一个过程。在我工作到第二年之前，一切都没有任何意义。这不是一夜成名的事情。如果那是你正在寻找的，那么不要浪费你的时间。

然而，这个职业改变了生活。几年前我处于人生的最低谷，成为一名自学成才的开发者让我找回了人生。你只需要以正确的方式去做，照顾好自己。

# 建立联系

诀窍是与其他开发者建立联系。

刚开始的时候，我不认识其他自学成才的开发者。我孤立自己，试图保持专注，但这是一条*孤独的*路*。*

作为人类，渴望陪伴是我们的天性。我们想要归属——不管我们如何否认。

但是仅仅为了拥有关系而试图建立关系也是不健康的。那么，对与错的界限在哪里？我还在学习中，但是从经验来看，和你走在同一条路上的人通常也会出人意料地支持你。

为了生存，你需要找到你的人，你的团队，那些懂代码的同伴。当你试图解释你所做的事情时，你的家人可能会觉得你很奇怪，但那些陌生人会把你视为明星——也试图通过编程来理解这个世界的人。

交朋友，建立联系，联系你钦佩的其他开发人员。找到你的背包。
