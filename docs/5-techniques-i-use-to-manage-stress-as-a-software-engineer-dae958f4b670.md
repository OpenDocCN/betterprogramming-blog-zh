# 作为一名软件工程师，我用来管理压力的 5 个技巧

## 你(很可能)不是和尚。压力是你生活的一部分

![](img/24962b8e02cc9fd3c26794982d34ee72.png)

普里西拉·杜·普里兹在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片。

我们经常面对它——在工作中，在家里，在各种生活环境中。如果我们不知道如何正确管理，我们可能会在焦虑中挣扎，在工作中效率低下。

以下是我用来管理压力的五种技巧。

# 1.晨报

这是我已经用了三年多的一种技术，它是一种神奇的黄金。

每天早上，我写两件事:

1.  大脑转储——所有我想写下来的想法、感觉和经历。
2.  今天的目标——无论如何，我今天必须实现的一个主要目标。

我每天早上做 5-10 分钟，到最后，我感觉完全不同了。我的心灵很平静。

如果你反思并写下你脑海中的一切(你所强调的)，你将会减少一整天的拖延，减少对你所面临的问题*的思考。*

我建议从“大脑倾倒”式的日志开始，每周早上写三次，看看效果如何。那么你可以每天都这样做。

# 2.步行

当我感到有压力时(例如，我找不到某个编码问题的解决方案)，我经常去散步 15 到 20 分钟，解决方案经常会在我的脑海中出现。我不带耳机。只是简单的散步。纯粹的魔法。

我注意到，当我做其他事情时，比如在休息时间看 YouTube 上的视频，几个小时后我会感到更有压力和焦虑。还有，在大自然(森林/低地水)走了一圈，感觉比其他地方都好。

所以试试对你有效的方法。

# 3.右侧休息

以前，我习惯戴上耳机，连续编码八个小时——几乎是连续的。但我不再那样做了。在这样的“艰苦工作”会议之后，我感到疲倦和压力更大。现在我以另一种方式运作。

当我们努力工作，然后休息时，我们的大脑工作得最好。而且也不是努力几个小时然后休息几天。它是关于努力 50-90 分钟，然后休息 15-22 分钟。

但这些并不像人们习惯的那样是“正常的”休息时间(例如，查看 insta gram/脸书或阅读新闻)。我说的是“正确的”休息。这些包括完全相反的活动。

在这样的休息时间，我经常做三件事:

1.  水合物
2.  伸展
3.  步行

我注意到，通过做这三件事，我感觉更放松，更专注，而且我可以工作 8 个多小时，最终不会感觉像废物(脑雾)。

所以在你的日常工作中加入休息时间。这项技术是编码人员的生产力瑰宝。

# 4.飞行模式

我曾经认为手机和电脑上的通知无关紧要。是的，它们在某种程度上分散了我的注意力，但这并不严重。

我错了。

每次通知出现在你的屏幕上时都要检查，这形成了一个坏习惯，叫做“害怕错过”我开始觉得我需要知道发生了什么。每当一个通知出现，我会立即点击它，停止我正在做的任何事情。

现在，我会关掉所有的通知四个小时，这样就没人能让我分心了。在那段时间里，我经常完成一天的大目标。

起初，这很难，因为我一直想知道世界上发生了什么。但是我注意到我不再担心它了。现在我感觉更专注了，也不那么焦虑了。有趣的是，这样一件简单的事情如何帮助我管理我的压力。

所以关掉通知，告诉人们不要在工作时间用不重要的事情分散你的注意力。

# 5.干净的桌子

不是显而易见的，但清晰的头脑是与清晰的环境相联系的。

我注意到当我有一张凌乱的桌子时，我很难找到解决编码问题的方法。这需要更多的时间，我对此感到压力更大。但是当我清理它的时候，我看到压力消失了。环境不再让我分心。我的头脑很清楚。

这个技巧花了我一分钟，我觉得从中受益匪浅。

每天早上或睡觉前清理你的桌子。
