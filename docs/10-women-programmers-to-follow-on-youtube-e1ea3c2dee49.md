## 有趣的编程内容、教程和技巧是由杰出的女性在 YouTube 上创作的

编程是这十年来最受欢迎的科目之一。我尽力寻找女性编程的渠道，但他们中的许多人只做关于技术的内容。几乎没有女性节目相关的频道。

据统计，编程领域由男性主导。然而，有越来越多的女性爱上了编程，她们作为榜样代表着世界加入 STEM(科学、技术、工程和数学)领域。

让我们看看 YouTube 上的一些女性程序员:

# 1) [琳达·柳卡斯——你好鲁比](https://www.youtube.com/channel/UCv_iIb0G-4JyFfm7IvqMk8Q)(327 万订户)

用简单易懂的方式教孩子(有时是成年人)编程。

Linda Liukas 是一位教育儿童编程的教育家。她把编程和孩子们的游戏时间联系起来。她就是这样一个甜美的教育家，总是面带微笑，向世界传递通俗易懂的编程理念。

看看她的 TedX 演讲。我想知道为什么观众没有被她的热情和积极性所鼓舞。

# 2) [井上真子](https://www.youtube.com/channel/UCEDkO7wshcDZ7UZo17rPkzQ) (48 万订户)

*通过浏览技术、职业建议和生活方式，帮助人们在科技行业找到自己的路。*

Mayuko 是著名的软件工程师 YouTubers 之一。她是一名 iOS 开发者，也是[汉科学院](https://academy.hanko.tech/)的创始人。她分享了自己在科技行业的经历。她的视频内容主要关注技术、职业和程序员的生活。

# 3) [蒂芙尼— Tiff In Tech](https://www.youtube.com/user/charnelle6) (12 万订户)

*分享技术、时尚和商业领域的有用技巧、动机和故事。*

Tiffany 是全栈开发者。她在模特和时尚行业开始了她的职业生涯。她帮助人们了解科技能提供什么。在科技频道的 Tiff 上，你会发现从科技到时尚到商业的各种话题。

# 4) [凯瑟琳·霍奇——布朗迪字节](https://www.youtube.com/c/KathrynHodgeblondiebytes/)(6.79 万订户)

*很多很多编程语言等方面的教程。*

Kathryn Hodge 分享了很多 Java、Javascript、Bootstrap、Python、AWS、GitHub、Node.js 等教程。她还涵盖了编程中的许多不同主题，比如代码审查和 vlogs。

# 5) [玛莎·兹韦列娃——打破陈规者](https://www.youtube.com/c/CodingBlonde)(4.48 万订户)

谈论科技职业、个人成长、心理健康以及女性在 STEM 领域打破陈规的运动。

玛莎·兹韦列娃是谷歌的前雇员。这个 YouTube 频道之前被命名为“编码金发女郎”Masha Zvereva 更名为该频道，提供更强大的内容和可操作的项目，以支持科学、技术、工程和数学领域的女性。这个频道有很多关于职业、编程、面试技巧、个人成长和心理健康的建议。

巴鲁的生活

# 6) [巴鲁·尤达辛纳——巴鲁的生活](https://www.youtube.com/c/LifeofLuba)(10.4 万订户)

*一个寻找技术、生活窍门、职业建议、企业家精神、动力和生活方式的地方。*

巴鲁·尤达辛纳在过去的六年里为 Airbnb 和 Yelp 等公司从事科技工作。她在频道上谈论编程、硅谷生活、生产力技巧、职业技巧、视频博客以及更多内容。

# 7) [布科拉](https://www.youtube.com/c/Bukola1)(23.4 万用户)

*进入科技行业的技巧、程序员的博客、个人理财和生产力技巧。*

布科拉是纽约的一名软件工程师。她为有兴趣加入科技行业的人分享了一些建议。有许多有趣的视频，如“编写一个 Tinder bot”和“用 YouTube 自动化 Spotify”

# 8)[Ania kubów](https://www.youtube.com/c/AniaKub%C3%B3w)(1.18 万订户)

*关于 JavaScript、React、React-Native、HTML、CSS、Node.js、Express、GraphQL、无服务器函数、数据库的教程。*

Ania Kubow 出生在波兰，在阿联酋长大。她毕业于迪拜学院，主修数学、物理和设计技术，后来到英国上大学。Ania Kubow 是一名软件开发人员。她喜欢通过构建复古游戏和令人敬畏的项目来教授代码。她也是 FreeCodeCamp 核心团队的一员。

# 9)[RealToughCandy](https://www.youtube.com/c/RealToughCandy)(4640 万订户)

*主要是关于 HTML、CSS & JavaScript、React、PHP、Linux + LAMP 栈、MERN 栈等技术和语言。*

她是一名讲师、YouTuber 用户、软件开发人员、作家和企业家。她还是 realtoughcandy.io 的创始人，该网站帮助人们提高水平并开始编程。

# 10) [玛雅·贝洛](https://www.youtube.com/c/MayaBello)(87000 订户)

*从《软件工程师的一天》到《编码教程》(大部分是 Python 语言)。*

Maya Bello 是一名软件工程师，她用 Python 创建了关于技术和编码教程的内容。

**注意**:通道的排列并不代表什么。
