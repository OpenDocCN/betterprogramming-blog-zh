我最近在开发名为“Pocket CPR”的Apple Watch应用程序，我注意到缺少涵盖SwiftUI功能的教程，例如专门针对Apple Watch的视图。

使用列表视图

SwiftUI中的List视图在垂直列中显示可滚动数据的列表。

您可以使用.listStyle修饰符向列表中添加更多的定制。

在这种情况下，我通过添加.listStyle(CarouselListStyle())创建了一个Carasouel视图。Carasouel列表视图的基本设置如下:

List {
                
  Text("One")
  Text("Two")
  Text("Three")
  Text("Four")
  Text("Five")
                
}.listStyle(CarouselListStyle())
