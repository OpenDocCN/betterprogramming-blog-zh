# 现在是学习的时候了

随着比特币和以太坊的价格创下历史新高，围绕区块链和 web3 开发的炒作也在升温。我在 9 月份接触了 web3，从那以后就迷上了它，学习了 crypto、区块链、NFTs 等等。通过这些教程，你可以学会做任何事情，从了解区块链技术的基本原理到编写智能合约和加密货币，到建立完整的 NFT 市场和代币交易所。如果你对 web3 开发感兴趣，我强烈推荐这些资源来帮助你开始。

我与这些产品或课程没有任何关系。这些只是我用来学习 web3 和区块链开发的资源。

# #1 —免费代码营

我一开始对区块链一无所知。幸运的是，我偶然发现了由 Patrick Collins 在 freecodecamp.org 举办的[可靠性、区块链和智能合约课程——从初级到专家 Python](https://www.youtube.com/watch?v=M576WGiDBdQ) 。Patrick 是一名优秀的教师，他在这门免费课程中讲授了超过 16 个小时的内容。检查他的媒体内容；两者都是不可思议的价值。

在最初的几个小时内，您将了解区块链，建立一个开发环境，并为开发智能合同打下基础。这个课程对我来说是完美的，因为我之前没有任何知识。我喜欢这门课程的另一个原因是，它使用 Python 来讲述材料。我看过的很多其他 web3 相关教程都使用了 JavaScript。如果你像我一样从零开始，并且有一点 Python 的经验，就从这个免费的在线课程开始吧。

# #5 —开发工具

除了视频资源之外，这里还有一些特定工具的资源，如果你深入 Web3 开发的话，你将会学到所有的东西。Web3 技术发展很快，但这些工具很受欢迎，而且背后有很棒的社区:

*   [Metamask](https://metamask.io/) —以太坊最流行的基于浏览器的加密钱包。也可轻松与 Moralis.io 集成。
*   [Solidity](https://docs.soliditylang.org/en/v0.8.10/) —一种面向对象的高级编程语言，用于实现智能合约。这是你将用来创建自己的硬币和 NFT。
*   Remix —一个用于编写和测试 Solidity 智能合约的在线 IDE。当你开始学习稳固性时，这个工具是非常神奇的。
*   [Truffle Suite](https://www.trufflesuite.com/) —像 Truffle 和 Ganache 这样的工具用来开发本地测试区块链，这样你就可以在推向生产之前在本地部署你的代码。
*   [Web3.js](https://web3js.readthedocs.io/en/v1.5.2/) —用于与以太坊客户端和智能合约交互的 JavaScript 库。
