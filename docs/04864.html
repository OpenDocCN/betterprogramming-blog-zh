了解此版本的新增功能和影响

React Native 0.63 RC0在几周前发布，为我们带来了一些真正令人兴奋的新功能。

本色

这里有两个新东西:PlatformColor和DynamicColor。

PlatformColor允许用户在iOS或Android上使用设备的原生颜色。

DynamicColor 允许用户根据手机的外观模式(亮模式/暗模式)配置颜色。

下面是一个如何结合使用这两者的示例:

<View
  style={{
    backgroundColor: DynamicColorIOS({
      light: PlatformColor("systemBlueColor"),
      dark: PlatformColor("systemRedColor"),
    }),
  }}
/>

可压制的

这个新版本中有一个新的组件Pressable。这个新组件旨在取代用于创建按钮的TouchableWithoutFeedback和TouchableOpacity。这里有一个例子:

<Pressable
  style={({ pressed }) => [
    {
      opacity: pressed ? 0.5 : 1,
      backgroundColor: pressed ? "red" : "orange",
    },
    styles.button,
    ,
  ]}
>
  <Text style={styles.buttonText}>Pressable</Text>
</Pressable>

我们不再需要包装和库存pressed状态。这是由组件处理的。
