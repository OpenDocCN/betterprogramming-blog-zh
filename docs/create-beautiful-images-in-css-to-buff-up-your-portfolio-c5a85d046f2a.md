## 云影

为了创建阴影，我们需要用额外的模糊来复制我们的云，把它涂成蓝色，然后缩小它。

![](img/72c83a224733eecf7493a10ec7dc7279.png)

带阴影的基础云

循序渐进:

*   用蓝色复制云彩。
*   添加模糊。
*   缩小它—变换:缩放(0.8)。
*   添加伪元素——两个白色圆圈来遮盖我们的阴影。
