# 从医学预科到计算机科学的过渡

在我大学的第一年，我在医学预科领域，专业未定。正如大多数人最初认为的那样，我曾认为生物很有趣，但在第一年一起经历了生物和化学之后，到了春季学期，我慢慢失去了兴趣。随着年龄的增长，我对成为一名医生有点兴趣，因为我会看一些电视节目，而且我对生物学感兴趣。想要帮助别人无疑是另一个原因，但是我的欲望在大一后的那个夏天发生了转变。我做过验光实习，这证实了我不再想从事医学，而是想探索一个不同的领域。

我决定转向计算机科学，因为我从小就热衷于解决问题和创造谜题。我的家人也非常支持我转到计算机科学的愿望，这也是一个激励因素。在参加了计算机科学 I 和 II 之后，我决定这个领域是我最想学习的领域，并以那种方式锻炼我的大脑。我的计算机科学助教是第一批支持我留在 CS 的导师之一。她鼓励我练习做黑客排名，这是练习技术面试问题，并继续申请实习，即使我当时是一名大二学生。我在课堂上也注意到，男生比女生多得多。

# 我的第一个技术实习和领导职位

![](img/f1cbdc87b37a85e471669c8a34120bcc.png)

由[克里斯多佛·高尔](https://unsplash.com/@cgower?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 拍摄的照片

大二结束时，我找到了一位来自西奈山健康医院的导师。他帮我联系了西奈山应用实验室(Mount Sinai App Lab)，这是医院里的一家技术创新型初创公司，在那里我可以作为数据分析师实习。这可能是我最大的学习经历之一，因为我能够获得使用 Python 预测 30 天内患者再次入院率的实践经验。我注意到在这次实习中，女性和男性的比例差不多，这和我在计算机科学课上看到的情况不同。

此外，我决定担任学校科技俱乐部(Tech @ LC)的主席，因为我想提高人们对计算机科学的认识，并激励像我一样的其他女性追求这一目标。作为技术俱乐部的主席是非常棒的，因为我给人们可以创造的项目做演示和演示。让来自实际技术行业的人加入进来，对我和许多成员来说也是一次大开眼界的经历。

# 快进到大学和 NBC 实习的第三年

![](img/303d69cd1cad9e813734b8aab7aeb01a.png)

在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上由 [Rodion Kutsaev](https://unsplash.com/@frostroomhead?utm_source=medium&utm_medium=referral) 拍照

在我大学的第三年，我仍然注意到班上的男生比女生多，尽管我确实注意到女生开始多于男生。大三的秋季学期是最紧张的时候，主要是因为我申请了许多不同的实习机会，希望能在夏天得到一份工作。

12 月是一个激动人心的时刻，因为我发现我将在 2019 年春天作为软件开发人员实习。在 NBC 实习是一个非常好的机会，因为我了解了创建全栈服务的所有软件开发过程，该服务将跟踪电视娱乐团队中每个工具发生的不同事件。在我的春季学期，我的移动和 app 编程课上也有一位教授不太鼓励我，说我的 CS 基础非常薄弱。这个时候冒名顶替综合征开始出现，因为他的课让我觉得自己在计算机科学方面不够好。但是实习证实了计算机科学是一个值得去的地方。

# 2019 年夏季

2019 年夏天是我最棒的学习经历之一，因为我已经从纽约搬到了夏洛特，我将为一家银行做软件开发。这是我第一次离开家，在一个不同的城市工作。在不同的城市实习帮助我在许多方面成长。我做了一个概念验证，证明一个软件比另一个好，能够将贷款处理从 30%增加到 50%。我在银行注意到，我的团队中男性比女性多得多，尽管女性毕业生的数量有所增加，这是我很高兴看到的。

我还在大三的时候申请成为了重写代码(RTC)的研究员，这是一个为女性科技工作者设立的非营利性组织。这是一次奇妙的经历，因为我能够和其他女大学生谈论她们在科技领域的经历，并询问与科技相关的问题。我鼓励其他女大学生作为会员或研究员加入重写代码的行列——这是一个很棒的社区！后来，我被选为夏洛特分部的中心领导，这很棒，因为我可以为其他人创造有趣的活动。

# 高三+格蕾丝·赫柏会议

![](img/12c3cb007327c1b7eb50b2cb4cb264a9.png)

由 [Samuel Pereira](https://unsplash.com/@samuelpereira?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

进入高年级时，我注意到我的计算机科学课程，如算法和计算理论，男女比例更接近 50:50，对此我很高兴。这次最有趣的经历是在佛罗里达州奥兰多的格蕾丝·赫柏会议(GHC)。由于霍普奖学金，我得以参加，在那里我花了 8 小时的时间去做志愿者，以换取一张免费的会议门票。这次会议是一个巨大的回报，因为我能够与许多其他人建立网络，听一些非常酷的演讲，并最终探索格蕾丝·赫柏各种各样的迪士尼公园。我会推荐任何有资格的人在未来几年申请。我还收到了银行的返聘通知，所以我非常高兴能在 2020 年以软件开发员的身份回到银行。

# 最后的收获和经验教训

我的高三在新冠肺炎结束，但回顾我作为一名女性技术人员的历程，我明白了机会总是在这个世界上，我只需要有力量去自己寻找它。通过经常与他人接触，与高年级学生和低年级学生建立联系，我建立了真实而持久的联系，我一直在前进，你们自己也应该这样做。我们需要更多的女性进入科技行业，所以你应该鼓励和激励你周围的人沉迷于科技世界。还有一个无限可能的世界有待发现！
