我最喜欢的React UI库的优缺点

今年年初，我决定将我的JavaScript技能提升到一个新的水平。厌倦了 jQuery ，我开始寻找更快、更现代、更可靠的 。

我选择了 React ，它在 Stack Overflow 的 2020开发者调查中被参与的65K开发者票选为最受喜爱和最想要的JavaScript框架。

关于React，我学到(也喜欢)的第一件事是，由于它的流行，你不需要重新发明轮子。有无数的UI套件为你准备好了。毫不奇怪，许多开发人员构建了React库，提供了几个组件来帮助您改进应用程序的UI。

在本文中，我将向您展示三个React UI库，根据我的经验，我会建议任何人从这三个库开始。

反应引导

react-bootstrap提供用世界上最流行的CSS框架构建的React组件: Bootstrap 。

“React-Bootstrap替换Bootstrap JavaScript。每个组件都是作为真正的React组件从头开始构建的，没有像jQuery这样不必要的依赖。作为最古老的React库之一，React-Bootstrap已经随着React一起发展壮大，成为您UI基础的绝佳选择。”— 反应引导

装置

要安装React引导程序，运行以下命令:

npm i react-bootstrap

蚂蚁设计

antd是一个React UI库，包含几个按照 Ant设计规范设计的高质量组件。

“Ant Design是一种企业级UI设计语言和React UI库，具有一组高质量的React组件，是企业最佳的React UI库之一。”— 蚂蚁设计

装置

要安装Ant Design ，运行以下命令:

npm install antd

定论

Ant Design是迄今为止我首选的React UI库。它有几十个组件，每个组件都有许多不同的属性。官方文档写得很好，很完整，并且充满了工作示例。根据我的经验，我认为antd是初学者的最佳选择。
