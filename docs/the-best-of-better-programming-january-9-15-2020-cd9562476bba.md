## JavaScript:

1.  [**Node . js 15**](https://medium.com/p/fc24e87e2590)中的新功能作者 [Jennifer Fu](https://medium.com/@jenniferfubook) —本周由 Jennifer Fu 带领我们，谈论 Node 15 中的新功能、新特性、错误修复等！
2.  [**React 的未来:服务器组件**](https://medium.com/p/90f6e3e97c8a) 作者 [Donovan So](https://medium.com/@donovanso) — Donovan 看看服务器端 React 会是什么样子。我自己对此也很兴奋。
3.  [**一种读取 Node.js 中文件的内存友好方式**](https://medium.com/p/a45ad0cc7bb6) 作者[Kasper Moskwiak](https://medium.com/@kaspermoskwiak)——Kasper 本周加入了时事通讯，教我们如何在没有内存泄漏的情况下读取 Node.js 中的千兆字节数据。好像有用！
4.  Michael Chi2021 年 React 学习路线图——最后，如果你刚刚开始学习 React，Michael Chi 将在今年教你如何开始以及如何成为 React 大师。

## 软件工程:

1.  [**4 种不同寻常的提高编程技能的方法**](https://medium.com/p/dbdc7864a5f4) 作者 [Szymon Adamiak](https://medium.com/@sadamiak) — Szymon 有一个有趣的提高编程技能的方法列表，这些方法可能与你想象的不同。我最喜欢的是第三条——限制自己。我有一些美好的编程记忆，在我下飞机的时候，我没有可靠的互联网，在事先没有任何想法的情况下，我做了一些很棒的东西。它通常只需要使用我的机器上的文件和大量的试验和错误，但我肯定在那次飞行中做得更好。
2.  [**5 个让你与众不同的非编码技能**](https://medium.com/p/a8f64d55a08e) 作者:[Fernando Doglio](https://medium.com/@deleteman123)——很容易认为成为一名优秀的开发人员就是写干净、简洁的代码，但其实远不止这些。费尔南多本周回来探索这一点。
3.  [**2021 年 PHP 还相关吗？**](https://medium.com/p/19580c75855) 由[布拉克·卡拉坎](https://medium.com/@burakkarakan)——“X 死了。X 万岁！”2021 年该不该学 PHP？当然可以！成千上万的公司使用它，社区很大，很活跃。Wordpress 仍然很庞大，为互联网的大部分提供动力，并且是建立在 PHP 之上的。不要沉迷于 TypeScript、Swift、Dart 或你的同事本周向你推销的任何东西。
4.  [**介绍和应对缓存挑战的策略**](https://medium.com/p/c619d51882c0) 作者 [Aastikta Sharma](https://medium.com/@aastiktasharma) —缓存是开发人员众所周知的棘手部分。我们中的许多人都遇到过这种情况，想知道为什么有些东西不能清除缓存，或者可能是在处理由于意外缓存数据引起的 bug。Aastikta 在这里教我们更多关于缓存的知识。
5.  [**理解工厂方法设计模式**](https://medium.com/p/f5ec631c99d8) 作者:[卡洛斯·卡瓦列罗](https://medium.com/@ccaballero) —卡洛斯本周回来教我们一个有用的设计模式——工厂。在您的项目或代码库中使用有意义吗？你是法官！

## 面试:

1.  [**如何在不到 10 分钟的时间内构建一个 LRU 缓存，100 行代码**](https://medium.com/p/fddad56d7af5) 作者[孙李贝蒂](https://medium.com/@sunnyb) —孙李本周在这里讲一个著名的面试问题，构建缓存。如果你正在为一个技术面试而学习，这可能值得一读！

## Python 和数据科学:

1.  [**Python 异步编程入门**](https://medium.com/p/2797129dd0b3) 作者:【崔永】 — Yong 本周回来谈论 Python 中的异步编程，你应该如何考虑它以及用 Python 进行异步编程的细节。
2.  [**Django、RabbitMQ 和 Celery 的分布式任务队列**](https://medium.com/p/703c7857fc17) 作者: [Louis de Bruijn](https://medium.com/@leddebruijn) — Louis 是本周新闻简报的新成员，谈论分布式系统以及如何在 Python、Django、RabbitMQ 和 Celery 中正确使用队列。

## 杂项:

1.  [**K8s:Vanilla All Way**](https://medium.com/p/9ba8c746e4f0)作者 [Luc Juggery](https://medium.com/@lucjuggery) — Luc 本周回来，谈论 Kubernetes 中的 Vanilla 堆栈，以及为什么它可能对您的 K8s 设置有意义。
2.  [**现在是 2021 年——这是你在艾伦·赫尔顿**](https://medium.com/p/73deeb0b1eb)的《技术 》中应该关注的地方——最后，艾伦本周回来谈论你的 API 标准化、一切自动化，以及总体上变得更加以 API 为中心。如果你正在偿还 API 技术债务，绝对值得一读。
