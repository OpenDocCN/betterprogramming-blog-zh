# 所以你想当经理？

## 在从工程师跳到经理之前，先回答这些问题

![](img/57a7a5a1787b6e0abc8192beb1bb7e67.png)

你受够了老板们对你颐指气使吗？持续的任务、可交付成果和时间表。难道现在不是你站出来自己做这项工作的时候了吗？

想想当你成为一名光彩照人的新经理时，你将拥有的金钱、名誉和影响力！看起来很容易，而且有很多额外津贴。

此外，你已经把所有这些工程术语锁起来了，对吗？既然你如此擅长你现在所做的事情，那么在公司阶梯上的下一步肯定会同样容易——只是更多相同的东西和更高的薪水。

不会那么难吧？

当一个人在当前职位上达到顶峰时，通常会想到管理。事实上，并不是所有优秀的工程师都能成为优秀的管理者。有些人就是不应该负责其他人类的产出。许多人都在努力理解和建立成为一名高效老板所需的不同技能。

因此，任何考虑寻求这一晋升的人都应该退后一步，回答几个关于他们认为成为一名经理是怎么回事的问题。

不仅仅是赚更多的钱。

是的，经理对决策有更大的影响力。经理通常会赚更多的钱。经理们还必须与人打交道而不是与计算机打交道，与感情打交道而不是与代码打交道，还必须适应在一大群人面前站着说话。

因此，如果你寻求职业发展，想从工程师一跃而下，确保你了解这个角色需要什么，以及作为一名新的工程经理你需要什么技能。

# 你能忍受自己的时间不属于自己吗？

当你步入管理岗位时，最难接受的事情之一就是你的时间不再属于你自己。集中注意力、集中精神、思考问题的时间已经过去了。有些时候，你会有喘息的空间去做一件事，有时是三件事。还会有更多的时候，没有空间或时间，需要立即做出决定。

作为一名经理，往往意味着你需要考虑行动和潜在后果，往往没有足够的时间来做出明智和深思熟虑的决定。

如果你非常依赖有喘息的空间来思考可能的后果，那么这种更快的决策速度一开始可能会很难。你不再在你的任务清单上工作，决定完成你面前任务的最佳方式。现在你要对整个团队的待办事项清单的输出负责，并对许多人正在努力创造的整体结果和产品负责。

这可能意味着更多的会议。虽然在商业世界中有很多会议遭到恶评的例子，但是对于一个优秀的经理来说，建立有效的会议技巧是更重要的技巧之一。

除了会议之外，还会有额外的日常开销，比如文书工作、假期审批、人力资源职能、评估，不要忘记，在所有这些额外工作之外，你还有一个老板，你要对他负责，他最有可能期待类似的职能工作成果。

任何新经理都需要快速提升时间管理和会议管理技能，以有效应对这些快速流动的激流。

# 你准备好成为一个外向的人了吗？

除非你已经是派对的灵魂人物，否则你的出现会影响到你周围的人。无论你多么不同意，向你报告的人内心深处都知道，有一天你可能不得不决定他们是否继续在那个组织谋生。

那是一个沉重的负担。

对于任何习惯于听耳机、独自专注于当前任务清单的人来说，这可能是一种粗鲁的文化冲击。然而，这不仅仅是站起来在一个人或一群人面前讲话。

作为一名经理，你的存在会影响讨论和态度。你的缺席也可以。然而，现在你既要处理个人对话，也要处理更大的以团队为中心的演示和讨论调节功能。这不是一个被动的角色。

管理者为团队定下基调。他们的态度和行为会影响团队中的每个人。处于权威的位置并不意味着你不能有糟糕的一天，而是意味着你应该知道你糟糕的一天会如何影响你周围的人。这就是为什么许多有经验的经理有一种神奇的能力，几乎在任何情况下都能保持冷静。剧透警告——这很难做到，需要大量的练习和耐心。

新任经理必须意识到，他们的情绪和行为超出了自己的承受能力，至少应该认识到他们的情绪会如何影响他人，即使他们仍在努力控制如何向他人表达这种情绪。

# 你考虑过竞选政治职位吗？

不，你不需要出去在别人的院子里张贴标语来争取一个工程经理的职位。然而，你需要认识到这个角色本身就有一定的政治水平。这不一定是负面的。

总的来说，至少在 2020 年初，政治越来越多地带有关于社会舆论的负面内涵。办公室政治在社会意识中也趋向于资产负债表的负面。这可能是由于公开传播的毒性故事和好莱坞对办公室政治的刻板印象，即通过各种喜剧和负面描绘而退化。

然而，这也不全是坏事。在许多组织中存在不利的表现，但是如果正确实施，管理的艺术和科学可以产生许多积极的结果。治国之道的核心是妥协，这对于任何管理者来说都是一项极其宝贵的技能。

办公室政治的其他好处包括让人们负起责任，奖励有建设性的行为。它可以培养更多的生产和沟通关系，更快的决策和整体组织凝聚力。

无论政治文化趋势是积极的、消极的，还是在整个范围内来回摇摆，任何希望进入管理者角色的人都将在一个新的舞台上工作。任何新的管理者都应该知道，在他们的新角色中，需要学习新的规则和方法来完成工作。

# 你的同理心水平有多高？

作为一名工程师，当一台计算机、一段代码或一个设备没有按照你的要求运行时，你拥有所有的权力。你可以放心地让它做你想做的事情，而不用担心它会有什么反应。作为一个经理，你没有那个闲心。作为一名经理，你现在面对的不是一块屈从于你意志的硅片，而是活生生的、会呼吸的、有感情的人。

当然，只要没有人指出他们最喜欢的星际小说不太现实，不如其他一些对外星人友好的星际飞船、电话亭或宏大的龙骑史诗，工程师们通常都很平静。

然而，即使是最理智、最理性、最冷静的一群人，也会有冲突。还有一种东西，人们称之为一种生活，这种生活发生在一天八九个小时之外，你们在一起的时间很短。去想想。

要成为一名好的领导者，你需要练习与重要的另一半建立成功关系所需的技能:成为一名好的倾听者。培养直接的个人纽带。愿意从他们的角度看问题。过度沟通。所有这些你在工程学校的课程中看不到的软技能。

没关系——你可以随时学习这些技能。也有很多关于职场同理心的质量管理培训课程。

作为一名新经理，不要犹豫要求培训或进行知识积累，以提高这方面的技能。他们对你的团队的成功和幸福至关重要。

# 你愿意为你参加派对的权利而战吗？

进入这个角色，你现在是老板了。虽然这意味着你可以直接影响你的团队每天的文化和乐趣，但这也意味着你现在要为发生的冲突和分歧负责。

虽然我们都想在聚会上玩得开心，但有时我们不得不应对到达那里的战斗。

作为一名工程师，你可以选择战斗，也可以选择不战斗。作为一名经理，你仍然可以做出选择，但是你失去了离开战斗的能力。当然——根据具体情况，你可以把它踢给职位更高的人或人力资源部的人。现在是你决定升级的责任了。你也有责任确保正确移交给你的上级。

管理者不能把头埋在沙子里，希望事情不了了之。

对于你这个角色来说，这可能不是一场重要的战斗。是的，软件工程师都会为整个团队应该使用什么样的编码标准而大动干戈，但是在担任经理角色之后，就很难对制表符和空格感到兴奋了。当您不是自己编写代码，而是需要更多地关注产品的最终功能，而不是代码中的简单间隔时，就会出现这种差距。

作为一名新经理，你将对这些斗争及其结果负责，无论你是作为战斗人员还是调解人参与其中。无论如何，你都不能忽视它们，而只是去别处找乐子。

# 你象棋下得怎么样？

有人想从工程师一跃成为工程经理有很多原因。更多的钱和更多的责任是两个关键驱动因素。然而，最大的机会之一是在决策过程中有更大的发言权。

作为一名工程师，大多数决策都围绕着眼前的职能工作。这些决定是战术性的，处理这里的*和现在的*的和*。我们如何解决这个问题？什么工具和库是完成这项任务的最佳方式？*

作为一名经理，前景远远超出了这里的*和现在的*以及对未来发展方向和潜在需求的总体认识。这就是通常所说的战略思维。

工程师关注的是如何实现路线图上的近期项目，而工程经理需要努力创建路线图。这需要从更大的角度来看待成功经营企业所涉及的组织、产品和客户。

除了额外的商业头脑，经理们还需要从战略的角度考虑他们所管理的团队。团队的士气如何？团队要成功，必须有哪些过程？当每个团队成员每天下班回家时，他们的文化和经历如何为他们提供最大的满足感？谁准备好站出来接受团队将面临的下一个重大挑战？

作为一名工程师，一个人开始关注他们周围的树，而在这个角色中的成长有助于建立一种理解，即那棵树是一个更大的森林的一部分。作为一名经理，不仅应该了解更广阔的森林，还应该了解其他森林、其他团队，它们会影响这个团队的人员和工作。战略思维是任何新任经理都应该掌握的一项非常重要的技能。

在商业世界的许多情况下，人们会认为我们必须走上一条向上的道路。企业界的经典形象是梯子。不幸的是，这导致许多人认为，仅仅因为某人在他们正在做的工作中表现出色，他们也将善于在公司的阶梯上更上一层楼。

这导致许多不具备管理技能或能力的人被推到这个角色上。不管这个人是否为这种改变所要求的日常职责和责任的巨大差异做好了准备，这种改变都会实施。

然而，当许多人被提供或被推到这个角色时，其他人寻找它，并寻找他们当前角色没有提供的额外的东西。我写这篇文章是为了这两个群体。

任何人都可以扮演经理的角色。只有那些愿意理解和接受这些不同的行为和技能的人，才会在个人和职业上获得成功和满足。正是这些人将拥有最强的团队和最出色的成绩。

对于那些即将成为经理的人，我向你们致敬。祝你好运！
