# 为什么没有一种方法在软件工程中工作得很好？

> 原文：<https://betterprogramming.pub/why-none-of-the-methodologies-works-well-for-software-engineering-b619c61d8e22>

## 与制造业不同，软件工程世界远非完美

![](img/d3783cc2fea01850c2bc0ab79ebafca7.png)

莱尼·屈尼在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

大多数经理(和一些员工)的梦想是拥有一个完善的软件工厂。

一条长长的(虚拟的)传送带，客户的需求从这里进来，这些需求被分解、分析。

工程师团队快速移动、敲击、点击各个位置，在传送带的末端，你就有了一个功能齐全、完美的产品。

那不是很好吗？

隐藏在这种欲望背后的问题是，“让我们达到这种制造水平精度的完美方法在哪里？”

显而易见的答案是——没有这样的方法，将来也不会有。句号。

答案超级简单。更复杂的是“为什么？”。我认为有两三个基本问题使得我们不可能想出如此可重复的东西。

# 每个产品的独特性

![](img/f16f304b0198590dbe74d5fdd9aeca68.png)

照片由[阿尔伯特·波杰斯](https://unsplash.com/@albertpotjes?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 拍摄

我一生中的大部分时间都在研究安全产品。因此，当我从一家公司转到另一家公司时，很多事情听起来都很熟悉——用户管理、策略、事件、控制所有这些的 UI、部署等等。感觉都很熟悉，但又总是不一样。

技术的选择是不同的(语言、框架、库)。它们的组合方式是不同的。边缘情况是不同的。特点不一样。诸如此类。

与体力劳动做个比较:你能想象如果你每次都是用完全不同的汽车零件手工组装一辆汽车，而汽车设计却是快速闪现的吗？

当你在这辆车上工作时，其他团队成员改变了一些零件。每个人都在非常宽松地使用 ducktape 和 WD-40，只是为了确保它能够运行，不会崩溃。

我刚刚描述了制造过程的一个反命题(所有部件都受到高度控制，设计不会在飞行中改变)。

# 工程师是不可互换的

![](img/51e79fcef6f6ab7c8b2c558a7f609d3d.png)

菲尔·肖在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

因此，制造过程首先需要的是可重复的汽车零件。它需要的另一件事是可重复的人。

如果你可以很容易地用另一个工人取代一个工人(也许只需要最低限度的在职培训)，你就可以拥有一个工厂。

但是，如果你所有的工人都有令人难以置信的技能组合和非常不同的生产力，经营一家工厂将是非常困难的。

我描述的正是软件工程师的情况。一个人是后端专家，在。NET，还有一个花了一辈子在汇编里摆弄嵌入式软件和梦想，第三个连一个 Hello World 都写不出来。因此，随机改变他们正在做的事情并期望可预测的结果是不可能的。

顺便说一句。我所说的也适用于质量保证。有一个人对思考所有边缘案例感到惊讶。另一个是自动化，第三个几乎不能执行别人写的测试。同样的事情，他们不容易互换。

# 一次行动的责任

制造业需要更深层次的专业化和分工。你可以让一个人负责一些琐碎的任务。

同样，在软件领域，我们有几乎相反的情况。首先，需要完成的所有任务没有明确的划分。有些任务不容易分成更简单的任务。人们拥有各种各样的任务来实现“制造”过程中的某一大块。

# 这个世界并不完美

![](img/3ac331465e0c7e0ec857a4b6448532b7.png)

里卡多·维亚纳在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

最后一件事。当你走向制造业的高端(例如，中央处理器或太空望远镜)，你会发现它们是在尽可能接近理想的条件下制造的。例如，制造 CPU 的洁净室比无菌的医院手术室少 1000 个空气中的微粒。

软件工程呢？咳咳……我不是提议把工程师放在一个洁净室(或真空室)里。我是说，工程师(软件制造过程中最关键的工具)工作在一个令人难以置信的(固有的)混乱环境中。

顺便说一句。我说的不是物理环境。然而，我很想和一个提出开放空间办公室想法的人聊几句。我说的更多的是人际关系、变向、政治、炒作等乱七八糟的环境。

# 摘要

我想我的博客中有很大一部分都是关于这个主题的:如何在一个不完美的世界中以一种好的方式运作？

我写的不是迷人的、挑衅的或病毒式的东西。更多的只是面包和黄油之类的东西。

许多文章可以总结为“这是人们认为它在完美世界中的工作方式。这就是它在一个不完美的世界中是如何工作的。以下是如何让它变得更理智的方法”。
