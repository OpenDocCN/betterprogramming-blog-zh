# 公司应该采取 6 项行动来支持科技行业的女性

## 国际妇女节即将到来。让我们期待一个改变

![](img/175aa2c3619e563270a19c4ca6152818.png)

作为一名技术领域女性的倡导者——我自己也是一名技术领域女性——公司和领导者曾数次向我寻求想法和最佳实践，以留住和提升该行业的女性人才。很高兴看到这一点，因为如果公司只关注聘用女性人才，而不花同样多的精力支持她们，创造一个让她们茁壮成长的环境，那就像用漏桶收集水一样。

事实是，科技行业的女性与其他行业的女性并没有什么不同。然而，众所周知，科技行业存在巨大的性别差距，许多女性在职业生涯中期退出科技行业，因为她们没有得到所需的支持。我和一个在人力资源部门工作的朋友聊过。众所周知，人力资源部拥有最多的女性人才。她想要的东西和我想要的一样，但她说她没有太多困难就能得到她想要的东西，在事业上茁壮成长。那么，为什么许多从事技术工作的女性说她们面临很多挫折，努力发展自己的职业生涯并留在职场中呢？

我想了想，这里有一个可能的原因:是因为技术领域的女性不多。当我们提出要求，谈论我们需要什么，或试图被听到时，我们必须更加努力，因为我们不是规范。我们并不代表大多数员工，我们经常被视为想要与我们大多数男性同事不同的东西。

因此，许多技术领导者和公司不知道如何适应和处理女性员工的问题和要求，只是因为她们没有经常经历这些问题和要求。

还有社会强加给我们的苛刻期望和标准。双重标准的一个经典例子是，当一个女人过于自信时，她被称为专横。当一个男人有主见时，他被认为是一个领导者。此外，当一位女性领导人体现出善良和同情等女性特征时，他们会被认为是“软”的。

为了庆祝国际妇女节(3 月 8 日)，我想和你们分享公司应该采取的六项行动，以便在留住和提升女性人才方面有所作为。我可以绝对自信地说这一点，因为我有基于性别的研究以及我作为一名技术女性、一名职业母亲、一名经理和该行业女性人才导师的个人经历作为支持。

# 1.提供灵活的工作环境

![](img/e7b44545a8e7c853491535a9220d7cc6.png)

在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上[站着的工作生活方式](https://unsplash.com/@standsome?utm_source=medium&utm_medium=referral)的照片。

根据*华尔街日报*关于[男人&女人如何看待工作场所的综合研究](http://graphics.wsj.com/how-men-and-women-see-the-workplace-differently/)，36%的女人和 7%的男人说他们在家做更多的工作。另一方面，7%的女性和 30%的男性表示，他们在家做的事情较少。

这些统计数据表明，如果家里或女人的家庭出现问题，她更有可能放下工作回家解决——除非她是 7%的女性中的一员，她们的伴侣在家里做得更多。这一数字与该行业高层女性较少的原因一致。

以下是我的个人经历:我经常看到，在女儿的学校活动中，帮忙和出席的母亲比父亲多得多。此外，我的家庭对我来说是最重要的，我非常乐意通过把他们放在我的优先列表的首位并随时为他们服务来展示我对家庭的承诺。即使作为一个以职业驱动和积极进取而闻名的人，如果我必须做出选择，我也会选择灵活性，而不是升职、高薪或更有趣的角色。

问题:贵公司是否提供灵活的工作时间，以及必要时在家工作的选择？

# 2.提供与高级管理层的接触

![](img/36f7c2cdec1ce2bed25c6c88674b649a.png)

照片由 [krakenimages](https://unsplash.com/@krakenimages?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄。

同一项研究显示，与女性相比，更多的男性表示，他们每周至少与高层领导就工作进行一次互动——在他们职业生涯的各个阶段，从初级到中级，一直到高级。

对女性来说，接近高级管理层意味着获得有价值的见解，以及谈论自己工作并获得反馈的机会。这也增加了知名度，打开了她可能没有的机会，如果她只是埋头苦干。不幸的是，妇女获得领导和非正式指导的机会不够多。

当我从一名个人贡献者变成一名经理时，我真正注意到了进入高级管理层对我职业生涯的影响。当我是一名个人贡献者时，我也有很好的想法和能力，但因为我从未被问过，而且我觉得高级管理层不会对我提供的东西感兴趣，所以我只是低着头，做好我的工作。没别的了。我无法与高级管理层分享我的想法和创意，但如果我这么做了，我本可以为公司做出更多贡献，或者在某些机构呆得更久。在我职业生涯的早期，我在任何一家公司的平均任期是两年。

问题:你的公司有积极帮助女性员工找到合适导师的导师-学员计划吗？

# 3.给予具体的赞美和认可

![](img/d602d84beec09f923f64ae86a7c93a51.png)

在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上由[克里斯蒂娜@ wocintechchat.com](https://unsplash.com/@wocintechchat?utm_source=medium&utm_medium=referral)拍摄的照片。

根据罗伯特·k·赫伯特的一项研究，发现女性(18.9%)比男性(3.3%)给予更多的赞美。

这意味着当女人在电话的另一端时，她们会有同样的期待。一句深思熟虑的赞美、一些赞美或认可，而不是泛泛而谈，会大有帮助。

定期的认可和表扬意味着她的工作得到了认可。当反馈是具体的(例如，“你对 X 项目所做的调查节省了几天的开发工作，因为我们更好地了解了这个领域”)而不是笼统的(例如，“你做得很好！”或者“了不起的工作！”)，这意味着其他人真正注意到并欣赏你为组织带来的价值。

女性也认为晋升是对她们工作的认可和肯定。总的来说，对他们来说，感觉被重视比感觉自己赢了更重要。

最后但同样重要的是，女性倾向于转移/逃避他人的正面评论，但这并不意味着她们不需要或不想要这些评论。所以如果你是一名女性员工的经理，不要误解这一点，继续给予周到的赞美。这样，你就会有一个敬业、快乐的员工。

我曾经认为认可对我来说并不重要。我总是被自己内心的渴望驱使着去做好，去做好人，去为他人服务。然而，当我不被认可时，我意识到认可对我职业生涯的影响。是的，我仍然像以前一样有动力，但我开始怀疑我的工作是否还会产生任何影响。结果，我开始在别处寻找一个我的工作会被重视的地方。

*问题:你的公司是否有一个被广泛认可的包容性认可体系？*

# 4.创建一个目标驱动的组织

![](img/8047128949823e02b265b9bb83a85c91.png)

蒂姆·马歇尔在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片。

> “与我们交谈过的女性对自己的工作质量非常满意，无论这意味着对一项研究的优雅和清晰感到自豪，还是以积极促进部门效率的方式重组行政系统。”— [剑桥大学](https://www.cam.ac.uk/women-at-cambridge/chapters-and-themes/chapter-1-the-achievements-that-matter-most-and-why)

女性对自己的工作感到非常自豪。这并不是说男性没有，但是男性和女性的自豪感来源可能会有很大的不同。

女性了解她们的工作如何对他人产生积极影响，无论是社区、顾客、组织中的其他人还是她们的团队。他们更有动力留在职场，甚至同一个职场。根据研究，他们经常被他们的目的所驱使，而不是像名誉和金钱这样的外在因素。

当我在产假后开始找工作时，我寻找有明确目的的地方，因为如果我不得不把一个月大的婴儿放在日托所去工作，那必须是一份有目的的工作！尤其是在科技行业，优秀的女性人才并不常见，每个公司都想雇佣我们，我们有很多选择。

*问题:你的公司有目标吗？所有员工——尤其是高层管理人员——每天都生活和呼吸着这个目标吗？*

# 5.鼓励支持网络

![](img/fc3ef9bd83a02315f531a89b6d6d4a6f.png)

[LinkedIn 销售导航员](https://unsplash.com/@linkedinsalesnavigator?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片。

女性在工作中需要一个支持网络——一个她们可以依赖的网络，在这个网络中，她们可以做自己而不被评判。他们不太擅长与人接触并形成自己的网络，因为他们通常不太敢说话。

他们希望成功，但不希望因为询问他们需要或想要的东西而被人以负面的眼光看待，理性，以成就为导向，或雄心勃勃，这与通常与女性相关的刻板特征相反:友好，敏感和欣赏。

研究表明，职场女性注重人际关系，具有很强的人际交往和协作能力。拥有一个他们可以利用这些技能的支持网络，将真正帮助他们脱颖而出。

我一直和男同事一起工作，和男同事交流没有问题。当我开始参与以女性为导向的团体时，无论是在工作中还是在社区中(如 Girls Geek Sydney)，我注意到男性和女性在处理不同问题和相互沟通方面的差异。这并不奇怪，因为男人和女人在心理上是不同的。我唯一的建议是接触和影响女性。同情我们，试着像我们一样思考。

*问:你们公司有专门针对女性员工的内部倡议或团体吗？*

# 6.提供指导

![](img/d431614687b3c2c5f6588ba3eee505c8.png)

[KOBU 社](https://unsplash.com/@kobuagency?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 拍摄的照片。

我曾经在工作中和一个男同事聊过，他开玩笑地说，“伊莎贝尔，告诉我你在我头上看到了什么。”当我什么也没说的时候，他问道:“你没有看到你在自己头顶上看到的玻璃天花板吗？”

的确，女性有很多限制性的信念和感知障碍。对工作场所性别平等有负面影响的主要障碍或信念之一是，女性没有意识到她们可以同时拥有家庭和事业。他们担心，如果他们被提升到高级职位，他们将需要工作更长的时间，可能无法花更多的时间与家人在一起。同样的担心也存在，因为它涉及到获得更多的报酬，这是为什么存在性别报酬差距的一个原因。女性担心自己无法兼顾两者，会让公司或家人失望。

*问题:贵公司是否提供教练研习班或会议，让女性员工可以公开谈论她们的担忧和关切，然后提供框架、工具和行动计划来打破那些限制性信念和感知障碍？*
