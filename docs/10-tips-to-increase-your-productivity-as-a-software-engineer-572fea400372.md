# 提高软件工程师工作效率的 10 个技巧

## 在有限的时间内完成更多工作

![](img/6e564d5a89868c1bf37c53144dbb1ffc.png)

卡尔·海尔达尔在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

大多数软件工程师每天有八个小时的工作时间。然而，他们在这几个小时里的进展却大相径庭。有些人在一年内就为下一次晋升做好了准备，而有些人可能需要很多年才能达到下一个级别。大多数程序员的工作时间都浪费在头疼、压力、脑残、无聊中。为这些时刻做计划会改变任何程序员的生活。在这篇文章中，我将分享一些在同样的工作时间内完成更多工作的想法。

# 1.有两份待办事项清单

有一个待办事项清单是一个伟大的生产力黑客。这有助于你像专业人士一样集中注意力并征服你的任务。然而，我给你的建议是把这个待办事项列表分成两个列表。第一个列表是需要集中注意力的困难任务，而另一个是即使不集中注意力也能完成的简单任务。只有你能决定什么任务需要关注，什么任务不需要，因为这与你的个性和偏好有关。

稍后，当你工作时，只要你感到专注，就在专注时间表上工作。每当你感到注意力不集中的时候，就是你的另一个清单的时候了。

# 2.上班时有书看

如果你爱读书，一定要选一本适合工作的书。和你的经理谈谈你在工作时间能做什么，不能做什么。我相信大部分公司都会允许你在工作时间阅读任何与你领域相关的内容。身边放一本书可以帮助爱书人在高效工作的同时处理压力和头痛。到年底，你可能会惊讶地发现，你完成了多少本书，而没有浪费时间或降低效率。

如果明智地选择书籍，它们将帮助你提高水平，同时在工作中获得乐趣。

# 3.参加在线课程，在工作中学习

如今，许多网站都提供免费的专业技术课程。总是有一门课程要完成:有许多课程可以增强你在你的领域的知识，为你开辟一个新的领域，增加你的软技能，帮助你学习一门语言，教你如何学习，或者任何其他可以随着时间的推移增强你的能力的东西。

书籍也是如此。你不必只专注于这些课程，就能在一周内完成。你所需要做的就是把这门课放在一边，这样你就可以在不做当前项目的时候继续上课。

# 4.教导他人

令人惊讶的是，提高你水平的最好方法是教书。当我们为他人总结我们所知道的东西时，我们组织我们的大脑，优化我们知道事情的方式。它将帮助你做出更多贡献，更好地沟通，了解更多。因此，偶尔向你的团队提供一次技术讲座或一份关于你所学到的新东西的总结报告。

# 5.写文章

这个技巧与上一点相似，但会进一步扩展你的知识。写一些你知道的科技文章。它可以是你学到的一项新技术，你修复的一个 bug，你读过的一本书，你谈论的一门课程，或者只是一篇大声思考的文章。你会帮助别人，会更加积极热情，你会学到更多。你会感受到工作对你的积极影响，因此你会对工作更加满意。一定要问你的经理你是否可以在工作时间这样做。

# 6.午餐时间社交

午餐时间的社交会让你精神焕发，让你的下午更有活力。会帮助你了解工作中身边的人。确保每周与不同的人见面，并愉快地谈论几乎所有的事情。如果你每周抽出一次午餐时间与你可以学习的人交谈，这将是有益的。与所有人交往:经理、实习生和与你同级的同事。

# 7.在会议和代码评审中互动

新开发人员通常觉得太害羞而不敢多交流。然而，成为每个会议和代码审查的重要成员将帮助你在很短的时间内提高你的水平。如果你是初学者，没有必要羞于在会议上提出你的问题，也没有必要在代码评审上分享你所能分享的。如果你的水平很高，也要分享你的担忧和反馈。如果你没有参加会议，通过参与讨论让你的团队想念你。此外，代码评审是展示你的专业知识和提高你的技能的好方法。不要羞于做出贡献，永远记住这是在工作中提升自己和展示热情的最佳方式。

# 8.制定促销计划

许多员工在上一次升职后很久才考虑升职。如果他们想过这个问题，他们会羞于在一对一的谈话中与经理讨论任何相关的事情。相反，我的建议是，一旦你加入一家公司或获得晋升，你就应该有一个晋升计划。这个计划需要从第一天开始就和你的经理讨论，并偶尔跟进。

要计划你的工作时间，你首先需要知道你应该达到的主要目标，以在各方面达到下一个水平。此外，要知道你的弱项，这样你就可以从第一天就开始努力。它会帮助你选择报读哪些课程，读哪些书，和哪些导师交往。

> “渴之前挖好井。”—中国谚语

# 9.有一个导师

从工作的第一天起就选择一位导师。一定要和他们讨论你的推广计划，让他们帮你发现自己的优缺点。偶尔向他们寻求跟进、建议和反馈。你可能有不止一个导师，但要确保明智地选择他们，并不时接受批评。

# 10.过好个人生活

一些开发人员认为他们需要工作八个小时以上才能在工作中出类拔萃，但事实并非如此。你需要享受你的个人生活，保持你生活中的优先权。这将帮助你感到精力充沛，充满热情，并拯救你的心理健康。最终，这将帮助你更专注，更有效率，更少的休假。

就是这样，希望这十个小技巧能帮助你在工作中出类拔萃。很好奇想知道大家有没有更多的小技巧。
