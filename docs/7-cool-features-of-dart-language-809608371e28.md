# 级联符号

级联(`..`、`?..`)允许您对同一对象进行一系列操作。这通常可以省去创建临时变量的步骤，并允许您编写更流畅的代码。

# 抽象类

使用`abstract`修饰符定义一个*抽象类(*一个不能被实例化的类)。抽象类对于定义接口很有用，通常有一些实现。

# 混合蛋白

混合是在多个类层次结构中重用类代码的一种方式。

为了实现 mixin，创建一个没有声明构造函数的类。除非你想让你的 mixin 像普通类一样可用，否则使用`mixin`关键字代替`class`。

要使用 mixin，请使用关键字 T6，后跟一个或多个 mixin 名称。

为了限制可以使用 mixin 的类型，使用`on`关键字来指定所需的超类。

# Typedefs

类型别名—是引用类型的一种简洁方式。通常用于创建项目中经常使用的自定义类型。

# 扩展方法

Dart 2.7 中引入的扩展方法是向现有库和代码添加功能的一种方式。

# 可选位置参数

通过将位置参数括在括号中，可以使它们成为可选参数。可选的位置参数总是在函数的参数列表的最后。除非您提供另一个默认值，否则它们的默认值为 null。
