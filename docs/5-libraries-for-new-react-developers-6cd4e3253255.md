库是可重用代码的集合，可以包含各种函数、对象和方法，具体取决于编写库的语言。虽然本文将关注对 JavaScript 开发人员有帮助的库，但是库绝不仅限于 JavaScript。使用库时，应用程序可以安装或链接到库，从而允许应用程序访问库的功能。

对于使用 JavaScript 和 JavaScript 框架的开发人员来说，有许多充满有用特性和功能的库。这些库可以包括各种处理字符串、日期、HTML DOM 元素、事件、cookies、动画、网络请求等的函数。

许多库允许开发人员定制他们创建的函数和元素的功能、风格和依赖数据。

库中的函数为开发人员提供了一种抽象的信息处理方式，使他们能够更快更容易地创建程序。一个库可以减少大约 20%的开发时间。

尽管库可以是预构建代码的一个极好的、非常有用的来源，但是库也会带来问题。库可能包含难以修复或解决的错误。不是所有的库都被维护，所以如果有一个 bug，不能保证它会被修复。也不保证库将被更新以考虑软件更新。

这里只是对使用 React 的开发人员有帮助的许多库的一部分。
