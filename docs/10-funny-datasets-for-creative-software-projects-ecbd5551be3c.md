# 1.大脚目击事件

我将从我自己用过的一个开始，以及这篇文章灵感背后的原因。

[大脚怪目击事件](https://data.world/timothyrenner/bfro-sightings-data)数据库由 [data.world](https://data.world/) 管理，使用由[大脚怪野外研究者组织](http://www.bfro.net/)汇编的数据——如果我们可以称之为大脚怪信息的话，这可能是最合法的来源。它包括世界各地目击这种神话生物的详细报告，包括日期、地点和天气情况。每个记录还包括确定事件性质的报告分类(直接目击、噪音、轨迹等)。).

我在过去的一个[个人项目](https://github.com/marianamv112/bigfoot-sightings-mgmt-app)中使用过这个数据集。这是一个在地图上显示所有大脚怪目击事件的平台，并允许任何人登记新的目击事件。为此，我使用大脚怪目击数据集创建了一个 API，作为我自己的数据库的种子。但我相信，如果你是一名数据科学家，就一定有可能预测下一次目击的最可能地点和日期。由于一些社区非常严肃地对待猎捕大脚野人的事情，这是一个让许多人感兴趣的想法。

# 2.纽约市的松鼠普查

中央公园里有多少只松鼠？纽约松鼠最常见的皮毛图案是什么？一只纽约客松鼠接近你的概率有多大？所有这些信息都可以通过使用[纽约市松鼠普查数据集](https://github.com/rfordatascience/tidytuesday/tree/master/data/2019/2019-10-29)来收集和/或计算。

幽默作家杰米·艾伦和包括科学家和野生动物专家在内的 300 多名志愿者在没有任何期望的情况下创建了这个数据集合。尽管这个项目开始时只是一个有趣的活动，但所采取的程序涉及到一些严肃的科学。有一点是肯定的:这是一个非常聪明和有趣的数据集，可以激发创造性的项目。

# 3.飞机野生动物袭击

[飞机野生动物袭击数据集](https://www.kaggle.com/faa/wildlife-strikes) 包含了 1990 年至 2015 年每一次报告的军用、商用或民用飞机野生动物袭击的记录，非常适合作为关于飞行事故的原创项目。

这些记录涉及每次事故的日期、航空器经营人、航空器制造商和型号、发动机制造商和型号、机场名称和位置、涉及的物种名称和数量以及造成的损害。通过这种方式，你可以尝试推断哪种鸟类对飞机造成了最大的损害，在你的下一次飞行中与动物发生事故的概率是多少，以及该概率是否高于飞机失事。

# 4.麦当劳菜单的营养事实

因为世界著名的快餐连锁店麦当劳在他们的网站上提供关于他们的菜单项目和营养事实的信息，所以使用这些数据编制一个数据库一定很容易。

使用[麦当劳菜单的营养事实](https://www.kaggle.com/mcdonalds/nutrition-facts)数据集，您可以找到平均麦当劳超值餐中饱和脂肪的百分比，以及烤鸡汉堡是否比脆皮汉堡更有营养价值，甚至可以找到满足一天营养需求所需的最少菜单项目。如果你想更进一步，你也可以整合来自星巴克菜单[营养事实](https://www.kaggle.com/starbucks/starbucks-menu)数据集的数据，创建一个更复杂的快餐项目。

# 5.UFO 目击事件

带摄像头的智能手机推出后，UFO 目击事件的数量真的减少了吗？让我们通过 [UFO 目击数据集](https://www.kaggle.com/NUFORC/ufo-sightings)来寻找答案，该数据集包含了上个世纪超过 80，000 份 UFO 目击记录。

你还可以确定何时何地有最好的机会看到不明飞行物，并预测它们何时会很快出现。

# 6.鞋子价格

谁在鞋子上花的更多:男人还是女人？每个性别最喜欢的鞋子品牌有哪些？什么月份买奢侈品鞋最划算？销售季节前会涨价吗？

这些问题可以通过分析和交叉引用[男鞋价格](https://data.world/datafiniti/mens-shoe-prices)和[女鞋价格](https://www.kaggle.com/datafiniti/womens-shoes-prices)数据集来回答！每个汇编包括 10，000 条关于鞋的记录，包括名称、品牌和价格，使其成为确定品牌偏好、定价策略和鞋类行业趋势的理想选择。

# 7.口袋妖怪

口袋妖怪是有史以来最受欢迎的动画和视频游戏之一，它是任何类型项目的灵感源泉。我甚至对那些受口袋妖怪启发的工作职位进行了代码评估！

[完整的口袋妖怪数据集](https://www.kaggle.com/rounakbanik/pokemon)是由 [Rounak Banik](https://www.kaggle.com/rounakbanik) 创建的，并且对于每一个口袋妖怪，[包括了你可以在 Pokedex](https://pokemondb.net/pokedex/pikachu) 上找到的信息:口袋妖怪名称、Pokedex 数量、攻击和防御基础等级、捕获率等。

你可能已经想出了数以千计的方法来使用这些数据集合，所以我将留给你作者提供的想法:

*   有没有可能建立一个分类器来识别传说中的口袋妖怪？
*   口袋妖怪的身高和体重与它的各种基础统计数据有什么关系？
*   有哪些因素影响经验增长和蛋步？这些量相关吗？
*   哪种类型整体最强？哪个最弱？
*   哪种类型最有可能成为传说中的口袋妖怪？
*   可以组建口袋妖怪梦之队吗？一队六只口袋妖怪造成的伤害最大，而其他六只口袋妖怪相对不受影响。

如果你更喜欢数码宝贝，请不要走！当然，也有一个很棒的 [Digimon 数据集](https://www.kaggle.com/rtatman/digidb)可供你使用。

# 8.截至 2021 年的世界幸福报告

你可以成为最终回答人类最大问题之一的人:幸福的秘密是什么？

带着这个目的，从分析截止到 2021 年的[世界幸福报告](https://www.kaggle.com/mathurinache/world-happiness-report-20152021) 数据集开始。它包含了关于[幸福指数](https://worldhappiness.report/faq/)的详细数据，幸福指数是一项全国性的生活质量评估，涉及人均 GDP、健康预期寿命、社会支持等因素。

由于数据集已经包含了 2021 年的数据，你甚至可以分析疫情是如何影响全球幸福的。

# 9.[文学作品]哈利波特

如果你是一个哈利波特迷，你肯定花了很多时间想知道你会被分配到霍格沃茨的哪个房间，你的魔杖的材料，或者你的守护神会变成什么样子。现在你可以通过分析这个[哈利波特数据](https://www.kaggle.com/gulsahdemiryurek/harry-potter-dataset) t 来找出答案，包括从电影剧本和网站上搜集的关于角色、咒语和药剂的详细信息。

# 10.百万首歌曲数据集

对于音乐爱好者来说，有一个数据库会立即激发你创建一个新项目:百万首歌曲数据集。

谁是有史以来最受欢迎的艺术家？明年听什么类型的会比较多？对下一个大热门的预测分析怎么样？

您开始编码所需的一切都在这个来自一百万首音乐曲目的音频特征和元数据的集合中。这个项目是通过对来自不同数据集的数据进行聚类而创建的，如 [Last.fm](http://millionsongdataset.com/lastfm/) 和 [musiXmatch](http://millionsongdataset.com/musixmatch/) ，其目的是[鼓励新的研究项目、算法和 API](http://millionsongdataset.com/)的开发，所以请随意使用它，并从中获得创造性！
