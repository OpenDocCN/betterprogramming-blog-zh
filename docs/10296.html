# 4——展示你的作品

软件业正在蓬勃发展，每天都有新的软件公司涌现。如果你想从所有找工作的软件开发人员中脱颖而出，你的技能是一流的，这很重要。

当软件开发近年来进步如此之大时，很难知道哪些技能会让你在2022年最具竞争力。你可能在UI设计或后端编程等领域发展了一些很棒的技能，但如果你想让人们在2022年看到你的简历，你需要跟上最新的发展。以下是一些关于如何在2022年被聘为软件开发人员的建议。

这些只是我的一点想法，我是根据我的个人经历来写的。

为什么软件开发人员需要最新的技能？

如今，要在软件开发中取得成功，不仅仅是软件的问题。这也是关于你的个人技能——你如何与其他团队成员合作，你快速适应新软件和解决问题的能力，等等。关键是软件开发人员需要能够快速思考和反应。由于软件世界总是在变化，学习新的东西对于在2022年保持敏锐的软件开发人员来说是必不可少的。

软件开发世界是不断变化的。每天似乎都会带来新的软件程序，软件开发人员需要从头开始学习或更新他们当前的技能。Web3、区块链开发等是软件开发的最新行业趋势——如果你现在不熟悉它们，几年后你可能会发现自己落后了。幸运的是，软件开发人员现在有很多可用的资源。电子学习软件使世界各地的人都可以进行软件开发，只需一台笔记本电脑和互联网连接，软件开发人员就可以比以往任何时候都更快地学习新技术。

2022年及以后的软件开发人员招聘是什么样的？

这些只是我的两美分，我可能完全错了。

在过去的几年里，大多数公司都在寻找有4-5年经验的软件开发人员。然而，在未来，这种情况可能会改变，公司将会寻找拥有更多最新技能的软件开发人员。他们希望软件开发人员能够立即投入工作，不需要太多的指导。

这并不意味着你不需要软件开发经验。如果希望软件开发人员快速掌握知识，那么未来的软件开发人员的工作可能不需要编程知识。然而，软件开发经验在你的简历上仍然是有价值的，因为它表明你对软件如何工作有所了解，并且能够适应新技术。

别忘了你的软技能！

1.软件开发人员需要具备的软技能

批判性思维

你需要跳出框框思考，想出创造性的解决问题的方法。

作为一名软件开发人员，您可以通过以下方式提高自己的批判性思维技能:

  • 问问题:总是问自己为什么事情是这样的，以及你如何能做得更好。
  • 挑战假设:不要只接受事物的表面价值——总是质疑别人告诉你的，看看是否有更好的方法来做事。
  • 创造性思维:不要害怕提出新的想法，即使这些想法一开始看起来很疯狂。

分析技能

你需要能够将复杂的任务分解成更小的步骤，并找出解决它们的方法。

要提高您的分析技能，您可以:

  • 将复杂的任务分解成更小的步骤，并找出解决它们的方法。
  • 使用图表帮助您更容易地理解数据。
  • 练习解决问题:尝试解决尽可能多的不同类型的问题，并从你的错误中学习。
  • 批判性思考:问问你自己为什么事情是这样的，你如何能让它变得更好。

解决问题

未来的软件开发人员不仅需要能够编写软件，还需要能够找到解决问题的方法。软件开发人员可以提高他们解决问题的技能的一种方法是通过发现其他人是如何解决相同的问题的，无论是在软件世界还是在软件世界之外。阅读关于软件开发或为不同目的编写软件的书籍也会有所帮助。软件开发人员应该经常问自己“下一步是什么？”并找出可能阻碍他们前进的障碍以及如何解决这些障碍。

合作

软件开发是一个团队过程，所以软件开发人员需要良好的沟通技巧才能顺利地一起工作。软件开发人员可以通过在学校或工作中参与小组项目来提高他们的协作技能。他们还可以加入软件开发论坛或在线社区，在那里他们可以讨论项目并与其他开发人员分享想法。

自治

软件开发人员需要能够独立工作，并拥有自己的项目。软件开发人员可以提高他们的自主技能的一个方法是通过独立项目工作。这些项目可以是从开发一个新的软件程序到创建一个新的网站。参与这些项目将帮助你学会如何掌控自己的项目，并独立做出决策。

沟通

软件开发人员需要能够与其他团队成员清楚地交流。软件开发人员提高沟通技巧的一个方法是参加以沟通为重点的研讨会和会议。这些研讨会可以帮助软件开发人员学习如何更好地与他们的队友沟通。此外，软件开发人员也可以通过练习积极倾听来提高他们的沟通技巧。

所有这些技能都可以通过不同的电子学习软件程序来学习和提高。对于批判性思维和解决问题，软件开发人员可以参加提供真实世界例子和挑战的在线课程。对于创造力，软件开发人员可以尝试不同的软件语言和工具，看看什么最适合他们。为了自治和交流，软件开发人员可以加入在线论坛和讨论组，从他们的同行那里获得反馈。有了正确的电子学习软件和一点点努力，任何人都可以成为一名熟练的软件开发人员！

2.建立一个强大的项目组合来展示你的技能。

建立一个强大的软件开发项目组合是被雇佣的关键。建立投资组合的最佳方式是为不同的组织从事开源软件项目，或者在像Upwork这样的网站上从事自由职业。这些软件开发项目将向雇主展示你是如何工作的，以及如何在他们的软件开发人员中应用他们需要的技能。记录你在具体项目中使用的软件也很重要，这样雇主就知道你擅长什么编程语言和工具。

在电子表格中记录软件开发项目，列出你用过的软件产品或技术，你工作过的软件公司，以及其他相关信息。

申请工作时，不要忘记在你的简历或求职信中包含所有这些数据！这样雇主可以很快看出你擅长什么样的软件开发项目

3.参加黑客马拉松和编程竞赛

软件开发人员真正脱颖而出的一个方法是参加黑客马拉松和编程竞赛。您可以在网上找到这些活动，它们是获得与世界各地软件开发人员合作经验的好方法。一些软件开发人员可能会犹豫是否要参加这些比赛，因为他们担心无法开发出足够好的软件。不过，别担心！你为这些比赛创造的软件不需要完美。只要确保它能工作，并且包含至少一个让你和其他参与者不同的功能。

在参加软件黑客马拉松之后，软件开发人员将会对如何管理他们的时间和从事软件项目有更好的想法。此外，这些软件开发活动也是与其他软件开发人员交流的好地方。

4.展示你的作品

向雇主展示你的软件开发技能的另一个好方法是在社交媒体上展示你的工作。像Twitter、LinkedIn和GitHub这样的平台是向世界展示你的软件开发技能的好方法。确保你在网上展示你最好的作品，并确保你积极参与。对于软件开发人员来说，通过建立一个包含他们所有项目的软件开发组合来展示他们的工作是很重要的。软件开发人员也可以在他们的博客或像Medium这样的网站上展示他们的软件开发技能。软件开发人员可能会考虑创建一个在线网站或博客，在一个地方展示他们所有的软件开发项目。这样，软件开发人员可以很容易地与雇主分享更新，并向潜在雇主展示他们最擅长什么！

雇主会希望雇佣以前做过软件的软件开发人员，所以展示你在这个作品集里最好的软件开发项目是很重要的。

5.加入软件开发社区

作为一名软件开发人员，提高技能的最好方法之一就是加入软件开发社区。这可能包括参加软件开发活动、加入在线论坛和讨论组，或者只是在社交媒体上关注软件开发人员。通过参与软件开发社区，您将能够向最优秀的人学习，并了解最新的软件开发趋势。

此外，还可以参与开源项目。这些项目是学习新编程语言和发展技能的好方法。许多雇主寻找活跃在软件开发社区并有开源项目工作经验的软件开发人员。

许多软件开发人员表示，获得聘用的最佳方式是获得一名软件开发导师(或者自己成为一名导师)。你可以通过在当地聚会或会议上与软件开发人员交流，加入在线论坛，或像sharpen Minds这样的付费网站来获得软件开发导师

结论

软件开发人员需求量很大，软件开发是一个不断增长的领域。如果你想被聘为软件开发人员，我们建议你参加软件竞赛，比如黑客马拉松或编程竞赛。

这些活动将让您有机会展示您的技能，同时与世界各地的其他软件开发人员建立联系！你也可以在Twitter和LinkedIn等社交媒体平台上展示你的工作，让潜在雇主看到你以前做过的所有伟大的事情。

如果所有这些都失败了，一定要通过参加会议或加入软件开发人员聚集的在线讨论组来参与软件社区。向那些成功找到软件开发工作的人学习的时间越多，成功的机会就越大！
