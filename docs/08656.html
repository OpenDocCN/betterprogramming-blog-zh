4.用Colorama美化你的命令行

用Colorama给你的命令行应用添加一些有用的颜色。

使用pip安装它:

pip install colorama

Colorama支持三种不同的颜色类型:

  • foreground为文本颜色
  • background为背景色
  • style用于一些额外的颜色造型
