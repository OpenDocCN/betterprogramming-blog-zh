# 作为软件工程师如何更快晋升

## 谁不想升职？

大多数软件公司给他们的员工头衔，并为他们提供从一个头衔晋升到另一个头衔的机会。每个头衔都反映了这个员工的专业知识，并决定了他们挣多少钱。不同的公司有不同的头衔，但是更高的头衔确实意味着更多的经验、金钱和对软件世界的更大贡献。所以你越快获得晋升，你就越成功。

在这篇文章中，我将告诉你如何做到这一点。

# 1.从第一天开始计划

许多软件工程师犯的一个大错误是没有从第一天开始为他们的下一次晋升做计划。他们会等到距离上次升职至少一年后再做打算。可悲的是，这使得他们在大多数情况下一年后才能升职。

你的计划越好，你的晋升就越快。

# 2.了解你当前水平的要求，并努力达到这些要求

许多开发人员只是在没有计划或愿景的情况下工作。给我一个任务，我就去做。他们忘记了他们的级别需要一套特定的职责。这些职责包括错误修复、设计、实现，有时还包括你做这些事情的速度。知道了要求你才会知道自己做的好不好。

直接与您的经理交谈，询问他们是否对您目前的水平有明确的要求。如果他们的回答不明确，而且看起来你的公司没有具体说明任何要求，那么是时候谷歌一下他们了。只需搜索“对一名*高级开发人员*的期望是什么？”例如，你会发现文章给你一个清晰的列表。别忘了把“*高级开发人员*”换成你的职位头衔。

一旦你有了这个清单，就把它写在你的计划里。努力达到那些要求，直到你成为大师，超过他们一个层次。

# 3.了解下一级的要求

一旦你擅长你目前的水平，即使你离掌握所有需要的技能只有几步之遥，也是时候知道下一步是什么了。当你有迹象表明你在下一个级别的表现如何时，是时候认真地与你的经理讨论升职了。

假设你是一名初级开发人员，你擅长指导他人、设计解决方案和快速交付。既然你知道每个级别的要求，你就知道这些品质不再是初级的了。了解这一点会让你在这些技能上更加努力。此外，这也是与你的经理讨论你的水平和产出的时候了。根据你的工作和水平，你应该得到提升。

一旦你达到了当前职位的水平，那就是升职的时候了。这就是为什么我建议非常了解每个级别的要求，并相应地从第一天开始计划。

# 4.了解你的优势和劣势

接受批评。向人们征求评论和反馈。知道自己的优点才能更加强化，知道自己的缺点才能改善。

即使你在其他方面领先，一些弱点也可能会让你的晋升延迟数年。解决办法很简单:有勇气听取他人的反馈，改变并努力克服自己的弱点。不过，这需要时间。所以越早开始越好。

# 5.让你的经理成为你的导师

对你的经理诚实。告诉他们你的计划，和他们一起大声思考，向他们寻求反馈和回顾。告诉他们为什么你从第一天开始就计划升职。告诉他们你正在努力克服哪个弱点。让与经理的一对一会谈成为确保你走上正轨的一种方式。

有时我们不想暴露我们缺乏经验，因为经理将是决定我们以后有多好的人。然而，还是要接受批评。让它成为你升级的途径。通过你的诚实，经理会知道你有多努力，你打算提高多少。

# 6.在您的团队中拥有发言权

从第一天起就成为讨论的一部分。当你参加一个会议时，不要只是对所有的团队讨论感到厌烦和困惑。成为它的一部分。如果你没有得到什么，就问清楚。如果你有想法，分享一下。如果你喜欢别人说的观点，在会议上告诉他们。

在你参加的任何会议中成为重要人物。这会让你更积极地参与每个项目和解决方案。

# 7.提交更多的代码评审

代码评审并不是每个公司都做的事情，但是如果你的公司做了，就专注于它们。在你工作的一年结束时，没有人会知道你做了多少工作，但是他们会知道你做了多少代码审查。

提交更多的代码评审意味着得到更多的批评、更多的建议和更多的勇气。此外，还做了更多的工作。

如果你对代码审查没有信心，只需谷歌更多关于如何做得更好的文章。永远不要推迟在这方面的工作。

# 8.有一个导师

除了你和经理的关系之外，选择一个在你想做得更好的方面是专家的人，请他们做你的导师。时不时和这位导师见面，听取他们的建议。告诉他们你的担忧，你的恐惧，让他们帮助你就你的计划应该包括什么进行一般性讨论。你可以有不止一个导师。

# 9.有一本日记

许多人喜欢为他们的个人生活写日志，但是他们忘记了日志对他们的工作也有很大的作用。我建议为你的职业生涯写一本日记。在这个日志里，你的计划被安排好了，有感谢美好机会的页面，有最奇怪的 bug 列表，还有每周的跟踪日志。让它成为一个最小的子弹日志。它将帮助你保持在正确的轨道上，并决定你想要关注的关键点。

# 10.成为某方面的专家

我的最后一条建议是选择一个非常具体的主题，并成为这方面的专家。选择是无限的。可以是对编程语言的深入了解，成为一名非常优秀的 scrum 大师，擅长文档写作。我不确定你有什么机会。目标是在某方面做得非常好，这样公司里的每个人在需要帮助时都会向你求助。这将有助于你快速超越自己的水平，成为公司的重要人物。

# 结论

这些是我快速提升的十个建议。也让我知道你的方法。但是，不要忘记，晋升本身从来都不是目标。这是晋升所展示的(即在我们的职业生涯中变得更好)。
