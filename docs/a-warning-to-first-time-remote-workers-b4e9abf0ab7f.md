# 对首次远程工作人员的警告

> 原文：<https://betterprogramming.pub/a-warning-to-first-time-remote-workers-b4e9abf0ab7f>

## 为什么“强迫的”分布式团队不能代表远程工作通常是什么样的

![](img/39fdd9d485f2a44f7196edb8b232660d.png)

Daria Nepriakhina 在 [Unsplash](https://unsplash.com/s/photos/remote-work?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上拍摄的照片

在疫情之后，许多公司都在寻找远程工作的答案。

很容易认为远程工作是解决这一危机的“即插即用”解决方案。远程工作也能保证员工的安全和工作效率。

不幸的是，我的预测是，许多工人将有一个悲惨的第一次经历。恢复到他们过去的工作效率并不容易。

不和谐的文化变化、有缺陷的 VPN 以及我们远程工作的背景将会给其他分布式工作的成功蒙上阴影。我担心“在家工作”会受到非议。

事实是，这是一个可怕而诡异的时代。这并不代表在家工作通常是什么样子。

# 首先，将会有一场文化冲击

![](img/764afaac7cc1a040edaa3908ed9e023b.png)

由[杰米·霍顿](https://unsplash.com/@haughters?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText)在 [Unsplash](https://unsplash.com/s/photos/shock?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上拍摄

与普遍的看法相反，许多团队已经习惯于让自己在线。我在福特的同事们以闪电般的速度关注着 Slack、电子邮件和 WebEx。我怀疑员工会发现很难让自己在线。

当电子邮件链变得太复杂而无法整理时，难的是*而不是*有能力“抓住一个房间”。

通过分发，你不会得到任何东西——所有的在线工具都还在。相反，你*失去了*面对面交流的能力。没有工具箱中的工具，找到解决方案将会更加困难。

乐观的一面是，视频通话和屏幕共享功能将减轻这一打击。对他人和自己有耐心是非常必要的。许多你常见的解决问题的外向工作流程将会改变。

为不同的“感觉”做好准备。你的问题可能不会很快得到回答。解决问题时，你可能会感到更加孤独。

# 缺乏硬件将是黑仔的第一周

![](img/03a2c4d588e1ff2bad7a918ea9d9727c.png)

照片由 [Unsplash](https://unsplash.com/s/photos/hardware?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上的 [Niclas Illg](https://unsplash.com/@nicklbaert?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText)

如果你计划去南极旅行，你需要带上合适的装备。你需要带一件漂亮的防风防水外套。或许一些保暖手套。肯定是用来喂企鹅的！

同样，在家工作也需要硬件——一个好的摄像头、麦克风和一些喂狗的东西！

我怀疑许多开发商和商人将会准备不足。

如果可以的话，在办公室关门前，尽可能多的从办公室拿些“东西”。购买显示器和鼠标不是你应该花时间和金钱去做的事情！

你不能准备的是 VPN 和互联网问题。许多公司没有互联网基础设施来支持远程员工。这是许多分布式团队不得不处理的烦恼。

# 新冠肺炎的背景只是让这种环境变得粗糙

![](img/429f2c7195137d7b742b315a52aaea1b.png)

由[安德里亚·莱奥帕尔迪](https://unsplash.com/@whatyouhide?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText)在 [Unsplash](https://unsplash.com/s/photos/running?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上拍摄

在家工作最大的好处是自由。

在办公室里，你可能会感到一些压力，需要更短的休息时间或更短的午餐时间。你可能会感到加班或早退的压力。

当你远程工作时，你会有一种掌控自己命运的感觉。只要你完成了工作，你就会有更多的自由。

也许“在家休息工作”看起来就像去健身房、遛狗或去杂货店购物。我怀疑许多第一次分布式团队不会欣赏这种文化。他们甚至可能要求团队成员严格地从早上 9 点到下午 5 点保持在线

此外，远程员工明白走出家门有多重要。他们可能在一个网络或一些共享的创意空间工作。他们去体育馆。他们去听音乐会。他们中午和下班后去餐馆。

在新冠肺炎时代，这些你都做不到。在家工作将会有更多的…家。这将是心理上的挑战。

我鼓励分散工作的员工尽可能多出去走走。请记住，通常分布式工作感觉比这要自由得多。

# 环境很重要—远程员工第一次需要适应

![](img/9a3f5c65dd43accbaa824ea1b8a30e18.png)

亚历山大·雷德尔在 [Unsplash](https://unsplash.com/s/photos/running?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上拍摄的照片

权力越大，责任越大。仅仅因为远程工作者有了新的自由，他们也需要锻炼自己优秀的时间管理技能。

在家里学习如何提高工作效率将会很困难——因为员工习惯于放松。背景*和*有关系吗？如果一名员工习惯在办公室的办公桌前工作，在那里很容易保持工作效率。

在你自己的家里创造一个工作空间是至关重要的。对于第一次远程办公的人来说，将“电子游戏桌”变成工作空间可能是一个棘手的环境转变。

说了这么多，我想许多第一次远程办公的员工会在某种程度上学会享受它。但肯定不是“即插即用”转向远程工作存在严重的风险和挑战。

此外，如果你最终讨厌远程工作，请理解这是一个不寻常的时代。对于远程工作到底是什么样的，这个社会实验并不是一个很好的测试。
