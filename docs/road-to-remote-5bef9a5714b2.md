我成为远程员工的旅程始于我大学毕业后的第一份工作。我居住的城市远离繁荣的科技中心。

我刚从软件专业毕业，我找的任何工作都涉及搬家或系好安全带上下班。和我还在上大学的女朋友(现在是妻子)约会，我决定继续住在我们的镇上。

我接受了堪萨斯城的一份工作，单程一小时。我对自己的第一份工作很兴奋，但这种动力似乎并没有让我兴奋。所以我开车，开车，开车。

像任何年轻和新就业的人一样，我决定买一辆新车。糟糕的决定。几年之内，我已经跑了 8 万英里。

我的老板有充分的理由让我们在家工作，但问题是公司将我们的远程工作限制为每年 30 天。因此，我通勤，直到我不能再这样做了。

# 新工作

有一次我获得了一些经验，我工作的那条街上一家公司的一个朋友鼓励我去申请。这个新地方非常时髦。提供一个库存充足的厨房，能够远程工作，并承诺有趣的项目工作。

我开始通勤到我的新办公室。主要是因为我已经习惯了。我正在和新同事交流，享受新办公室的福利。

然后，我开始每周有一天在家工作。然后两天。不知不觉中，我每周只去办公室两次。我团队里的人会拿这个开玩笑，但是每天花两个小时在路上是一个巨大的生产力杀手。

更不用说与之相关的其他费用了。从来没有影响过我的工作。我要汇报的客户在一个完全不同的城市。我和办公室里的人唯一的互动就是偶尔吃顿午饭，或者打一场乒乓球！

# 鼓励远程工作

那份工作结束后，我加入了同一地区的另一家公司。再说一次，离我家一小时路程。只是这家公司与众不同。

首席执行官*鼓励我们远程工作。只要我们完成高质量的工作，他就不会在乎我们白天坐在哪里。这是什么！在家工作没有负罪感。我们的老板甚至出钱让我们建立家庭办公室。*

古老的格言“屁股坐在座位上意味着工作正在进行”已经过时了。他们确实鼓励在办公室过星期一。这是一次偶然的面对面的会面，但主要是一起吃午饭。

# 远程工作的好处

*   花在通勤上的时间更少。
*   提高员工保留率。
*   接触更多的申请人。
*   提高生产力。
*   更快乐的工作生活。
*   避免在会议室呆几个小时。
*   更好地控制你的时间表。

# 为什么有些雇主可能不提供远程工作

我认为有几个原因:

*   经理喜欢亲自见你。他们把你的实际存在等同于正在完成的工作。你不可能像在家那样偷懒...只要你投入了八个小时。
*   一些经理也喜欢控制你的一天。你在哪里，你在做什么，等等。远程工作让他们更加困难。
*   也可能是信任问题。他们不相信你一个人能成为好员工。作为一名经理，如果你很难信任你的员工，那么你就雇错了员工。
*   请记住，远程工作并不适合所有人。
*   在商业层面，许多公司认为他们拥有或租赁的昂贵办公楼是一种浪费，如果你不在那里使用它们，而不是看到办公室的真实面貌，这是一种沉没的成本。
*   如果公司投资建设漂亮的办公室，让你可以工作和与同事合作，那是再好不过了。如果你的员工在其他地方更快乐、更有效率，不要难过。

# 回到办公室

我已经远程工作了四年。大约一年前，一个朋友提出了在镇上找间办公室的想法。他拥有自己的生意，对分摊租金很好奇。

我拒绝了他一段时间。我为什么要回到一个办公室，尤其是一个我必须付钱的办公室？我的生活发生了一些变化。在家完成工作越来越难了。

于是，有一天我同意了。他已经在寻找办公空间，并且很高兴我能加入他的工作。我们去年秋天搬进来，从那以后一直在那里工作。

这间办公室对我来说有点不同。我的通勤时间是五分钟，当我两岁的孩子爬到我身上时，我可以逃离房子，当我回家时，我可以关掉工作。

我偶尔还是会离开那里，比如今天，我坐在一家咖啡店里。

# 结论

远程工作并不适用于每一种职业。如果你在制造业工作，我相信你必须亲自到某个地方去生产。

如果你是一名高中老师，那么，你会明白的。

我对这篇文章的看法是——如果允许的话，按照你想要的方式工作。如果你是一名经理，如果你的员工快乐且富有成效，你会从他们身上得到更多。
