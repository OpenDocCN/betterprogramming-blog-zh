作为开发人员，我们通常喜欢自己管理软件和服务。虽然在学习一项技术时，花费时间和精力是值得的，但管理自己的服务器并每天执行那些枯燥重复的维护任务很快就会成为一种痛苦。如今，云服务越来越受欢迎，大大取代了自托管和自管理的应用程序。

# 云托管的优势

*   **更简单的服务器管理**。您不必为您的服务设置任何硬件或操作系统。您的云提供商将为您减负。
*   **烦恼更少。**您不必处理服务器停机和潜在故障。云提供商通常保证最低的服务器正常运行时间和快速的问题修复。
*   **更快的服务。您租用的服务器和容器通常具有很高的带宽和很低的延迟。**
*   **降低成本**。保持服务器运行可能会导致高功耗，这通常比您通过自托管所能节省的还要多。
*   **安全性提高。**提供商通常会分配大量资源来保证服务及其用户的安全。他们有完整的、非常有经验的团队在研究它。

# 为什么许多提供商提供免费计划？

提供免费产品供个人使用或免费试用是一种营销策略，其重点是建立一个潜在客户网络，这些客户可能会为商业环境中的许可和服务付费。公司特别投资于学生，他们学习如何使用某种产品，更有可能在工作中使用其付费版本。

一个简短的声明:我不是这些公司的附属机构，也不是它们的赞助商。这篇文章的目的是告诉读者他们可以利用的免费服务。

这些公司和服务提供商都是很好的选择，可以满足不同的需求。其中一些是等效的，而另一些则有其他用途。根据您的项目需求选择哪个提供商由您决定。它们的排列没有特定的顺序。

# 赫罗库

[Heroku](https://heroku.com) 是基于容器的云平台即服务( [PaaS](https://en.wikipedia.org/wiki/Platform_as_a_service) )。它可用于部署、管理和扩展 web 应用、API 等。

Heroku 是完全托管的，所以开发人员不需要专注于服务器的设置和维护。该服务提供商通过免费的[构建包](https://devcenter.heroku.com/articles/buildpacks)提供多语言支持。它可以与 GitHub 集成，用于自动应用程序部署。

Heroku 使用起来相当简单，但是功能非常丰富。您可以通过免费和付费的附加组件轻松扩展其功能。免费帐户每月有多达 1000 小时的活动服务器正常运行时间。Heroku 曾经与 GitHub 合作提供学生计划，但截至目前，它不接受新的应用程序。

# Pythonanywhere

[Pythonanywhere](https://www.pythonanywhere.com/) 是基于 Python 编程语言的在线集成开发环境(IDE)和 PaaS。它可以与数据库和框架结合起来创建一个完整的应用程序。

这个平台非常简单，但是非常容易使用，对初学者也很友好。它有一个免费的网络应用程序，有限的带宽和较低的 CPU 速度。尽管如此，对于不需要大量资源或可伸缩性的简单项目来说，它还是不错的。

# 雷普利特

[Replit](https://replit.com) 是一个基于 web 的 IDE，它允许你在 web 浏览器上编写、协作和执行代码。

免费计划包括团队协作、500 MB 内存、500 MB 存储和有限的 CPU 能力。您的应用程序可以立即轻松部署，并可通过提供给您的 URL 进行访问。

Replit 简单直观的设计使得远程编写和运行代码变得轻而易举。对于那些没有足够强大的计算机来同时运行 IDE、web 浏览器和本地实际应用程序的人来说，它是完美的。

# 亚马逊网络服务(AWS)

[AWS](https://aws.amazon.com/) 是亚马逊的子公司。它提供各种各样的云服务，包括服务器托管、云功能、数据库、分析、虚拟机等等。

AWS 是世界领先的云服务提供商之一，保证了高用户安全性和高可靠性。此外，与其他较小的托管公司相比，它们提供了更高的带宽和 CPU 速度。自动按需应用扩展是您在选择服务提供商时应该考虑的另一个核心特性。

众所周知，AWS 还提供非常便宜但非常棒的产品。它有许多服务的免费计划，限时免费试用，以及从初始注册日期到 AWS 的 12 个月免费计划。除了这些免费的选项之外，他们还有一个随用随付的方法，所以你不必浪费钱在你不使用的资源上。

# 微软 Azure

与 AWS 非常相似，Azure 是一种云计算服务，提供许多解决方案和产品，包括 PaaS、基础设施即服务( [IaaS](https://en.wikipedia.org/wiki/Infrastructure_as_a_service) )、软件即服务( [SaaS](https://en.wikipedia.org/wiki/Software_as_a_service) )，支持众多编程语言和技术。

Azure 也是世界上最大的云服务提供商之一，我谈到的关于 AWS 的行业领先公司的所有性能、可扩展性和安全优势也适用于 Azure。

显然，有一个免费计划，其中包括 12 个月的免费试用，外加 200 美元的信用额度，可以在第一个月使用。此外，许多产品总是免费的，或者你可以选择现收现付的模式。除此之外，如果你是一名学生，你可以利用 GitHub 的学生开发者包，并获得 100 美元的免费积分，用于第一年的订阅和许多免费服务。点击了解学生版 Azure 的更多信息[。](https://azure.microsoft.com/en-us/free/students/)

# 谷歌云平台(GCP)

[谷歌云平台](https://cloud.google.com/)是另一项领先的云计算服务，提供服务器托管、机器学习、分析、数据库等等。

像这个列表中的其他大公司一样，谷歌提供了开箱即用的可靠性、性能和安全性。您还可以获得高度可扩展的服务和现收现付的方法。

谷歌的免费计划包括 300 美元的信用额度，可在订阅后的前 90 天内使用，此外还有许多每月使用限额不会过期的产品。除此之外，学生每年可以获得高达 300 美元的学分，另外还可以无限制地免费使用教育实验室和 Coursera 上的 13 门免费课程。点击了解更多[。](https://cloud.google.com/edu/students)

GCP 也是 Google APIs 及其所有相关功能的核心。它们与 Google 的所有服务完美集成，并提供一个完整的环境来部署和运行您的应用程序。

# GitHub 页面

[GitHub Pages](https://pages.github.com/) 是一个静态站点托管服务，直接从存储库的分支提供 HTML、CSS 和 JavaScript 文件。

GitHub 页面完全免费，没有资源限制。它的最大优势是让主持项目演示网站、文档手册或个人作品集变得轻而易举。它还可以与云功能和外部 API 配对，以扩展其功能，而不仅仅是静态文件服务和有限的前端 JavaScript。

# 结论

作为开发人员，云服务提供商可以成为您最好的朋友，因为他们减轻了您自己管理服务器和安全性的负担，并提高了您的应用程序的性能。他们负责所有无聊的任务，改善开发体验。
