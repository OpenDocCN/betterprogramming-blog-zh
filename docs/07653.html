为什么是Web开发？

在这个数字时代，每天都有越来越多的公司上线。这意味着每个公司都需要一个需要开发的网站。这些网站通常是由web开发者开发的。即使是现在，你也可能在由一个web开发团队开发的web浏览器上滚动浏览媒体。在这个不断发展的数字时代，网站对企业乃至一个国家的发展起着重要作用。大多数社交媒体网站，如Instagram、Twitter、脸书，甚至YouTube，都是由web开发者创建的。

公共福利协会的兴起

现在，随着智能手机的兴起，2021年将有38亿智能手机用户，对网站有着巨大的需求。

当智能手机在2007年开始流行时，网站大多由internet explorer驱动，浏览网页并不是一种很好的体验。在那段时间里，智能手机具有GPS、摄像头和其他只有本地应用程序才能使用的功能。本机应用程序是专为特定设备构建的应用程序，可以直接安装。

现在，随着渐进式网络应用程序的兴起，如果应用程序在Play Store或App Store上，这些pwa能够提供与原生应用程序相同的功能，而无需遵循谷歌或苹果的切割。因此，随着PWAs的兴起，对网络开发人员的需求只会增加。如果你开发一个web应用，它将是跨平台的，不像创建一个原生应用。

1.HTML、CSS和JavaScript

当我们谈论web开发时，我们需要了解的主要基础知识是HTML、CSS和JavaScript。

我们来比较一下网站的要素和人体的要素。HTML定义了网页的结构，你可以把它想象成你身体的骨架。JavaScript就像大脑一样，使用它可以执行复杂的操作，如播放视频、显示交互式内容等。CSS用于给你的网页添加样式。你可以把它看作是你穿的不同类型的衣服。这是web开发入门的必备技能。

2.反应

根据 Stack Overflow 的 2020开发者调查，React是最受欢迎的web框架。

这是一个非常强大的JavaScript库，用于构建交互式用户界面。它使用基于组件的设计，因此一切都被分成组件。我们有父组件，它呈现子组件。区别于其他前端框架的特性是虚拟DOM。

真正的DOM(文档对象模型)基本上就是你在浏览器上看到的HTML结构。虚拟DOM和真实DOM是一样的。我们基本上可以假设虚拟DOM是真实DOM的副本。它具有与DOM对象相同的属性和方法，但是它不能控制屏幕上显示的内容。虚拟DOM中的操作非常快。

当代码发生变化时，虚拟DOM中的每个组件都会被重新呈现，然后与先前虚拟DOM状态的快照进行比较。只有被改变的元素会在真正的DOM中重新呈现。这个过程也被称为对账。因此，我们可以使用React构建快速高效的用户界面。

它非常简单和容易，因此它有一个指数学习曲线。

如果您学习React，您也可以通过学习React Native进入本机应用程序开发，React Native与React非常相似，并使用与React相同的基于组件的设计。如果你知道反应，学习母语是很容易的。

您当前所在的网页也使用React。

3.饭桶

无论是前端还是后端，Git对每个开发人员来说都是必不可少的。Git是一个分布式版本控制系统。所以你可能会问版本控制系统到底是什么。举个例子，假设你在手机上下载了一个应用(我们姑且认为它的版本是1.0)。一周或一个月后，根据应用程序的不同，您会在名为版本1.1的更新中获得新的额外功能。这不过是版本控制。

假设你正在做自己的项目。您构建了一个具有一些特性的项目，并且您的项目运行良好。然后可以使用Git保存这段代码。稍后，您可以继续添加更多功能。然后，您意识到您添加的新特性正在导致您的应用程序崩溃。Git允许您保存项目的不同版本，所以如果您愿意，您可以随时返回到您之前的代码。因此，它被称为版本控制系统。

4.Redux

在React中，我们有一个基于组件的设计，我们使用父组件来呈现子组件。我们将数据(也称为道具)从父组件发送到子组件。当我们需要将这些数据从父母发送给孙子或曾孙或更远的时候，我们会遇到一个被称为道具钻探的问题。这意味着我们总是需要添加所有中间子组件的数据。为了解决这个问题，我们有一个名为Redux的解决方案。

Redux 用于前端框架中的应用状态管理。我们可以通过使用Redux store 来解决这个问题，Reduxstore拥有可以从UI的任何部分访问的所有所需数据，因此可以处理prop drilling。在有大量数据的应用程序中，我们需要一个真实的来源，我们可以使用Redux。

随着React中Redux挂钩(useDispatch( )和useSelector( ))的引入，在项目中实现Redux变得前所未有的简单。

5.以打字打的文件

根据栈溢出的2020年开发者调查，TypeScript是第二受欢迎的编程语言。

" TypeScript是JavaScript的超集，它具有可选的类型，可以编译成普通的JavaScript。“是微软在2012年发布的，而且是开源的。由于JavaScript是一种高度灵活的语言，一些小错误可能会导致大问题。使用TypeScript有助于我们在编译时而不是运行时找到代码中的类型错误。TypeScript帮助我们创建具有类型的接口，并使我们的代码类型安全。

我们还可以使用JavaScript的新特性，比如可选链接和nullish合并，而不用担心兼容性，因为它将被编译成ES5 JavaScript。因为TypeScript帮助我们更快地发现问题和错误，所以它是最受欢迎的语言之一。

6.Rest API

应用程序编程接口(API)主要帮助不同的应用程序相互通信。在这个时代，我们几乎在所有事情上都使用API，因为我们的产品使用了很多服务和功能。因此，API使我们能够使用其他应用程序开发的特性和功能，并将它们集成到我们的应用程序中。

Rest代表表象状态转移。它帮助我们使用HTTP方法来创建、读取、更新和删除(CRUD操作)所需的数据。我们每天都用这些。您正在阅读这篇文章，因此您发出了一个GET请求，从服务器获取这篇文章，以便您能够阅读它。您可以假设我使用了POST请求来添加和发布这篇文章。如果需要，我可以使用PUT请求来更新本文。最后，我可以使用删除请求来删除这篇文章。因此，使用Rest API确实让我们的生活变得更加轻松，这无疑是一项重要的技能。

7.数据库ˌ资料库

我们的项目需要一个数据库。这可能是 MongoDB 、 MySQL 、 PostgreSQL ，甚至是 Firebase 。Firebase是迄今为止最容易使用和实现的。您可以使用Firebase添加新功能，如身份验证、推送通知、云功能等等。您甚至可以使用Firebase部署您的应用程序。

使用Firebase，您可以获得一个可以在项目中实现的实时数据库。有了MongoDB，您可以使用 Pusher 为您的应用程序添加实时同步。所以这是你需要具备的技能之一。
