# ******实体模型工具******

****![](img/5c829504b5412b8e1d63037599809e25.png)****

****[UX 店](https://unsplash.com/@uxstore?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍照****

****模型是一种概念工具，特别是在 web 开发中使用。它基本上是一个网站或网络应用程序的早期草案。****

****图帮助你对应用程序的外观和感觉有一个正确的概念。如果你需要一个简单的网站组件，需要整个网站的模型工具，Figma 可以满足你。****

# ******通用工具******

****![](img/35619f7f21d217ba385bada31035f278.png)****

****[斯科特·格雷厄姆](https://unsplash.com/@homajob?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片****

****一些将军的工具是:****

*   ****[电子邮件](https://www.emailjs.com/)****
*   ****[can use](https://caniuse.com/)****
*   ****[图像压缩器](https://imagecompressor.com/)****
*   ****[Css 招数](https://css-tricks.com/)****

## ****[**EMAILJS**](https://www.emailjs.com/)****

****EmailJS 允许你直接用 JavaScript 发送邮件，不需要任何服务器代码。这是非常有用的建立一个投资组合，并希望有一个联系形式，但如果你不想为此创建一个完整的后端，那么你可以使用 emailJS。****

## ****[can use](https://caniuse.com/)****

****它检查你的语法是否在所有设备和浏览器上被支持，比如 flexbox。****

## ****[**图像压缩器**](https://imagecompressor.com/)****

****当你正在部署你的网站，你想提高你的网站的加载速度，你可以压缩图像。****

## ****[**CSS-Tricks**](https://css-tricks.com/)****

****这个网站将帮助你探索新的和有趣的方法来处理 CSS。你可以找到关于制作神奇卡片和许多其他主题的信息。****
