如何安装

我用家酿送来了mimiq。请确保你已经安装了家酿软件——点击了解更多。

brew install wendyliga/core/mimiq

如何使用

您可以通过执行以下命令开始记录:

mimiq

然后按回车键停止。你可以在你的桌面上得到GIF结果。

您可以添加几个选项，比如为多模拟器设置指定模拟器。您可以通过执行以下命令来检查要记录的可用模拟器:

mimiq list

然后选择模拟器UDID，并开始记录:

mimiq --udid 6ABC274C-76BB-483B-8DD6-8A9F8E486E0F

要了解更多细节，请查看 mimiq库。
