## 为或多或少的知名公司工作了 7 年

为一家初创公司工作和为一家更成熟的公司工作可能是不分昼夜的。或者差异可以不太明显，比如黄昏和黎明。

任何公司的工作文化都没有经验法则，无论其规模大小。然而，在比较两者时，根据公司的情况，有一些地方需要特别注意。

在过去的七年里，我仔细观察了 T4 好的一面，坏的一面*，是的，还有硬币两面的丑陋的一面*。以下是那些希望进入或脱离创业环境的人应该知道的一些事情。无论如何，这些都不是普遍真理。相反，它们是我观察到的模式。肯定会有很多公司不适合下面的评估。话虽如此，这里去…**

# **钱钱钱**

**![](img/11a81e1c3a88b033c4f87c805a6ce19b.png)**

**照片由 [Fabian Blank](https://unsplash.com/@blankerwahnsinn?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 在 [Unsplash](https://unsplash.com/s/photos/money?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上拍摄**

**实话实说吧。不管你对工作有多热情，工作是达到目的的手段；也就是为了谋生。当然，这并不意味着你不能在你的手艺上投入太多。也许你的公司从事可再生能源，也许它向慈善机构捐款或奉行企业社会责任政策，但归根结底，公司需要盈利才能生存并支付员工工资。**

**成熟的公司通常是自给自足的。他们有固定的客户群，或者相当稳定的收入来源。当你寻求财务稳定时，这无疑是风险较小的选择，尽管可能不那么令人兴奋。晋升和现金奖励，如果发放的话，往往会符合公司的标准，比如根据你所在公司或项目的表现，每年加薪 5%或发放年终奖金。**

**另一方面，初创公司通常在一开始没有足够的利润来支付运营成本。每一个创业公司，无论产品或愿景多么伟大，都是一场赌博。你无法控制的因素太多了，每个行业都不一样，你无法对未来的成功做出任何结论性的陈述。**

**为初创公司工作，这种冒险既是危险的，也是诱人的。大多数创业公司都会失败，原因多种多样，*但是*如果一家创业公司成功了或者被收购了，你就能大获全胜。请注意:胜算对你不利。**

**还有另一个有时被忽视的不确定性需要注意。如果一家初创公司做得很好，并开始自给自足，它肯定会经历成长的烦恼。**

**成功的创业公司对投资者或董事会负有责任，一旦红色数字开始变黑，人情可能需要偿还。在成为可持续发展企业的跑道上，有些决策需要快速做出。在这方面，一旦公司有了喘息的空间，可能需要降低成本或优化利润，这通常会导致在产品完成或服务启动、运行并发挥作用后裁员。请注意，尤其是如果你不是核心团队的一员(也就是可有可无)，你可能会成为*内部重组的受害者，*就在你认为你帮助创立的公司发展良好的时候。最糟糕的是，你无法与解雇你的人交谈、协商，甚至责怪他们，因为他们也可能不是出于自愿。**

# **工作文化**

**![](img/5fb3dac02b70068683d8c3bd58785f0b.png)**

**在 [Unsplash](https://unsplash.com/s/photos/foosball?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上[鲍炯](https://unsplash.com/@ba0_tru0ng?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText)的照片**

**不足为奇的是，创业公司的工作文化通常更加宽松。**

**着装规范、守时，甚至在某种程度上，你投入的时间都不如完成任务重要。只要你及时有效地完成你负责的每一件事，人们就会很高兴。这可能是福也可能是祸。**

**在没有人监督你的情况下，你必须站出来。创业生活并不容易——事实上，你的工作量可能会超出要求。取决于你创业的行业，这个领域可能会竞争激烈。这意味着你的雇主经常会改变策略——雇佣你是希望你能调整。从这个意义上说，用简单性、可扩展性和模块化的思想来构建系统也更重要，以便能够相对容易和快速地从一个部分切换到另一个部分。所有这一切的另一个潜在表现是，你和你的团队可能会比大公司的同事更加紧密(但即使在不同的创业公司之间也可能有所不同)。**

**另一方面，更大的雇主可能会要求你穿商务休闲装，更准时，更严格地遵守编码标准。后者尤其不是一件坏事——随着团队的成长，结构变得更加重要。除此之外，你可能还会有经理和绩效评估会议，这在初创公司是没有保证的。根据你创业公司的资金情况，你甚至可能不得不[成为自己的项目经理](https://levelup.gitconnected.com/software-development-sans-project-manager-3924c43fdc6a)。**

# **增长潜力**

**![](img/1c29473fb92ef2f33248fe299a9ee382.png)**

**安妮·斯普拉特在 [Unsplash](https://unsplash.com/s/photos/growth?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上的照片**

**求职时，公司如何促进职业和个人成长是一个重要的考虑因素。一个好的雇主会在这两方面都有所帮助。**

**在更大、更成熟的公司中，这通常以*晋升*的形式发生，但也可能有其他机会，例如让员工完成云、QA 或其他技术认证，邀请他们代表公司参加世界各地的开发者大会，或者给他们一笔津贴，用于购买例如编程书籍或其他有助于他们更好地工作的东西。**

**根据公司参与的工作类型，它还可以让员工决定他们希望参与哪些项目——如果后端开发人员可以加入项目的前端团队，他们就可以这样做。**

**创业公司可用于员工成长的资源往往更少。是的，可以有促销，但在利润或投资者数量有限的地方很少。创业公司的闪光点是你将被雇佣去做的工作类型。初创公司通常使用尖端技术，而非初创公司倾向于坚持使用已经存在更长时间、经受住时间考验或拥有更大社区的工具。获得最新框架等方面的实践经验可以让你在一个新兴领域有一个良好的开端。**

**换句话说，当在较小的团队中工作时，人们可以与同事建立更好的关系。如果你想扩大你的职业网络，而不仅仅是泛泛之交，这将是一个很大的好处。你可能会在下班后和你的团队一起出去喝酒的时候遇到并真正了解某人，而不是在演讲之间的五分钟休息时间在昂贵的会议上认识新朋友，从一个阶段跑到另一个阶段。**

# **管理**

**![](img/5d3f4dbb2c347ccd86b5617d4797af4e.png)**

**照片由[德鲁·比默](https://unsplash.com/@drew_beamer?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText)在 [Unsplash](https://unsplash.com/s/photos/meeting?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上拍摄**

**自然，落户的公司有更多的结构。这种结构有多种形式，其中之一就是等级制度。这些可能或多或少是平的，但在绝大多数情况下，你会有一个经理、主管或类似角色的人。也可以有几个经理，负责不同的工作领域，比如一个总经理和一个技术经理。除此之外，可能还有项目经理、团队领导和其他各种头衔的人，他们都包含类似的职责。**

**和清单上的其他项目一样，这既不是好的*也不是坏的事情——舒适的结构取决于许多个人和职业环境，每个人都不一样。如果你喜欢有明确的需求和结果的定义良好的标签，那么更多的结构可能更适合你。***

**另一方面，创业公司通常在你如何处理事情上给你更多的自由。会议经常不定期召开，没有人告诉你应该如何解决问题，有时你甚至不得不帮助找出一家初创公司应该追求的总体商业战略。门票可能会有一个空白的描述，你必须找出并填写。**

**这种结构和自由之间的妥协往往会定义创业文化，即使在不同的创业公司之间，也会有很大的不同。加入新公司或帮助创建新公司的部分吸引力在于，你对这种妥协应该是什么样子有发言权。这对团队来说是一个持续的挑战，也是一次很好的学习经历，这将因团队的组成而异。**

# **利益**

**![](img/01c12a6a82d32a26d7631b05ed7a7809.png)**

**丹尼尔·弗兰克在 [Unsplash](https://unsplash.com/s/photos/dentist?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上的照片**

**一家没有稳定收入来源的公司可能无法为员工提供福利。这里没什么可说的了。如果你不喜欢风险，或者需要医疗保险等福利，那么现在一家年轻的初创公司可能不适合你。**

**大公司肯定会提供好处。然而，对于一些人来说，在初创公司获得一份体面的薪水可能是容忍福利缺失的足够理由。也许你绝对肯定创业即将起飞，你投入的时间和精力即将得到回报。但无论你的理由是什么，这个决定都不应该轻易做出。**

# **回顾往事**

**![](img/34915f1cca7a9601f354b95104bfc1ad.png)**

**照片由[莫尼克·卡拉蒂](https://unsplash.com/@moniquecarrati?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText)在 [Unsplash](https://unsplash.com/s/photos/chocolate?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 拍摄**

**回顾过去，我很享受在初创公司和大公司的时光。**

**当然，这只是我的主观经验，但对我来说，我是否会选择创业公司，总是取决于我的个人情况。我需要的稳定性越少，我就越享受创业生活。在创业公司，输赢往往比其他地方更重要。**

**最后，人们需要意识到一个简单的事实:大多数创业公司都会失败。他们可能不会很快失败，他们可能会存在一段时间，但尽管如此，大多数创业公司都没有成功。原因也是多方面的，而且往往完全无法控制。**

**在不再是初创公司的公司里，生活也可以是愉快的。有时候，这些工作与创业公司可能使用的技术一样令人兴奋，也一样使用尖端技术。初创公司通常缺乏的结构可能是一把双刃剑，但我总的来说发现它的利大于弊。随着我在职业和个人方面的发展，我发现结构变得越来越重要，但我认为这种情况会随时改变。**

**生活就像一盒巧克力，你的控制永远不是绝对的。灵活变通——拥有 B 计划和 C 计划——将是你在进退两难时避免溺水的生命线！**
