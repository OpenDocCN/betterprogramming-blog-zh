我是谷歌新加坡公司的高级软件工程师，经常有人问我应该学习哪种编程语言

有时候这个问题表现为“我应该学Java、Python还是JavaScript？”

老实说，我是在阅读了埃里克·冯·赖因哈德在文章《2022年最好学的编程语言？—谷歌首席开发人员解释。事实上，我决定写这个是因为我对这个问题有非常不同的看法，所以请不要介意灵感的标题。

哪一个？

我推荐“没有具体的一个”。

不管你是从C++、Java、Python、Javascript还是其他语言开始。学习一门新语言不需要很长时间，也不应该是首要任务。

编程语言是达到目的的手段(我再说一遍！)在实践中，我们倾向于不断切换到一个或另一个来完成我们的工作。

我在高中开始了我的C++之旅，在大学花了很多时间做Javascript。我在微软的工作要求我做大量的C#/。Net和Typescript，当我切换到谷歌时，我开始开发Android相机应用程序——所以我开始使用Java。很快我转向了相机的计算摄影方面，这些天我写了很多C++(产品代码)和Python来做原型。

在整个过程中，学习一门新语言的成本从来都不高。

我不得不承认，随着时间的推移，我们已经在如何使用编程语言方面建立了不同的“种类”。一些语言纯粹用于标记，如HTML或XML，而其他语言纯粹用于脚本，如shell脚本或bat脚本，一些语言本质上是功能性的，其他语言不是，等等。

当从一种“类型”转换到另一种类型时，可能会涉及到一定的学习成本。对这些“种类”有了基本的了解，学习一门新语言的这个障碍就会变得很薄。

等等…那我们关注什么？

在我看来，最重要的技能是——解决任何给定问题陈述的能力

  • 如果它应该是一个web应用程序，你能构建它吗？
  • 你有一个安卓应用，能提升性能吗？
  • 你有一个潜在的问题，可以通过一个训练有素的神经网络来解决——你能继续做下去吗？

即使你以前“从来没有”做过！以及完成任务所需的任何编程语言！

最后，感觉都是一样的，所以继续努力，专注于学习新技术，尝试更深入地挖掘看起来像魔术的概念，在开始时，尝试探索许多不同的类型，直到所有看起来都一样。最终，挑一个自己真正喜欢的，深入下去！

好的，但是一些具体的建议会更好！

对于职业生涯不同阶段的程序员，我有不同的推荐。

如果你刚刚开始学习编程

我建议选择一种你想从事的技术——可以是web，可以是移动(Android / iOS)，可以是游戏开发，可以是其他应用程序开发，可以是训练ML模型，构建桌面应用程序等等。

但关键是要拿起一项技术，在那里学习相关的语言。如果可能的话，通过构建随机软件来学习，并在这上面花些时间。如果可能的话，继续质疑你所看到的不同魔法实际上是如何工作的。

最后，开始挖掘你正在使用的库所抽象的更基本的概念，如并发或多线程、数据库、图形渲染、图像处理、网络等。

你看，我们用任何语言编写的程序要么直接编译成汇编指令，要么编译成中间字节码，最终再编译成汇编指令供硬件执行。如果你接受这个事实，不同语言的语法将开始看起来非常相似。

某个需求，比如说“将数组的每个值增加1”可以用不同的语法来处理:

在Python中:

y = [i + 1 for i in x]
## or
for i in range(0, len(x)):
  x[i] = x[i] + 1

在JavaScript中:

for (var i = 0; i < x.length; ++i) x[i]++;
// Or
var y = x.map(val => ++val);
// Or
var y = Array.from(x, val => ++val);

在Go中:

for i:= range x {
  x[i]++
}

所有这些或多或少会编译成这样的内容:

increment(int*, int):
  cmp     w1, 0
  ble     .L1
  mov     x2, x0
  add     x1, x0, w1, sxtw 2
.L3:
  ldr     w0, [x2]
  add     w0, w0, 1
  str     w0, [x2], 4
  cmp     x2, x1
  bne     .L3
.L1:
  ret

在这一系列示例中，更有趣的是，其中一些会导致数组复制，而另一些不会——我建议重点学习更多关于这些类型的问题及其对程序的影响。

如果你有经验并且精通某些编程语言，但是觉得你错过了

我还是会推荐同样的方法——退一步想想让你兴奋的问题陈述。然后思考你需要学习的新方法。

例如，在使用相机应用程序时，我有机会在相机应用程序中使用“夜间模式”，这需要我们捕捉一系列图像，并以稳健的方式将它们合并，以产生曝光良好的低噪声图像。这在计算上非常昂贵，我们需要我们的算法在一定的延迟预算内在资源受限的硬件上运行。

这对我来说是一个非常令人兴奋的问题陈述，要求我重新学习现代C++，图像处理概念等。

但是这太有趣了，花在学习上的努力并不觉得是一件麻烦的事。我建议找到这样的方向，开始学习需要付出的一切。

TL；DR；不要费心去寻找一种编程语言，它将是你2022年的赌注。
