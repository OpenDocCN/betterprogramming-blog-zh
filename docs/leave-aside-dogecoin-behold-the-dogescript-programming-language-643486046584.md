# 让我们开始编码吧

每个人都知道对您的代码进行注释是一个很好的实践:

```
shh good doge
```

我们刚刚在 DogeScript 中创建了一个评论。如果你还有话要说呢？

```
quiet
  much say
  doge to the moon 
  barkingly wow
loud
```

这就相当于经典的:

```
/*
  boring comment
  that has
  multiple lines
*/
```

让我们进入一些更严肃的东西，比如声明变量和操作:

```
shh here we are declaring our variables
very bigDoge is 5
very smallDoge  is 3
very allDogeshh this is how we create a sum function
such getAllDoge much a b
wow a+bshh and here we clearly see 8 will be the value of allDoge
allDoge is plz getAllDoge with smallDoge bigDoge
```
