# 该领域的当前阶段

量子技术并不是一个新领域。最近媒体和不同公司的关注可能使它看起来是这样，但事实是量子计算的发展始于 20 世纪 80 年代初。

从那以后，这个领域取得了长足的进步。今天，你可以用 Python 编写一个实际的量子计算机，并得到一些结果。

是的，结果不会很准确，因为硬件的当前阶段是非常有缺陷的，但你可以得到结果的事实将该领域从理论推向了实践。在实际的量子硬件上开始编程和运行代码，这是一个非常令人鼓舞的事实。

2017 年，IBM 发布了 Python 库 [Qiskit](https://qiskit.org/) (量子信息科学工具包)。这个库允许您编写可以在云上的 IBM 机器上执行的 Python 代码。这意味着你可以为现实生活中的应用实现许多流行的量子算法，如 Shor 和 Grover，并在由 [Slack workspace](https://medium.com/u/12b2fb01d834#/) ，其中包括来自开发团队、教授和该领域的研究人员，以及来自所有年龄和背景的好奇人士。这个 Slack 工作空间非常活跃，对新手很有支持。

除了 Slack，还有许多[脸书团队](https://www.facebook.com/groups/qinfo.scientists.unite)分享该领域的最新研究论文以及来自世界各地的任何即将到来的网络研讨会。技术论文的作者通常非常欢迎读者的问题和评论，并乐意尽他们所能提供帮助。

# 可用资源

由于各种年龄和背景的人们越来越关注，已经开发了许多材料，并且正在开发许多材料来简化复杂的概念，并以易于掌握的方式传递它们。

这些资源大部分都是免费的，可以从任何地方访问。Qiskit 有自己的 YouTube 频道，其中包含如何使用该库的视频以及涵盖不同应用的量子计算基本概念的视频。

回到 2020 年 8 月，IBM 提供了为期两周的虚拟暑期学校，涵盖了从量子计算基础到其在化学领域的应用的所有内容。这个暑期学校的[录音](https://www.youtube.com/playlist?list=PLOFEBzvs-VvrXTMy5Y2IqmSaUjfnhvBHR)现在每个人都可以在网上获得。

IBM 并不是量子计算学习资料的唯一来源。你可以在 [edX](https://www.edx.org/learn/quantum-computing) 、 [Coursera](https://www.coursera.org/learn/quantum-computing-algorithms) 和 [FutureLearn](https://www.futurelearn.com/courses/intro-to-quantum-computing) 上找到许多提供的课程，你可以免费旁听，学习很多关于量子计算的基础甚至高级主题。

麻省理工学院也在它的[开放式课程](http://www.mit.edu/~aram/advice/quantum.html)上提供了关于量子计算的各种主题的课堂讲稿。

# 近期的需求

量子计算是一个快速发展的领域，每个人都试图建立完美的硬件，为当前的算法找到新的应用，甚至开发新的算法。正因为如此，不久的将来对量子程序员和研究人员的需求将会增加。

许多政府和工业机构已经拨出大量资金来开发量子技术。[量子日报](https://thequantumdaily.com/) (TQD)估计目前量子计算的市场大约为 2.35 亿美元。预计到 2025 年，这一数字将大幅增长至 62.5 亿美元。

这种难以置信的资金数量导致学术界、政府和行业职位数量的增加。几乎所有的技术公司都在改变他们的商业模式，以适应量子技术产生影响的时候。

TQD 还补充说，美国劳工统计局估计，到 2020 年为止，量子软件开发职位比申请人能够填补的职位多出约 140 万个。

2019 年，麻省理工学院发表了一篇名为[“Q&A:量子计算人才短缺”](http://news.mit.edu/2019/mit-william-oliver-qanda-talent-shortage-quantum-computing-0123)的文章，解决了该领域目前面临的不同挑战。后来，它开发了 MIT xPRO，这是一个解决现实问题的小组，学生不是唯一对学习量子信息的不同方面感兴趣的人。

# 最后的想法

量子技术现在正处于它的黄金阶段——还没有开始，还没有完全发展。它目前正从理论走向实践，为那些对量子技术感兴趣的人提供了如此多的工作和机会。

如果你曾经想知道量子计算机如何工作，如何编程，或者甚至好奇想了解更多，现在是时候让你参与并加入这个领域了。

这个领域非常开放，欢迎新的人才和好奇的人，他们想投身其中，但担心自己可能没有所需的背景，或者只是认为自己不够聪明，无法进入量子计算机。

作为一个去过那里并有这些疑问的人，我告诉你，这个社区从来没有让我觉得自己是个骗子，每个人都欢迎并支持我和我的问题。

所以，没什么好怕的。如果你想做，那你就可以。你所要做的就是大胆一试。
