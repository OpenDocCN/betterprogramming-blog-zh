# 升华文本键映射和设置导入程序

![](img/7015049adaa3738a6baee2e619c766a9.png)

[https://marketplace.visualstudio.com/items?itemName = ms-vs code . sublime-key bindings](https://marketplace.visualstudio.com/items?itemName=ms-vscode.sublime-keybindings)

该扩展将 Visual Studio 代码绑定转换为与 ST3 相同的。旋转一下，试试这个:

`cmd ⌘ + P` — Mac

`ctrl + P` —视窗

![](img/0b6806200c449569b2e198f9b1bb430b.png)

你可以进入文件，如果你在搜索前添加了`>`符号，你就可以搜索操作——比如切换内置终端、安装扩展等等。

对于我们这些习惯了 ST3 捷径的人来说，这是一个巨大的胜利。
