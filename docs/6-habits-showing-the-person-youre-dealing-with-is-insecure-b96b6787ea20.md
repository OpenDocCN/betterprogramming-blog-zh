# 为什么情绪不稳定的人很少成为有效的领导者

## 能够发现人们情绪上的不安全感将有助于你采取措施保护精神和情绪健康

![](img/af49d5eabea774b1ee852ca4fe64fcc7.png)

与缺乏安全感的老板、经理或领导一起工作或相处会对你的精神和情感健康产生负面影响，让生活变得难以忍受。

情绪上的不安全感代表了一种紧张的感觉，这种感觉可能是由感觉到自己在某些方面脆弱或自卑而引发的。

当一个人情绪不稳定时，他们通常不能有效地管理自己的情绪来应对挑战、压力和压力源。

正因为如此，他们倾向于从事一些行为，这些行为会使环境变得有毒，并妨碍他人的工作效率。

在他的书 [*建立你的领导力简历*](http://www.leadershipnow.com/leadershop/9780805449648.html)*中，约翰尼·亨特概述了缺乏安全感的领导者的典型行为，这些行为会削弱他们提供有效领导的能力。*

# *他们很少信任别人*

*给予信任是正确的，但对于缺乏安全感的领导者来说，这并不容易。*

*缺乏安全感的领导者认为表扬是一种无法收回的成本。他们认为给予会让他们失去一些东西，或者减少他们的价值。*

*相反，赞美别人是回馈所有人的礼物。*

*当一个领导者欣赏团队正在做的事情时，就会提升他们的动力，鼓励他们更加努力。*

> *“事情不顺利的时候责怪自己，事情顺利的时候把功劳归于别人。建立团队需要的是给予他人信任的过程。”*
> 
> *—弗兰克·伦茨*

# ***2)他们隐瞒信息***

*为了有效地完成工作，人们需要获得他们需要的所有信息。他们需要知道他们的责任是什么。*

*事实上，不仅仅是意识到这些信息，他们还需要被领导者尽可能清晰连贯地传达。*

*缺乏安全感的领导经常对下属隐瞒信息。他们通常不太信任自己的员工，不会将信息保密。*

*如果不是这样，人们就不知道对他们的期望是什么。这限制了他们的表现，损害了组织。*

> *“当你发布信息时，你向他人传递了信任和信心。当你隐瞒这一点时，你传达的信息恰恰相反:没有信任，没有信心。”*
> 
> *约翰尼·亨特*

# *他们是微观管理者*

*微观管理是领导者控制组织中所有事情的倾向。*

*缺乏安全感的领导什么事都插手，很难授权。他们无法理解下属在没有他们同意的情况下采取主动。*

*   *他们想让别人知道他们是负责人。*
*   *他们想知道每个人的下落。*
*   *他们希望总是对别人做的事情有第二种看法，“只是为了确定。”*

*这创造了一个充满压力的环境，使人们失去动力，效率降低。*

# *他们太需要表扬了*

*虽然他们自己不能给予赞扬，但他们要求得到赞扬。他们追求它，当他们得不到它的时候经常立案。*

*正如约翰尼·亨特所说:*

> *当有人需要他的追随者总是告诉他他有多棒时，他的工作与领导的心跳完全相反——融入他人的生活。*

*为什么他们渴望表扬？是因为他们“我或我”的心态。他们认为如果没有他们，什么也做不成。因此，他们告诉自己，他们应该为自己的努力获得荣誉。*

# ***5)他们不为他们领导的人提供安全保障***

*人们需要一个安全的环境来保持最佳状态。*

*这并不意味着他们有安全官员安装警卫。相反，这是一种有人支持他们的*感觉。**

*当办公室的气氛和环境充满恐惧、事后批评和自我怀疑时，这通常是不自信的领导者通过激发恐惧来领导的表现。*

# *他们拿的比给的多*

*缺乏安全感的领导者是索取者。*

*沃顿商学院管理学教授亚当·格兰特是第一个将关系分为三类的人——给予者、接受者和匹配者。*

*对于接受者来说，任何关系都是零和的；关键是他们能得到多少，而其他人得到的很少或没有。*

*他们的利益和需求比任何人都重要。对他们来说，领导力是交易性的，而不是关系性的，在这种情况下，其他人不被视为人，而是被用来促进个人利益的工具。*

*如果你不能再提供给接受者他们想要的东西，他们会很快处理掉你，然后去找其他能提供的人。就这么简单。*

# *最后的想法*

*不安全感不利于有效的领导，并阻碍建立真正的关系。*

*最成功的领导者是那些能够控制自己情绪并能够帮助他人控制情绪的人。*

*他们不会通过让其他人难以发挥作用来限制他们的组织，也不会在履行职责时发挥最佳水平。*
