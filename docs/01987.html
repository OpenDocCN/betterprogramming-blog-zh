数组

数组是JS中最常用的数据类型之一，幸运的是，我们有一套非常好的方法来使任何数组操作变得轻而易举。

目标

“在JavaScript中，对象为王。如果你理解对象，你就理解了JavaScript”——W3Schools

JS中的一切都可以被认为是一个对象。但是在这种情况下，我们将像在普通JSON object lit中一样讨论对象上的方法。

创建和复制对象

Object.assign()方法可用于将所有可枚举的自身属性值从一个或多个源对象复制到一个目标对象。

此方法也可用于将现有对象克隆到新对象中。

如果您正在处理嵌套对象，您应该注意到Object.assign() 将只创建一个浅拷贝，它将复制顶层属性，但是嵌套对象在原始对象和复制对象之间共享，即复制对原始对象的引用。

解决JS中这一缺点的最快方法之一是将JSON字符串化，并在将其分配给新对象之前再次解析它。

这给了我们一个包含所有嵌套属性(除了方法)的原始对象的副本，并且它创建了这个副本而没有引用原始对象。因此，对新对象所做的任何更改都不会反映在原始对象中。

注意:其中一些方法可能无法在IE等传统浏览器中运行。检查浏览器兼容性，如果需要支持这些浏览器，请确保在项目中使用transpilers或polyfills 。
