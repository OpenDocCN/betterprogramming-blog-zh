## 斗争 5——人们认为他们缺乏领导素质

我在职业生涯中与两种类型的安静程序员合作过:一种选择维持他们的冷静，保持安静；另一个人觉得大声说话有挑战性，因此保持沉默。

第一类程序员对自己的能力有很高的自信。他们知道什么时候大声说话，获得适当的知名度，并抓住最好的机会。在事业上取得成功对他们来说不是问题。

然而，第二类开发人员在他们的工作场所面临许多障碍。他们发现很难表达自己的想法和担忧。最终，他们安静的天性使他们很难在事业上取得进展。

让我们来看看后一类安静程序员的日常挣扎。

# 1.他们在会议中没有发言的空间

很难想象一个软件开发人员的工作日没有任何会议。团队成员聚在一起讨论日常状态，计划发布，集思广益，计划活动，并执行更多您需要发言来分享您的观点的活动。

随着在家工作的情况继续存在，即使是早期的桌面讨论也转向了全面的 Zoom 通话。至少在物理会议上，你可以观察他人的肢体语言，并计划何时发表声明。通常在虚拟会议中，你没有这种优势。

安静的程序员甚至觉得参与面对面的讨论都很有挑战性。想象一下，在虚拟通话中，没有人能看到他们想说些什么，这对他们来说是多么艰难的事情。

我们公司都有这样的人，他们不停地说话，不在乎别人的时间和观点。至少可以说，这些情况对于安静的开发者来说是不方便的。他们发现自己很无助，不得不努力腾出空间来表达自己的观点。

# 2.他们的工作没有得到关注

软件工程的职业很像娱乐业。展示你的才华的演讲技巧对于这个职业的成长是绝对必要的。你越能展示你的想法，突出你的能力，你就能抓住越多的机会。

这对所有开发人员来说都是真实的，不管他们在行业中有多少年的经验。你需要不断地与你的领导、客户、业务领导等互动。，为自己和工作赢得更好的知名度。但这是安静的程序员最挣扎的地方。

假设两个人的技能和知识相当，经理会更喜欢说话总是很自信的开发人员。这些开发人员可以毫不费力地传达信息。这样的机会对于创造一个最终支持开发人员职业抱负的良好印象至关重要。

一个安静的程序员可能擅长演示，但他们的经理不会冒险和一个很少在会议上发言或参与讨论的人在一起。由于这个原因，安静的程序员很难有机会展示他们的才华。这种机会的缺乏阻碍了他们的职业发展。

# 3.人们认为他们懒惰

安静不等于懒惰。以我的经验来看，安静的程序员都很努力，很真诚。他们专注于分配给他们的任务，不会试图打断他们周围发生的每一次谈话。

然而，并不是每个人都对安静的开发者有相同的看法。保持冷静并远离大多数讨论会让其他人认为他们缺乏主动性。这种看法反过来使人们相信安静的程序员不想承担责任。

对于不爱说话的程序员来说，在办公室和其他人交往是很耗费精力的。他们更愿意把精力花在富有成效的工作上。当团队中的其他人利用休息时间围坐在咖啡桌旁聊天时，他们喜欢阅读一些新的东西，或者了解他们每天的技术更新。

远离社交活动的偏好使得安静的程序员被团队孤立。他们无法与办公室里的其他人分享自己的观点和兴趣。因此，人们很容易认为安静的程序员只是懒惰。而这种偷懒的形象伤害了他们的绩效考核和工作分配。

# 4.他们发现很难分享他们的想法

安静的程序员寻求避免冲突。他们非常专业，并期望从别人那里得到同样的礼貌。但在这个竞争激烈的职业中，很难指望每个人都投桃报李。

通常，雄辩者会接管讨论。他们善于感同身受地表达自己的想法。他们的自信和表达方式使他们很容易分享自己的想法，并说服他人同意他们的观点。

有时，我看到安静的程序员大胆发言，提供他们对主题的观点。但通常，他们的观点会被讨论中声音更大、分量更重的人提出的论点所取代。

如果不能分享他们的想法，安静的程序员会感到没有动力。他们认为团队不重视他们的意见。最终，他们在未来的讨论中不再发表自己的观点，这进一步使他们与群体隔离开来。

# 5.人们认为他们缺乏领导素质

在软件工程专业中，领导素质是非常重要的。它可以是技术领导、职能领导或人员领导能力。如果你有其中的任何一个，你一定会在你的职业生涯中蒸蒸日上。

即使你更愿意成为一名个人贡献者，成长轨迹也包括诸如技术领导或架构师这样的职位。这些职位需要领导技能。为了提升你的职位，你的经理需要相信你的领导能力。

要成为任何领域的领导者，你需要与人沟通，并有能力不受阻碍地分享你的想法。但是正如我们已经知道的，安静的程序员很难在一群人中表现自己。他们也不喜欢自我推销的想法，不管它有多微妙。

成为任何领域的领导者也意味着你必须不断地和许多人一起工作。而当你与人共事时，必然会有对立的想法和摩擦。但是安静的程序员天生就在冲突处理中挣扎。

# **最终想法**

如果你是一个安静的程序员，在你的工作场所面对以上的挣扎，你就能理解我在说什么。由于很少或没有可见性，你的职业发展速度将比拥有类似编程技能的同龄人慢得多。

安静的程序员在编码和高质量交付方面绝不逊色。我和几个有这种特质的程序员一起工作过，他们绝对令人愉快。因此，如果你对自己的职业发展感到满意，就没有必要改变。

然而，安静的程序员很少会成长到超过某个水平。如果你觉得你的成长由于你的个性而停滞不前，你需要提高你的知名度。这可能需要一些时间，但你可以克服任何挑战，用毅力实现你想要的职业目标。
