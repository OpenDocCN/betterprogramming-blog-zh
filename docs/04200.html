用这些新鲜而有创意的应用程序想法来练习你的编码技能

要获得优秀的编码技能，必须付出努力和时间。为什么不今天就开始呢？从下面的列表中选择一个你觉得有趣的想法，然后开始努力。

事不宜迟，这里有一个创造性的有趣的编码挑战列表，你可以从今天开始编码。

涂鸦地点克隆

doodle-place是一个由动画涂鸦居住的网络世界。你可以四处逛逛，看看世界各地的用户创作的涂鸦，或者贡献你自己的涂鸦。

你将从构建涂鸦克隆中学到什么

  • 如何使用 JavaScript Canvas API 在屏幕上绘制图形，比如涂鸦
  • 如何使用2D图形和 WebGL API
  • 处理用户输入，例如绘制涂鸦并将结果保存在数据库中

将CSV文件解析成JSON

这个应用程序使用d3-dsv(客户端)解析CSV文件。d3-dsv是一个实用函数，作为 d3 (一个流行的数据可视化库)的助手，但也可以单独使用，如此处所示。

通过构建CSV-into-JSON解析器，您将学到什么

  • 如何使用不同的数据源，比如CSV和JSON，以及如何解析每组数据
  • 亲身体验d3-dsv图书馆。这里是d3-dsv 文档。

阻止站点浏览器扩展克隆

网站拦截器是一个网页扩展，允许你通过添加一个准确的域名来拦截一个或多个网站。享受没有干扰的额外时间。

您将从构建站点拦截器克隆中学到什么

  • 所有关于如何建立浏览器扩展
  • 用于构建浏览器扩展的JavaScript、HTML和CSS
  • Chrome/Firefox/Safari浏览器市场如何运作

概念克隆

概念是一个轻便、快速、不受干扰的软件，可以用来记笔记等等。我用它来记录我的任务和想法。我喜欢的是对编辑器的降价支持。

没有任何限制；你可以构建桌面版本的概念，以及移动或网络。挑一个你最喜欢的环境或者你最想学习的环境。

通过构建概念克隆，您将学到什么

  • 如果你想更进一步，使用 CSS网格。CSS grid非常擅长创建布局。
  • 使用数据库。你需要将笔记存储在数据库中，以便以后再次获取。
  • DevOps。完成后，将应用程序部署到web或应用程序商店。向世界展示你的成就。

苹果播客克隆

用于iOS和macOS的播客应用使用应用底部的导航图标，可以轻松访问流行和特色播客。

你将从构建苹果播客克隆中学到什么

  • 如何从API获取数据？数据是这个应用程序的播客。使用此API获取所有播客。
  • 如何构建mac OS桌面或iOS应用并部署到应用商店。如果你喜欢为网络而建，那也很好！
  • 本机布局如何工作，例如，如何在屏幕上呈现内容。
