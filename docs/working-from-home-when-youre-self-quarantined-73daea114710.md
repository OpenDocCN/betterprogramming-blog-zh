作为一个内向的人，我不怎么出门。不管怎样，我在过去几周与人交流时注意到一件奇怪的事情:在所有白领工作中，在家工作是仅次于新冠肺炎的第二大热门话题。

全球各地的公司越来越多地将重点转向远程工作，以限制传染性冠状病毒的传播。世卫组织已正式宣布冠状病毒为疫情。

虽然有些人对在家工作的前景感到兴奋，但也有一些人担心不确定的持续时间。长期自我隔离的想法会令人沮丧。

我就直说了吧:在家工作有时会感觉像住在监狱里，这显然会令人沮丧。为了避免这种心态，你需要接受不适，保持积极，在社交生活中付出更多努力。

# 为什么是我？

我一生中的大部分时间都是作为一名自由职业者远程工作，尽管不断变化的企业工作文化模式今天没有影响到我，但我确实不时会经历在家工作的高潮和低谷。

我花了一些尝试和错误来找到我的最佳状态，并找出最适合我的方法。

对于门外汉来说，在家工作是一把双刃剑。看似有利可图的提高生产率的交易(通过节省通勤时间)往往也会带来分心和孤立。除了手头的主要工作，它还考验你的耐力和组织能力。保持自律和理智——尤其是当你长期处于其中的时候——可能是一个挑战。

我总结了一些技巧，这些技巧应该有助于让远程工作成为每个人的成功体验，从刚刚起步的人到像我这样的老手。

# 早点开始

当你在家工作时，你节省了通勤的宝贵时间。起床开始工作很容易，但那只是懒惰。

早点开始，就像你去办公室一样。利用没有通勤时间的机会进行中等强度到高强度的锻炼。这对你的身心健康都有积极的影响。虽然锻炼有助于增强你的免疫系统，提高力量和疾病预防，但很少有人意识到它带来的认知益处。

提高注意力和集中力，更好的记忆力，降低压力，增强精神耐力和创造力，加速学习只是锻炼带来的一些心理健康好处。许多人在工作中经历焦虑和抑郁。疲劳和倦怠在当今所有职业中都很普遍。进行一次良好的锻炼有助于减少这种可能性，同时也为一天的工作开了一个好头。

# 设定你的规则

在家工作模糊了工作和个人生活之间的界限。

有了办公室，比平时工作更长时间是很容易的。过度工作不仅对你的健康有害，还会降低你的工作效率。每个人都需要暂停——为了家庭，你的个人生活，或者只是为了补充能量。

为了避免过度工作，你需要设定一个明确的完成时间。向你的经理和同事明确截止时间，并收拾好你的工作工具，就像你离开办公室时一样。设定一个每天结束时的例行程序，在其中检查你的邮件或为第二天建立待办事项清单会有所帮助。

在个人方面，你需要确保你家庭的其他成员在同一页上。将他们聚集在一起，建立你的工作边界。为了避免不合时宜的打扰，让他们知道你安排了会议和电话。这无疑有助于减少工作与家庭的重叠。

# 找到你的空间

工作表现和你所处的环境有很大关系。虽然有些人(包括我)可以整天穿着睡衣工作，但其他人可能会发现这是一种分散注意力的方式，并且只需点击一下鼠标就能体验到网飞的低效率。

找到一个能让你专注于工作的空间至关重要。不需要固定的地方。在家工作会导致幽居病。如果搬家能给你一个有利于更好工作的环境，那就去做吧。如果你在沙发上感觉很舒服，就使用它(尽管从长远来看，符合人体工程学的椅子更好)。如果建立一个类似你办公室的指定工作场所能提高你的注意力，那就去做吧。你的家给了你尝试不同工作空间的奢侈。

# 起床，休息，孵化

在传统的办公室里，每个人都会时不时地喘口气——尤其是当他们需要理清思路的时候。无论是散步还是玩桌上足球，当你在家工作时，也允许自己这样做。

安排有规律的休息时间，远离你的屏幕和工作。伸伸腿，带着你的狗去散步，喝杯咖啡。长时间盯着屏幕会导致眼睛疲劳和颈部问题。这对你的大脑空间也没有好处。当被一个问题困住时，短暂的休息有助于重振你的思维，消除思维障碍。你的潜意识足够聪明，可以通过重新聚焦和评估信息来解决问题，从而产生更新的想法。

你会惊讶地发现快速休息可以恢复你的精力。如果你是一个上网就忘了时间的人，使用应用程序或网络浏览器扩展作为提醒。

# 规划问题—更重要的是确定优先级

在家工作意味着你是自己的经理——至少从组织和时间管理的角度来看是这样。这确实会带来一种自由感，你的大脑会欺骗你，让你认为你有充足的时间。事情很容易走下坡路，如果你的一天没有时间表或结构，这就为拖延打开了大门。

设定现实可行的目标。找出你最有效率的时间，把它们用在优先任务上。围绕它们安排你剩余的时间表。对于刚加入在家工作俱乐部的人来说，日子很容易就过去了。时光飞逝。有一个每日/每小时的时间表会有所帮助。使用经过验证的方法快速高效地完成工作，不管是吃青蛙！还是番茄工作法。

番茄工作法对做其他家务也很有用:回复电子邮件、洗衣服或整理你的餐具室。虽然事先有结构感很重要，但不要过度。呆在家里可以让你灵活地重新安排你的日常生活。利用这一点。

# 沟通

互联网是远程工作中的主要通信方式。因此，确保您拥有自己选择的带宽至关重要。

尽管有很多可用的协作工具，如 Slack、Skype、Hangouts 和 Trello，但基于文本的交流对一些人来说可能很棘手——尤其是如果在家工作对你来说是一种新的体验。这也说得通，因为发短信不是在说话。事情可能会被曲解。决策需要时间。为了避免这样的麻烦，赶快打电话。

在家工作很容易感到孤独和寂寞。人类渴望社会互动。这是自然的，远程工作在某种程度上是没有的。这使得使用你所掌握的交流工具变得至关重要。定期利用空闲的频道和视频对话来保持乐观的情绪，从而弥补面对面交流的不足。

# 结束语

记住，每个人都有不同的工作方式。把更多的注意力放在找出你的风格上，而不是把上面的建议作为硬性规定。新冠肺炎的爆发迫使全世界进行了一次大规模的在家工作的实验。没有人知道事情会如何发展。

一些公司可能会让远程工作成为未来的常态，将传统办公室视为一种奢侈品/开销。今天让远程工作成为一种有益的体验，肯定会对明天有所帮助。

在家工作了五年多，我大致有以下习惯:

*   早点开始或者晚点下班(取决于我所在的时区)。
*   去健身房或者做几组俯卧撑来开始我的一天。
*   我家里有一个指定的工作场所。耳机当然有助于减少噪音。
*   我每 30-45 分钟走一圈，可能会在解决问题的时候不经意地打打网球。它让我不断前进。
*   在特定的时间放松。快速跟踪一天的进度，看看事情是否步入正轨。剩下的时间和家人朋友一起度过。

我希望这些建议对你有所帮助，并希望你保持健康。感谢阅读。
