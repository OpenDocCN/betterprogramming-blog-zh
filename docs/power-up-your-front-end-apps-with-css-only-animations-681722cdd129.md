我已经有一段时间没有尝试提高我的 CSS 技能了，大约一周前，我写了这篇关于如何开始创建 CSS Ar 的文章。虽然我对简单的几行 CSS 代码所能做的事情感到惊讶，但我们在创建前端时通常不会做太多 CSS 艺术。

让我们来看看如何创建和使用 CSS 动画，结果会自己说话:*我们需要更多地使用它们*。
