距离我发布上一篇关于Android库的文章已经一年多了。现在，我带着2019年发布(或大幅刷新)的牛逼库和项目汇总回来了。他们不是按等级顺序排列的。我们开始吧！

1。线圈

Coil 是一种在Android上加载图像的全新方法。这个库的首字母缩写代表协程图像加载器。该库速度快、重量轻(~1500种方法)、易于使用且现代，尤其是与Glide、Picasso或Fresco等库相比。它支持gif和SVG，可以进行四种默认变换:模糊、圆形裁剪、灰度和圆角。

下面是代码示例:

imageView.load(“https://www.example.com/image.jpg") {
 crossfade(true)
 placeholder(R.drawable.image)
 transformations(CircleCropTransformation())
}

文档和代码样本足以让你毫不犹豫地开始使用它。一切都很清楚和容易。目前这个库的版本是0.9.5。

有趣的事实:在过去的三个月里，我面试了14位候选人，没有人知道Coil。保持冷静，通过提及线圈给面试官留下深刻印象。甚至谷歌在他们的“高效加载大位图”文章中写及Coil时，也犯了一个关于作者的错误。不管怎样，现在已经修好了。

20.轮廓

Contour 是来自 Square Engineering 的用于Android上复杂布局的类型安全、Kotlin-first API。

根据他们的 GitHub页面:

“Contour的目标是尽可能薄地包装Android的布局API。它允许您在纯Kotlin中构建复合视图，而不使用不透明的布局规则——而是通过自己挂钩到布局阶段。Contour的最佳比较是constraint layout——但不是在XML中定义约束，而是将它们作为可执行的lambdas提供。”

这是一个非常有趣的想法，尤其是因为这个库试图反对旧的XML-Android方法。文档非常好，但是请记住:

"这个项目目前是实验性的，API可能会在没有通知的情况下发生重大变化."
