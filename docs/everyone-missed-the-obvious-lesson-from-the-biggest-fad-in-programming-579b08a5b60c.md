“结对编程已经席卷了科技行业，”亚当·费舍尔在他 2016 年关于主题的博客[中写道。](https://www.adams.life/blog/2016/09/alternative-pair-programming-solutions/)

像任何其他行业一样，软件工程行业很容易成为时尚——一些聪明的思考者对一个普通的问题提出了一个激进的解决方案，突然每个人都认为它完全改变了游戏规则。然后，当然，人们实际上开始做这件事，并意识到…嗯，[它也有问题](https://medium.com/the-non-traditional-developer/pair-programming-the-good-the-bad-and-the-ugly-5fc3f9c3663c)。

*结对编程*是一种流行的编码策略，两个人同时对同一问题进行编码。一个人开车，做编码。另一个人观察、检查输入的每一行代码。当然，这意味着在这个问题上花费了两倍的时间，但是根据一些研究，这也导致了错误率的降低。

如果你从来没有在有人坐在你的肩膀上纠正你打字时编码过，让我告诉你:你需要高度的同理心和耐心来扮演好这两个角色。如果你们不认识对方，或者不熟悉正在做的工作，这两个职位都令人伤脑筋。但是人们称赞它是在更短时间内完成更多工作的终极解决方案。

结对编程非常有效地让你为意想不到的原因编码。是的，拥有另一个视角和减少知识孤岛是很有用的。但是最能改变游戏规则的是一个和佛教一样古老的概念:心流。

# 结对编程有助于流动

每个人都知道[](https://en.wikipedia.org/wiki/Flow_(psychology)#cite_note-Csikszentmihalyi1990-1)*是什么，即使不是用那个术语。这就是进入状态。它被激光聚焦了。当你全神贯注于需要完成的事情时，时间会在你不知不觉中流逝。*

*它非常有效地帮助你完成工作，不幸的是人类并不擅长进入心流状态。我们容易分心。我们饿了，需要零食。我们喜欢检查我们的通知，以确保我们没有错过任何重要的东西，尽管我们几乎从来没有错过。*

*我第一次尝试结对编程时，我坐下来做了同样的编码任务，我已经尝试做了一个多星期，但没有成功。然而，不知怎么的，这次开始打字的时候，手指飞了起来，脑子嗡嗡作响。我吸收了我的合作伙伴的评论，并毫不费力地将它们合并到我的代码中。*

*当分配的 60 分钟过去后，我带着一些惊讶浮出水面，看着我写的高效、实用、整洁的代码行。他们成功了。*

*神奇的调味汁在成对的搭档中不起作用。不是现场反馈。这两个都很有用，但事实是让我如此高效的是这是我第一次尝试在没有任何*干扰的情况下完成任务。**

*显然，我不会查看我的 Instagram。当然，我不会看我的邮件。我有 60 分钟的时间和我的搭档一起尽可能多的完成工作——我不会无理取闹而停止工作。*

# *你不需要结对编程来获得同样的好处*

*第二天，当我的搭档生病不能和我一起做结对编程时，我崩溃了。老实说，心流的感觉很奇妙——就像第一次戴上眼镜，意识到今天之前看到的一切都有点模糊。我想再次投入进去。*

*但是我意识到结对编程有点像和一个搭档在健身房锻炼。你很难独自承担责任。有理由把分心的事情放在一边一段时间，去完成一些事情。还有一个提醒是，如果你现在努力工作，你可以完成更多的工作，然后放松一下。然而，这些都不是你需要另一个人去做的事情。只会让事情变得更简单。*

*结对编程有独立于流的好处，但对我来说，真正改变游戏规则的是理解一个小时专注的、不分心的工作比一天中断的编码更有价值。*

*没有伴侣也能重现心流的状态。*

# *如何创造你的心流*

*你需要做的第一件事是决定你的意图。你会用这段时间做什么？*

*它不一定是代码:你可以用它来进行生活管理，比如回复邮件、整理账单或打扫浴室。所有这些任务都将受益于不间断的关注，就像脚本一样。*

*接下来，向某人宣布，并要求他们在你的焦点块之后检查你。只要知道有人会问这个问题，你就会有动力去做。*

*第三，消除一切潜在的分心来源。对我来说，这意味着吃点零食，喝点饮料，把我的猫锁在外面，把我的手机移出视线范围，打开我的音乐。*

*第四，设置定时器。记住你只有有限的时间来实现你的目标。如果你有一整天的时间，你会花一整天。如果你有一个小时的时间，并且你知道这一点，你会发现专注于手头的任务要容易得多。*

# *结论*

*结对编程有一定的规律性，但令我惊讶的是，在大多数情况下，它的主要好处被广泛忽视了。*

*专注于自己的任务而不分心，在当今这个注意力分散的世界里是一种非常难得的状态，无论是对自己还是对工作。结对编程除了帮助你达到心流状态之外还有其他好处，但事实是你会做得更多，只是因为你被迫关注你的任务，而不是其他。*

*有一个伴侣会有所帮助，但是你自己也有可能找到这种状态。你所需要的是确定你在做什么，让别人对你负责，消除分心的诱惑，并承诺一个时间块。*

*找到你的心流，看看你能完成什么。*
