过去一个月里我们发现的最好的故事

嘿大家好，

随着我们步入五月，我希望你的四月充满了有意义的工作。在这个月度版本中，我们已经编辑了一些令人难忘的教程、全面的指南、发人深省的帖子以及我们在整个平台上发现的一些小建议。

事不宜迟，我们开始吧。

编程；编排

Python 3.11预计2022年10月问世。虽然你还不需要将你的应用升级到最新版本，但是，如果你想学习新的东西，崔永教我们一些在Python 3.11 的最新alpha版本中引入的新的 类型注释特性。如果你想学习更多的新内容， Giorgos Myrianthous 的Python 3.11指南是一个很好的起点。

毫无疑问，Python开发人员正享受着他们一生中最美好的时光——随着Anaconda团队在PyCon US 2022上发布PyScript，情况更是如此。想从你的HTML写Python？Anancoda的高级数据科学家Sophia Yang为您提供了完美的参考指南。

构建一个Chrome扩展听起来有些吓人。 Bartosz Salwiczek 的端到端教程— 如何开发React Chrome扩展用于媒体 —显示情况并非如此。这是我上个月最喜欢的教程之一——这并不是因为我是一个统计迷。我觉得学习编程的最好方法是通过构建东西。

谈到构建东西，赫伯特·沃尔弗森概述了一些游戏开发 Rust的入门工具。说到Rust，大卫·德拉苏斯的短文帮助 C/C++开发人员理解Rust工具链，而赫梅·加西亚为编译器工程师们分享了一些 Rust技术。Rust很棒，如果你需要更多的灵感来搭上火车，除了Python之外， Qiskit 团队已经开始使用它来提高性能。

正如我们已经提到的C++， Joseph Robinson，博士的视觉指南值得一看，如果你对指针的概念还很模糊的话。

写能写代码的程序是不是感觉太元了？这就是元编程。阅读迈赫迪·亚尔的kot Lin元编程指南，了解更多概念。

顺便说一句，如果你的外置硬盘有一天坏掉了，把它收藏起来。巨蟒来救援了！

转向swift ui——处理导航和NavigationView一直是iOS开发者头疼的问题。事情正在变得越来越好，尼克·麦康奈尔提出了一个流畅的解决方案，用SwiftUI 在你的代码库中有效地实现导航。与此同时， Rob Sturgeon 的最新作品展示了如何将SwiftUI视图和模型转移到单独的Swift包中。
