如何拒绝并保持利益相关者的支持

作为一个项目经理，接收范围之外的请求是理所当然的。这种事情发生的比我想承认的还要频繁。

计划和优先级会改变，但是忠于目标并确保所做的事情增加价值对于避免范围蔓延和确保项目成功是至关重要的。

虽然事情可能很紧急，但并不是所有事情都很重要(你好，艾森豪威尔！).所以你怎么能说不而不让人失望呢？此外，您如何在保持利益相关者支持的同时做到这一点？

一句话:说不的时候你必须考虑周全。取决于是谁问你，这可能会让你失去一个重要的盟友。

在我的职业生涯中，两种方法我都尝试过。起初，我对一切都说好，纯粹是因为没有经验。后来，我开始对一切说不——也是因为缺乏经验，但这一次夹杂着傲慢。

在我职业生涯的早期，我应该听取美国最高法院法官鲁斯·巴德·金斯伯格的意见:

“为你关心的事情而奋斗，但要以一种能让别人加入你的方式去奋斗”。

这就是我开发这种“推回”技术时所追求的。

我想坚守阵地，关注大局，保护项目的目标和目的。同时，我知道我正步入危险的水域。

项目的顺利完成离不开发起人和利益相关者的支持，所以不能惹怒他们！

这种方法是基于多年的反复试验。十个，如果有人在数的话。我已经将这些学习整理成一系列实际的行动和行为，并循环进行。他们在这里:

  1. 没有提供借口的过度解释。
  2. 避免防御性。
  3. 创造界限。
  4. 小心掉名字。

在谈判中，目标是达成一个双赢的局面，双方都觉得达成了一笔满意的交易。

有时斗争超出了你的工资级别，或者甚至不值得，所以即使你在技术上“输了”，你仍然可以向所有利益相关者强化你的过程——这是一个巨大的胜利。

请记住，所有这一切的更高目的是教育人们下次做正确的事情。

1.没有提供借口的过度解释

总的感觉是涉众知道所有项目中发生的一切。当然，我们确实会发送状态报告，但这不足以让人们随时了解最新情况。

直到几年前，当我的一位经理抱怨我的电子邮件时，我才有了这种感觉。他们说，"你必须停止回复邮件说，'根据下面的邮件…'没有人会花时间仔细阅读一个线程，然后回复你。"

这击中了我的头。

首先，让我的经理叫我出去的耻辱。第二，什么？如果阅读电子邮件符合人们的利益，他们难道不会这样做吗？回答:不，他们没有。他们只是瞥了他们一眼。

那位经理然后告诉我写一个简短的总结，给出我想要/需要什么东西的背景和原因——每次我发送或回复电子邮件时。

我知道这听起来令人生畏，但我保证这是值得的。这个习惯为我节省了很多时间，因为它帮助我避免了无休止的讨论。这是对人说不的圣杯。

当你给出很多关于你的项目的背景和信息时，你就摆脱了主观性和情感。

例如，下次你需要拒绝一个请求时，提供具体的证据说明为什么你的资源被分配给了其他任务，因此你不能吸收另一个请求(这次)。

最后，提供两个令人信服的替代方案:

  1. 你可以承担这项任务，但是这个人需要给你一个现实的期限。
  2. 主动帮助这个人分解任务，重新安排工作量的优先顺序。你或许可以快速分享一个 MVP 。

在理想的世界里，我们不会屈服。但是如果你的目标是建立关系，这不是最好的方法。记住金斯伯格大法官的话:带领别人加入你。

2.避免防御性

你听说过动物闻到恐惧的说法吗？利益相关者也是如此。你一表现出不安，他们就知道你赢了。

这就是为什么听起来和感觉自信是至关重要的。

你可以通过避免使用诸如“我认为”、“我感觉像”或“只是”这样的软弱的语言来使你的交流更有说服力

用激发自信和信任的话语来代替它们。更强的选项是“我有信心”、“我确信”、“我期待”和“我相信”

另外，不要陷入过度道歉的陷阱。你没有做错任何事，所以拒绝新的请求时为什么要说对不起呢？我曾经有这个可怕的习惯。在我看来，我是有礼貌的，但相反，我只是透露出我的不安全感。

在会议上或回复邮件时，避免防御性，以显示你知道自己在说什么。展示你很了解你的项目并且诚实。

在未来，如果你说某件事做不到，那是因为你做了功课。人们会相信你的话。

3.创建边界

项目经理是给予者。他们的存在是为了促进互动，确保一切顺利进行。

给予者提供帮助，分享知识，为人们增加价值，但如果他们不创造界限，他们会被视为受气包。那里有一条细线。

现在，你既不能取悦每个人，也不能惹恼他们，那么你如何扭转这个问题呢？我找到了组织心理学家亚当·格兰特提出的解决方案。

格兰特认为，在谈判中，给予者可以从“慷慨的针锋相对”的策略中获益。

在他的书给予和接受中，格兰特解释说给予者可以吸收他人的一些特点，同时保持开放，一旦有机会就重新变得慷慨。

例如，如果你觉得对方表现得咄咄逼人或不合理，你需要保持坚强，不要退缩。没有必要咄咄逼人，重复有毒的行为，但重要的是要坚持自己的立场。

在这本书里，格兰特举了很多例子来说明给予者是多么糟糕的代表他们自己的谈判者。为了克服这种倾向，记住你是在代表项目谈判——而不是你自己。

再一次，你会把情感从等式中剔除。你会把事情看得更远，这对于想出创造性的解决方案来创造新的双赢可能性是至关重要的。

4.小心掉名字

我理解当你想要什么的时候，说出重要人物的名字的吸引力。然而，这样做，你只是暴露了你的不安全感。

每当有人向我提出项目范围之外的要求时，我会向他提出大量问题。我经常使用 5个为什么技巧来找出问题的根源。

如果在评估结束时，这个人所能做的最好的回答是，“哦，实际上我的经理希望这件事尽快完成”，我知道这并不着急。

现在轮到你过度解释为什么不能按照要求行动了。然后，提供其他解决方案。

现在，根据研究，如果你是一个爱掉别人名字的人，请记住，人们会认为你不够称职。

我不是说它有时不起作用。显然，如果有人指名道姓地说出首席执行官的名字，就没有多少提问的余地了。但是记住谁是“冒犯者”,并为下一次做好准备。

最后的想法

当我们专注于建立关系，而不是将工作视为一场竞赛时，事情会进展得更好。确保你着眼于大局，引导人们加入你，而不是把他们推开。

在某些情况下不确定如何行动是非常正常的。这就是为什么当你处于困境时，这些屡试不爽的策略是救命稻草。
