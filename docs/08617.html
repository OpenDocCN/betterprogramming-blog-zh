7.已排序()

使用sorted()函数对列表或元组等可迭代对象进行排序。

  • 字符串按字母顺序排序
  • 数字是按数字排序的。
  • 您也可以选择升序或降序。

以下是一些例子:

排序字符串

numbers = ["C", "A", "B"]
print(sorted(numbers))

输出:

['A', 'B', 'C']

以逆序排序字符串

numbers = ["C", "A", "B"]
print(sorted(numbers, reverse=True))

输出:

['C', 'B', 'A']

排序数字

在上面的例子中，你对列表进行了排序。让我们对一组数字进行排序:

numbers = (3, 1, 7)
print(sorted(numbers))

输出:

[1, 3, 7]
