这是我第一篇将“我”放在显著位置的文章，因为关注读者是我的主要使命。但是，在这个疫情，我们都在经历艰难的时刻，我觉得给非常私人的问题一个千篇一律的解决方案是不公平的。

今天，我只想提请大家注意我在过去六个月里，坐在电脑前试着编码时处理负面情绪的过程。

因为你看，当你不工作时，处理不好的感觉和焦虑是不同的。你可以找到一些容易安抚你的事情，比如看电影或者只是躺下来让你的想法离开。

但是我的老板不喜欢那样，也不付钱让我看电影。他们付钱给我，让我提高工作效率，让我每周花 40 个小时解决自己的问题。所以我必须像一个专业人士那样处理这个问题。

编码不是你可以在分心的时候做的事情。你必须时刻牢记下一步，考虑同时应用一千种不同的最佳实践。这是一种非常以自我为中心的做法，因为除了编辑之外，你不能让你的大脑在任何地方游荡。

以下是我在困难时期作为急救包使用的方法。他们帮助我恢复了工作效率，甚至可能保住了我的工作。

# 日志

这种做法受到了世界各地心理健康专家的称赞，因为它有助于头脑更加清醒，生活更加幸福。它可以让你更好地理解你的愿望和担忧，获得对你真实想法的感知。对我来说，写日记还有另外一个好处:它给我一种自我进步的感觉，这在现在似乎整个世界都停滞不前的时候非常重要。

写日记不是我早上才做的事情，而是每当我有时间就写下几行。它们看起来会像这样:

*   我觉得…
*   我担心…
*   我很高兴，因为…
*   我对…很生气
*   我想完成…

# 休息 20 分钟

有时情况比那要困难得多。有时候发现自己坐在电脑前，完全写不出东西。思绪散乱，担心得什么都集中不起来。我可以打一些东西，但是感觉我总是忘记东西，犯一些愚蠢的错误。

这是我认识到我需要一个短暂的休息来处理这种情况。否则，我知道一整天的工作都会被浪费掉，而且在结束时我会感觉更糟。

我会休息一会儿，比如离开键盘 20 分钟左右。我问自己一个我在其他文章中已经提到过的问题:“[我现在需要什么？](https://byrslf.co/the-only-question-you-will-only-have-to-ever-ask-yourself-e2d9793c47c1)

如果答案是我担心我母亲在这种情况下的健康，我会打电话给她，确保她没事，并采取预防措施。如果我对在家里与某人发生的争吵感到难过，我会打电话回家，试着面对自己和那个人，以确保一切都好，我们会度过难关。

这不是心理学会议。只是安抚自己的一个基本能力。比如自我拥抱。它可以帮助每个人渡过难关，并给自己一个机会来挽救你的工作日和生产力，因为现在你觉得你对这种情况有更多的控制权。你不再是受害者了。

# 学习新东西

我是一名程序员。我喜欢学习与我职业相关的新东西。例如，目前我正在学习网页设计原理以及如何使用 Adobe XD。这是我的激情所在，也是我生存的动力。它给了我一种比我在工作中更伟大的感觉。每次我学到新的东西，我相信它会在未来回报我。这让我对它充满了希望。

# 结论

在这些困难时期，担心或悲伤是完全正常和可以接受的。不幸的是，在现实世界中，我们不能总是停下来。我担心我的生产力和工作，因为思想会让我夜不能寐，或者让我无法为自己写代码。这就是我的处理方式。希望这篇文章为你们中的一些人提供了一些如何面对这种情况的见解。
