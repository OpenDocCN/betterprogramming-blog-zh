在 Python 中，元组用于存储不同数量的元素。与可变的列表不同，元组是 Python 中不可变的数据容器。这意味着一旦它们被创建，你就不能改变它们的值。因此，Python 开发人员经常使用它们来表明它们不打算被更改。此外，它们的不变性防止了意外的变化。此外，与列表相比，元组通常更小，因此更易于存储，因为列表的可变性需要额外的开销。

当处理元组时，我们经常需要访问其中的单个项。Python 为我们提供了以不同方式检索项目的灵活性。在这篇文章中，我想回顾一下最重要的。总的来说，它们被称为元组解包。

# 2.用下划线表示不需要的项目

我们并不总是想使用一个元组对象中的所有条目。例如，我们可能只对访问用户的姓名和性别感兴趣。如前所述，这些数据包含在元组中。

不需要的项目的下划线

*   我们在年龄信息对应的位置使用下划线。
*   应该注意的是，下划线实际上是一个有效的 Python 变量名。实际上，它保存年龄数据，如下所示:

```
>>> print(f"Age: {_}")

```

*   但是，建议使用这种用法，而不是使用显式变量名，因为它清楚地表明了不需要该特定位置的项的意图。

# 结论

在本文中，我们回顾了与 Python 中元组相关的六种重要的解包技术。这里有一个快速回顾:

*   当元组没有太多的项并且我们想要使用所有的项时，我们可以使用一对一的解包来创建多个变量来表示每个项。
*   当有我们不想使用的项目时，我们使用下划线来明确表示我们不想要的项目。
*   当我们对元组中的连续项感兴趣时，我们可以使用带星号的表达式来提取这些项。
*   当不需要连续的项目时，我们可以将下划线和带星号的表达式结合起来。
*   元组打包和解包是声明和调用接受可变数量的位置参数的函数的有用技术。
*   带星号的表达式可以用来解包元组的所有项，就像其他可重复项一样。
