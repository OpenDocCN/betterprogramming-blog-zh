去年的这个时候，我坐在卧室的角落里，关着灯，在我的开发团队站立的时候，努力不吵醒我的小弟弟。我是三个高级软件开发人员的(虚拟)房间里唯一的实习生。现在，我是一名全职软件开发人员，站在一个房间里(这次是一个真实的房间！)的高级开发人员，而我的弟弟正在家里安详地睡觉。

获得全职工作并不是偶然的。这要求我走出自己的舒适区，作为一名开发人员真正挑战自己。现在，我已经做了几个月的全职软件开发人员，我想我应该分享一下我在转变过程中所学到的东西，以及我是如何从实习生晋升为正式开发人员的。

# 问问题，不管有多蠢

开始实习可能会令人望而生畏。你加入了一个非常有才华的团队，并被期望为你几乎一无所知的代码库做出贡献。你可能会感到害怕，甚至有点傻。我记得在我实习的第一周，我感到不知所措，以至于我确信他们雇佣我是个错误。

在这种情况下，你可能会因为害怕觉得自己很蠢而不敢问问题。然而，那是你最不想做的事。提问是你学习的方式。这就是你的团队中所有有才华的人变得如此有才华的原因，而不问问题，你所做的一切都是在延迟你的知识增长。

# 你是*所有者*的代码，而不是贡献者

我可能应该先解释一下代码所有者和代码贡献者之间的区别。简单地说，贡献者是为代码库增加价值的人。这通常采用添加新功能或修复 bug 的拉请求的形式。另一方面，所有者的任务是通过防止代码气味和错误的引入来保持代码库的健康。一个好的代码所有者还应该帮助其他人对代码库做出贡献，并遵循最佳实践。当然，除了这些职责之外，所有者还贡献新的代码。

作为一名实习生，我很容易陷入贡献者的心态——尤其是在刚开始的时候，我感觉自己淹没在没有救生衣的代码海洋中。虽然一开始贡献者的心态可能是好的，但是如果你想成长为一名软件开发人员，开始像所有者一样思考是很重要的。阅读和重读你团队的投稿指南，或者建议他们制作一份(如果不存在的话)。看看资深开发者在别人的拉取请求上留下的评论。请记住，您不仅仅是为回购贡献代码。你拥有回购协议。维护代码质量、防止错误和保持代码库健康是你的责任。如果你看到一些需要改进的地方，不要害怕对 PR 提出建议，或者如果你觉得合并 PR 会对代码基础产生负面影响，不要阻止它们。

# 做出持久的贡献

想想你能改善现状的方法。这可能是改进开发人员的工作流程，引入新技术，或者其他任何即使在实习结束后也能让你的团队受益的事情。

例如，在我实习的几个月后，我发现我们团队的部署流程有所欠缺。步骤太多，人为失误的空间太大。我发现很多手动步骤可以自动化，所以我创建了一个 Slack bot 来处理部署中更繁琐的部分。当我向团队演示时，我的开发同事都非常兴奋，我也很兴奋，因为这给了我展示自己能力的机会。我创建这个机器人已经有几个月了，但它仍在继续提供价值，因为我们仍在使用它进行部署。

如果你正在寻找一种做出持久贡献的方式，一个很好的选择是改进你团队的入职文档，因为他们总是可以做得更好。记录你的入职流程和你遇到的任何困难，然后与你的团队分享这些信息。下一个实习生或新员工几乎肯定会为此感谢你。

# 让别人听到你的声音

你可能是个实习生，但这并不意味着你没有发言权。*使用它。*

你实习的主要动机可能是学习新的技能(或者赚一大笔钱)，但是与团队的其他成员分享你的想法和观点也同样重要。对我来说，一开始这样做很吓人。我以为我的反馈对一群比我更有经验的资深开发者不会有帮助。然而，随着时间的推移，我了解到他们重视我的意见，我现在经常在讨论中发表意见。

每个人都有一些有价值的东西可以贡献给团队，包括你。公司最初雇佣实习生的原因之一是培养新的想法。如果你有一些你认为可能对他人有益的想法，不要犹豫，说出来吧。

# 别忘了花时间在 Y *ou* 上

仅仅因为你的团队使用 Scrum 并不意味着你只能做板上的事情。高级开发人员做的许多工作在 sprint 中永远不会出现。花些时间学习一项新技术，提高你对代码库的理解，或者任何你认为可以提高你作为软件开发人员效率的事情。

也许举个例子会有帮助。我很幸运，我的团队已经有了一个叫做 10%时间的过程，我们可以用 10%的时间做我们想做的事情。对我来说，这意味着在周五学习新的工具和框架。我可能是一名 React 开发人员，但在周五，我会学习移动框架，构建 Slack 机器人，并尝试新的语言。见鬼，我甚至花了一天时间学习如何在区块链上编码。虽然这些事情都与我的工作没有直接关系，但它们有助于建立跨领域的联系。这种交叉培训可以帮助你想出创新的解决方案，否则你是不会想到的。我可以诚实地说，我在周五花的时间让我成为了一名更好的开发人员。

# 结束语

像最好的实习一样，我的实习充满了学习新事物和作为开发人员成长的机会。我完成了很多，犯了很多错误，最重要的是，感谢一个令人敬畏的团队、经理和一群同事，我学到了超过我预期的东西。正是因为他们的宝贵经验，我才成为了今天的软件开发人员，希望我至少能够将其中的一些经验分享给你们。

感谢阅读。现在出去充分利用你的实习机会吧！
