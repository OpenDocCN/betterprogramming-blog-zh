## 不需要前期成本或资金投入

疫情在经济和精神上影响了我们所有人，人们现在比以往任何时候都更希望建立不同的收入来源。

如果你是一名开发人员，有许多行之有效的方法和途径可以让你马上开始赚钱。

幸运的是，没有前期成本。然而，值得注意的是，在这些方法上花一点钱肯定可以加快这个过程——但这不是强制性的。

下面是五种不用工作就开始赚钱的方法。

# 1.电子书

![](img/f7dfa3f70577ee6c7f24556d1af24c7e.png)

照片由 [freestocks](https://unsplash.com/@freestocks?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 拍摄

写一本电子书不仅是作为一名开发者成长的一个好方法，也是挣钱的一个好方法。

虽然一开始看起来很难，尤其是如果你以前没有写过电子书，但事实是这并不难。

你可以使用[谷歌文档](https://www.google.com/docs/about/)来编写和格式化你的电子书。写作时不要专注于造型。在第一份草稿中，把注意力集中在内容上，尽可能多地表达连贯的想法。

有时候，当你在写一章的时候，你会意识到你已经在前几章中涵盖了很大一部分。因此，我建议在直接开始写作之前，先列出一个粗略的内容索引。

至于封面，Canva 是一个免费轻松自由设计你的电子书封面的好地方。

蒂姆·丹宁有一篇关于出版电子书时可能面临的挑战的精彩文章。

至于电子书的格式化，你可以使用 [Reedsy](https://blog.reedsy.com/how-to-format-a-book/) 。

现在你已经完成了电子书，是时候出版它了。

你可以很容易地建立一个登陆页面，并在那里出售你的电子书。(由于本文面向开发人员，所以我不会深入介绍如何设置登录页面。)

许多开发者已经从他们的电子书中获利，有些人甚至在点击发布按钮之前就已经获利了！

你也可以利用像 [Gumroad](https://gumroad.com/) 、 [Payhip](https://payhip.com/) 和 [KDP](https://kdp.amazon.com/) 这样的平台。

# 2.当自由职业者

![](img/0df69ebe99f19c38825823835090ab34.png)

照片由[达维德·巴拉迪](https://unsplash.com/@davideibiza?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

自由职业是一种流行的兼职方式，甚至有可能取代朝九晚五的工作。

你可以提供制作应用程序或网站，优化代码，以及更多你觉得舒服的事情，并获得报酬。

此外，自由职业是增强你的投资组合的好方法。

可以尝试类似 [Upwork](https://www.upwork.com/) 和 [Freelancer](https://www.freelancer.com/?gclid=Cj0KCQiA34OBBhCcARIsAG32uvO4HaTkv6rwXt9RLytrK1TrntyxqAGkWNVXtqXzMWL43masPlDymfgaAtfPEALw_wcB&ft_prog=ABL&ft_prog_id=381510452128) 的各种平台，但是对于开发者来说，获得自由职业项目的选择更多。Stack Overflow 有一个[作业门户](https://stackoverflow.com/jobs/remote-developer-jobs)，在那里你可以找到符合你时间表的远程作业。

此外，你甚至可以去做顾问或短期项目工作。

我建议你一次尝试一个平台并坚持下去，而不是一次尝试所有的平台。

# 3.博客

与流行的观点相反，即使在 2021 年，博客仍然是高度相关和有益的，而且它不会很快消失。

值得注意的是，博客已经发展了很多年，随着所有伟大的内容在网上免费提供，博客不再仅仅是写单个的帖子。

网上有一些平台(比如你现在所在的 Medium)欢迎作家，并努力创建一个作家社区。

尽管如此，这总是开始写博客的好时机。对于开发人员来说，这是一个很好的方式来展示您令人惊叹的辅助项目的幕后，撰写操作方法文章，甚至讨论开发人员面临的困难。

软件开发和编程总是在不断发展，并且总会有大量的读者可供你在博客中瞄准。

你可以创建自己的博客网站，或者使用像 [Medium](https://www.medium.com) 这样的平台，完全免费。

# 4.应用程序

![](img/e4b44b62aa4770fcad9c4c533375953e.png)

照片由[威廉·胡克](https://unsplash.com/@williamtm?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

到 2026 年，全球应用市场预计将达到 4073.1 亿美元，所以这是带着你的想法进入应用市场的好时机。

如果你使用 JavaScript，你有很多选择来构建跨平台的应用，使你能够同时瞄准多个应用市场。

事实上，通过结合 JavaScript 和其他领域，如机器学习、游戏开发等，可以构建大量的应用程序。

这也为你打开了新的机会之窗。

然而，值得一提的是，应用程序市场充斥着劣质的应用程序和服务。你的应用创意必须非常独特和有效，才能让人们首先尝试。

另外值得一提的是，发布 app 并不是一个 100%免费的过程。虽然有很多平台可以免费托管你的网站，但在谷歌 Play 商店发布需要一次性支付 25 美元，而苹果公司每年收取 99 美元。

# 5.录像

制作自己教学的视频，提供一对一的咨询，讨论编程方面的事情等。，是建立观众群和获得关注的好方法。

当我说制作视频时，我不仅仅是指在 YouTube 上，也包括像 T2、Udemy 和 T4 这样的平台。

对于外行来说，Udemy 和类似的平台允许你上传特定主题的付费和免费课程。

你可以非常深入地教授你的专业语言，并上传到各种网站，如 Udemy 和 YouTube。这不仅能让你赚钱，还能巩固你的知识。

# 最后的想法

开发人员有很多工具可供他们使用，可以快速自由地开始从不同来源获得收入的整个过程。

从制作视频到撰写电子书，开发者可以传授他们的知识和经验。已经有大量的观众想要这种类型的内容。

大多数开发人员已经知道如何建立登录页面，实现货币化，以及如何营销自己。但是现在，他们将所有这些结合起来，为自己创造多种收入来源。

建立各种收入来源至关重要，疫情更加强调了这一点的重要性。
