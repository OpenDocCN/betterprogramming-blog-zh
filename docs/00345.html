用HTML和Javascript制作滑块的教程

我正在编写一个关键用户流的原型，需要为测试用户创建自定义blob外观的能力。对于原型控件，我决定使用 d3.js 来创建测试用户可以操作的滑块。本教程介绍了我构建滑块的步骤，以及我发现有用的资源。

建立你的工作空间。

在目录中设置您的文件:

  • index.html
  • 主页. js
  • main.css

安装d3.js

下载d3.js 并解压缩到您的项目文件夹，或者您可以链接到html文件底部的脚本:

<script src="https://d3js.org/d3.v5.min.js"></script>

通过带有npm install d3-simple-slider的终端安装D3-简单滑块或者你可以从GitHub 下载。

设置您的HTML和CSS

我会把CSS留给你，但是这里有一个基本的index.html设置，有2个div元素，我们将在main.js文件中调用它们。

第一个是id="mood"，包含一个SVG，这是我们将使用滑块操作的内容。(我使用 Blobmaker 生成了SVG)第二个id="slider-color-picker"，目前为空。JavaScript将完成构建滑块的工作。
