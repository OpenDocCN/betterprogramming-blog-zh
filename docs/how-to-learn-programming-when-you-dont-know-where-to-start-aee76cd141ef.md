# 从选择编程语言开始

当谈到选择要学习的第一门编程语言时，有一些事情需要考虑。编程语言的清单很长，而且每年都在增长。没有简单的建议，比如:“你应该学 ____”。

选择你的第一门编程语言不应该掉以轻心，因为它很可能需要数百个小时的练习才能达到一半的水平。尽管大多数有经验的程序员都会同意高级脚本语言相对容易学习。

在学习五种不同的语言之前，我认为先学习一种很重要。而且你需要学好它！

最终，当你作为程序员变得更有经验时，学习不同的是明智的。但是，因为你刚刚开始，一开始就坚持一个。

以下是需要考虑的事项:

1.  学起来有多难？
2.  有多受欢迎？有就业市场吗？
3.  学习这种语言有多少资源？
4.  这种语言的长期前景如何？

## 1.学起来有多难？

并非所有的编程语言都有相同的学习曲线。因为你是一个没有任何经验的新手，所以最好开始学一门被认为相对容易学的课程。正如我之前所说，高级脚本语言(比如 JavaScript、 [PHP](https://php.net/) ，或者 [Python](https://www.python.org/) )是一个很好的起点。

## 2.流行的编程语言

你可以在这里找到流行编程语言的列表。这个列表可以让你了解某种语言目前有多流行。但是请记住，你不应该仅仅因为一门编程语言受欢迎就去学习它。

## 3.有多少资源？

当你即将开始学习的时候，你会非常依赖别人的知识。与相对较新的语言相比，已经存在了较长时间的编程语言可能会有更多的资源可供您使用。

可供学习的资源:

*   [中等](https://medium.com/)
*   [堆栈溢出](https://stackoverflow.com/)
*   [YouTube](https://www.youtube.com/)
*   编程语言的官方文档。
*   培训网站，如[Lynda.com](https://www.lynda.com/)(现领英学习)。
*   [W3Schools](https://www.w3schools.com/)
*   书

## 4.长期前景

当谈到编程语言的前景时，您可以尝试回答以下几个问题:

*   它存在多久了？
*   有多少大项目？
*   有哪些大公司在用？

# 开始编程

如果你已经选择了你的编程语言，你终于可以开始编程了——激动人心的部分开始了。

这里有三个建议可以帮助你:

## 1.耐心

是的，编程很难学。那是因为有很多东西要学。

你开始时一片空白，所以有些话题对你来说毫无意义是不可避免的。这种情况可能会一次又一次地发生。你学习如何编程的成功取决于你如何处理这些情况。

如果你不理解某个话题，不要远离它。呆在原地。如果你因为不理解而开始跳过某些部分，你以后会有麻烦的。

大多数时候，当你学习一门编程语言时，主题是累积的。没有地基，建造房屋的墙壁是没有意义的。

如果你第一次没有理解一个题目，不要气馁！耐心是关键。有时候，阅读不同人的解释有助于你理解。

## 2.编者ˌ编辑

一个好的编辑可以让你更好地理解正在发生的事情。不要从记事本开始写代码，给自己找个像样的编辑器。让自己轻松一点。

大多数编辑器都有语法高亮显示(或者可以通过插件添加)，并且会显示基本错误。这将极大地帮助你开始编码。有很多免费的编辑器可用。

## 3.你好世界！

开始动手的最好方法是尽可能编写最基本的代码。我们称这个基本例子为 *Hello World！*

这段代码所做的就是输出文本 *Hello World！*在你的屏幕上。谷歌 a *Hello World！*你决定学习的编程语言的教程。
