# 为什么每个开发人员都需要学习写得更好

> 原文：<https://betterprogramming.pub/why-every-developer-needs-to-learn-to-write-better-d5974e38e64c>

## “如果你试图在几个人之间做出选择来填补一个职位，总是要雇佣更好的作者。”

![](img/b9036c23cf7696254d2a008f4b130663.png)

由[尼克·莫瑞森](https://unsplash.com/@nickmorrison?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

我在读“杰森·弗里德的《变得真实》”，我对这一部分感到惊讶:

> “如果你试图在几个人当中决定一个职位的人选，总是要雇佣更好的作家。不管那个人是设计师、程序员、营销人员、销售人员还是其他什么，写作技巧都会有回报。有效、简洁的写作和编辑会产生有效、简洁的代码、设计、电子邮件、即时消息等等。
> 
> 那是因为成为一个好作家不仅仅是文字。好的作家知道如何交流。他们让事情变得容易理解。他们可以设身处地为别人着想。他们知道该省略什么。他们想得很清楚。而这些才是你需要的品质。"

这是我第一次注意到，当你只是一名软件工程师时，作为一名作家是多么重要。然而，一天一天，我越来越记得这个片段，我只是觉得我需要与周围的每个人分享它。

我们许多人学习技术写作，但我在这里谈论更多的是创造性写作。人类正常书写的方式。写作对于软件工程师来说是必不可少的，这里有几点关于你为什么需要写得更好的原因:

# 1.写作有助于解决问题

解决任何问题的最好方法是用简单的语言来表达问题本身。一旦你的问题被清楚地表达出来，解决方案就会随之而来。

同样，为你的问题写好伪代码也需要一个好的作者。

# 2.干净的代码是给作者的

干净代码的一个重要提示是以人类可以阅读的方式编写。这包括选择更好的变量名和注释。如果你语言不好，`x`和`y`总是比较容易。然而，作为一个痴迷于文字的人，你会喜欢将你的代码转换成任何人都能读懂的东西。

# 3.良好的沟通

正如 Jason Fried 所说，首要原因是良好的沟通。良好的沟通意味着你的团队一切都更好。你可以向他们解释你做了什么，你计划做什么，你被什么卡住了。你可以更好地说服客户，也可以更好地与客户沟通。此外，好的语言有助于你更好地理解他人。

更好的写作意味着更好的文档、更好的电子邮件和更好的聊天信息。

# 4.简化远程工作

许多开发人员不擅长远程工作，因为他们不擅长写作。他们觉得他们无法通过电子邮件和短信在网上充分表达自己，也无法仅仅通过文字进行良好的沟通。一个好的作家会更喜欢远程工作。

# 5.找份工作——尤其是自由职业合同

我是一名自由职业者，为了得到任何合同，我都要写一封求职信。我想招聘客户仅仅通过检查求职信就能从 100 多名自由职业者中决定面试哪一个。成为一名优秀的作家会帮助你从第一句话就获得这个客户，或者永远失去他们。

一些开发人员可能是专家和专业人士，但因为他们写不好，他们永远得不到这份工作。

# 6.分享你的知识

一些程序员是专家，然而他们的知识只是他们的知识，他们只是不能分享，因为他们没有写作技能。其他程序员不是这方面的专家，但他们非常出名，对社区有帮助，因为他们写并分享他们知道的每一件事。

# 7.赚更多的钱

这不仅仅是分享知识，写作可能会帮你赚更多的钱。写文章是挣钱的一种方式，写电子书是另一种方式，有时写评论也是一种方法。对于软件工程师来说，依靠写作赚钱还有很多其他方式。

# 8.更多代码审查

代码审查现在是许多公司的重要组成部分。进行更多的代码评审证明你有多专业，并且对社区有帮助。可悲的是，阻碍软件工程师编写代码评审的最重要的事情之一，就是不能很好地表达他们的评审。因此，学习如何成为一名更好的作家会有所帮助。

# 9.更好的设计文档

许多公司提供有效的写作课程，因为它们对软件工程师帮助很大，尤其是在设计文档方面。设计文档不仅仅是关于 UML 图，它是关于你如何描述你的设计，不仅是技术读者，也包括非技术合作伙伴。

# 10.营销和为你的项目寻找投资者

如果你试图推出一个只有讲述故事能力的个人项目，你能获得更多的用户和投资者吗？非技术用户不关心你有多专业，或者你的项目编码有多好，他们关心的是故事。你的故事可能会把你的项目变成一个拥有 100 万用户的产品，或者一个只有家人和朋友会尝试的产品。

# 最后的想法

有效和创造性的写作对每个软件开发人员都很重要。作为学生，这是开发人员从第一天起就需要掌握的技能。作为一名开发人员，它将提升你的领域、经验和日常生活。它会清晰地表达你的想法和观点，并在工作中创造更好的关系。这也有助于获得更好的想法和清晰的思考。
