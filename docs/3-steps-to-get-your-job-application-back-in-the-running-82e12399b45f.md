# 让你的工作申请重回正轨的 3 个步骤

## 取消简历托管

![](img/4a5b15b2e5a1447be136383a99fc6f8d.png)

马库斯·温克勒在 [Unsplash](https://unsplash.com/s/photos/resume?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上的照片。

无论你是申请全职工作还是实习，简历审核都是申请过程的第一步。正是在这个阶段，公司会评估你的项目、技能、之前的经验和教育程度，以决定他们是否要继续面试。

对很多人来说，这是过程开始的地方，也是结束的地方。简历“鬼影”是一种常见的情况，公司拒绝了你的简历，但从未给你回邮件。很多人可能会认为这是一种拒绝，并认为他们的过程就此结束，但我实际上认为这是一线希望。

如果你在提交简历后收到了正式的拒绝，那你就完了。你真的没有什么可做的来继续你的申请了。但是如果你被鬼缠身，你会进入一种`NULL`状态。你从未被正式拒绝过(据你所知)，但你的申请不再被追踪。

虽然这不仅仅是接受一个拒绝，你应该更进一步。这可能意味着——对我来说已经意味着——永远得不到回音和得到一份工作之间的区别。

# 被捉鬼后我应该做的第一步是什么？

如我所说，这里没有什么可担心的。你要走的路已经从简单的*等待回音*变成了更加深思熟虑、亲自动手的搜索。

你应该做的第一件事就是看看是否有公司代表与你的大学合作。他们的工作是帮助你在面试过程中取得成功。联系职业服务机构，询问是否有这样的机构。

一旦你有了自己的电子邮件地址，就发送一封电子邮件，内容如下:

> 嗨 __，
> 
> 我就读于 __ 大学，最近申请了 __ 大学，但没有得到回音。我想知道我的申请情况如何？我真的很想和 ___ 一起前进，因为我认为那里的工作很棒，我非常适合这个角色。
> 
> 谢谢大家！
> 
> 我的名字
> 
> 我的电子邮件
> 
> 附上我的简历

在这封电子邮件中，你向你的大学代表表达了对该公司的兴趣，并让他们知道你已经申请并希望继续下去。我知道很多人都使用过这个策略，而且效果很好。

如果你觉得自己有一份被忽视的好简历，附上简历是关键。如果你有一份很棒的简历，但却没有回音，很可能没人看。通过将它附在这里，您可以保证至少有一个简短的外观，这可能就是您所需要的。

# 我没有大学代表

这里的关键是不要陷入任何挫折，继续前进。

如果你没有大学代表，那根本没什么大不了的。我上的是一所公立大学，它在大多数公司都没有代表，我从未发现它对我有太大的阻碍。

你现在要做的是与招聘人员取得联系——最好是大学招聘人员或与你申请的职位相关的招聘人员(如实习招聘人员等)。).

你应该尽你所能去接触他们。LinkedIn 也行。电子邮件更好。

一旦你和他们联系上了，给他们发一封和上面那封非常相似的邮件，解释你已经申请了，表达了兴趣，给出了联系方式和你的简历。

然而，在这封邮件中，你可能想在邮件正文中多谈一点你的简历。招聘人员与成千上万的人打交道，不像你的大学代表，他们已经有很多他们正在跟踪的候选人，对你的学校没有偏好。所以你要尽可能的友好和直接。

一旦你发出这封邮件，给它一两个星期。这个策略非常有效，因为你会得到一个真正的招聘人员来看你的简历。如果看起来有希望，他们会让你快进到技术面试。如果他们这样做了，你就成功地将你的状态从`NULL`转变为`Under consideration`。

# 我找不到招聘人员

虽然我会敦促你尝试使用上述两种策略中的一种，但仍然有一些选择——尽管它们都不太可能有效。我将快速浏览一些我认为关键的问题。

## **找一个会给你时间的员工**

这些公司的大多数员工都忙于工作，没有时间去处理大学生向他们寻求帮助的问题。所以尽量找一个你认识的人(可能是实习生)或者是你学校校友的全职员工，和他们谈谈可能的推荐或者进入面试程序。

## **在学校参加公司活动**

虽然可能非常拥挤，但有时这些公司活动有一个链接或表格，你可以填写，让招聘人员知道你参加了，并对公司感兴趣。填好那个。有时你也可以把你的简历堆起来，或者交给招聘人员。如果可以的话，试着去做。

在 COVID 时代，许多公司不再去校园，所以他们举办网络系列。我仍然会尝试参加这些活动，因为它们可能会提供相同的好处。

## 参加招聘会或虚拟招聘会，即使你已经申请了

招聘会是一种被低估的将你的申请推到前台的方式。我认识许多学生，他们甚至去了招聘会，带着聘书离开。所以即使你已经申请了，也不要羞于去他们的展位再次表达兴趣。

## **参加黑客马拉松**

许多公司会派公司代表参加黑客马拉松，这是一个与他们交流的好环境。让他们知道你悬而未决的申请，你想继续前进。

# 我已经做了所有这些，但仍然没有得到回应

如果你已经在一家公司经历了所有这些，仍然不能得到一个面试机会，那么可以很有把握地假设他们此刻没有把你看作一个潜在的雇员。

去另一家可能对你和你的申请更感兴趣的公司吧。记住，不要对任何拒绝过于沮丧，不要继续装神弄鬼。这发生在我们每个人身上——你所需要的只是一个机会。
