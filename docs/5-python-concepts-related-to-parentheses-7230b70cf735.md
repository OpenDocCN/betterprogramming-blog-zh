# 3.遗产

当我们定义类时，我们通常只需要使用`def ClassName`作为第一行来表示新 Python 类的声明。尽管没有明确说明，但默认情况下，所有 Python 类都是`object`类的子类。换句话说，这些类继承自父类`object`，如下所示:

```
>>> import inspect
>>> inspect.getmro(Person)
(<class '__main__.Person'>, <class 'object'>)
```

当我们说继承时，这意味着子类默认具有与超类相同的行为，除非在子类中被覆盖。例如，通过继承 object 类，我们的`Person`类自动访问各种内置函数，比如`id`和`hash`:

```
>>> id(person)
4544585536
>>> hash(person)
284036596
```

# 结论

在本文中，我们回顾了 Python 中与括号使用相关的五个重要概念。这里有一个快速回顾:

*   我们使用括号作为调用操作符来调用函数。
*   与函数类似，类也是可调用的。通过调用一个类，我们创建了该类的实例对象。
*   当我们定义类时，我们使用括号来表示当前类的超类。子类和超类之间的关系叫做继承。
*   括号与表示内置数据类型(元组)的值密切相关。要记住的一件重要的事情是，如果我们创建一个单元素元组，我们需要在元素后面包含一个逗号。
*   与使用方括号的列表理解不同，生成器表达式使用一对括号，尽管这两种技术在方括号或括号中有相同的表达式。
