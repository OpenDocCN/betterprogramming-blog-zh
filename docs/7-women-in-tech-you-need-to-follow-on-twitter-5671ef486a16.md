多元化、公平和包容(DE&I)比以往任何时候都更加成为关键对话、头条新闻和社交媒体的焦点。

女性领导者，尤其是整个科技行业的女性领导者，正在进行公开对话，讨论如何让自己和她们工作的公司在多元化方面负起责任。技术领导者利用他们的平台最终创造工作场所，在这里每个员工都可以在个人和职业上茁壮成长，并了解自己的价值。

Vantage Point 编制了一份名单，列出了六位领导和倡导者，他们正在引领科技行业的变革，同时在社交媒体上分享他们的愿景。

# **艾丽卡·乔伊·贝克**

Erica Baker 是著名的[项目 Include](https://projectinclude.org/) 的主要创始人之一，该项目旨在通过接纳女性和其他少数族裔，使现代工作场所更具包容性。Erica 在创造一个更加性别中立的技术环境方面发挥了关键作用。

# **金伯利·布莱恩特**

[金伯利·布莱恩特](https://twitter.com/6Gems)是[黑人女孩代码](http://www.blackgirlscode.com/)的创始人兼执行董事，该组织旨在增加数字空间中有色人种女性的数量。该计划使 7 至 17 岁的有色人种女孩能够成为 STEM 领域的创新者和所在社区的领导者。

# **琥珀色康维尔**

Amber Conville 是一名软件开发人员，通过她的网站 [crebma](http://crebma.com/about) 组织会议。她是 Integral.io 的工程总监，也是 Self.org 会议的创始人。

# 米娜·马卡姆

米娜·马卡姆积极参与科技社区，为[黑人女孩代码](http://blackgirlscode.com/)授课，并创立了[女孩发展组织的达拉斯分会](http://girldevelopit.com/)和[DFW·萨斯](http://www.meetup.com/DFW-Sass/)。她还是面向开发者、设计师、企业家和管理者的 web 技术会议 [Front 门廊](http://frontporch.io/)的共同组织者。

# 简·麦克戈尼格尔

[简·麦克戈尼格尔](https://twitter.com/avantgame)是一名[游戏设计师和教育家](http://janemcgonigal.com/)。她因在虚拟现实游戏领域的工作而闻名。除了在游戏领域的工作之外，她还是一位多才多艺的作家。她的新书是*现实被打破:为什么游戏让我们变得更好，以及它们如何改变世界*。

# 阿米莉亚·格林哈尔
