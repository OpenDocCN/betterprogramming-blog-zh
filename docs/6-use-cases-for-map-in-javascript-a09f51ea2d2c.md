`map()`函数通过调用用户提供的回调函数创建一个新数组。该函数访问调用数组中的每个元素。您可以看到`map()`方法正在经历一个循环，并在回调函数中编写语句来构造一个新数组。

# 参数是什么？

参数是回调函数和在执行回调函数时用作“this”的值。

## 回调函数

`callBackFunction`:对数组中的每个元素都调用该函数，当回调函数完成执行时，返回值被添加到将使用`map()`构造的新数组中

`currentValue`:回调函数所遍历的数组的当前元素。

`index`:回调函数正在处理的当前元素的索引。

`array`:回调函数遍历的数组。

## 这

`thisArgument` —执行`callBackFunction` **时，作为`this` 的值。**
