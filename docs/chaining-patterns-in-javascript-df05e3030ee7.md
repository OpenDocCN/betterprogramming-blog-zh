# 责任链模式

责任链(COR)是一种允许多个对象发送、接收和处理某些请求的模式。这些对象(仅仅是函数)既不依赖于前一个请求的实现细节，也不依赖于下一个请求的实现细节，并且可以决定在运行它的执行时做什么。它们也可以中止整个链，或者决定让请求继续到链中的下一个对象(或函数)。
