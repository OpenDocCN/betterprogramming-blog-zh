老实说，在我拿到工作机会之前，我就开始写这篇“我如何在谷歌找到工作”的文章了——事实上，在我参加最后的面试之前。这并不是说我认为我会得到它；更多的是记录过程供我自己使用，并了解如何为下次改进。哎呀，我得到了那份工作！积极的视觉化(意外地)是如何工作的？

此外，在我自己准备的过程中，我阅读了所有我能找到的类似这样的文章。我想我是一名软件工程师，我擅长观察模式，如果我读了足够多的成功故事，我会看到那些做得好的人的模式，我会提出我自己的惊人的学习计划。

而且成功了！我能够从每篇文章中获取零碎的建议，并将它们应用到我自己的学习中。有些成功了，有些失败了，但所有这些都让我觉得这是一个可以实现的目标，从而增强了我的信心。

所以事不宜迟，我非常荣幸能把我的故事加入其中。我希望它能帮助到外面的人。

# 首先:没有计算机科学学位的人请注意

当我在 LinkedIn 上宣布这份工作时，令我震惊的是，许多人对谷歌会雇佣一个没有计算机学位的人感到惊讶。我是来辟谣的，说是必须的要求。因为它不是。自从一周前开始在谷歌工作以来，我已经见过各种背景的软件工程师(*比如*电气工程师、律师、动画设计师)。我在加州大学圣巴巴拉分校获得了机械工程学士和硕士学位，我认为这对我的简历很有帮助。但要明确的是，我从来没有上过任何数据结构或算法的课程。(我通过 [Udacity 全栈 Web 开发人员 Nanodegree](https://imp.i115008.net/g2y45) 训练营学习了我的大部分编码。我在科技创业公司磨练了 2.5 年的编程技能。)

但值得庆幸的是，我们生活在一个知识惊人、革命性的时代。知识无处不在。它就在你的指尖。过去有各种各样的障碍需要跨越——无论是社会经济的，地理的，等等。—接触学校、大学、教师、教授或图书馆。但是现在，你只需要上网。

谷歌知道这一点，更重要的是，他们有兴趣开发一种产品来满足不同人群(整个地球)的需求。做到这一点的最好方法是雇佣一群多元化的人。

所以不要让非计算机学位的事情阻碍你。

事实上，永远不要让任何事情阻碍你！让谷歌决定你是否属于，即使你被拒绝了，记住很多谷歌人在被接受之前申请了两次、三次甚至更多次。所以你在一个好公司里。

好了，我们开始吧。

# 方法/计划

我本质上是个学生。如果我的余生可以作为一名学生得到报酬，我会的。这就是我选择软件工程的原因，因为这是最接近的东西。你不断地学习，与他人合作，创造性地解决复杂的问题。

但我最喜欢学校的一点是，你总是为成功而努力。我不是说学校很容易，但我是说，如果你足够努力，你通常会得到你需要的正确资源来通过那门课。

在现实世界中，并不总是有如此明确的成功路线图。没有学习指导，没有作业问题，也没有课程表。所以我决定给自己做一个。

## 我把我的谷歌面试学校化了

我参加谷歌面试就像大学课程的期末考试一样。我写了自己的教学大纲，创建了每周要涵盖的主题，找到了教科书，给自己布置了家庭作业，安排了期中考试，然后拼命学习。

# 教学大纲和作业时间表

教学大纲是任何大学课程的路线图，它用两个重要的东西为学期制定了日程表:

*   涵盖的主题
*   家庭作业

在我看来，制定一个教学大纲(并坚持下去)是学习中最重要的一部分。这也是最困难的一个，因为有太多的方法、数据结构和算法需要学习，很难将它们组织起来并易于管理。如果你没有一个计划，它会变得势不可挡。

怎么知道研究什么课题？怎么知道练什么样的题？你怎么知道你是否在浪费时间学习那些永远不会要求你做的事情呢？这该死的考试会有什么内容？

想象一下，去一个班级，你的整个成绩是基于一次期末考试，但他们不给你任何讲座或家庭作业。这有点像进入谷歌的感觉。我需要的是一个可以遵循的行动方案，该方案彻底涵盖了我需要了解的所有相关主题。

## 为自己设定最后期限，让自己走上正轨

截止日期是有帮助的，因为它们提供了重要的检查点。如果你落后了，截止日期会促使你按时完成主题。如果你比预期更快地掌握了一个话题，你可以庆祝这个里程碑。无论哪种方式，截止日期都有助于你保持正轨，这样你就不会没有时间了。

如果你发现你在一个话题上的挣扎比预期的多，你可能需要调整，所以在结束时给自己几个星期的额外时间来考虑这个问题。

## 练习解释你的方法

一旦你掌握了一个话题或问题，别忘了练习用 REACTO 格式回答问题。记住编码是不够的。你的面试官也在评估你的沟通和合作能力。这需要和你面试的编码部分一样多的练习。所以在你编码之前练习讨论你的方法。想出你想要测试的例子和边缘案例。讨论空间和时间的复杂性。诸如此类。要获得更详细的指南，请查阅这篇文章。

# 期中考试

我知道在学校里，期中考试是在学期中间进行的，但在这种情况下，我们实际上要把期中考试看作是对真实交易的模拟测试。我会在真正面试前至少两周计划这些。这样，如果你模拟面试搞砸了，你还有一些时间来改正。

## 模拟面试

这帮如此，那么也不多。请不要打折。我保证他们会很尴尬，可能会很尴尬，但是非常有帮助。

首先，他们会让你在比赛日不那么紧张。面试压力非常大，尤其是科技类的。问完技术问题后的前十秒，我感觉自己有一种灵魂出窍的体验，就像漂浮在电脑上方五英尺的某个地方。这种感觉不会真的消失，但随着练习会变得更好。所以把模拟面试当成真事来对待，

其次，如果你能设法与 FAANG 的实际员工进行模拟面试，他们会给你一个很好的真实问题来研究(比如，来自绝密公司金库的问题)。如果你没有得到答案，不要感到尴尬——这是另一个你在实际面试中不会搞砸的问题。

第三，不可避免地，在随意的交谈中，他们会告诉你一些超级有用的建议。例如，我在 FB 的一个朋友告诉我，永远要做建议测试用例的人。不要等到面试官问你，要积极主动。我的谷歌模拟面试官告诉我要像对待队友一样对待面试官。与他们合作，而不是反对他们。他们也希望你成功。

这是一种很难在网上找到的建议，但是，哇，真的有影响吗？

如果你在 FAANG 公司找不到人，试试 pramp.com 的[或者 interviewcake.com 的](http://www.pramp.com)。我听说这些是很好的资源。或者如果你有一个朋友也在读书，就互相面试。

# 死亡周

死亡周是你面试前的一周。在这一点上，你可能不应该学习太多新的东西。相反，是时候回顾和尝试你已经学过的问题的变体了。

第一步是对我已经解决的 AlgoExpert 问题重做几次。正如他们所说，肌肉记忆。我知道很多人说不要背，你也不应该背。但对我来说，重复做同样的问题真的有助于将基础知识铭刻在我的记忆中，并在我复习时建立我的信心。然后你可以去找问题的变体。这就是 Leetcode 公司模块派上用场的地方。

确保你复习了所有的主题。不要只关注那些你不擅长的或者你擅长的。

# 面试那天

重要的一天终于到来了！面试当天，一定要醒来，做你平常的运动和喝咖啡。试着记住，今天你将和一位聪明的软件工程师一起讨论有趣的问题。多酷啊。

## 做动力姿势

我的谷歌招聘人员在我面试的那天早上发给我这个视频，它完全改变了游戏规则。观看 Ted 演讲，听听它背后的科学原理，在面试前做一些有力量的姿势。他们真的可以有所作为。

## 假装面试官已经是你的队友

记住面试官是你的团队成员。这样对待他们，因为在一天结束的时候，这是他们评估你的依据。他们正在评估是否希望你加入他们的团队，成为团队成员需要的不仅仅是会编码。它需要交流、合作和总体愉悦。

# 一般建议

*   **不要累垮，也不要临时抱佛脚，耗尽时间。**找到你的金发女郎的时间量。不会太久。不要太短。给自己适当的时间去掌握所有不同的领域。
*   **集中注意力，不要掉进兔子洞。**学习时容易走神。前一分钟，您还在学习如何使用广度优先搜索来计算岛屿，但随后您会想为什么不使用深度优先搜索呢？两个我都可以用，但是哪个更好呢？接下来你知道的是，你在 YouTube 上了解哪个谷歌自助餐厅有最好的中国菜，以及你一个月能得到多少次免费按摩。
*   **保持专注。我发现一个非常有用的技巧是，当这些问题出现时，把它们都记下来，以便记住以后再来回答它们。我指的是快速笔记——最多 3 到 5 个单词的要点。这样你就可以把 100%的注意力放在手头的任务上，然后再回到细节上。记住，细节很重要，只是没有基础重要。分清主次，我的朋友！**
*   玩得开心。不要因为做运动、和朋友一起吃饭、看电视或电影而感到难过。学习时休息是必要的。它有助于整个不烧坏的事情。只要确保保持一个良好的，平衡的时间表。
*   **运动。我发现以两英里的跑步开始我的一天对我在学习过程中更有效率非常有帮助。有大量的书籍和文章探讨了为什么锻炼对大脑有好处，所以我不会深入探讨，但我会说它是有效的。在你学习之前，试着做些事情让血液流动起来。健康饮食。它比你想象的更有用。**
*   不要转动你的车轮。首先尝试问题是好的，但是如果十分钟后你真的一点概念都没有，就看答案吧。举例来说，我不可能凭直觉找出 DFS 或 BFS。所以盯着这个问题一个小时也没用。
*   **拓展你的人际网络。**他们说这需要一个村庄，他们是对的。当我发现我得到了谷歌电话面试的机会时，我告诉了我以前的大学教授/研究顾问这个好消息。你猜怎么着？他只是碰巧有一些在谷歌的学生。他帮我联系了一个人，这个人最终给了我一些重要的建议，并对我进行了一些模拟面试。当我联系一位在脸书工作的大学老朋友时，也发生了同样的事情。他甚至主动推荐我去脸书！
*   **不要忘记行为问题。你可能会专注于技术问题，因为它们不熟悉，但不要忘记为行为面试进行练习。**
*   放松，记住这不是你唯一的机会。如果你紧张，那太好了。这意味着你想要它，你在乎它，你把自己放在那里——所有的好东西，太棒了！但是记住，如果你没有得到它，你可以再试一次。不像申请大学，你实际上还有一次机会。每一次面试，你都离你的目标更近了一步。因为希望每次你好转的时候。

感谢阅读。我希望这能帮助你建立自己的教学大纲/学习指南并坚持下去。记住，你得到了这个！
