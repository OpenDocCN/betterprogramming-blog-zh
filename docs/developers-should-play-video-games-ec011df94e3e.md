## 编程与生活

## 视频游戏通常被认为是浪费时间，但它们可以在许多方面给开发者带来好处

电子游戏通常被认为是一种没有什么好价值的娱乐来源。也许社会就是这么看的。当然，整天玩游戏对任何人都没有好处。

但是它也有好的一面，这一点经常被忽视。具体来说，对于开发者来说，视频游戏有时会是一个福音。

开发人员往往有严格的工作时间表，大部分时间都处于紧张状态。在这种情况下，电子游戏有助于提高精神健康。这就像是大脑的一次锻炼。

让我们讨论一下电子游戏给开发者带来的一些好处。可以转化到工作中的技能。

# 它提高了你的思维能力

编程和电子游戏有很多共同点。例如，两者都需要遵循一组预定义的规则来达成解决方案。

电子游戏经常迫使我们思考如何绕过障碍。他们这样做了无数次。这增强了我们的横向思维，这在寻找解决复杂算法的新想法或消除不太直接的错误时非常有用。

认为电子游戏帮助我们变得更好的另一个重要方面是批判性思维。

批判性思维要求你根据呈现给你的事实做出理性和逻辑的决定。

虽然练习代码是加强这种技能的最明显的方式，但是以关卡(相互连接)的形式设计的视频游戏通常鼓励你应用你的批判性思维技能。

# 它能提高注意力和多任务处理能力

编程是一项刺激智力的工作。它需要高度的专注。某些视频游戏，尤其是像反恐精英这样的动作游戏，有助于实现这一点，因为它们需要大量的注意力。

拥有良好的注意力技巧对提高开发人员的工作效率和表现大有帮助。

玩一小时[电子游戏](https://news.uark.edu/articles/40981/one-hour-of-video-gaming-can-increase-the-brain-s-ability-to-focus)有助于促进大脑活动，提高注意力。

每个开发人员都必须在某个时候处理多任务。视频游戏可以确保你随时准备好处理多项任务。无论是要记住的组合键和目标列表，还是与多个敌人战斗，多任务处理几乎在所有视频游戏中都经过了测试。

# 这是一种很好的减压方式，可以促进大声思考

软件开发是一项耗费脑力的工作。通常，这需要你在高压环境下工作。视频游戏的身临其境和引人入胜的体验让你通过逃避到一个不同的现实中来喘口气。

电子游戏对心理健康的益处比人们可能意识到的要深。它们在本质上是激励性的，在某种程度上是为了挑战你每一个通过的级别，给你一种成就感。有规律的游戏时间有利于恢复自我，减少工作压力。

研究证明了某些游戏如何让人们更容易应对压力、抑郁和其他心理健康问题。

此外，玩游戏促进大声思考。大声思考不仅有助于[缓解压力](https://www.nature.com/articles/s41598-017-04047-3)，也有助于理清自己的思路。

游戏玩家倾向于大声思考，尤其是当他们陷入任务时。今天，许多技术面试官鼓励应聘者也这样做。它有助于评估思维过程，并消除任何误解的范围。

视频游戏是一个练习大声思考的好地方，尤其是当你不习惯的时候。

# 它促进团队建设和生产力

产品是团队工作的反映。每个公司都希望拥有高效的团队和积极的工作氛围。为了实现这一目标，他们在团队建设活动(体育运动、每周郊游等)上投入了大量资金。

虽然这些都很棒，但是协作视频游戏已经被科学证明可以提高团队生产力和办公室的士气[。](https://aisel.aisnet.org/thci/vol10/iss4/2/)

它能创造信任，让员工们相互交流，这对新组建的团队来说是一个奇迹。

多人游戏还要求你在不同的时间承担不同的角色和责任，这有助于你更好地协调、领导和其他基于团队的技能。此外，它还能让你保持竞争精神。

如今，办公室乐于投资专用游戏区，用于社交和放松目的。这是一个集思广益或无意中解决问题的好地方。

# 结束语

视频游戏在其他方面也有助于开发者。它们增强了我们的逻辑思维、计划和决策能力，同时也教会了我们一两件关于耐心和毅力的事情。

虽然电子游戏有很多好处，但长时间沉迷于电子游戏被认为是有害的。它并没有真正提高你的编程技能。或者教你如何编码。

此外，重要的是要注意，不是一个游戏玩家无论如何都不是一个缺点。在软件开发中没有放之四海而皆准的规则，玩电子游戏当然不会决定一个开发者的命运。

我希望电子游戏的上述好处将鼓励你在某个时候尝试它，并分析它对你工作效率的影响。再说一次，如果你不喜欢游戏，这完全没关系。
